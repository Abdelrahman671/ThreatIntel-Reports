<div class="wpb_wrapper">
<p>
<p>The DoNot APT (aka APT-C-35) has been active since 2016. They have attacked many individuals and organisations in South Asia. DoNot APT is reported to be the main developers and users of frameworks for developing Windows and Android malware<a href="https://ti.qianxin.com/blog/articles/stealjob-new-android-malware-used-by-donot-apt-group-en/"><sup>[1]</sup></a>.</p>



<p>This group mainly targets organisations in India, Pakistan, Sri Lanka, Bangladesh and other South Asian countries<a href="https://unit42.paloaltonetworks.com/updated-backconfig-malware-targeting-government-and-military-organizations/"><sup>[2]</sup></a><a href="https://www.eset.com/int/about/newsroom/press-releases/research/eset-research-investigates-donot-team-cyberespionage-targeting-military-governments-in-south-asia/"><sup>[3]</sup></a>. They focus on government and military organisations, foreign ministries, and embassies.</p>



<p>For the initial access, DoNot APT uses phishing emails containing malicious attachments. To get to the next stage, they execute a macro embedded in an MS Office document which drops a PE file and executes them. We have witnessed this PE file being a DLL file in the past campaigns.&nbsp; There is a change in the initial access of this APT group. The <a href="https://twitter.com/RedDrip7/status/1619267505650036738">RedDrip Team</a>, tagged a ZIP (filename: “Day .zip”) file being part of the DoNot APT campaigns. The lure was themed to leverage geopolitical tensions between India and Pakistan as shown in Figure 1. The ZIP file is bundled with a WinRAR SFX executable, which contains a DLL file and PDF files under a folder named “<em>Kashmir</em>”.</p>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="460" height="142" src="https://labs.k7computing.com/wp-content/uploads/2023/02/Figure-1-1.png" alt="" class="wp-image-25447" srcset="https://labs.k7computing.com/wp-content/uploads/2023/02/Figure-1-1.png 460w, https://labs.k7computing.com/wp-content/uploads/2023/02/Figure-1-1-300x93.png 300w" sizes="(max-width: 460px) 100vw, 460px" /><figcaption>Figure 1: Components within in WinRAR SFX file</figcaption></figure></div>



<p></p>



<p>The SFX executable executes that DLL as shown in Figure 2. The DLL file is responsible for connecting to the C2. The C2 Server was down at the time of writing this blog.</p>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="520" height="56" src="https://labs.k7computing.com/wp-content/uploads/2023/02/Figure-2-2.png" alt="" class="wp-image-25448" srcset="https://labs.k7computing.com/wp-content/uploads/2023/02/Figure-2-2.png 520w, https://labs.k7computing.com/wp-content/uploads/2023/02/Figure-2-2-300x32.png 300w" sizes="(max-width: 520px) 100vw, 520px" /><figcaption>Figure 2: Process execution</figcaption></figure></div>



<p></p>



<p>The domain registrar for the C2 domain (briefdeal.buzz) is <em>NameSilo Inc,</em> which seems to be a pattern since September 2022. The DLL contains 2 export functions. The first function&nbsp; creates mutex (“olgui1pigg”) as a marker for infection and then establishing connection to C2 and the next function is for self copying itself to the %TEMP% folder and creating a scheduled task for persistence by setting a new Scheduled Task that runs every four minutes. The action assigned to the task is to run the first export function. The C2 server was down during our analysis.</p>



<p>It tries to send the basic info of the Victim’ PC to the C2 like Username, Computer Name, Processor ID.</p>



<p>Here we mentioned the timeline of samples which are used by DoNot APT from Sep 2022 to Jan 2023.</p>



<figure class="wp-block-table"><table><tbody><tr><td><strong>Time</strong></td><td><strong>Samples File Name</strong></td></tr><tr><td>Jan 2023</td><td>Kashmir Solidarity Day Material .exeRequirement of pattenization data for seamanship items.xls</td></tr><tr><td>Dec 2022</td><td>spreadsheet.xlsdttcodexgigas.xlsSam.pptAccounts.xlsattachment.xlsff.xlstrix.xls</td></tr><tr><td>Nov 2022</td><td>Requirement list of spares.xls</td></tr><tr><td>Sept 2022</td><td>bodli.doc</td></tr></tbody></table></figure>



<p>The C2 URL TLD has been <em>.buzz </em>since the September 2022 campaign. The DNS registrar for the C2 URL is <em>NameSilo Inc.</em> and the IPs resolved to the ASN number <em>399629</em>. Based on the ASN number we were able to deduce the ISP as BL Networks, and from there we were able to track it to a Virtual Private Server (VPS) provider called <a href="https://bitlaunch.io">BitLaunch</a>, which accepts crypto payments. We found evidence of this particular VPS service being used for malicious campaigns in the past. We found this pattern to be consistent since September 2022.</p>



<p>We at K7 labs provide detection against such threats. Users are advised to use a reliable security product such as “K7 Total Security” and keep it up-to-date so as to safeguard their devices.</p>
