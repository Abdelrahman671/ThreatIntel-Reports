<p><sub>Monday, November 2, 2015 3:03 PM</sub></p>
<p>Hello,</p>
<p>I am currently only able to view the History for my Scheduled Tasks going back only 1 day on Windows Server 2012 R2.<br>
Is there a way to increase this to at least 7 days?</p>
<p>I have noticed that on Windows Server 2008 R2 the History seems to at least go back 3 days.</p>
<p>Any help here would be greatly appreciated!</p>
<p>Kind regards,<br>
Daniel</p>

<p><sub>Tuesday, November 3, 2015 8:05 AM âAnswered | <strong>3 votes</strong></sub></p>
<p>Hi Daniel,</p>
<p>I believe you need to increase the below&nbsp;eventlog's Properties to retain more logs.</p>
<p>History&nbsp;pulls eventID 140 from here. <a href="https://technet.microsoft.com/en-us/library/cc722006.aspx" data-linktype="external">A task's history is tracked by events in the Microsoft-Windows-TaskScheduler event log.</a></p>
<p>Microsoft-Windows-TaskScheduler/Operational</p>
<p><a href="/en-us/url?sa=t&amp;rct=j&amp;q=&amp;esrc=s&amp;frm=1&amp;source=web&amp;cd=1&amp;cad=rja&amp;uact=8&amp;ved=0CBwQFjAAahUKEwiWjb_d5PPIAhVGpZQKHRuOBqA&amp;url=https%3A%2F%2Ftechnet.microsoft.com%2Fen-us%2Flibrary%2Fcc721981.aspx&amp;usg=AFQjCNFPLNuLjcTsNOEsrfHNYLqk8eIcFA&amp;bvm=bv.106379543,d.dGo" data-linktype="absolute-path">Set Log Retention Policy - TechNet - Microsoft</a>&nbsp;- Default is 10MB of data, if you run lots of schtsk then it needs to be increased.</p>

<p><sub>Wednesday, April 29, 2020 2:58 PM</sub></p>
<p>In WIN10 (and Server2012) this is a property of the event log.&nbsp;</p>
<p>First open the event viewer..</p>
<p>Applications and Services Logs/ Microsoft/ Windows / TaskScheduler</p>
<p>There you can pick Maintenance or Operational - Properties.. Extend the size of the log file. (operational is 10MB by default)</p>
