          <h1>Breakdown of Tycoon Phishing-as-a-Service System</h1>          
          
        </div>
        <div class="blog-details-header-right">
          
            
            <a href="#" class="theme-changer">Change theme to light</a>
            
            
            <img src="https://www.trustwave.com/hs-fs/hubfs/Web/Spiders/2024/spider_01-mob.png?width=450&amp;height=250&amp;name=spider_01-mob.png" loading="eager" alt="Breakdown of Tycoon Phishing-as-a-Service System" class="hero-spider">
            
          
        </div>
      </div>
      <div class="toc-content content-wrapper">
        
        <div class="wrapper-main_content">
          
          <div class="blog-details-metadata">
            <span class="date-time">
              <time datetime="2024-02-20 16:14:52">February 20, 2024</time>
            </span>
            <span class="minread">6 Minute Read</span>
            
              <span class="blog-author">by Rodel Mendrez</span>
            
          </div> 
          
          
          <div class="wrapper-blog-content">
            <span id="hs_cos_wrapper_post_body" class="hs_cos_wrapper hs_cos_wrapper_meta_field hs_cos_wrapper_type_rich_text" style="" data-hs-cos-general-type="meta_field" data-hs-cos-type="rich_text"><p><span>Just weeks after Trustwave SpiderLabs <a href="https://www.trustwave.com/en-us/resources/blogs/spiderlabs-blog/trustwave-spiderlabs-detects-spike-in-greatness-phishing-kit-attacks-on-microsoft-365-users/">reported&nbsp;</a> on the Greatness phishing-as-a-service (PaaS) framework, SpiderLabs’ Email Security team is tracking another PaaS called Tycoon Group.</span></p>
<!--more-->
<p>The team found Tycoon Group during a regular investigation into a phishing incident, and its distinctive method of communication to its phishing server convinced the team to further explore this active PaaS operation.</p>
<p>&nbsp;</p>
<h3 style="font-weight: bold;">Tycoon Group</h3>
<p>Tycoon Group PaaS is sold and marketed on Telegram for as low as $120. Its key selling features include the ability to bypass Microsoft two-factor authentication, achieve "link speed at the highest level," and leveraging Cloudflare to evade antibot measures, ensuring the persistence of undetected phishing links.</p>
<p>&nbsp;</p>
<p><img loading="lazy" src="https://www.trustwave.com/hs-fs/hubfs/Blogs/SpiderLab_Blog_Images/figure1.png?width=451&amp;height=128&amp;name=figure1.png" width="451" height="128" alt="figure1" style="height: auto; max-width: 100%; width: 451px;" srcset="https://www.trustwave.com/hs-fs/hubfs/Blogs/SpiderLab_Blog_Images/figure1.png?width=226&amp;height=64&amp;name=figure1.png 226w, https://www.trustwave.com/hs-fs/hubfs/Blogs/SpiderLab_Blog_Images/figure1.png?width=451&amp;height=128&amp;name=figure1.png 451w, https://www.trustwave.com/hs-fs/hubfs/Blogs/SpiderLab_Blog_Images/figure1.png?width=677&amp;height=192&amp;name=figure1.png 677w, https://www.trustwave.com/hs-fs/hubfs/Blogs/SpiderLab_Blog_Images/figure1.png?width=902&amp;height=256&amp;name=figure1.png 902w, https://www.trustwave.com/hs-fs/hubfs/Blogs/SpiderLab_Blog_Images/figure1.png?width=1128&amp;height=320&amp;name=figure1.png 1128w, https://www.trustwave.com/hs-fs/hubfs/Blogs/SpiderLab_Blog_Images/figure1.png?width=1353&amp;height=384&amp;name=figure1.png 1353w" sizes="(max-width: 451px) 100vw, 451px"></p>
<p align="center" style="font-size: 14px;"><em>Figure 1. Tycoon Group phishing-as-a-service as advertised on Telegram</em></p>
<p align="center" style="font-size: 14px;">&nbsp;</p>
<p>SpiderLabs has investigated<span>&nbsp;</span><a href="https://urlscan.io/result/55f79f7e-c6b5-4be4-9e2f-d544dc463918" rel="noopener" target="_blank">scanned sessions<span>&nbsp;</span></a>in<span>&nbsp;</span><em>Urlscan.io</em><span>&nbsp;</span>containing artifacts and filenames related to this PaaS. The scans show the earliest Tycoon Group’s phishing page submission occurred on August 25, 2023, suggesting that the group likely introduced this service around that timeframe.</p>
<p>In mid-October 2023, the group released an update claiming that “<em>link and attachment will be smooth,</em>” which coincides with the integration of WebSocket in their phishing page to enhance browser-to-server communication. This information corresponds with<span>&nbsp;</span><a href="https://urlscan.io/result/8a913716-247a-4aef-9f72-70801618a8b0/#transactions" rel="noopener" target="_blank">URLScan.io scanned sessions<span>&nbsp;</span></a>, highlighting the use of socket.io, a WebSocket JavaScript library, in their phishing pages starting around late October 2023. This allows the phishing page to transmit data to the actor’s server in a more streamlined fashion.</p>
<p>In February 2024, Tycoon Group introduced a new capability to bypass two-factor authentication targeting Gmail users. The release also includes Gmail “Display” login page and Google Captcha expanding its potential target list to non-Microsoft 365 users.</p>

<h3 style="font-weight: bold;">“Myscr” JavaScript Component</h3>
<p>The second component is a JavaScript file prefixed with the name “myscr.” This script also uses multiple obfuscation techniques to evade bot crawlers and antispam engines. One obfuscation method involves a very long array of characters represented as decimal integers. Each integer value undergoes conversion from decimal to character and is then concatenated to construct the HTML source code of the phishing page. In addition, the script uses an obfuscation technique known as an "<a href="https://en.wikipedia.org/wiki/Opaque_predicate" rel="noopener" target="_blank">opaque predicate</a>," inserting unnecessary code in the program flow to obscure the underlying logic of the script.</p>

<h3 style="font-weight: bold;">Exfiltration Via WebSocket</h3>

It uses a JavaScript WebSocket library known as<span>&nbsp;</span><a href="https://socket.io/" rel="noopener" target="_blank"><strong>socket.io</strong><span>&nbsp;</span></a>to establish communication with the phisher's server, enabling the exfiltration of the data entered into the fake sign-in page.</p>

<h3 style="font-weight: bold;">Tycoon Group Admin Panel</h3>
<p>The Tycoon Group PaaS provides an admin panel accessible to subscribers/renters, enabling them to log in, generate, and oversee campaigns, as well as manage phished credentials.</p>
<p>Depending on their subscription plan, subscribers may access the panel for a limited duration. Within the settings section, users can generate new campaigns, selecting the desired phishing theme and toggling various PaaS features on or off. Additionally, subscribers can manage phished credentials, including usernames, passwords, and session cookies. The service further also allows subscribers to forward phishing results to their Telegram account.</p>

<h3 style="font-weight: bold;">Conclusion</h3>
<p>The rise of the phishing-as-a-service model as offered by groups like Tycoon Group and Greatness Phishing has made sophisticated phishing attacks very accessible, even for unskilled criminals. This ease of use is demonstrated by the greater number of phishing attacks using these types of services that we have observed. What makes the Tycoon Group distinctive is the use of WebSocket in the phishing page that allows a more efficient transmission of data between the browser and the phisher’s server. Furthermore, it also allows actors who subscribe to this service to easily manage their campaign and phished credentials.</p>
