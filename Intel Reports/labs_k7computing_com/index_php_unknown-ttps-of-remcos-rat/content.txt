<div class="wpb_wrapper">
<p>
<p>Typically spread through malicious attachments, drive-by downloads, or social engineering, Remcos RAT has been active since 2016. Initially presented by <a href="https://breakingsecurity.net/remcos/">BreakingSecurity</a>, a European company, as a legitimate remote control tool, it has since been exploited by threat actors for nefarious purposes, despite claims of restricted access for lawful use.</p>



<p>On analyzing a few samples from VirusTotal, we got one interesting sample which was a .vhd file. Let’s analyze how threat actors have crafted the VHD (Virtual Hard Disk).</p>



<p>&nbsp;After extracting the .vhd file we got a bundle of files shown in Figure 1.&nbsp;</p>



<div class="wp-block-image"><figure class="aligncenter size-large"><img loading="lazy" width="1024" height="570" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-1-1024x570.png" alt="" class="wp-image-27335" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-1-1024x570.png 1024w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-1-300x167.png 300w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-1-768x427.png 768w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-1.png 1229w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption>Figure 1: Extracted VHDfile</figcaption></figure></div>



<p></p>



<p>The shortcut file has the following powershell command line in target, pointing to the <em>MacOSX.ps1 </em>script. Its deconstructed components are depicted in Figure 1,</p>



<pre class="wp-block-code"><code> &#91; \\localhost\C$\Windows\System32\cmd.exe /c powershell.exe -ExecutionPolicy Bypass -File ".MacOSX/MacOSX.ps1" ]</code></pre>



<p>&nbsp;While analyzing the script we got to know it had several operations in it. Some of the functionality seems to be remnants of old TTP.</p>



<ul><li>Download a PDF file as PNG file (Figure 1)</li><li>Create a Task to download and execute a powershell script. (Figure 2)</li></ul>



<p>&nbsp;We found some key functionalities for this script</p>



<ul><li>AMSI Bypass (Figure 3)</li><li>Download a PNG file which is a VB script. (Figure 4)</li></ul>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="673" height="369" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-2.png" alt="" class="wp-image-27336" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-2.png 673w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-2-300x164.png 300w" sizes="(max-width: 673px) 100vw, 673px" /><figcaption>Figure 2: Downloading PDF</figcaption></figure></div>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="671" height="506" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-3.png" alt="" class="wp-image-27337" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-3.png 671w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-3-300x226.png 300w" sizes="(max-width: 671px) 100vw, 671px" /><figcaption>Figure 3: Schedule task</figcaption></figure></div>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="614" height="562" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-4.png" alt="" class="wp-image-27338" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-4.png 614w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-4-300x275.png 300w" sizes="(max-width: 614px) 100vw, 614px" /><figcaption>Figure 4: AMSI Reaper</figcaption></figure></div>



<p></p>



<p><strong>AMSIReaper </strong>which is an open source tool available in <a href="https://github.com/h0ru/AMSI-Reaper.git">GitHub</a>.</p>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="671" height="388" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-5.png" alt="" class="wp-image-27339" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-5.png 671w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-5-300x173.png 300w" sizes="(max-width: 671px) 100vw, 671px" /><figcaption>Figure 5: Downloading PNG file (elana.png)</figcaption></figure></div>



<p></p>



<p>The command in <em>$binaryData ,</em> in Figure6<em>,</em><strong><em> </em></strong>downloads a file <em>(elana.png</em>) from a specified URL (<em>hxxps://bitbucket.org/openheartplayercertlover/certlover2/downloads/elana.png</em>) . The PNG file is a VB script file base64 encoded file which is decoded and saved as under %Programdata% as <em>second.ps1</em>.</p>



<p>On further analyzing the png file we got to know it was actually a VB script.</p>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="699" height="863" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-6.png" alt="" class="wp-image-27340" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-6.png 699w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-6-243x300.png 243w" sizes="(max-width: 699px) 100vw, 699px" /><figcaption>Figure 6: VB script in elana.png</figcaption></figure></div>



<p></p>



<p>It defines a function <em>peopaias </em>which creates an instance of Internet Explorer (apegadas), navigates to a blank page, and waits until the page is fully loaded. It then sets up the browser window properties, including position and size. The function waits until a user input element is available on the page and then retrieves the input value before quitting the browser. It also defines a function <em>convertibilidade </em>which takes a string parameter <em>cytiso </em>containing script code and executes it using <em>ExecuteGlobal</em>. From this URL(&#8220;hxxp://paste.ee/d/azfhe&#8221;) we are able to get the base64 encoded VB script .</p>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="721" height="496" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-7.png" alt="" class="wp-image-27341" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-7.png 721w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-7-300x206.png 300w" sizes="(max-width: 721px) 100vw, 721px" /><figcaption>Figure 7: Encoded VB script</figcaption></figure></div>



<p></p>



<p>After decoding the VB script, we got a PowerShell script which was encoded with base64.</p>



<div class="wp-block-image"><figure class="aligncenter size-large"><img loading="lazy" width="1024" height="572" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-8-1024x572.png" alt="" class="wp-image-27342" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-8-1024x572.png 1024w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-8-300x168.png 300w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-8-768x429.png 768w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-8.png 1187w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption>Figure 8: Encoded PowerShell script in decoded VB script</figcaption></figure></div>



<p></p>



<p>On executing the script and capturing the output we realized that it was a&nbsp; VB script.</p>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="644" height="503" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-9.png" alt="" class="wp-image-27343" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-9.png 644w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-9-300x234.png 300w" sizes="(max-width: 644px) 100vw, 644px" /><figcaption>Figure 9: VB script in decoded PowerShell script&nbsp;</figcaption></figure></div>



<p></p>



<p>The VB script downloads an image file from the particular URL (<em>&#8216;hxxps://uploaddeimagens.com.br/images/004/731/958/original/new_image.jpg?1707143673</em><strong>&#8216;</strong>).</p>



<div class="wp-block-image"><figure class="aligncenter size-large"><img loading="lazy" width="1024" height="682" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-10-1024x682.png" alt="" class="wp-image-27344" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-10-1024x682.png 1024w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-10-300x200.png 300w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-10-768x511.png 768w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-10.png 1272w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption>Figure 10: Downloading PNG</figcaption></figure></div>



<p></p>



<p>On&nbsp; analyzing this image file we found that it was a base64 encoded value.&nbsp;</p>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="609" height="563" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-11.png" alt="" class="wp-image-27346" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-11.png 609w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-11-300x277.png 300w" sizes="(max-width: 609px) 100vw, 609px" /><figcaption>Figure 11: Base64 value in PNG overlay</figcaption></figure></div>



<p></p>



<p>After decoding it we came to know that it was a PE file.</p>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="377" height="274" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-12.png" alt="" class="wp-image-27348" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-12.png 377w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-12-300x218.png 300w" sizes="(max-width: 377px) 100vw, 377px" /><figcaption>Figure 12: .net binary</figcaption></figure></div>



<p></p>



<p>While analyzing the .NET DLL file in Dnspy we got to know how attackers are using the main payload and where they are using it.&nbsp;</p>



<p>The DLL is used to perform the process hollowing that is injecting the Remcos payload into a newly-created “<em>RegAsm.exe</em>” process. Once the function is invoked, it finds “RegAsm.exe” from below locations on the victim’s device &#8220;C:\Windows\\Microsoft.NET\Framework\v4.0.30319\RegAsm.exe&#8221;.</p>



<p>The following APIs are used for process hollowing:&nbsp;</p>



<p>CreateProcess() with CREATE_SUSPENDED flag(0x4), GetThreadContext(), ZwUnmapViewOfSection, .VirtualAllocEx(), WriteProcessMemory(),SetThreadContext(), ResumeThread().</p>



<p>&nbsp;The URL, where the Final Payload is hosted, is stored in a reverse format in the VB script as shown in Figure 13.</p>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="849" height="422" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-13.png" alt="" class="wp-image-27349" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-13.png 849w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-13-300x149.png 300w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-13-768x382.png 768w" sizes="(max-width: 849px) 100vw, 849px" /><figcaption>Figure 13: Reverse URL string</figcaption></figure></div>



<p></p>



<p>The main payload Remcos is a VC8 compiled binary.</p>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="472" height="275" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-14.png" alt="" class="wp-image-27350" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-14.png 472w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-14-300x175.png 300w" sizes="(max-width: 472px) 100vw, 472px" /><figcaption>Figure 14: C++ binary payload</figcaption></figure></div>



<p></p>



<p>It first decrypts a RC4 encrypted blob in the resource section, named “SETTINGS”.</p>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="415" height="122" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-15.png" alt="" class="wp-image-27351" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-15.png 415w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-15-300x88.png 300w" sizes="(max-width: 415px) 100vw, 415px" /><figcaption>Figure 15: Getting resource</figcaption></figure></div>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="546" height="435" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-16.png" alt="" class="wp-image-27352" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-16.png 546w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-16-300x239.png 300w" sizes="(max-width: 546px) 100vw, 546px" /><figcaption>Figure 16: Manipulating resource&nbsp;</figcaption></figure></div>



<p></p>



<p>In the blob, the first byte “3E” is the size of the RC4 key and the rest is the encrypted Remcos configuration block.</p>



<div class="wp-block-image"><figure class="aligncenter size-large"><img loading="lazy" width="1024" height="708" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-17-1024x708.png" alt="" class="wp-image-27353" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-17-1024x708.png 1024w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-17-300x207.png 300w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-17-768x531.png 768w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-17.png 1273w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption>Figure 17: Decoded RC4 in setting</figcaption></figure></div>



<p></p>



<p>From this configuration block we can get the C2, malware activities etc.</p>



<p>It sets the “RUN” registry for the persistence.</p>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="736" height="380" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-18.png" alt="" class="wp-image-27354" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-18.png 736w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-18-300x155.png 300w" sizes="(max-width: 736px) 100vw, 736px" /><figcaption>Figure 18: Persistence</figcaption></figure></div>



<p></p>



<p>The designated filename for logging victim keystrokes and clipboard data, various settings instructing Remcos on how to initiate its functionalities on the victim&#8217;s device, and the authentication details employed for establishing a connection to the C2 server were all crucial components.</p>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="794" height="347" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-19.png" alt="" class="wp-image-27355" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-19.png 794w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-19-300x131.png 300w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-19-768x336.png 768w" sizes="(max-width: 794px) 100vw, 794px" /><figcaption>Figure 19: Keylogging</figcaption></figure></div>



<p></p>



<p>It also creates a mutex to avoid multiple entries of this binary. Remcos also records the audio input from the victim&#8217;s microphone.</p>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="511" height="617" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-20.png" alt="" class="wp-image-27356" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-20.png 511w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-20-248x300.png 248w" sizes="(max-width: 511px) 100vw, 511px" /><figcaption>Figure 20: Stealing audio</figcaption></figure></div>



<p></p>



<p>Remcos RAT connects with a URL to collect geolocation information.</p>



<div class="wp-block-image"><figure class="aligncenter size-full"><img loading="lazy" width="574" height="97" src="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-21.png" alt="" class="wp-image-27357" srcset="https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-21.png 574w, https://labs.k7computing.com/wp-content/uploads/2024/03/Figure-21-300x51.png 300w" sizes="(max-width: 574px) 100vw, 574px" /><figcaption>Figure 21: geolocation&nbsp;</figcaption></figure></div>



<p></p>



<p>The other capability of Remcos RAT is&nbsp;</p>



<ul><li>Capturing screenshots of the victim&#8217;s screen upon startup.</li><li>Disabling User Account Control (UAC) on the victim&#8217;s device.</li><li>Sending data to C2.</li></ul>



<p>Attackers are always finding fresh strategies to evade the Antivirus (AV) and Endpoint Detection and Response (EDR) system,to&nbsp; secure their ongoing attacks.</p>
