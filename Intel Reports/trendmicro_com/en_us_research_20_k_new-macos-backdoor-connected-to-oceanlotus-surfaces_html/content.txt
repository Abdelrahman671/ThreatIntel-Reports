		<div>
			<p>We recently discovered a new backdoor we believe to be related to the <a href="/en_us/research/18/d/new-macos-backdoor-linked-to-oceanlotus-found.html" target="_self">OceanLotus group</a>. Some of the updates of this new variant (detected by Trend Micro as Backdoor.MacOS.OCEANLOTUS.F) include new behavior and domain names. As of writing, this sample is still undetected by other antimalware solutions.</p>
<p>Due to similarities in dynamic behavior and code with previous OceanLotus samples, it was confirmed to be a variant of the said malware. </p>

		</div>

    <div class="richText">
	
    


	
		<div>
			<p>OceanLotus was responsible for <a href="/en_us/research/18/d/new-macos-backdoor-linked-to-oceanlotus-found.html" target="_self">targeted attacks</a> against organizations from <a href="https://www.volexity.com/blog/2017/11/06/oceanlotus-blossoms-mass-digital-surveillance-and-exploitation-of-asean-nations-the-media-human-rights-and-civil-society/" target="_blank">industries</a> such as media, research, and construction. Recently they have also been discovered by researchers from Volexity to be using <a href="https://www.volexity.com/blog/2020/11/06/oceanlotus-extending-cyber-espionage-operations-through-fake-websites/" target="_blank">malicious websites</a> to propagate malware.</p>
<p>The attackers behind this sample are suspected to target users from Vietnam since the document’s name is in Vietnamese and the older samples targeted the same region before.</p>
<h2><span class="body-subhead-title">Arrival</span></h2>
<p>The sample arrives as an app bundled in a Zip archive. It uses the icon for a Word document file as a disguise, attempting to pass itself off as a legitimate document file.</p>

		</div>
	

</div>

			<p>Another technique it uses to evade detection is adding special characters to its app bundle name. When a user looks for the fake doc folder via the macOS Finder app or the terminal command line, the folder's name shows &quot;ALL tim nha Chi Ngoc Canada.doc&quot; (“tìm nhà Chị Ngọc” roughly translates to “find Mrs. Ngoc’s house”). However, checking the original Zip file that contains the folder shows 3 unexpected bytes between &quot;.&quot; and &quot;doc&quot;.</p>

<p>The operating system sees the app bundle as an unsupported directory type, so as a default action the “open” command is used to execute the malicious app. Otherwise, if the postfix is .doc without special characters, Microsoft Word is called to open the app bundle as a document; but since it is not a valid document, the app fails to open it.</p>

<p>2)      Attempt to remove file quarantine attribute of the files in the system.</p>
<p>3)      Copy &quot;ALL tim nha Chi Ngoc Canada.?doc/Contents/Resources/configureDefault.def(doc)&quot; to &quot;/tmp/ALL tim nha Chi Ngoc Canada.doc(doc)&quot;</p>

<p>6)      Change access permission of second-stage payload to execute the launch of the second-stage payload</p>
<p>7)      Delete the malware app bundle &quot;ALL tim nha Chi Ngoc Canada.?doc&quot;</p>
<p>8)      Copy &quot;/tmp/ALL tim nha Chi Ngoc Canada.doc(doc)&quot; to &quot;{execution directory}/ALL tim nha Chi Ngoc Canada.doc&quot;</p>
<p>9)      Delete &quot;/tmp/ALL tim nha Chi Ngoc Canada.doc&quot;</p>
<h2><span class="body-subhead-title">Second-stage payload</span></h2>

<p>1)      Drop third-stage payload to ~/Library/User Photos/mount_devfs</p>
<p>2)      Create persistence for the sample by creating ~/Library/LaunchAgents/com.apple.marcoagent.voiceinstallerd.plist</p>

			<h2><span class="body-subhead-title">Recommendations</span></h2>
<p>Threat groups such as OceanLotus are actively updating malware variants in attempts to evade detection and improve persistence. The following best practices can be applied to defend against malware:</p>
<ul>
<li><span class="rte-red-bullet">Never click links or download attachments from emails coming from suspicious sources</span></li>
<li><span class="rte-red-bullet">Regularly patch and update software and applications</span></li>
<li><span class="rte-red-bullet">Use security solutions suitable for your operating system</span></li>
</ul>
<p>To protect systems operating on macOS, we recommend <a href="/en_us/forHome/products/maximum-security.html" target="_self">Trend Micro Home Security for Mac</a>, which offers comprehensive and multi-device protection against malware and other cyberthreats.</p>
