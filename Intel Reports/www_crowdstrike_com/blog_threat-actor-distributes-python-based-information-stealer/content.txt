">On July 23, 2024, CrowdStrike Intelligence identified a malicious ZIP file containing a Python-based information stealer now tracked as </span><i><span style="	font-weight: 400;
">Connecio</span></i><span style="	font-weight: 400;
">. A threat actor distributed this file days after the July 19, 2024, single content update for CrowdStrike’s Falcon sensor — which impacted Windows operating systems — was identified and a fix was deployed. The ZIP file uses the filename </span><code><span style="	font-weight: 300;
">CrowdStrike Falcon.zip</span></code><span style="	font-weight: 400;
"> in an attempt to masquerade as a Falcon update. CrowdStrike Intelligence does not currently attribute the analyzed activity to any named adversary.</span></p>
<h2>Technical Analysis</h2>
<p>Technical analysis is based on the ZIP archive (SHA256 hash: <code>5ba542fcfa45d50c0d65dda4dbbd7a28f737a2fc53841ddaab7f68ae1cdf5183</code>) that was uploaded to a public malware repository on <code>2024-07-23 12:09 UTC</code> with the filename <code>CrowdStrike Falcon.zip</code>. This archive contains the files shown in Table 1.</p>
<table class="orange"><tbody><tr class="top-row bg-orange"><td><b>Filename</b></td><td><b>SHA256 Hash</b></td></tr><tr><td><code>Readme.txt</code></td><td><code>56cbd8ce60f18d4cececfa703a92c0188dd81ed97b4de12e3f120d7ce736225a</code></td></tr><tr><td><code>CrowdStrike Falcon.exe</code></td><td><code>21653e267a6c7e4f10064ad2489dba54e04612cc7ce4043b8c8dcaf8b39210d6</code></td></tr></tbody></table>
<p><b>Table 1. Archive CrowdStrike Falcon.zip content</b></p>
<p>The file <code>CrowdStrike Falcon.exe</code> is a self-extracting RAR (SFX) that contains and executes a Python-compiled executable; this executable contains the <i>Connecio</i> information stealer.</p>
<p>The text file <code>Readme.txt</code> contains instructions for potential victims to disable Windows Defender and execute the malicious executable present in the archive (Figure 1).</p>
<div id="attachment_218592" style="	width: 951.0px;
" class="wp-caption aligncenter"><img class="wp-image-218592 size-full" src="https://www.crowdstrike.com/wp-content/uploads/2024/07/Intel-blog-4-Fig-1-Read-Me.png" alt width="941" height="249" /><p id="caption-attachment-218592" class="wp-caption-text">Figure 1. Readme.txt content</p>
</div>
<p> </p>
<p>Upon execution, the malicious file <code>CrowdStrike Falcon.exe</code> unpacks and invokes an information stealer written in the Python programming language that collects system information, the system’s external IP address (by making an HTTP <code>GET</code> request to the URL <code>http[:]//ipinfo[.]io/[]?token&#61;7fb82f5925fe8b</code>), and data from multiple web browsers.</p>
<p>Then, the information stealer resolves its command-and-control (C2) servers by making HTTP <code>GET</code> requests to the Pastebin dead-drop URLs shown in Table 2.</p>
<table class="orange"><tbody><tr class="top-row bg-orange"><td><b>Pastebin Dead-Drop URL</b></td><td><b>Description</b></td></tr><tr><td><code>https[:]//pastebin[.]com/FHfj1fYD</code></td><td>C2 server configuration</td></tr><tr><td><code>https[:]//pastebin[.]com/HeVWATy1</code></td><td>Simple Mail Transfer Protocol (SMTP) accounts to exfiltrate collected information via email</td></tr><tr><td><code>https[:]//pastebin[.]com/2diVWcDQ</code></td><td>List of cryptocurrency address patterns and potential replacements; the addresses are likely used for clipboard-hijacking cryptocurrency theft, as they match cryptocurrency addresses and the malware uses the Python library <code>pyperclip</code> to control the clipboard<sup>1</sup></td></tr></tbody></table>
<p><span style="	font-weight: 400;
"><sup>1 </sup>https[:]//pypi[.]org/project/pyperclip/</span></p>
<p><b>Table 2. Pastebin dead-drop URLs</b></p>
<h2>Recommendations</h2>
<p>These recommendations can be implemented to help protect against the activity described in this report.</p>
<ul><li>Only accept updates delivered through CrowdStrike official channels and adhere to CrowdStrike support teams’ technical guidance</li><li>Train users to avoid executing files from untrusted sources</li><li>Check website certificates on the download page to ensure downloaded software originates from a legitimate source</li><li>Use browser settings to enable download protection that can issue warnings regarding potentially harmful websites or downloads</li><li>Block SMTP network connections to unknown servers when not required within the organization</li><li>Block Pastebin web access when not required within the organization</li></ul>
