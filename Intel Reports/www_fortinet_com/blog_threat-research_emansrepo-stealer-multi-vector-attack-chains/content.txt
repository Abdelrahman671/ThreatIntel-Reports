  <p style="margin-left: 80.0px;"><b>Affected Platforms: </b>Microsoft Windows<br />
<b>Impacted Users: </b>Microsoft Windows<br />
<b>Impact: </b>The stolen information can be used for future attack<br />
<b>Severity Level: </b>High<br />
</p>
<p>In August 2024, FortiGuard Labs observed a python infostealer we call Emansrepo that is distributed via emails that include fake purchase orders and invoices. Emansrepo compresses data from the victim’s browsers and files in specific paths into a zip file and sends it to the attacker’s email. According to our research, this campaign has been ongoing since November 2023.</p>
<p>The attacker sent a phishing mail containing an HTML file, which was redirected to the download link for Emansrepo. This variant is packaged by PyInstaller so it can run on a computer without Python.</p>
