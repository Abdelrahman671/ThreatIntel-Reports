<div id="c234756" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-center ce-below"><div class="ce-bodytext"><header class=" "><h2 class=" "> The Web Panel </h2></header><p>We were able to a get a glimpse at the web panel for the malware. The panel provides diverse options for creating malware stubs with customizable settings. Certain features are accessible only to users with 'higher level' subscriptions.&nbsp;Customization options include naming the malware, selecting an icon, and configuring the Telegram channel that receives notifications once information theft is successful. Additional choices include making the stub difficult to detect, disabling Windows Defender and deploying an extra malicious payload from a specified URL.&nbsp;</p></div><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><a href="/fileadmin/_processed_/9/9/GDATA_AilurophileStealer_Webpanel1_2977046056.jpeg" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/_processed_/9/9/GDATA_AilurophileStealer_Webpanel1_4ee7ae60be.jpeg" width="1436" height="863" loading="lazy" alt="" /></a><figcaption class="image-caption"> Customization options for the malware stubs for Ailurophile </figcaption></figure></div></div></div></div> <div id="c234757" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-center ce-below"><div class="ce-bodytext"><p>The following part of the panel is supposed to show a summary of which hosts are infected and the extent of information stolen for the ‘customerʼ of the malware:&nbsp;</p></div><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><a href="/fileadmin/web/general/images/blog/2024/08/Gdata_AilurophileStealer_Webpanel2.jpeg" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/web/general/images/blog/2024/08/Gdata_AilurophileStealer_Webpanel2.jpeg" width="982" height="126" loading="lazy" alt="" /></a></figure></div></div></div></div> </div></div> <div data-magellan-target="c234758" class="small-12 column"><div class="row"> <div id="c234758" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-center ce-above"><div class="ce-bodytext"><header class=" "><h2 class=" "> Stealing capabilities </h2></header><p>Supported browsers include Chrome, Edge, Brave, Yandex, Opera and CocCoc. The stealer can steal the following information from the supported browsers&nbsp;</p><ul class="list-black-square"><li>Auto fill information&nbsp;</li><li>Cookies&nbsp;</li><li>Stored passwords&nbsp;</li><li>Browsing history&nbsp;</li><li>Stored credit card information&nbsp;</li><li>Wallet data from browser wallet extensions&nbsp;</li></ul><p>In addition, the malware steals specific files that contain a designated set of keywords in their filenames, have particular extensions, and are located within a specified group of directories.&nbsp;</p></div></div></div> </div></div> <div data-magellan-target="c234759" class="small-12 column"><div class="row"> <div id="c234759" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-center ce-below"><div class="ce-bodytext"><header class=" "><h2 class=" "> Stealer Overview </h2></header><p>The stealer is completely written in PHP. This fact requires the victim machine to be able to execute PHP code, which is not the usual case. Therefore, the malware is packaged into an executable using a commercially available third-party software called ‘ExeOutputʼ. The malware is additionally virtualized using a commercially available software called BoxedApp.&nbsp;</p></div><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><a href="/fileadmin/web/general/images/blog/2024/08/GDATA_AilurophileStealer_StealerOverview.jpeg" class="fancybox"><img class="image-embed-item " src="/fileadmin/web/general/images/blog/2024/08/GDATA_AilurophileStealer_StealerOverview.jpeg" width="1212" height="725" loading="lazy" alt="" /></a></figure></div></div></div></div> <div id="c234760" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-center ce-above"><div class="ce-bodytext"><p>BoxedApp is used to produce a self-contained executable that has its own virtual file system, virtual registry and even virtual processes. The boxed app code intercepts I/O and other system calls from the application and redirects them to the respective virtual entities that it has created. When the malware runs, the boxed app code runs and extracts the required files into the virtual file system (the files are memory resident in this case). The files include the PHP interpreter and required runtime files as well as the config and malware components&nbsp;</p><p>A virtual process is then launched where the extracted PHP interpreter is used to run the code in ‘index.phpʼ&nbsp;</p><p>&nbsp;</p><pre><code class="language-bash">C:\\Users\\muser\\Desktop\\Data\\php.exe  -c \'C:\Users\muser\Desktop\Data\php-cli.ini\' \'C:\\Users\\muser\\Desktop\\Data\\index.php\' 

 </code></pre><p>&nbsp;</p><p>&nbsp;</p></div></div></div> </div></div> <div data-magellan-target="c234761" class="small-12 column"><div class="row"> <div id="c234761" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-intext-left"><header class=" "><h2 class=" "> Configuration file </h2></header><div class="image-gallery-wrap"><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><a href="/fileadmin/web/general/images/blog/2024/08/GDATA_AilurophileStealer_ConfigurationFile.jpeg" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/_processed_/4/c/GDATA_AilurophileStealer_ConfigurationFile_ad56296a6d.jpeg" width="300" height="111" loading="lazy" alt="" /></a></figure></div></div></div><p>The configuration is JSON based. The file stores settings that were set via the web panel. The ‘welcome’ field is absent in the web panel. &nbsp;The web panel advises to contact the malware author on telegram for fine-tuned customizations. Fields `key_decrypt` and ‘user_id’ are internal options. The malware code reads the config file and fetches the settings from it when the malware starts to run. config.json. The code collects the IP, host name, operating system information and machine architecture. It defines interesting folders, file name keywords, file extensions , wallet data directories and browser paths. As a next step, the required modules are loaded to execute functionality housed in them. It then initiates the collection of credit card, cookie, browsing history, browser auto fill information, wallet data and telegram session data. data-magellan-target="c234766" class="small-12 column"><div class="row"> <div id="c234766" class="fce column large-12 small-12 frame-layout-0 " ><header class=" "><h3 class=" "> GetCookie.php </h3></header><p>Collects cookies stored in various browsers and stores them under %LOCALAPPDATA%\Ailurophile\Cookies\. The information is collected from ‘password.db’ file which is an SQLite database. Password information is stored separately for each browser application.&nbsp;</p><p>&nbsp;</p></div> </div></div> <div data-magellan-target="c234768" class="small-12 column"><div class="row"> <div id="c234768" class="fce column large-12 small-12 frame-layout-0 " ><header class=" "><h3 class=" "> GetHistory.php </h3></header><p>Collects stored browser history and stores them under&nbsp;%LOCALAPPDATA%\Ailurophile\History\. The information is collected from ‘history.db’ SQLite database file.&nbsp;&nbsp;</p></div> </div></div> <div data-magellan-target="c234769" class="small-12 column"><div class="row"> <div id="c234769" class="fce column large-12 small-12 frame-layout-0 " ><header class=" "><h3 class=" "> GetFile.php </h3></header><p>The folders Desktop, Documents and Downloads are searched for files with the file type extensions<strong> 'rdp’, ‘txt’, ‘doc’, ‘docx’, ‘pdf’, ‘csv’, ‘xls’, ‘xlsx, ‘keys’, ‘ldb’ and ’log’.</strong><br> Those files are copied over for uploading if they contain any of the following <u><strong>keywords</strong></u> in their name: &nbsp;<br><strong>'default', 'backup', 'passw', 'mdp', 'motdepasse', 'acc', 'mot_de_passe', 'login', 'secret', 'bot', 'atomic', 'account', 'acount', 'paypal', 'banque', 'bot', 'metamask', 'wallet', 'crypto', 'exodus', 'discord', '2fa', 'code', 'memo', 'compte', 'token', 'backup', 'secret', 'seed', 'mnemonic', 'memoric', 'private', 'key', 'passphrase', 'pass', 'phrase', 'steal', 'bank', 'info', 'casino', 'prv', 'privé', 'prive', 'telegram', 'identifiant', 'identifiants', 'personnel', 'trading', 'bitcoin', 'sauvegarde', 'funds', 'recup', 'note'. </strong></p><p>Stolen files are archived into a zip file and stored to %LOCALAPPDATA%\Ailurophile\stolen_files.zip.&nbsp;</p></div> </div></div> <div data-magellan-target="c234770" class="small-12 column"><div class="row"> <div id="c234770" class="fce column large-12 small-12 frame-layout-0 " ><header class=" "><h3 class=" "> GetCard.php </h3></header><p>Collects stored credit card information from ‘web.db’ sqliteSQLite database stored by cChromium-based browsers. The information is then stored in %LOCALAPPDATA%\Ailurophile\Cards.txt.&nbsp;</p></div> </div></div> <div data-magellan-target="c234771" class="small-12 column"><div class="row"> <div id="c234771" class="fce column large-12 small-12 frame-layout-0 " ><header class=" "><h3 class=" "> GetTelegram.php </h3></header><p>The script terminates Telegram if it is running by employing the system utility taskkill.exe. All files under %APPDATA%\Telegram Desktop\tdata\ copied over for stealing. However, some subfolders are excluded namely ‘emoji’, ‘user_data’, ‘user_data#2’, ‘user_data#3’, ‘user_data#3’, ‘user_data#4’, ‘user_data#5’. Collected files are stored under %LOCALAPPDATA%\Ailurophile\Telegram\.&nbsp;</p></div> </div></div> <div data-magellan-target="c234772" class="small-12 column"><div class="row"> <div id="c234772" class="fce column large-12 small-12 frame-layout-0 " ><header class=" "><h3 class=" "> CryptoWallet.php </h3></header><p>There are numerous crypto wallet extensions available for cChromium-based browsers. The script has an extensive list of data directories of such extensions (under %LOCALAPPDATA%\Google\Chrome\User Data\Default\Local Extension Settings\) and simply copies over and steals the data under them. The stolen data is stored under %LOCALAPPDATA%\Ailurophile\Wallets\&lt;wallet name&gt;\.&nbsp;</p></div> </div></div> <div data-magellan-target="c234773" class="small-12 column"><div class="row"> <div id="c234773" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-center ce-below"><div class="ce-bodytext"><header class=" "><h3 class=" "> Delivery.php </h3></header><p>The script is executed only if the delivery option is enabled in the configuration file. This script has two purposes. One is to disable Microsoft Defender, if possible, by modifying Windows policies in the registry and by employing the PowerShell command “Set-MpPreference” to disable various Microsoft Defender features. Disabling Microsoft Defender is attempted only when the current user account has administrator privileges. &nbsp;</p></div><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><a href="/fileadmin/web/general/images/blog/2024/08/GDATA_AilurophileStealer_DeliveryPHP2.jpeg" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/web/general/images/blog/2024/08/GDATA_AilurophileStealer_DeliveryPHP2.jpeg" width="714" height="317" loading="lazy" alt="" /></a><figcaption class="image-caption"> Code that disables Windows Defender and its associated features </figcaption></figure></div><div class="column column-block"><figure class="image"><a href="/fileadmin/web/general/images/blog/2024/08/GDATA_AilurophileStealer_DeliveryPHP1.jpeg" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/web/general/images/blog/2024/08/GDATA_AilurophileStealer_DeliveryPHP1.jpeg" width="714" height="88" loading="lazy" alt="" /></a></figure></div></div></div></div> <div id="c234774" class="fce column large-12 small-12 frame-layout-0 " ><p>The second purpose is to download and execute an additional payload from the URL obtained from the configuration.&nbsp;</p></div> </div></div> <div data-magellan-target="c234775" class="small-12 column"><div class="row"> <div id="c234775" class="fce column large-12 small-12 frame-layout-0 " ><header class=" "><h2 class=" "> Note and closing remarks </h2></header><p>At least one sample that we examined did not contain cryptowallet.php. Furthermore, the config file differed slightly. This is in comparison to the newer malware stubs that we got hold of. It suggests that the malware is currently being actively developed. Investigation of the malware files and the file names used indicate that the malware is distributed posing as software cracks and porn.&nbsp;</p></div> </div></div> <div data-magellan-target="c234776"
