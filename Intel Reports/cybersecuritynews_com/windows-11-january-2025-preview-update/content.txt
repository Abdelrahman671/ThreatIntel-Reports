                            <ul class="td-category"><li class="entry-category"><a  href="https://cybersecuritynews.com/category/cyber-security/">Cyber Security</a></li><li class="entry-category"><a  href="https://cybersecuritynews.com/category/cyber-security-news/">Cyber Security News</a></li></ul>                            <h1 class="entry-title">Windows 11 January 2025 Preview Update Disconnects Remote Desktop Sessions</h1>

                            

                            <div class="td-module-meta-info">
                                <div class="td-post-author-name"><div class="td-author-by">By</div> <a href="https://cybersecuritynews.com/author/guru/">Guru Baran</a><div class="td-author-line"> - </div> </div>                                <span class="td-post-date"><time class="entry-date updated td-module-date" datetime="2025-03-27T17:02:28+00:00" >March 27, 2025</time></span>                                                                                            </div>

            </div> <!-- /.td-pb-row -->

            <div class="td-pb-row">
                                            <div class="td-pb-span8 td-main-content" role="main">
                                <div class="td-ss-main-content">
                                    



        <div class="td-post-content tagdiv-type">
            <div class="td-post-featured-image"><a href="https://i1.wp.com/blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEhEMjnn65Ni-6Pfj7z3nk2676n6r1Oe0Bha16U5oi0_Rps1KJYNU7fBOW9HNh-PIc3a0vp6qboTDI4mSPZIUT81w7TBATPDuHW2jfVEC0nwn5_ZDXe2baBvdBMkP3aSCFLRxgxQPPOgK6Upe8qKxZ0QX5Dt3ZAWRSJsBiehWeKlnAQ6qsfhlU1DhAhghQe3/s16000/Windows%2011%20January%202025%20Preview%20Update.webp?w=1600&resize=1600,900&ssl=1" data-caption=""><img post-id="97515" fifu-featured="1" width="696" height="" class="entry-thumb td-modal-image" src="https://i1.wp.com/blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEhEMjnn65Ni-6Pfj7z3nk2676n6r1Oe0Bha16U5oi0_Rps1KJYNU7fBOW9HNh-PIc3a0vp6qboTDI4mSPZIUT81w7TBATPDuHW2jfVEC0nwn5_ZDXe2baBvdBMkP3aSCFLRxgxQPPOgK6Upe8qKxZ0QX5Dt3ZAWRSJsBiehWeKlnAQ6qsfhlU1DhAhghQe3/s16000/Windows%2011%20January%202025%20Preview%20Update.webp?w=696&resize=696,0&ssl=1" alt="Windows 11 January 2025 Preview Update Disconnects Remote Desktop Sessions" title="Windows 11 January 2025 Preview Update Disconnects Remote Desktop Sessions" title=""/></a></div>
            
<p>Microsoft&#8217;s January 2025 Windows preview update (<a href="https://aka.ms/WinServer2025/5050094" target="_blank" rel="noreferrer noopener nofollow">KB5050094</a>) for Windows 11 version 24H2 has caused significant issues with Remote Desktop Protocol (RDP) sessions, including Remote Desktop Services (RDS).</p>



<p>Users have reported unexpected disconnections, particularly after the release of the March 2025 Windows security update (<a href="https://aka.ms/WinServer2025/5053598" target="_blank" rel="noreferrer noopener nofollow">KB5053598</a>).</p>



<p>The problem primarily affects Windows 11 version 24H2 devices connecting to RDS deployments running Windows Server 2016 or earlier versions. Users experience RDP session disconnections after approximately 65 seconds when establishing <a href="https://cybersecuritynews.com/windows-server-2025-security-update/" target="_blank" rel="noreferrer noopener">UDP-based</a> connections.</p>



<p>While Windows Server 2025 systems are not directly impacted as RDS hosts, they may encounter disconnects when functioning as RDP clients connecting to older servers.</p>



<p>Microsoft has acknowledged the issue and observed a substantial increase in reports following the March 2025 security update. The company has taken steps to address the problem using Known Issue Rollback (KIR), a feature designed to reverse problematic non-security updates delivered via Windows Update.</p>



<h2 class="wp-block-heading"><strong>The Issue</strong></h2>



<p>To resolve the issue on affected Windows enterprise-managed devices, IT administrators are advised to install and configure a special Group Policy. The policy, named &#8220;Windows 11 24H2 and Windows Server 2025 KB5053598 250314_20401 Known Issue Rollback,&#8221; can be found under Computer Configuration > Administrative Templates in the Group Policy Editor.</p>



<p>For most home and professional users, the rollback should be applied automatically. However, restarting the device may expedite the process. IT administrators looking to deploy the fix immediately should follow Microsoft&#8217;s guidelines for using Group Policy to implement a Known Issue Rollback.</p>



<p>It&#8217;s important to note that this Group Policy serves as a temporary solution, disabling the change that caused the issue. Microsoft is actively working on a permanent resolution, which will be included in a future Windows update. Once this update is released, organizations will no longer need to rely on the Group Policy to address the problem.</p>



<p>The Remote Desktop issues are not isolated to this particular update. Recent <a href="https://cybersecuritynews.com/windows-server-2025-security-update/" target="_blank" rel="noreferrer noopener">Windows Server 2025</a> updates have also caused Remote Desktop freezes, with users experiencing unresponsive mouse and keyboard inputs shortly after connecting to a session.</p>



<p>Microsoft resolved a similar issue for Windows 11 24H2 systems with the KB5052093 optional update released on February 25, 2025, and plans to roll out a fix for Windows Server 2025 in an upcoming update.</p>



<p>These Remote Desktop problems highlight the ongoing challenges Microsoft faces in maintaining seamless functionality across its diverse Windows ecosystem. </p>



<p>As the company continues to release updates and new versions of its operating systems, users and IT administrators must remain vigilant and prepared to implement temporary solutions when issues arise.</p>



<p>For those experiencing difficulties with the KB5050094 update, Microsoft has provided troubleshooting steps, including running Windows Update troubleshooter, addressing service issues, and using the System File Checker (SFC) and Deployment Image Servicing and Management (DISM) tools.</p>



<p>As Microsoft works towards a permanent solution, users are advised to stay informed about the latest updates and apply the necessary fixes to ensure smooth Remote Desktop operations across their Windows environments.</p>
