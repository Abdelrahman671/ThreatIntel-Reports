There are indications that katana may be associated with an  HTTP banking botnet in the future.</p><p>We’ve <a href=https://www.avira.com/en/blog/how-is-the-mirai-variant-infecting-the-iot-landscape target=_blank rel="noopener noreferrer">previously looked at how Mirai</a>, an IoT botnet, has evolved since its source code became public. A recent analysis of IoT attacks and <a href=https://www.avira.com/en/blog/malware-threat-report-q2-2020-statistics-and-trends target=_blank rel="noopener noreferrer">malware trends</a> shows that Mirai&#8217;s evolution continues. For example, variants of Mirai can be bought, sold, or sourced via YouTube channels, in this case, <a href=https://www.youtube.com/channel/UC6ZLky15GgnkPOl7Co50jPg/featured target=_blank rel="noopener noreferrer">VegaSec</a>. These, and other changes, enable unskilled attackers to create malicious botnets, resulting in an increase in attacks.</p><h2>Analysis</h2><p>The Katana botnet attempts to exploit old vulnerabilities via Remote Code Execution/Command Injection vulnerabilities such as LinkSys and GPON home routers.
