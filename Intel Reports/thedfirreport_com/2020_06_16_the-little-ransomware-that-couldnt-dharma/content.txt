			<h1 class="entry-title">The Little Ransomware That Couldnâ€™t (Dharma)</h1>
		<div class="entry-meta">
			<span class="posted-on"><a href="https://thedfirreport.com/2020/06/16/the-little-ransomware-that-couldnt-dharma/" rel="bookmark"><time class="entry-date published" datetime="2020-06-16T11:52:41+00:00">June 16, 2020</time></a></span>		</div><!-- .entry-meta -->

	<div class="entry-content">
		
<p>Ransomware continues unabated in the year of continually mounting pressure. But for every big game actor out there compromising Fortune listed companies there are the little guys that maybe just aren&#8217;t as skilled.</p>
<h3>Initial access:</h3>
<p>Threat actor logged in from 217.138.202.116 as a local admin at 0858 UTC.</p>
<h3>Reconnaissance:</h3>
<p>The threat actor brought their own enumeration and pivoting tool set NS.exe. We&#8217;ve seen this tool used time and time again to scan and map file shares. Executed at 0936 UTC.Â </p>
<p><img fetchpriority="high" decoding="async" class="alignnone size-full wp-image-484" src="https://thedfirreport.com/wp-content/uploads/2020/06/d1.png" alt="" width="979" height="515" srcset="https://thedfirreport.com/wp-content/uploads/2020/06/d1.png 979w, https://thedfirreport.com/wp-content/uploads/2020/06/d1-300x158.png 300w, https://thedfirreport.com/wp-content/uploads/2020/06/d1-768x404.png 768w" sizes="(max-width: 979px) 100vw, 979px" /></p>
<pre><span title="" data-original-title="">%USERPROFILE%\Desktop\Oc\NS.exe</span></pre>
<h3>Action on Objectives:</h3>





<p>First was the execution of a bat file called shadow.bat, which deletes shadow files</p>



<figure class="wp-block-image size-large"><img decoding="async" width="1044" height="486" class="wp-image-439" src="https://i0.wp.com/thedfirreport.com/wp-content/uploads/2020/06/image-3.png?fit=1024%2C477&amp;ssl=1" alt="" srcset="https://thedfirreport.com/wp-content/uploads/2020/06/image-3.png 1044w, https://thedfirreport.com/wp-content/uploads/2020/06/image-3-300x140.png 300w, https://thedfirreport.com/wp-content/uploads/2020/06/image-3-1024x477.png 1024w, https://thedfirreport.com/wp-content/uploads/2020/06/image-3-768x358.png 768w" sizes="(max-width: 1044px) 100vw, 1044px" /></figure>
<p>&nbsp;</p>
<pre>vssadmin delete shadows /all</pre>



<p>Seconds later logdelete.bat is run which clears all log files.</p>



<figure class="wp-block-image size-large"><img decoding="async" width="1070" height="491" class="wp-image-440" src="https://i1.wp.com/thedfirreport.com/wp-content/uploads/2020/06/image-4.png?fit=1024%2C470&amp;ssl=1" alt="" srcset="https://thedfirreport.com/wp-content/uploads/2020/06/image-4.png 1070w, https://thedfirreport.com/wp-content/uploads/2020/06/image-4-300x138.png 300w, https://thedfirreport.com/wp-content/uploads/2020/06/image-4-1024x470.png 1024w, https://thedfirreport.com/wp-content/uploads/2020/06/image-4-768x352.png 768w" sizes="(max-width: 1070px) 100vw, 1070px" /></figure>



<p>&nbsp;</p>
<p>Following this, closeapps.bat was run which loops through various common applications to try to prevent open file lockouts keeping critical files from being encrypted.</p>
<p><img loading="lazy" decoding="async" class="alignnone size-full wp-image-488" src="https://thedfirreport.com/wp-content/uploads/2020/06/d2.png" alt="" width="539" height="911" srcset="https://thedfirreport.com/wp-content/uploads/2020/06/d2.png 539w, https://thedfirreport.com/wp-content/uploads/2020/06/d2-177x300.png 177w" sizes="auto, (max-width: 539px) 100vw, 539px" /></p>





<p>Registry run keys and two startup folders were then created for the primary ransomware file 1pgp.exe.</p>
<pre>C:\Users\Administrator\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup\1pgp.exe<br />C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp\1pgp.exe<br />HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Run\1pgp.exe</pre>
<!-- /wp:paragraph -->

<!-- wp:paragraph -->
<p>Finally the ransomware executed and locked up the system.</p>
<p>This ransomware can be linked to the Dharma/Crysis family of ransomware based on the pdb path present in the file strings.</p>
<p><img loading="lazy" decoding="async" class="alignnone size-full wp-image-515" src="https://thedfirreport.com/wp-content/uploads/2020/06/image-1-1.png" alt="" width="346" height="103" srcset="https://thedfirreport.com/wp-content/uploads/2020/06/image-1-1.png 346w, https://thedfirreport.com/wp-content/uploads/2020/06/image-1-1-300x89.png 300w" sizes="auto, (max-width: 346px) 100vw, 346px" /></p>
<p>A rather sparse ransom note was left behind.</p>
<!-- /wp:paragraph -->

<!-- wp:image {"id":438,"sizeSlug":"large"} -->
<figure class="wp-block-image size-large"><img loading="lazy" decoding="async" class="alignnone size-full wp-image-499" src="https://thedfirreport.com/wp-content/uploads/2020/06/1-1.png" alt="" width="271" height="111" /></figure>
<!-- /wp:image -->

<!-- wp:paragraph -->
<h3>Â </h3>
<h3>Lateral Movement</h3>
<p>Even though the threat actor was running as a domain administrator they did not attempt to move laterally or spread their ransomware.</p>
<h3>IOC&#8217;s</h3>
