It is hosted on the server with IP <b>8.209.99.41</b>.</li></ul><p>This same server also hosts the soon-to-be-expired domain <b>gstatlcs[.]com</b>, which was registered on <strong>July 23rd, 2019</strong>.</p><h2>Conclusion</h2><p>Web skimmer operators are always actively searching for new methods to prevent detection of their malware on compromised websites.</p><p>This time, we found them combining four popular tricks to conceal their malicious code:</p><ol><li>Including requests to usually benign static content (e.g. stylesheets or images) that are normally less scrutinized in traffic monitoring or static file analysis. (<a href="https://blog.sucuri.net/2020/01/web-swiper-in-image-title.html">1</a>, <a href="https://blog.sucuri.net/2016/10/magento-credit-card-swiper-exports-image.html">2</a>)</li><li>Planting malicious code inside real images. (<a href="https://blog.sucuri.net/2018/07/hiding-malware-inside-images-on-googleusercontent.html">1</a>, <a href="https://blog.sucuri.net/2013/07/malware-hidden-inside-jpg-exif-headers.html">2</a>, <a href="https://blog.sucuri.net/2015/11/exif-php-joomla-backdoor.html">3</a>)</li><li>Hosting malicious files on popular legitimate websites such as GitHub. (<a href="https://blog.sucuri.net/2017/12/malicious-cryptominers-from-github.html">1</a>, <a href="https://blog.sucuri.net/2018/03/github-hosts-lokibot-infostealer.html">2</a>, <a href="https://blog.sucuri.net/2018/03/github-hosts-infostealers-part-2-cryptominers-and-credit-card-stealers.html">3</a>).</li><li>Using misleading variable names, filenames, and domains to make people believe they belong to a reputable popular service (in this case, Google Tag Manager). (<a href="https://blog.sucuri.net/2019/08/magento-skimmers-from-atob-to-alibaba.html">1</a>, <a href="https://blog.sucuri.net/2019/02/google-analytics-and-angular-in-magento-credit-card-stealing-scripts.html">2</a>).</li></ol><p>While this approach may make it more difficult to spot the malware for third-party researchers, webmasters who implement <a href="https://docs.sucuri.net/plugins/core-integrity-check/">integrity control checks</a> or <a href="https://sucuri.net/malware-detection-scanning/">website monitoring services</a> should be able to detect addition of new files to the system or changes in existing files.</p></div><div class="post-author"><div class="author-wrap"><div class="author"><div class="author-avatar"> <a href="https://blog.sucuri.net/author/denis" rel="author"> <img alt='' src='https://secure.gravatar.com/avatar/c9ef50b85bd345ea4e0d8da558816f3d?s=120&#038;d=mm&#038;r=g' srcset='https://secure.gravatar.com/avatar/c9ef50b85bd345ea4e0d8da558816f3d?s=240&#038;d=mm&#038;r=g 2x' class='avatar avatar-120 photo' height='120' width='120' loading='lazy' decoding='async'/> </a></div><div class="author-description"><h5 class="title-author"> <span class="fn"> <a href="https://blog.sucuri.net/author/denis" rel="author"> Denis Sinegubko </a> </span></h5><p class="note">Denis Sinegubko is Sucuri’s Senior Malware Researcher who joined the company in 2013.
