          <h1>Trik Spam Botnet Leaks 43 Million Email Addresses</h1>
          <div class="cz-news-story-title-section">
            <div class="cz-news-title-left-area">
                            By <h6><a rel="author" href="https://www.bleepingcomputer.com/author/catalin-cimpanu/" class="author"><span itemprop="author" itemscope itemtype="https://schema.org/Person"><span itemprop="name">Catalin Cimpanu</span></span></a></h6>
            </div>
	    <meta itemprop="articleSection" content="Security"><div class="cz-news-title-right-area">
              <ul><li class="cz-news-date">June 12, 2018</li>
                <li class="cz-news-time">11:22 AM</li>
                
                <li class="cz-news-comment"><a href="#comments">3</a></li>
              </ul></div>
          </div>
	  <div class="articleBody">
	   
<p>Over 43 million email addresses have leaked from the command and control server of a spam botnet, a security researcher has told Bleeping Computer today.</p>

<p>The leaky server came to light while a threat intelligence analyst from <a href="http://cybersecurity.vertek.com" rel="nofollow" target="_blank">Vertek Corporation</a>, was looking into a recent malware campaign distributing a version of the Trik trojan, which was later infecting users with a second-stage payload &mdash;the GandCrab 3 ransomware.</p>

<p>The Vertek researcher discovered that Trik and GandCrab would download the malicious files that infected users' systems from an online server located on a Russian IP address.</p>

<p style="text-align:center"><img alt="Trik campaign" height="494" src="https://www.bleepstatic.com/images/news/u/986406/Malware/Trojans/Trick-Phorpiex-campaign.jpg" width="800"></p>

<p>The researcher told Bleeping Computer that the group behind this operation misconfigured its server and left its content accessible to anyone accessing the IP directly.</p>

<p>On this server, he discovered 2201 text files, labeled sequentially from 1.txt to 2201.txt containing chunks of roughly 20,000 email addresses, each.</p>

<p>The Vertek researcher believes the operators of this server have been using these recipient lists to service other crooks who contracted their services to distribute various malware strains via malspam campaigns.</p>

<p style="text-align:center"><img alt="Leaky Trik server" height="488" src="https://www.bleepstatic.com/content/posts/2018/06/12/Trick-Phorpiex-leak.png" width="895"></p>

<h2>Server leaks 43,555,741 unique email addresses</h2>

<p>"We pulled all of them to validate that they are unique and legitimate," the researcher told Bleeping Computer earlier today. "Out of 44,020,000 potential addresses, 43,555,741 are unique."</p>

<p>The researcher is now working with Australian security expert Troy Hunt, the owner of the Have I Been Pwned service, to determine how many of these emails are new and how many have been previously leaked in other data dumps.</p>

<p>"The email addresses are from everywhere," the researcher told us. " There were 4.6 million unique email domains. Everything from .gov to .com, and domain of several private businesses."</p>

<p>The Vertek researcher has analyzed the files and broke down the email addresses per domain. In a list the researcher shared with us earlier today (embedded at the bottom of this article), he points out that the vast majority of email addresses are old, from antiquated email services such as Yahoo (10.6 million) and AOL (8.3 million).</p>

<p>Surprisingly, while there are many custom email domains included in the leak, there are very few Gmail addresses included, suggesting the email addresses database is either incomplete, or this malware campaign intentionally targeted users using older email services.</p>

<h2>The Trik trojan</h2>

<p>The Trik trojan is a classic malware downloader. It infects computers and assembles them into a giant botnet. The botnet's operators use these computers to send out new spam campaigns, or they sell "install space" to other crooks, allowing them to deliver more pontent threats to Trik victims, similarly to how they rented install space to the GandCrab crew for the campaign Vertek stumbled on.</p>

<p>The Trik trojan has been an active threat for at least a decade but has recently seen a resurgence, according to this <a href="https://www.proofpoint.com/us/threat-insight/post/phorpiex-decade-spamming-shadows" rel="nofollow" target="_blank">Proofpoint report</a>.</p>

<p>In its earlier days, the malware operated primarily as a worm that self-spread via removable USB storage devices, Skype, or Windows Live Messenger chats. These worm-based variants had previously been tracked under the name of <a href="https://blog.trendmicro.com/trendlabs-security-intelligence/shylock-not-the-lone-threat-targeting-skype/" rel="nofollow" target="_blank">Phorpiex</a>.</p>

<p>The malware evolved into a fully-fledged trojan years later, when it forked the codebase of the SDBot trojan and started using email spam as its main delivery &amp; infection mechanism, while also switching to an <a href="https://www.johannesbader.ch/2016/02/phorpiex/" rel="nofollow" target="_blank">IRC-controlled botnet architecture</a>.</p>

<p>Trik is not the first spam botnet to leak its email addresses database. <a href="https://benkowlab.blogspot.com/2017/08/from-onliner-spambot-to-millions-of.html" rel="nofollow" target="_blank">In August 2017</a>, a spam operation known as Onliner leaked 711 million email addresses that it was using to spam users.</p>

<p>At the time of writing, the Trik C&amp;C server that's leaking email addresses keeps going offline at intermittent intervals.</p>

function cz_strip_tags(input, allowed) {
  allowed = (((allowed || '') + '')
    .toLowerCase()
    .match(/<[a-z][a-z0-9]*>/g) || [])
    .join(''); // making sure the allowed arg is a string containing only tags in lowercase (<a><b><c>)
  var tags = /<\/?([a-z][a-z0-9]*)\b[^>]*>/gi,
    commentsAndPhpTags = /<!--[\s\S]*?-->|<\?(?:php)?[\s\S]*?\?>/gi;
  return input.replace(commentsAndPhpTags, '')
    .replace(tags, function($0, $1) {
      return allowed.indexOf('<' + $1.toLowerCase() + '>') > -1 ? $0 : '';
    });
}
function cz_br2nl(str) {
	var regex = /<br\s*[\/]?>/gi;
    //var pure_str = str.replace(regex,"\n");
    var pure_str = str.replace(regex,"");
    return cz_strip_tags(pure_str,'');
}
