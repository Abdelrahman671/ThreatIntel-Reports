<h1><span id="hs_cos_wrapper_name" class="hs_cos_wrapper hs_cos_wrapper_meta_field hs_cos_wrapper_type_text" style="" data-hs-cos-general-type="meta_field" data-hs-cos-type="text">Protestware taps npm to call out wars in Ukraine, Gaza</span></h1>
<div id="hs_cos_wrapper_module_1577203992502740" class="hs_cos_wrapper hs_cos_wrapper_widget hs_cos_wrapper_type_module" style="" data-hs-cos-general-type="widget" data-hs-cos-type="module">
<h2 class="post-header__secondary-head blog__post-secondary-h2">ReversingLabs researchers have discovered npm packages that hide scripts broadcasting messages of peace related to the conflicts in Ukraine and in Israel and the Gaza Strip.</h2>

</div>

<div class="blog__post-content html-content">
<span id="hs_cos_wrapper_post_body" class="hs_cos_wrapper hs_cos_wrapper_meta_field hs_cos_wrapper_type_rich_text" style="" data-hs-cos-general-type="meta_field" data-hs-cos-type="rich_text"><p><img src="https://www.reversinglabs.com/hs-fs/hubfs/Blog/Protestware-taps-npm-to-call-out-wars-in-Ukraine-and-Gaza.webp?width=1400&amp;height=732&amp;name=Protestware-taps-npm-to-call-out-wars-in-Ukraine-and-Gaza.webp" width="1400" height="732" loading="lazy" alt="Protestware taps npm to call out wars in Ukraine, Gaza " style="height: auto; max-width: 100%; width: 1400px;" srcset="https://www.reversinglabs.com/hs-fs/hubfs/Blog/Protestware-taps-npm-to-call-out-wars-in-Ukraine-and-Gaza.webp?width=700&amp;height=366&amp;name=Protestware-taps-npm-to-call-out-wars-in-Ukraine-and-Gaza.webp 700w, https://www.reversinglabs.com/hs-fs/hubfs/Blog/Protestware-taps-npm-to-call-out-wars-in-Ukraine-and-Gaza.webp?width=1400&amp;height=732&amp;name=Protestware-taps-npm-to-call-out-wars-in-Ukraine-and-Gaza.webp 1400w, https://www.reversinglabs.com/hs-fs/hubfs/Blog/Protestware-taps-npm-to-call-out-wars-in-Ukraine-and-Gaza.webp?width=2100&amp;height=1098&amp;name=Protestware-taps-npm-to-call-out-wars-in-Ukraine-and-Gaza.webp 2100w, https://www.reversinglabs.com/hs-fs/hubfs/Blog/Protestware-taps-npm-to-call-out-wars-in-Ukraine-and-Gaza.webp?width=2800&amp;height=1464&amp;name=Protestware-taps-npm-to-call-out-wars-in-Ukraine-and-Gaza.webp 2800w, https://www.reversinglabs.com/hs-fs/hubfs/Blog/Protestware-taps-npm-to-call-out-wars-in-Ukraine-and-Gaza.webp?width=3500&amp;height=1830&amp;name=Protestware-taps-npm-to-call-out-wars-in-Ukraine-and-Gaza.webp 3500w, https://www.reversinglabs.com/hs-fs/hubfs/Blog/Protestware-taps-npm-to-call-out-wars-in-Ukraine-and-Gaza.webp?width=4200&amp;height=2196&amp;name=Protestware-taps-npm-to-call-out-wars-in-Ukraine-and-Gaza.webp 4200w" sizes="(max-width: 1400px) 100vw, 1400px"></p>
<p>Newly discovered open source software packages on the npm platform contain scripts that broadcast peace messages related to ongoing conflicts in Ukraine and on the Gaza Strip when they are deployed, according to research conducted by ReversingLabs.&nbsp;<br><br>The packages are just the latest examples of so-called “protestware,” a recurrent issue in the open source software ecosystem in which application developers conceal political messages inside open source code, often designing it to display to the user after an application is installed or when it is executed.</p>
<p>Although the latest packages are not malicious, they underscore a persistent risk in open source software, in which unintended and malicious features can lurk undetected — even in widely used applications.</p>
<!--more-->
<h2>Protest package for Palestine</h2>
<p>The npm package <a href="https://www.npmjs.com/package/e2eakarev" rel="noopener" target="_blank" style="font-style: italic;">e2eakarev</a>, version: 7.1.0, published in late October, describes itself as a “free Palestine protest package” and was posted by the npm user ~updater.downloader. It has a total of 8 downloads. ReversingLabs researcher Lucija Valentić found that installing the package launches a postinstall script, <span style="font-style: italic;">index.js</span>, which checks to see if the package is being launched in Israel. If it is, an English language message calling for peace is displayed in the terminal.</p>
<p>“The world has witnessed too much suffering in Palestine, and it is time for us to take a stand,” the message reads in part. It calls on those reading the message to raise awareness of the “Palestinian struggle,” donate to humanitarian aid and back the “Boycott, Divest, Sanction" (BDS) movement, which seeks to encourage individuals and businesses to divest from Israel.</p>
<p>The message is signed “The Anonymous Protester.” And that… is about it. The package was not observed to perform any other actions — malicious or not, Valentić said.</p>
<h2>Protestware: Sneaky!</h2>
<p>Another npm package that was found recently to contain protestware is <a href="https://www.npmjs.com/package/@snyk/sweater-comb" rel="noopener" target="_blank" style="font-style: italic;">@snyk/sweater-comb,</a> version 2.1.1, which was first published in August, 2023 by the firm Snyk. <span style="font-style: italic;">Sweater-comb</span> is described as a “tool which provides guardrails with automation” of Snyk APIs by applying custom Optic CI rules to Snyk's OpenAPI specifications. Version 2.1.1 includes the <span style="font-style: italic;">module es5-ext</span>, <a href="https://www.npmjs.com/package/es5-ext" rel="noopener" target="_blank">ECMAScript 5 extensions</a>, a common module with more than 7.3 million weekly downloads. This is the last version of the Snyk package that includes the <span style="font-style: italic;">es5-ext</span> module, with protestware functionality. All package versions since have opted out from using it as a dependency.<br><br>What does this protestware do? After installation, <span style="font-style: italic;">es5-ext</span> launches a postinstall script, <span style="font-style: italic;">_postinstall.js</span>, which attempts to determine the geographic location of the host. If the host machine is located in Russia, it shows a message, in Russian, that is critical of Russia's invasion of neighboring Ukraine, and in Russian when the package is installed in Russia.&nbsp;<br><br>“The whole world condemned the unjustified invasion and decided to impose unprecedented sanctions against Russia,” it reads. “With each new day they will be felt stronger and stronger among civilians.” Russian speakers reading the message are encouraged to download the Tor browser, and visit a web page on how to circumvent censorship in the country.&nbsp;<br><br>As it turns out, <span style="font-style: italic;">es5-ext</span> is a package with a long history. The firm Checkmarx<a href="https://checkmarx.com/blog/new-protestware-found-lurking-in-highly-popular-npm-package/" rel="noopener" target="_blank"> reported on it in March 2022</a>, noting that the protestware feature was added to the <span style="font-style: italic;">es5-ext</span> package in early March of that year, shortly after Russia invaded Ukraine. The same report also highlighted another module, <span style="font-style: italic;">styled-components</span>, that was likewise updated to include a post-install script that offered pro-Ukraine protest messages.&nbsp;<br><br>ReversingLabs researchers have subsequently found the <span style="font-style: italic;">es5-ext</span> used in other popular applications, including the installation executable for Signal Messenger for Windows. With about 10 million weekly downloads, the reach of the <span style="font-style: italic;">es5-ext</span> package is broad. Within developer forums, the issue of the protestware features <a href="https://github.com/medikoo/es5-ext/issues/186" rel="noopener" target="_blank">frequently crops up</a>, as developers utilizing the <span style="font-style: italic;">es5-ext</span> module contend with antivirus programs flagging their wares as infected with a virus.<br><br>Since July 2023, ReversingLabs detected 179 npm packages using the <span style="font-style: italic;">es5-ext</span> package (a count that includes every version of each npm package). That isn’t a huge number. However, that list includes many legitimate packages like <em>sweater-comb</em>, Signal and version 6.4.0 of the <a href="https://www.npmjs.com/package/shut" rel="noopener" target="_blank">shut framework</a> with substantial distributions and user bases.&nbsp;</p>
<a href="https://blog.reversinglabs.com/hubfs/ReversingLabs-report-for-Signal-Messenger-Windows-installation-executable.webp" class="lightbox-image"><img src="https://www.reversinglabs.com/hs-fs/hubfs/ReversingLabs-report-for-Signal-Messenger-Windows-installation-executable.webp?width=1400&amp;name=ReversingLabs-report-for-Signal-Messenger-Windows-installation-executable.webp" alt="ReversingLabs" width="1400" style="width: 1400px;" srcset="https://www.reversinglabs.com/hs-fs/hubfs/ReversingLabs-report-for-Signal-Messenger-Windows-installation-executable.webp?width=700&amp;name=ReversingLabs-report-for-Signal-Messenger-Windows-installation-executable.webp 700w, https://www.reversinglabs.com/hs-fs/hubfs/ReversingLabs-report-for-Signal-Messenger-Windows-installation-executable.webp?width=1400&amp;name=ReversingLabs-report-for-Signal-Messenger-Windows-installation-executable.webp 1400w, https://www.reversinglabs.com/hs-fs/hubfs/ReversingLabs-report-for-Signal-Messenger-Windows-installation-executable.webp?width=2100&amp;name=ReversingLabs-report-for-Signal-Messenger-Windows-installation-executable.webp 2100w, https://www.reversinglabs.com/hs-fs/hubfs/ReversingLabs-report-for-Signal-Messenger-Windows-installation-executable.webp?width=2800&amp;name=ReversingLabs-report-for-Signal-Messenger-Windows-installation-executable.webp 2800w, https://www.reversinglabs.com/hs-fs/hubfs/ReversingLabs-report-for-Signal-Messenger-Windows-installation-executable.webp?width=3500&amp;name=ReversingLabs-report-for-Signal-Messenger-Windows-installation-executable.webp 3500w, https://www.reversinglabs.com/hs-fs/hubfs/ReversingLabs-report-for-Signal-Messenger-Windows-installation-executable.webp?width=4200&amp;name=ReversingLabs-report-for-Signal-Messenger-Windows-installation-executable.webp 4200w" sizes="(max-width: 1400px) 100vw, 1400px"></a>
<p style="text-align: center;"><br><span style="font-weight: bold; font-size: 16px;">Figure 1: ReversingLabs’ report for Signal Messenger Windows installation executable showing the presence of protestware</span></p>
<p>That is concerning, because even though the postinstall script displaying the pro-Ukraine message isn’t malicious, it is unneeded.&nbsp;<br><br>“The problem is that this package is doing more than it advertises. It not only extends es5 with extra methods and shims, it also implements a "Call for peace" message. I think it's an unacceptable practice that slows done [sic] installation process,” a developer with the handle Rush commented on GitHub. &nbsp;<br><br>And that’s a problem. The ability of sketchy and unnecessary code like the latest protestware in <em>es5-ext</em> to work its way into widely used and legitimate applications and platforms like Signal, the shut framework and <em>sweater-comb</em> suggests that the cyber risks lurking in software supply chains are real.</p>
<h2>Righteous? Yes. Malicious? Maybe.</h2>
<p>Both of the examples of protestware cited above were just that: software that is used to convey a message of protest. ReversingLabs did not observe malicious functionality designed to disrupt the systems running the protestware, steal sensitive information or engage in lateral movement in either case.&nbsp;<br><br>However, not all protestware is benign. In the immediate aftermath of Russia’s invasion of Ukraine in February 2022, for example, an update to the <a href="https://github.com/vuejs/vue-cli/issues/7054" rel="noopener" target="_blank" style="font-style: italic;">vue-cli open source module</a> was updated to include functionality that identified systems running in Russia and <a href="https://krebsonsecurity.com/2022/03/pro-ukraine-protestware-pushes-antiwar-ads-geo-targeted-malware/" rel="noopener" target="_blank">then wiped them of sensitive files and information</a>.&nbsp;<br><br>As violent conflicts crop up around the globe, expect more instances of protestware that takes up the cause of both innocent victims and warring parties. In some cases, that protestware may come in the form of updates to benign and even well-established modules. A certain percentage of it will cross the line from benign display messages to actually inflicting damage on targeted systems.&nbsp;</p>
<h2>Dig deep on package security</h2>
<p style="font-size: 18px;">The lesson for organizations is that the days of whistling past the graveyard when it comes to software supply chain threats is over. Developers inclined to make hasty conclusions about the security or integrity of any piece of software based on its past performance, age or popularity are setting themselves up to be disappointed, said Tomislav Peričin, the co-founder and Chief Software Architect for ReversingLabs.&nbsp;&nbsp;</p>
<blockquote>
<p><span style="font-size: 24px; font-style: italic;">“We have seen an increase in the number of software supply chain attacks, impactful vulnerabilities, and other incidents. The risks that developers and software consumers face have never been higher, and that includes political messages. Having software perform random acts of political activism does little for the specific cause. But it does decrease the private sector’s already shaky trust in software."</span><br><span style="font-size: 24px; font-style: italic;">—<a href="https://www.linkedin.com/in/tomislav-peri%C4%8Din-746064286/" rel="noopener" target="_blank">Tomislav Peričin</a></span></p>
</blockquote>
<p>If you’re lucky, that disappointment may come in the form of a postinstall script throwing up an unexpected appeal for peace. At the worst, the protest may take the form of a wiped system, stolen data or a fast-spreading worm.&nbsp;<br><br>The presence of political protest messages in open source code is just one more indicator of the need for development organizations to dig deeper into the function and integrity of the open source, as well as third-party and proprietary code they rely on. Postinstall scripts like the ones found in <em>e2eakarev</em> and <em>es5-ext</em> may not cause damage, but they are often an indication of the willingness of the package developers to use their code as a platform for their own words, actions and ideas. When that happens without adequate disclosure to the users of the module (as was the case with <em>es5-ext</em>), it is cause for concern and additional scrutiny for any development organization.&nbsp;<br><br>ReversingLabs is constantly hunting for suspicious package behaviors. Our Software Supply Chain Security platform uses behavior indicators that can track if packages have <em>postinstall</em> scripts, allowing us to spot protest packages like <em>e2eakarev</em> and <em>es5-ext </em>and determine whether they are benign, malicious or somewhere in between. That kind of scrutiny of software dependencies is increasingly important as both malicious and political actors look to software supply chains to make their presence felt.&nbsp;</p>
<h2>Next steps</h2>
<p>As the discovery of these latest examples of protestware indicate, development and application security teams need the ability to peer deeper inside software packages and distinguish malicious functions from legitimate ones.&nbsp;<br><br>This can be challenging with <a href="/blog/why-app-sec-testing-alone-is-not-enough-to-secure-your-software-supply-chain" rel="noopener">legacy application security testing tools</a>, and demands specialized skills and knowledge that many organizations lack. Modern tools like <a href="/products/software-supply-chain-security" rel="noopener">ReversingLabs Software Supply Chain Security</a> can bridge the gaps, and help development and application security teams protect their supply chain from compromise.</p>
