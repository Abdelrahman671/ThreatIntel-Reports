                    <div class="node-body article-body"><div></div><p><b>A</b>. Below is a summary of all the net.exe usage methods.</p>                              <p><b>net accounts</b></p>                              <p>Used to modify user accounts. Specified on its own will give information                              about the current logon.</p>                              <p>Options: </p>                              <table width="100%" border="1">                              <tr>                              <td width="185">/forcelogoff:&lt;minutes or no&gt;</td>                              <td>Minutes until the user gets logged off after logon hours expire. No means a                              forced logoff will not occur</td>                              </tr>                              <tr>                              <td width="185">/lockoutthreshold:&lt;number of failed attempts&gt;</td>                              <td>This parameter allows you to configure the number of failed logon attempts                              before the account is locked. The range is 1 to 999. </td>                              </tr>                              <tr>                              <td width="185">/lockoutduration:&lt;minutes&gt; </td>                              <td>This parameter specifies the number of minutes accounts remain locked                              before automatically becoming unlocked. The range is 1 to 99999. </td>                              </tr>                              <tr>                              <td width="185">/lockoutwindow:&lt;minutes&gt; </td>                              <td> This parameter lets you configure the maximum number of minutes                              between two consecutive failed logon attempts before an account is locked. The                              range is 1 to 99999.</td>                              </tr>                              <tr>                              <td width="185">/minpwlen:&lt;length&gt;</td>                              <td>Minimum number of characters for the password. Default is 6, valid range is                              between 0 and 14</td>                              </tr>                              <tr>                              <td width="185">/maxpwage:&lt;days&gt;</td>                              <td>Maximum number of days a password is valid. Default is 90, valid range is                              between 0 and 49710</td>                              </tr>                              <tr>                              <td width="185">/minpwage:&lt;days&gt;</td>                              <td>Number of days that must occur before the password can be changed. Default                              is 0, valid range is between 0 and maxpwage</td>                              </tr>                              <tr>                              <td width="185">/uniquepw:&lt;number&gt;</td>                              <td>Password may not be reused for number attempts</td>                              </tr>                              <tr>                              <td width="185">/sync</td>                              <td>Forces a domain sync</td>                              </tr>                              <tr>                              <td width="185">/domain</td>                              <td>Performs any of the above actions on the domain controller</td>                              </tr>                              </table>                              <p><b>net computer</b></p>                              <p>Used to add and remove computer accounts from the domain.</p>                              <p>Options: </p>                              <table width="100%" border="1">                              <tr>                              <td width="185">\\&lt;computer name&gt;</td>                              <td>Name of the computer to be added or removed</td>                              </tr>                              <tr>                              <td width="185">/add</td>                              <td>Add the specified computer</td>                              </tr>                              <tr>                              <td width="185">/del</td>                              <td>Removes the specified computer</td>                              </tr>                              </table>                              <p><b>net config server</b></p>                              <p>Allows modifications to the server service. Entered with no parameters give                              details of the current configuration</p>                              <p>Options:</p>                              <table width="100%" border="1">                              <tr>                              <td width="185">/autodisconnect:&lt;minutes&gt;</td>                              <td>Number of minutes an account may be inactive before disconnection. Default                              is 15, valid range between 1 and 65535. -1 means never disconnected.</td>                              </tr>                              <tr>                              <td width="185">/srvcomment:"text"</td>                              <td>Set the comment for the machine</td>                              </tr>                              <tr>                              <td width="185">/hidden:&lt;yes or no&gt;</td>                              <td>Specified is the computer is hidden in the listing of computers</td>                              </tr>                              </table>                              <p><b>net config workstation</b></p>                              <p>Allows modifications to the workstation service. Entered with no parameters                              give details of the current configuration</p>                              <p>Options: </p>                              <table width="100%" border="1">                              <tr>                              <td width="185">/charcount:&lt;bytes&gt;</td>                              <td>Number of bytes to be collected before data is sent. The default is 16,                              valid range is between 0 and 65535.</td>                              </tr>                              <tr>                              <td width="185">/chartime:&lt;msec&gt;</td>                              <td>Number of milliseconds NT waits before sending data. If charcount is also                              set whichever is satisfied first is used. Default is 250, valid range is                              between 0 and 65535000.</td>                              </tr>                              <tr>                              <td width="185">/charwait:&lt;seconds&gt;</td>                              <td>Number of seconds NT waits for a communications device to become available.                              Default is 3600, valid is between 0 and 65535.</td>                              </tr>                              </table>                              <p><b>net continue &lt;service name&gt;</b></p>                              <p>Restarts the specified paused service.</p>                              <p><b>net file</b></p>                              <p>Lists any files that are open/locked via a network share.</p>                              <p>Options: </p>                              <table width="100%" border="1">                              <tr>                              <td width="185">id</td>                              <td>Identification of the file (given by entering net file on its own)</td>                              </tr>                              <tr>                              <td width="185">/close</td>                              <td>Close the specified lock</td>                              </tr>                              </table>                              <p>See <b><a href="/web/20150511162820/http://windowsitpro.com/article/john-savills-windows-faqs/faq.asp?faqnumber=1586">Q. Without parameters will list global                              groups.</p>                              <p>Syntax:</p>                              <p>net group &lt;group name&gt; \[/command:"&lt;text&gt;"\]                              \[/domain\]<br/>                              net group &lt;group name&gt; \[/add \[/comment:"&lt;text&gt;"\] or                              /delete\] \[/domain\]<br/>                              net group &lt;group name&gt; &lt;user name&gt; /add or /delete \[/domain\]</p>                              <p>Options:</p>                              <table width="100%" border="1">                              <tr>                              <td width="185">groupname</td>                              <td>Name of the global group</td>                              </tr>                              <tr>                              <td width="185">/comment:"&lt;text&gt;"</td>                              <td>Comment if a new global group is created. Up to 48 characters</td>                              </tr>                              <tr>                              <td width="185">/domain</td>                              <td>Performs the function on the primary domain controller</td>                              </tr>                              <tr>                              <td width="185">username</td>                              <td>Username to which apply the operation</td>                              </tr>                              <tr>                              <td width="185">/add</td>                              <td>Adds the specified user to the group or the group to the domain</td>                              </tr>                              <tr>                              <td width="185">/delete</td>                              <td>Removes a group from a domain or a user from a group</td>                              </tr>                              </table>                              <p><b>net localgroup</b></p>                              <p>Performs actions on local groups. Same parameters as net group.</p>                              <p><b>net name</b></p>                              <p>Adds/removes a name to which messaging may be directed to.                    </tr>                              <tr>                              <td width="185">/add</td>                              <td>Add the name</td>                              </tr>                              <tr>                              <td width="185">/delete</td>                              <td>Remove the name</td>                              </tr>                              </table>                              <p><b>net pause &lt;service name&gt;</b></p>                              <p>Used to pause a service from the command line.</p>                              <p><b>net print</b></p>                              <p>Used to list/modify print jobs.</p>                              <p>Options: </p>                              <table width="100%" border="1">                  </tr>                              <tr>                              <td width="185">sharename</td>                              <td>Name of the printer queue</td>                              </tr>                              <tr>                              <td width="185">job</td>                              <td>The job number to modify</td>                              </tr>                              <tr>                              <td width="185">/hold</td>                              <td>Pauses a job on the print queue</td>                              </tr>                              <tr>                              <td width="185">/release</td>                              <td>Removes the hold status of a job on the print queue</td>                              </tr>                              <tr>                              <td width="185">/delete</td>                              <td>Deletes a job off of the print queue</td>                              </tr>                              </table>                              <p><b>net send</b></p>                              <p>Sends a message to a computer, user or messaging name.</p> Omitting a computer name will                              close all sessions</td>                              </tr>                              </table>                              <p><b>net share</b></p>                              <p>Used to manage shares from the command line.</p>                              <p>Syntax:</p>                              <p>net share
