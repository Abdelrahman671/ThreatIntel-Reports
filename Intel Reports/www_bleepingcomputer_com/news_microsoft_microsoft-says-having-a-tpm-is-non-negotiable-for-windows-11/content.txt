          <h1>Microsoft says having a TPM is "non-negotiable" for Windows 11</h1>
          <div class="cz-news-story-title-section">
            <div class="cz-news-title-left-area">
                            By <h6><a rel="author" href="https://www.bleepingcomputer.com/author/sergiu-gatlan/" class="author"><span itemprop="author" itemscope itemtype="https://schema.org/Person"><span itemprop="name">Sergiu Gatlan</span></span></a></h6>
            </div>
	    <meta itemprop="articleSection" content="Microsoft"><div class="cz-news-title-right-area">
              <ul><li class="cz-news-date">December 4, 2024</li>
                <li class="cz-news-time">07:44 PM</li>
                
                <li class="cz-news-comment"><a href="#comments">16</a></li>
              </ul></div>
          </div>
	  <div class="articleBody">
	   
<p style="text-align:center"><img alt="Windows 11" height="900" src="https://www.bleepstatic.com/content/hl-images/2024/10/02/Windows-11-headpic.jpg" width="1600"></p>

<p>Microsoft made it abundantly clear this week that Windows 10 users won't be able to upgrade to Windows 11 unless their systems come with TPM 2.0 support, stating it's a "non-negotiable" requirement.</p>

<p>TPM 2.0 (Trusted Platform Module 2.0) is a dedicated processor on modern computers that provides hardware-based security functions and serves as a trusted hardware component for storing sensitive data, including encryption keys and other security credentials.</p>

<p>In June 2021, Microsoft announced <a href="https://www.bleepingcomputer.com/news/microsoft/windows-11-wont-work-without-a-tpm-what-you-need-to-know/" target="_blank">TPM 2.0 as a mandatory requirement</a> for installing or upgrading to Windows 11, saying it makes systems more resistant to tampering and sophisticated cyberattacks.</p>

<p>However, that didn't prevent Windows users from creating various tools, scripts, and techniques&nbsp;<a data-sk="tooltip_parent" data-stringify-link="https://www.bleepingcomputer.com/news/microsoft/how-to-bypass-the-windows-11-tpm-20-requirement/" delay="150" href="https://www.bleepingcomputer.com/news/microsoft/how-to-bypass-the-windows-11-tpm-20-requirement/" rel="noopener noreferrer" target="_blank">to bypass this requirement</a>, although there's no guarantee that Windows 11 will work without a TPM in the future.</p>

<p>In a new article published on Tuesday, senior product manager Steven Hosking tagged it as "a non-negotiable standard for the future of Windows" and shared how TPM 2.0 tackles new security challenges with improved industry-standard cryptography and seamless integration with Windows security capabilities like Secure Boot and Windows Hello for Business.</p>

<p>"TPM 2.0 plays a crucial role in enhancing identity and data protection on Windows devices, as well as maintaining the integrity of your system," <a href="https://techcommunity.microsoft.com/blog/windows-itpro-blog/tpm-2-0-%e2%80%93-a-necessity-for-a-secure-and-future-proof-windows-11/4339066" target="_blank" rel="nofollow noopener">said</a> Hosking.</p>

<p>"TPM 2.0 is essential to counteracting present-day cyber risks. As such, you should not disable TPM checks on Windows 11 endpoints. [..] TPM 2.0 also helps future-proof Windows 11. One way it does so is by helping to protect sensitive information as more AI capabilities come to physical, cloud, and server architecture."</p>

<p>However, although Windows 10 will reach the end of support in less than a year, on October 14, 2025, leaving systems without new bug fixes or security updates, Statcounter Global data shows that more than 61% of all Windows systems worldwide <a href="https://gs.statcounter.com/os-version-market-share/windows/desktop/worldwide" target="_blank" rel="nofollow noopener">still run Windows 10</a>, while less than 35% run Windows 11, three years after the latter's <a href="https://www.bleepingcomputer.com/news/microsoft/windows-11-is-released-what-you-need-to-know-and-new-features/" target="_blank">October 2021 launch</a>.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Windows market share worldwide" height="285" src="https://www.bleepstatic.com/images/news/u/1109292/2024/Windows%20market%20share%20worldwide.jpg" width="700"><figcaption><em>Windows market share worldwide (Statcounter)</em></figcaption></figure></div>

<p>&#8203;Even though Windows 10's end of support is rapidly approaching, the company announced on October 31 that Windows 10 <a href="https://www.bleepingcomputer.com/news/microsoft/microsoft-wants-30-if-you-want-to-delay-windows-11-switch/" target="_blank">home users will be able to delay the switch</a> to Windows 11 for one more year if they're willing to pay $30 for Extended Security Updates (ESU).</p>

<p>"The ESU program for consumers will be a one-year option available for $30. Program enrollment will be available closer to the end of support in 2025," said Yusuf Mehdi, Microsoft Executive Vice President and Consumer Chief Marketing Officer.</p>

<p>However, Long-Term Servicing Branch (LTSB) and Long-Term Servicing Channel (LTSC) releases that cater to specialized devices, like industrial and medical devices, will still receive updates beyond October 2025.</p>

<p>For instance, Windows 10 2016 LTSB <a href="https://learn.microsoft.com/en-us/lifecycle/products/windows-10-iot-enterprise-ltsb-2015" target="_blank" rel="nofollow noopener">will reach</a> its extended end date on October 13, 2026, while Windows 10 IoT Enterprise LTSC 2021 <a href="https://learn.microsoft.com/en-us/lifecycle/products/windows-10-iot-enterprise-ltsc-2021" target="_blank" rel="nofollow noopener">will reach</a> its extended end date six years later, on January 13, 2032.</p>

<p>Today, Microsoft also revealed that Windows 11 24H2 has entered "a new stage of availability" as it's now being rolled out&nbsp;to more eligible devices running Windows 11, versions 22H2 and 23H2.</p>

<p>"To check if the update is available for your device, simply select&#8239;Settings &gt; Windows Update and select Check for updates. If your device is ready for the update, you'll see the option to&#8239;Download and install," a Windows release health update <a href="https://learn.microsoft.com/en-us/windows/release-health/status-windows-11-24h2" target="_blank" rel="nofollow noopener">states</a>.</p>

function cz_strip_tags(input, allowed) {
  allowed = (((allowed || '') + '')
    .toLowerCase()
    .match(/<[a-z][a-z0-9]*>/g) || [])
    .join(''); // making sure the allowed arg is a string containing only tags in lowercase (<a><b><c>)
  var tags = /<\/?([a-z][a-z0-9]*)\b[^>]*>/gi,
    commentsAndPhpTags = /<!--[\s\S]*?-->|<\?(?:php)?[\s\S]*?\?>/gi;
  return input.replace(commentsAndPhpTags, '')
    .replace(tags, function($0, $1) {
      return allowed.indexOf('<' + $1.toLowerCase() + '>') > -1 ? $0 : '';
    });
}
function cz_br2nl(str) {
	var regex = /<br\s*[\/]?>/gi;
    //var pure_str = str.replace(regex,"\n");
    var pure_str = str.replace(regex,"");
    return cz_strip_tags(pure_str,'');
}
