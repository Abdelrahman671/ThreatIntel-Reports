observed changes in the distribution scheme of SpyAgent (A.K.A. TeamSpy/TVRat/TeamBot/Sheldor), a malware that likely originated over a decade ago based on the historical timeline below. </p><p>SpyAgent is a malware that abuses legitimate, well-known remote access tools (RAT). The recent changes observed by our team allow the malware to stay stealthy while bypassing and evading many security products. </p><p>Attackers evading existing security controls is a trend we see increasing. This is a problem for the industry as most security solutions that were developed with an “assume breach” mindset will miss these stealthy attacks until is it far too late to stop the damage.  </p><h3>Historical Background on SpyAgent: </h3><ul><li>In a report from <a href="https://web.archive.org/web/20111226101507/go.eset.com/us/resources/white-papers/CARO_2011.pdf" target="_blank" rel="noopener">2011</a>, a malware named “Sheldor” used DLL search order hijacking to abuse TeamViewer 5.0 for malicious activities.</li><li>In a report from <a href="https://web.archive.org/web/20130508171555/https:/www.crysys.hu/teamspy/teamspy.pdf" target="_blank" rel="noopener">2013</a>, a malware named “TeamSpy” used DLL search order hijacking to abuse TeamViewer 6.0 for malicious activities. The report also shows a relationship between “Sheldor” and “TeamSpy.”</li><li>In a report from <a href="https://vms.drweb.com/virus/?_is=1&amp;i=8415393" target="_blank" rel="noopener">2016</a>, a malware named “Spy-Agent” used DLL search order hijacking to abuse TeamViewer 6.0 for malicious activities. The report contains unique URI patterns that the malware uses to communicate with the C&amp;C server.</li><li>SpyAgent’s main capabilities are leveraged to enhance the usage of TeamViewer by hooking some of the functions used by legitimate applications.</li><li>The most important thing that SpyAgent does is obtain the client&#x27;s unique ID, which is required to connect to a computer using TeamViewer.</li><li>Other hooks disable logging and hide the GUI of the application to make it stealthy and avoid detection.</li></ul><h3>10 years later: </h3><ul><li>A report published in <a href="https://www.trendmicro.com/en_us/research/21/k/campaign-abusing-rats-uses-fake-websites.html" target="_blank" rel="noopener">2021</a> showed that the “Spy-Agent” malware has been observed shifting from hijacking TeamViewer to hijacking “Safib Assistant,” a Russian replica of TeamViewer.</li><li>The report demonstrates the distribution scheme of the malware, using fake crypto applications as a theme.</li><li>The fake applications are actually downloaders of multiple malware families, which include different RATs and stealers, such as RedLine. “SpyAgent” is one of the downloaded malware families. </li></ul><h3>Recent Changes </h3><p>In May 2022, a small change was observed in the fake crypto campaign first seen in 2021. Instead of using NSIS or Inno Setup droppers, executable files over 700MB were used to evade detection. This is because many security products limit the size of scanned files to compensate for performance issues. This is similar to the way the “Quartz.dll” SpyAgent files are inflated to 1GB+, as noted in the 2021 report. This is done by simply appending a large overlay of zeros to the file, and this technique allows the file to be compressed to the actual size without the overlay.  </p><p></p><figure style="float:none;width:auto;max-width:auto"><img asset_uid="blt102c6360c03796fb" alt="Figure 1: Overlay of zeros appended at the end of the file to inflate its’ size to avoid detection" size to avoid detection</figcaption></figure><p></p><p>Due to their large size, the files are not stored in public malware repositories or sandboxes. </p><p>The large overlayed executables are droppers which use Microsoft’s .NET ClickOnce launch utility “<a href="https://app.any.run/tasks/379dee24-c7f4-4255-a581-64d218fbe9f5/" target="_blank" rel="noopener">AppLaunch.exe</a>” to download and execute malware files from the web. </p><p>More malicious files related to this change can be found by <a href="https://www.virustotal.com/gui/file/0cfdd88e61fc93fbdc189de21b2cdaec357c2d8a82cddb532db894530d7d9a66/relations" target="_blank" rel="noopener">searching</a> the included txt file inside the zip: </p><p></p><figure style="float:none;width:395px;max-width:395px"><img asset_uid="blt8ce5dfe10c64c982" alt="Figure 2: Contents of the text file included in the zip archive" everyday because of the profit! </li></ol><p>In June 2022, additional changes were observed. </p><p>The SpyAgent theme is no longer related to crypto applications. </p><p>The dropper files are once again Inno Setup files, however, they no longer download any additional malware except “SpyAgent” bundled with “Safib Assistant.” </p><p>Since “Safib Assistant” is a legitimate tool, similar to TeamViwer, just less known, this change lowers the detection rate for the campaign as the only real malware is “SpyAgent.” </p><p>However, as previously was reported, the “SpyAgent” DLL files are very large. </p><p><a href="https://attack.mitre.org/techniques/T1027/001/" target="_blank" rel="noopener">T1027.001</a> is a known MITRE technique that adversaries use to “decrease the effectiveness of certain tools and detection capabilities that are not designed or configured to scan large files. This may also reduce the likelihood of being collected for analysis. Public file scanning services, such as VirusTotal, limits the maximum size of an uploaded file to be analyzed.” </p><p>Example
