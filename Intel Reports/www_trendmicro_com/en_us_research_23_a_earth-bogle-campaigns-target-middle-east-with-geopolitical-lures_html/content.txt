	<h1 class="article-details__title">Earth Bogle: Campaigns Target the Middle East with Geopolitical Lures</h1>
	<p class="article-details__description">We discovered an active campaign ongoing since at least mid-2022 which uses Middle Eastern geopolitical-themed lures to distribute NjRAT (also known as Bladabindi) to infect victims across the Middle East and North Africa.</p>
	<p class="article-details__author-by">By: Peter Girnus, Aliakbar Zahravi
		
			<time class="article-details__date">January 17, 2023</time>
		
		
		<span>Read time:&nbsp;</span><span class="eta"></span> (<span class="words"></span> words)
	</p>

            <div class="richText">
	
    


	
		<div>
			<p>While threat hunting, we found an active campaign using Middle Eastern geopolitical themes as a lure to target potential victims in the Middle East and Africa. In this campaign we have labeled Earth Bogle, the threat actor uses public cloud storage services such as files.fm and failiem.lv to host malware, while compromised web servers distribute <a href="https://www.trendmicro.com/vinfo/tmr/?/us/security/news/cybercrime-and-digital-threats/lord-exploit-kit-rises-delivers-njrat-and-eris-ransomware">NjRAT</a>.</p>
<p>NjRAT (<a name="_Int_ILx1T3XH" id="_Int_ILx1T3XH"></a>also known as <a href="https://www.trendmicro.com/vinfo/tmr/?/us/threat-encyclopedia/malware/BLADABINDI/">Bladabindi</a>) <a name="_Int_Mxz9rjEZ" id="_Int_Mxz9rjEZ"></a>is a remote access trojan (RAT) malware first discovered in 2013. It is primarily used to gain unauthorized access and control over infected computers and has been used in various cyberattacks to target individuals and organizations in the Middle East. Users and security teams are recommended to keep their systems’ security solutions updated and their respective cloud infrastructures properly secured to defend against this threat.</p>
<p><span class="body-subhead-title">Routine</span></p>

		</div>
	

</div>
            <div class="image">
	
    


	<figure class="image-figure">
		
			<img src="/content/dam/trendmicro/global/en/research/23/earth-bogle-campaigns-target-the-middle-east-with-geopolitical-lures/figure1-earth-bogle-campaign-target-middle-east-njrat.png" alt="fig1-earth-bogle-target-middle-east-africa-njrat-campaign-using-geopolitical-lure"/>
		
		<div class="caption-image-container ">
			<figcaption>Figure 1. Attack kill chain</figcaption>
			
		</div>
	</figure>

</div>
            <div>




    
    
    <div class="richText">
	
    


	
		<div>
			<p>The malicious file is hidden inside a Microsoft Cabinet (CAB) archive file masquerading as a “sensitive” audio file, named using a geopolitical theme as a lure to entice victims to open it. The distribution mechanism could be via social media (Facebook and Discord appear to be favored among these campaigns), file sharing (OneDrive), or a phishing email. The malicious CAB file contains an obfuscated VBS (Virtual Basic Script) dropper responsible for delivering the next stage of the attack.</p>
<p>Once the malicious CAB file is downloaded, the obfuscated VBS script runs to fetch the malware from a compromised or spoofed host. It then retrieves a PowerShell script responsible for injecting NjRat into the compromised victim’s machine.</p>
<p><span class="body-subhead-title">Use of Middle Eastern Geopolitical Themes as Lures</span></p>
<p>The initial CAB files have exceptionally low detection rates on Virus Total (SHA256: a7e2b399b9f0be7e61977b51f6d285f8d53bd4b92d6e11f74660791960b813da and 4985b6e286020de70f0b74d457c7e387463ea711ec21634e35bc46707dfe4c9b), which allows the attackers to remain undetected and spread their attack across the region. The group behind the campaign uses public cloud hosting services to host malicious CAB files and uses themed lures to entice Arabic speakers into opening the infected file.</p>

		</div>
	

</div>


    
    
    <div class="image">
	
    


	<figure class="image-figure">
		
			<img src="/content/dam/trendmicro/global/en/research/23/earth-bogle-campaigns-target-the-middle-east-with-geopolitical-lures/figure2-earth-bogle-campaign-target-middle-east-njrat.png" alt="fig2-earth-bogle-target-middle-east-africa-njrat-campaign-using-geopolitical-lure"/>
		
		<div class="caption-image-container ">
			<figcaption>Figure 2. Malicious CAB file hosted on cloud sharing services</figcaption>
			
		</div>
	</figure>

</div>


    
    
    <div class="richText">
	
    


	
		<div>
			<p>One of the malicious CAB files’ <a name="_Int_aX9ZJcxV" id="_Int_aX9ZJcxV"></a>filename translates to “A voice call between Omar, the reviewer of the command of Tariq bin Ziyad’s force, with an Emirati officer.cab”. The attacker uses the lure of a supposedly sensitive voice call between an Emirati military officer and a member of the Tariq bin Ziyad (<a href="https://www.amnesty.org/en/documents/mde19/6282/2022/en/">TBZ</a>) Militia, a powerful Libyan faction. The file lures victims in the region into opening the file by insinuating a false link between the UAE and a group associated with war crimes, appealing to political interests and emotional appeals. These lures are consistent with a campaign <a name="_Int_5SbahHnS" id="_Int_5SbahHnS"></a>disclosed in <a href="https://di.sclosu.re/en/njrat-malware-spreading-through-discord-cdn-and-facebook-ads/">December 2022</a> that used Facebook advertisements on spoofed Middle Eastern news outlets’ pages, which were shared and pushed to other users by unsuspecting mules.</p>
<p>This malicious CAB file <a name="_Int_VzmDBSoY" id="_Int_VzmDBSoY"></a>contains an obfuscated VBS script that functions as the agent responsible for delivering the next payload. When a victim opens the malicious CAB file and runs the VBS file, the second stage payload is retrieved.</p>
<p><span class="body-subhead-title">Delivering the PowerShell Payload</span></p>
<p>The second stage payload is an obfuscated VBS script file masquerading as an image file (SHA256: 6560ef1253f239a398cc5ab237271bddd35b4aa18078ad253fd7964e154a2580). When this malicious file is run, a malicious PowerShell script is retrieved.</p>

		</div>
	

</div>


    
    
    <div class="image">
	
    


	<figure class="image-figure">
		
			<img src="/content/dam/trendmicro/global/en/research/23/earth-bogle-campaigns-target-the-middle-east-with-geopolitical-lures/figure3-earth-bogle-campaign-target-middle-east-njrat.png" alt="fig3-earth-bogle-target-middle-east-africa-njrat-campaign-using-geopolitical-lures"/>
		
		<div class="caption-image-container ">
			<figcaption>Figure 3. Malicious VBS file fetches malicious PowerShell script</figcaption>
			
		</div>
	</figure>

</div>


    
    
    <div class="image">
	
    


	<figure class="image-figure">
		
			<img src="/content/dam/trendmicro/global/en/research/23/earth-bogle-campaigns-target-the-middle-east-with-geopolitical-lures/figure4-earth-bogle-campaign-target-middle-east-njrat.png" alt="fig4-earth-bogle-target-middle-east-africa-njrat-campaign-using-geopolitical-lures"/>
		
		<div class="caption-image-container ">
			<figcaption>Figure 4. Deobfuscated VBS script</figcaption>
			
		</div>
	</figure>

</div>


    
    
    <div class="richText">
	
    


	
		<div>
			<p>The domain delivering the malicious PowerShell script is an infected or spoofed host with documented affiliations with the Libyan Army, and a quick check on the domain gpla[.]gov[.]ly shows it was registered in 2019.</p>

		</div>
	

</div>


    
    
    <div class="image">
	
    


	<figure class="image-figure">

		<div class="caption-image-container ">

		</div>
	</figure>

</div>


    
    
    <div class="richText">
	
    


	
		<div>
			<p>Similar campaigns have suggested the creation, use, and abuse of fake social media accounts claiming to belong to reputable organizations to serve advertisements with links to public cloud sharing platforms which <a name="_Int_SA1pCkdr" id="_Int_SA1pCkdr"></a>contain malicious payloads to unsuspecting victims. This allows the threat actors to:</p>
<ol>
<li>Infect users directly through clicks on these malicious links.</li>
<li>Use geopolitical-themed lures and abuse social sharing features to deliver malicious payloads and spread to a wider audience.</li>
</ol>
<p>We also noted that the domain gpla[.]gov[.]ly has a <a href="http://www.zone-h.org/mirror/id/34787739">history of compromise</a> going back to at least 2021.</p>

		</div>
	

</div>


    
    
    <div class="image">
	
    


	<figure class="image-figure">

    <div class="richText">
	
    


	
		<div>
			<p><span class="body-subhead-title">Second stage Dropper Overview</span></p>
<p>The second stage dropper (SHA256: 78ac9da347d13a9cf07d661cdcd10cb2ca1b11198e4618eb263aec84be32e9c8) is an obfuscated PowerShell script that drops five files in total: two binaries, a VBS script, a PowerShell script, and a Windows batch script.</p>
<p>Each module has the following functionality:<br />
</p>
<ul>
<li><span class="rte-red-bullet">Payload_1: Process injector</span></li>
<li><span class="rte-red-bullet">Payload_2: NjRAT</span></li>
<li><span class="rte-red-bullet">gJhkEJvwBCHe.vbs: Executes rYFFCeKHlIT.bat</span></li>
<li><span class="rte-red-bullet">rYFFCeKHlIT.bat: Executes KxFXQGVBtb.ps1</span></li>
<li><span class="rte-red-bullet">KxFXQGVBtb.ps1: Load Payload_1 and Payload_2 into the memory and inject NjRAT into the aspnet_compiler.exe via payload_1</span></li>
</ul>
<p>Upon execution, the second stage dropper kills the following .NET-related processes on the infected system. After which, “KxFXQGVBtB.ps1” <a name="_Int_FcZINXG5" id="_Int_FcZINXG5"></a>executes the “aspnet_compler.exe” in conjunction with the process injector to inject NjRAT.       <br />
</p>
<p><span class="blockquote">[Reflection.Assembly]::Load($<b>MyS</b>).GetType('<b>NewPE2.PE</b>').'GetMethod'('<b>Execute</b>'<a name="_Int_QYGyhSEP" id="_Int_QYGyhSEP"></a>).Invoke($<a name="_Int_weA9pnWx" id="_Int_weA9pnWx"></a>null,{[<a name="_Int_61fb0ALe" id="_Int_61fb0ALe"></a>OBJECT[]]}, ($<b>JKGHJKHGJKJK</b>,$<b>serv</b>));      </span></p>

		</div>
	

</div>


    
    
    <div class="image">
	
    


	<figure class="image-figure">

    <div class="richText">
	
    


	
		<div>
			<p>The dropper further drops &quot;rYFFCeKHlIT.bat&quot; in C:\Users\Public and creates a directory called &quot;WindowsHost&quot; in C:\ProgramData\ to store the VBScript file &quot;gJhkEJvwBCHe.vbs&quot;. On deobfuscation, gJhkEJvwBCHe.vbs runs the rYFFCeKHlIT.bat file, responsible for executing another PowerShell script called &quot;KxFXQGVBtb.ps1&quot; that contains a bypass PowerShell execution policy flag.</p>

		</div>
	

</div>

    <div class="richText">
	
    


	
		<div>
			<p>&quot;KxFXQGVBtB.ps1&quot; is the final PowerShell dropper responsible for loading the NjRAT binary into memory and injecting it into the legitimate .NET binary file called &quot;aspnet_compiler.exe&quot; via the process injector. The PowerShell script uses the “[Reflection.Assembly<a name="_Int_DQDh0liw" id="_Int_DQDh0liw"></a>]::Load&quot; method to load the process injector “($Mys)” into the memory. It then invokes a method called 'Execute' with two parameters. The first parameter is a full path to the PEfile to inject (&quot;C:\Windows\Microsoft.NET\Framework\&lt;VERSION&gt;\aspnet_compiler.exe&quot;), and the second parameter is the primary payload NjRAT ($serv).</p>

		</div>

		<div>
			<p><span class="body-subhead-title">Conclusion</span></p>
<p>This case demonstrates that threat actors will leverage public cloud storage as malware file servers, combined with social engineering techniques appealing to people’s sentiments such as regional geopolitical themes as lures, to infect targeted populations. Furthermore, governments weakened by regional conflict are at a higher risk for compromise, wherein threat actors and advanced persistent threat (APT) groups could compromise and use government infrastructure in targeted campaigns. This is compounded by the ability to share cloud storage content via advertising and social media, presenting an opportunity for threat actors and APT groups to reach a wider infection radius.</p>
<p>Organizations can protect themselves by remaining vigilant against phishing attacks and skeptical regarding sensational topics and themes abused online as lures. Users should be wary of opening suspicious archive files such as CAB files, especially from public sources where the risks of compromise are high. Security teams should be aware of the dynamic nature of conflict zones when considering a security posture. Organizations can also consider a cutting edge <a href="/en_us/business/products/user-protection/endpoint-security.html">multilayered defensive strategy</a> that can detect, scan, and block malicious URLs.</p>
