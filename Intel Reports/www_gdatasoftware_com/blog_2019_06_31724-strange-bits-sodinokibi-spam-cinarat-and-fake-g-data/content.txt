of old. Instead, the most intriguing bits of modern research will at some point contain the phrase “That’s strange…”, followed by more prodding and poking and – hopefully – a lightbulb moment. This series that we call "Strange Bits" contains many findings that struck our analysts as odd, either because they do not seem to make any sense at the time or because a malicious program exhibits behaviors that none of us have seen before. Maybe these findings will spark ideas in other fellow researchers – maybe those findings are just what it says on the tin: Strange….</i></p></div> </div></div> <div data-magellan-target="c191356" class="small-12 column"><div class="row"> <div id="c191356" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-center ce-below"><div class="ce-bodytext"><header class=" "><h2 class=" "> Sodinokibi Ransomware Spam Campaign targets Germany </h2></header><p>Sodinokibi ransomware was known so far for being installed via Oracle WebLogic exploit (see <a href="https://blog.talosintelligence.com/2019/04/sodinokibi-ransomware-exploits-weblogic.html" target="_blank" rel="noreferrer">Talos' article</a>). A new campaign uses spam emails with attached MS Office Word document to download Sokinokibi to the target system.&nbsp;<a href="https://twitter.com/JAMESWT_MHT/status/1133701006238375937" target="_blank" rel="noreferrer">JamesWT</a> found the first sample, <a href="https://twitter.com/sculabs" target="_blank" rel="noreferrer">Sculabs</a> another one<sup>[1]</sup>. The email pretends to be a warning letter from the fee collection center of public-law broadcasting institutions in the Federal Public of Germany and demands 213.50 EUR payment.</p></div><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><img class="image-embed-item " src="/fileadmin/user_upload/Presse/Deutschland/2019/05/StrangeBitsGEZ.png" width="914" height="893" loading="lazy" alt="" /></figure></div></div></div></div> <div id="c191363" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-center ce-below"><div class="ce-bodytext"><p>The attached document has the file name "Mahnbescheid - Antwortbogen - Aktenzeichen 4650969334.doc"[2]. It uses a recursive acronym for its name: "CinaRAT (Cina is not a Remote Access Tool)". The sample is packed with VMProtect.</p><p>The malware developer, who goes by the name <i>Wearelegal</i>, states on their GitHub repository that the project is based on QuasarRAT. We compared the client stub code of CinaRAT 1.0.1.0<sup>[5]</sup> to the client stub code of QuasarRAT 1.3.0.0<sup>[6]</sup> with a diff tool <i></i><a href="https://www.telerik.com/justassembly" target="_blank" rel="noreferrer">JustAssembly</a>. We found only minor differences, e.g., for the name of the malware, but no added features. That means CinaRAT is a rebranded QuasarRAT.</p><p>CinaRAT exists at least since October 2017 which is the release date on the Github "Release" tab. The previous version of CinaRAT had a different name: Yggdrasil.</p></div><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><a href="/fileadmin/user_upload/Presse/Deutschland/2019/05/StrangeBits_CinaRAT_Yggdrasil.png" class="fancybox"><img class="image-embed-item " src="/fileadmin/user_upload/Presse/Deutschland/2019/05/StrangeBits_CinaRAT_Yggdrasil.png" width="1022" height="487" loading="lazy" alt="" /></a><figcaption class="image-caption"> The first version of CinaRAT when it was still named &quot;Yggdrasil&quot;. The image belongs to the malware author&#039;s demo screenshots. </figcaption></figure></div></div></div></div> </div></div> <div data-magellan-target="c191250" class="small-12 column"><div class="row"> <div id="c191250" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-center ce-below"><div class="ce-bodytext"><header class=" "><h2 class=" "> Gh0stRAT imitates G DATA </h2></header><p>We discovered a malware sample that uses G DATA icons and version information<sup>[7]</sup>.
