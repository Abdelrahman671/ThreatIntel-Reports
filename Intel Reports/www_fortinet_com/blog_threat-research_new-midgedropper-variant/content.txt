  <p style="margin-left: 80.0px;"><b>Affected Platforms: </b>Windows<br />
<b>Impacted Users:</b> Windows users<br />
<b>Impact:</b> Potential to deploy additional malware for additional purposes<br />
<b>Severity Level: </b>Medium</p>
<p>One of the most exciting aspects of malware analysis is coming across a family that is new or rare to the reversing community. Determining the function of the malware, who created it, and the reasons behind it become a mystery to solve. The previously unseen dropper variant we recently found, named MidgeDropper, has a complex infection chain that includes code obfuscation and sideloading, making it an interesting use case. Although we couldn’t obtain the final payload, this blog will still explore what makes this dropper tick.</p>
<h2><span style="font-weight: normal;">Initial Infection Vector</span></h2>
<p>The initial infection vector was not available to FortiGuard Labs at the time of our investigation. However, we strongly suspect it to be a phishing e-mail because we have access to an RAR archive—!PENTING_LIST OF OFFICERS.rar—that would have been the likely attachment to an e-mail.</p>
<h2><span style="font-weight: normal;">!PENTING_LIST OF OFFICERS.rar</span></h2>
<p>Two files are in the !PENTING_LIST OF OFFICERS.rar archive: “Notice to Work-From-Home groups.pdf” and “062023_PENTING_LIST OF SUPERVISORY OFFICERS WHO STILL HAVE NOT REPORT.pdf.exe” (Figure 1).</p>

<div class="cmp cmp-text aem-GridColumn aem-GridColumn--default--12">
  <h2><span style="font-weight: normal;">Notice to Work-From-Home groups.pdf</span></h2>
<p>The “Notice to Work-From-Home groups.pdf” file is exactly what it appears to be: a PDF file. It contains an image of an error message that falsely indicates that the PDF document failed to load. It is designed to act as a decoy and shift the recipient’s attention to clicking on and executing the “062023_PENTING_LIST OF SUPERVISORY OFFICERS WHO STILL HAVE NOT REPORT.pdf.exe” file. Since file extensions are hidden by default in Windows, it is unlikely that anyone reviewing the contents would see the “.exe” and would instead assume they were opening another PDF file.</p>

<p>At 6.7MB, the ”062023_PENTING_LIST OF SUPERVISORY OFFICERS WHO STILL HAVE NOT REPORT.pdf.exe” file is large by malware delivery standards. This executable primarily functions as a dropper for the following stages of infection.</p>

<div class="cmp cmp-text aem-GridColumn aem-GridColumn--default--12">
  <h2><span style="font-weight: normal;">VCRUNTIME140_1.dll</span></h2>
<p>“VCRUNTIME140_1.dll” is a legitimate DLL that is part of the Microsoft Visual C++ runtime package. Unfortunately, the particular version used here is malicious.</p>
<p>Due to “VCRUNTIME140_1.dll” being a Dynamic Link Library, it doesn’t exist as a separate executable. It has to have assistance via another application to load its code into memory and execute it. “seAgnt.exe” is that application. This technique is called sideloading (<a href="https://attack.mitre.org/techniques/T1574/002/">https://attack.mitre.org/techniques/T1574/002/</a>) because a dependency of a legitimate application is highjacked to allow the malicious code to load.</p>


</div>
