class="article__content"><h1>Disgruntled ransomware affiliate leaks the Conti gang&#x27;s technical manuals</h1><span class="wysiwyg-parsed-content"><p>A disgruntled member of the Conti ransomware program has leaked today the manuals and technical guides used by the Conti gang to train affiliate members on how to access, move laterally, and escalate access inside a hacked company and then exfiltrate its data before encrypting files.</p>
<p>Leaked on an underground cybercrime forum named XSS earlier today, the files were shared by an individual who appears to have had an issue with the low amount of money the Conti gang was paying them to breach corporate networks.</p>
<p><div class="relative w-full my-[.5rem]"><img alt="2021-08-XSS-Conti-leak.png" loading="lazy" width="500" height="500" decoding="async" data-nimg="1" class="z-10" style="color:transparent;object-fit:contain;width:100%" srcSet="https://cms.therecord.media/uploads/format_webp/2021_08_XSS_Conti_leak_6fb39d6257.png?w=640 1x, https://cms.therecord.media/uploads/format_webp/2021_08_XSS_Conti_leak_6fb39d6257.png?w=1080 2x" src="https://cms.therecord.media/uploads/format_webp/2021_08_XSS_Conti_leak_6fb39d6257.png?w=1080" /></div></p>
<p><div class="relative w-full my-[.5rem]"><img alt="2021-08-XSS-Conti-leak-Cobalt.png" loading="lazy" width="500" height="500" decoding="async" data-nimg="1" class="z-10" style="color:transparent;object-fit:contain;width:100%" srcSet="https://cms.therecord.media/uploads/format_webp/2021_08_XSS_Conti_leak_Cobalt_c06b8856f5.png?w=640 1x, https://cms.therecord.media/uploads/format_webp/2021_08_XSS_Conti_leak_Cobalt_c06b8856f5.png?w=1080 2x" src="https://cms.therecord.media/uploads/format_webp/2021_08_XSS_Conti_leak_Cobalt_c06b8856f5.png?w=1080" /></div></p>
<p>In messages spammed across the forum, the individual shared screenshots of IP addresses where the Conti gang hosts Cobalt Strike command-and-control servers, which Conti affiliate members use to access hacked company networks.</p>
<figure class="wp-block-embed is-type-rich is-provider-twitter wp-block-embed-twitter"><div class="wp-block-embed__wrapper">
https://twitter.com/pancak3lullz/status/1423324601346629635
</div></figure>
<p>In addition, the individual also published a RAR archive named &quot;<strong><em>Мануали для работяг и софт.rar</em></strong>,&quot; which roughly translates to &quot;<strong><em>Manuals for hard workers and software.rar</em></strong>.&quot;</p>
<p><a href="https://www.virustotal.com/gui/file/03e9b8c2e86d6db450e5eceec057d7e369ee2389b9daecaf06331a95410aa5f8/detection" target="_blank" rel="noopener noreferrer">This archive</a> contains 37 text files with instructions on how to use various hacking tools and even legitimate software during a network intrusion.</p>
<p>For example, the leaked manuals contain guides on how to:</p>
<ul><li>configure the <a href="https://rclone.org/" target="_blank" rel="noopener noreferrer">Rclone</a> software with a MEGA account for data exfiltration</li><li>configure the <a href="https://anydesk.com/en" target="_blank" rel="noopener noreferrer">AnyDesk</a> software as a persistence and remote access solution into a victim&#x27;s network [<a href="https://twitter.com/AltShiftPrtScn/status/1423188974298861571" target="_blank" rel="noopener noreferrer"><em>a known Conti tactic</em></a>]</li><li>configure and use the <a href="https://www.cobaltstrike.com/" target="_blank" rel="noopener noreferrer">Cobalt Strike</a> agent</li><li>use the NetScan tool to scan internal networks</li><li>install the <a href="https://www.metasploit.com/" target="_blank" rel="noopener noreferrer">Metasploit</a> pen-testing framework on a virtual private server (VPS)</li><li>connect to hacked networks via RDP using a <a href="https://ngrok.com/" target="_blank" rel="noopener noreferrer">Ngrok</a> secure tunnel</li><li>elevate and gain admin rights inside a company&#x27;s hacked network</li><li>take over domain controllers</li><li>dump passwords from Active Directories (NTDS dumping)</li><li>perform SMB brute-force attacks</li><li>brute-force routers, NAS devices, and security cameras</li><li>use the <a href="https://www.secura.com/blog/zero-logon" target="_blank" rel="noopener noreferrer">ZeroLogon exploit</a></li><li>perform a <a href="https://www.qomplx.com/qomplx-knowledge-kerberoasting-attacks-explained/" target="_blank" rel="noopener noreferrer">Kerberoasting</a> attack</li><li>disable Windows Defender protections</li><li>delete shadow volume copies</li><li>how affiliates can configure their own operating systems to use the Tor anonymity network, and more</li></ul>
<p><div class="relative w-full my-[.5rem]"><img alt="2021-08-XSS-Conti-leakk-files-1024x258.png" loading="lazy" width="500" height="500" decoding="async" data-nimg="1" class="z-10" style="color:transparent;object-fit:contain;width:100%" srcSet="https://cms.therecord.media/uploads/format_webp/2021_08_XSS_Conti_leakk_files_1024x258_2e8c9174b8.png?w=640 1x, https://cms.therecord.media/uploads/format_webp/2021_08_XSS_Conti_leakk_files_1024x258_2e8c9174b8.png?w=1080 2x" src="https://cms.therecord.media/uploads/format_webp/2021_08_XSS_Conti_leakk_files_1024x258_2e8c9174b8.png?w=1080" /></div></p>
<p>Leaks from Ransomware-as-a-Service (RaaS) operations are extremely rare; however, the data shared today isn&#x27;t anything that security researchers would describe as groundbreaking.</p>
<p>The leaked files contain guides for basic offensive tactics and techniques that the Conti and other ransomware gangs have used during previous intrusions for years.</p>
<p>However, the leak will help some security firms put together stronger defensive playbooks that they can recommend to their customers in order to improve their ability to detect Conti intrusions—now knowing exactly what operations Conti affiliates might execute.</p></span><div class="social-share-icons"><ul><li><a target="_blank" rel="noreferrer"
