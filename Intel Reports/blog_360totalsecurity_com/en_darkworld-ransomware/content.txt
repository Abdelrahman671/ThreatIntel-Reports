								<p><strong>Overview</strong></p>
<p>Recently, 360 Security Center detected a ransomware that disguised commonly used software and appeared on the network. The virus called itself DarkWorld in the ransom letter.</p>
<p>What to be mentioned is that, unlike the conventional ransomware method that only encrypts ransomware in the past, the virus also plays a dual role as an information stealer while ransoming. The information stolen by the virus author may be further sold for profit.</p>
<p>From this point, the destructive mode of ransomware has changed. It is foreseeable that traditional malicious software may gradually cease to be satisfied with a single malicious behavior, but will enrich itself into a multi-functional malicious code integrator, causing multiple destructive consequences.</p>
<p>After the virus encrypts the victim&#8217;s files, it will ask for a Bitcoin ransom equivalent to $300. However, users do not need to worry, the 360 Total Security can intercept and kill the ransomware before  problems occur.</p>
<p><a href="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.1.png"><img class="aligncenter size-full wp-image-27310" src="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.1.png" alt="" width="582" height="340" srcset="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.1.png 582w, https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.1-300x175.png 300w" sizes="(max-width: 582px) 100vw, 582px" /></a></p>
<p><strong>Trojan horse behavior and encryption method</strong></p>
<ol>
<li>Poisoning</li>
</ol>
<p>After the DarkWorld ransomware runs, it will encrypt the file using the Rijndael encryption algorithm, and then add the suffix of the encrypted file &#8220;.dark&#8221;, and create &#8220;Important.txt&#8221; as a ransom letter. The ransom letter requires the victim to send &#8220;1EdxGR5fxRjhWtxNSbyDHv4nVdx5BP54L2&#8221; this The wallet address remits the equivalent of 300 USD in Bitcoin ransom, and sends the victim id to the virus author&#8217;s mailbox darksimo@protonmail.com to obtain the decryption key and decrypt the victim&#8217;s files.</p>
<p><a href="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.2.png"><img class="aligncenter size-full wp-image-27315" src="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.2.png" alt="" width="516" height="453" srcset="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.2.png 516w, https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.2-300x263.png 300w" sizes="(max-width: 516px) 100vw, 516px" /></a></p>
<p style="text-align: center;">DarkWorld Blackmail Letter Important.txt Blackmails Bitcoin</p>
<p><a href="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.3.png"><img class="aligncenter size-full wp-image-27320" src="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.3.png" alt="" width="458" height="179" srcset="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.3.png 458w, https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.3-300x117.png 300w" sizes="(max-width: 458px) 100vw, 458px" /></a></p>
<p>Encrypted files are added with &#8220;.dark&#8221; file extension</p>
<ol start="2">
<li>Blackmail encryption and information theft</li>
</ol>
<p>The Trojan pretends to be a third-party software file name and icon to induce users to double-click to run it, and then begins to traverse the file to encrypt and steal information through the Rijndael encryption algorithm.</p>
<p>Before starting encryption, the virus will first send the generated random encryption key and other information to the virus author server. Afterwards, it began to traverse the files and encrypt the files with the specified suffix through the Rijndael encryption algorithm. At the same time, pay attention to avoiding the specified system folder to avoid affecting the operating system. For txt files smaller than 2M, stolen and uploaded to the author&#8217;s server and then encrypted.</p>
<p><a href="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.4.png"><img class="aligncenter size-full wp-image-27325" src="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.4.png" alt="" width="854" height="1068" srcset="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.4.png 854w, https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.4-240x300.png 240w, https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.4-768x960.png 768w, https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.4-819x1024.png 819w" sizes="(max-width: 854px) 100vw, 854px" /></a></p>
<p>Before starting encryption, the virus first generates an encryption key consisting of a string of &#8220;random number + current time&#8221;, and then obtains the victim&#8217;s machine name, plus the victim&#8217;s id, and sends it to the Trojan server.</p>
<p><a href="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.5.png"><img class="aligncenter size-full wp-image-27330" src="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.5.png" alt="" width="579" height="277" srcset="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.5.png 579w, https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.5-300x144.png 300w" sizes="(max-width: 579px) 100vw, 579px" /></a></p>
<p>The virus also pays attention to avoiding the specified system file directory to avoid affecting the operating system, and then encrypts the file with the specified extension.</p>
<p><a href="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.6.png"><img class="aligncenter size-full wp-image-27335" src="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.6.png" alt="" width="770" height="165" srcset="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.6.png 770w, https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.6-300x64.png 300w, https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.6-768x165.png 768w" sizes="(max-width: 770px) 100vw, 770px" /></a></p>
<p style="text-align: center;">DarkWorld files avoid system critical directories</p>
<p><a href="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.7.png"><img class="aligncenter size-full wp-image-27340" src="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.7.png" alt="" width="324" height="549" srcset="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.7.png 324w, https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.7-177x300.png 177w" sizes="(max-width: 324px) 100vw, 324px" /></a></p>
<p style="text-align: center;">Target file extension</p>
<p>In the file encryption thread of the virus, the size of the txt file is also judged to determine whether to steal the user&#8217;s local file to the server. For txt files smaller than 2M (2097200 byte), the encryption thread will steal it and upload it to the Trojan server.</p>
<p><a href="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.8.png"><img class="aligncenter size-full wp-image-27345" src="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.8.png" alt="" width="698" height="353" srcset="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.8.png 698w, https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.8-300x152.png 300w" sizes="(max-width: 698px) 100vw, 698px" /></a></p>
<p style="text-align: center;">DarkWorld steals local files</p>
<p><a href="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.9.png"><img class="aligncenter size-full wp-image-27350" src="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.9.png" alt="" width="890" height="369" srcset="https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.9.png 890w, https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.9-300x124.png 300w, https://static.ts.360.com/blog/wp-content/uploads/2021/02/25.9-768x318.png 768w" sizes="(max-width: 890px) 100vw, 890px" /></a></p>
<p style="text-align: center;">DarkWorld uses Rijndael encryption algorithm</p>
