    <section class="post-content-wrapper mt-5">
        <div class="post-content">
            <p>This blog was authored by <a href="https://blog.talosintelligence.com/author/paul-rascagneres/">Paul Rascagneres</a> and <a href="http://blogs.cisco.com/author/warrenmercer">Warren Mercer</a> with contributions from <a href="https://blogs.cisco.com/author/emmanueltacheau">Emmanuel Tacheau</a>, <a href="https://blog.talosintelligence.com/author/vanja-svajcer/">Vanja Svajcer</a> and <a href="https://blog.talosintelligence.com/author/martin-lee/">Martin Lee</a>.</p><h2 id="executive-summary-talos-continuously-monitors-malicious-emails-campaigns-we-identified-one-specific-spear-phishing-campaign-launched-against-targets-within-palestine-and-specifically-against-palestinian-law-enforcement-agencies-this-campaign-started-in-april-2017-using-a-spear-phishing-campaign-to-deliver-the-micropsia-payload-in-order-to-remotely-control-infected-systems-although-this-technique-is-not-new-it-remains-an-effective-technique-for-attackers-">Executive Summary <br> Talos continuously monitors malicious emails campaigns. We identified one specific spear phishing campaign launched against targets within Palestine, and specifically against Palestinian law enforcement agencies. This campaign started in April 2017, using a spear phishing campaign to deliver the MICROPSIA payload in order to remotely control infected systems. Although this technique is not new, it remains an effective technique for attackers.</h2><p>The malware itself was developed in Delphi; in this article, we describe the features and the network communication to the command and control server used by the attackers. The threat actor has chosen to reference TV show characters and include German language words within the attack. Most significantly, the attacker has appeared to have used genuine documents stolen from Palestinian sources as well as a controversial music video as part of the attack.</p><h2 id="spear-phishing-campaign-we-identified-a-spear-phishing-campaign-that-appeared-to-be-targeting-members-of-the-palestinian-law-enforcement-services-as-well-as-other-palestinian-targets-in-the-public-sector-the-email-purports-to-be-sent-from-an-individual-named-yasser-saad-but-includes-a-mismatch-between-the-apparent-name-and-the-email-address-yassersaaid01-gmail-com-suggesting-yasser-saaid-">Spear Phishing Campaign <br> We identified a spear phishing campaign that appeared to be targeting members of the Palestinian law enforcement services as well as other Palestinian targets in the public sector. The email purports to be sent from an individual named "Yasser Saad" but includes a mismatch between the apparent name and the email address '<a href="mailto:yassersaaid01@gmail.com">yassersaaid01@gmail.com</a>' suggesting "Yasser Saaid".</h2><figure class="kg-card kg-image-card"><img src="https://blog.talosintelligence.com/content/images/-3a42reuT9u0/WUftPtxDtnI/AAAAAAAAATo/4DKV8eEdl5wfFFexC9q8euth1clcSj1fwCLcBGAs/s640/image2.png" class="kg-image" alt loading="lazy"></figure><p>The subject of the email translates to "Brothers security officers and directors", with the text content "Kindly to view and circulate under the responsibility:<br> The Council of Ministers' Decision on the Use of the Internet in Government Institutions"</p><p>Attached to the email is a .r10 file, which suggests that the file is a tenth part of a split RAR archive. However, this isn't the case. The attachment is a simple RAR file. Despite the unusual file name extension, this file can be opened by many RAR archive handlers without modification.</p><p>The RAR archive contains a single executable file named: InternetPolicy_65573247239876023_3247648974234_32487234235667_pdf.exe</p><p>The .r10 file extension may have been chosen in order to confuse automated file parsing systems that check for malicious contents of archives with known file name extensions. Similarly, the long name of the file within the archive, along with the ending '_pdf.exe' may have been used to convince victims into thinking that the file is a real PDF file. It is worth keeping in mind that by default Windows will not show the .exe extension to the user. The icon of executable file itself is that commonly used for PDF files, enhancing the idea that the contents of the archive is a PDF.</p><p>When the executable is launched it extracts the decoy document embedded as the PE resource named Resource_1 and opens it.</p><h3 id="decoy-document-the-decoy-document-displayed-internetpolicy-pdf-is-a-scanned-document-by-the-ministry-of-interior-of-the-state-of-palestine-signed-by-dr-alaa-mousa-minister-of-communications-technologies-">Decoy Document <br> The decoy document displayed, InternetPolicy.pdf, is a scanned document by the Ministry Of Interior of the State Of Palestine, signed by Dr Alaa Mousa, Minister of Communications &amp; Technologies:</h3><figure class="kg-card kg-image-card"><img src="https://blog.talosintelligence.com/content/images/-VTBikCk-Rdw/WUftVP1oi-I/AAAAAAAAATs/B711nmvYeRcpehemiGN1VouC_YQ2ON3-ACLcBGAs/s640/image8.png" class="kg-image" alt loading="lazy"></figure><p>The decoy document contains 7 pages describing new internet usage policies. The first page (shown above) informs governmental departments of the policy, and instructs them to follow it. A handwritten note stated to be from the Director of the IT Department adds his 'seal of approval' to the document.</p><p>In the background, the malware MICROPSIA is executed on the infected system. This malware is a Remote Administration Tool (RAT), it will be described later.</p><h2 id="associated-campaigns-we-have-also-identified-drive-by-download-campaigns-which-are-distributing-variants-of-the-same-malware-but-with-different-decoy-documents-">Associated Campaigns <br> We have also identified drive-by download campaigns which are distributing variants of the same malware, but with different decoy documents.</h2><p>URLs used in this campaign include:</p><ul><li>http://sheldon-cooper[.]info/Fuqha_NewDetails_docx.r10</li><li>http://feteh-asefa[.]com/pc/public/Fuqha_NewDetails_docx.r10</li><li>http://feteh-asefa[.]com/pc/public/Altarnatevs.r10</li><li>https://sheldon-cooper[.]info/attachment.r10</li></ul><p>As with the spear phish, the archives also have the same .r10 extension. The first two archives contain the file: Fuqha_NewDetails_874918321795_39778423423094_1988734200039_docx.exe which although the file name suggests a .docx file, the icon is that of a PDF document.</p><p>The second two archives contain the file: Altarnatives_Palestine_89840923498679852_9879483278432732489_pdf.exe Again being an executable file with a PDF style icon.</p><h3 id="decoy-documents">Decoy Documents <br></h3><h4 id="altarnatives_palestine-document-the-pdf-decoy-document-is-study-from-the-palestinian-center-for-policy-research-and-strategic-studies-masarat-">Altarnatives_Palestine Document <br> The .pdf decoy document is study from the Palestinian Center for Policy Research and Strategic Studies (MASARAT):</h4><figure class="kg-card kg-image-card"><img src="https://blog.talosintelligence.com/content/images/-yROszkBAVzY/WUftbDF0ubI/AAAAAAAAATw/FrjxSdzUrkYXLm_AEZxO7PUTlsKBytzegCLcBGAs/s640/image6.png" class="kg-image" alt loading="lazy"></figure><p>This 22 pages long research document addresses the current level of threat &amp; security issues within the West Bank for 2016 &amp; 2017. It contains chapters relating to human rights, data from Arab World for Research and Development center, violence center report etc.</p><h4 id="fuqha_newdetails-document-this-8-page-long-document-appears-to-be-an-intelligence-report-based-on-interviews-documents-and-public-information-the-document-mentions-an-assassination-report-of-one-of-the-highest-ranked-officers-of-the-al-qassam-group-s-millitary-wing-of-hamas-aka-armed-militia-the-document-contains-a-single-image-an-illustration-of-the-leadership-of-hamas-hierarchical-security-subgroups-">Fuqha_NewDetails Document <br> This 8 page long document appears to be an intelligence report based on interviews, documents and public information. The document mentions an assassination report of one of the highest ranked officers of the Al Qassam group's (Millitary Wing of HAMAS, aka Armed Militia). The document contains a single image, an illustration of the leadership of Hamas, hierarchical security &amp; subgroups:</h4><figure class="kg-card kg-image-card"><img src="https://blog.talosintelligence.com/content/images/-i67ZadIH5Gk/WUftfUvtGFI/AAAAAAAAAT0/zI9t6pCeQYEoWXaHnMEwX3SodZEnGHfGACLcBGAs/s640/image11.png" class="kg-image" alt loading="lazy"></figure><h2 id="further-campaigns-we-also-identified-additional-related-malicious-documents-which-conform-to-the-same-naming-pattern-and-which-share-indicators-with-the-original-malware-we-discovered-">Further Campaigns <br> We also identified additional, related malicious documents which conform to the same naming pattern and which share indicators with the original malware we discovered.</h2><h4 id="plan_palestine-document-plan_palestine_898409266595123498679852_9879483278432732489_pdf-exe">Plan_Palestine Document <br> Plan_Palestine_898409266595123498679852_9879483278432732489_pdf.exe</h4><p>The decoy document of this sample is a word document. It presents the strategic objectives, policies and interventions concerning security units (aka Police), including how to face the challenges, how to train police, new weapons etc.</p><figure class="kg-card kg-image-card"><img src="https://blog.talosintelligence.com/content/images/-4kIGYDSNWy8/WUftj5JCdXI/AAAAAAAAAT4/wXxw3SP7zFAJp66w5ry1--Ki_OIv-HgEQCLcBGAs/s640/image3.png" class="kg-image" alt loading="lazy"></figure><h4 id="diwan2017_palestine-document-diwan2017_palestine_89840923498679852_9879483278432732489_pdf-exe">Diwan2017_Palestine Document <br> Diwan2017_Palestine_89840923498679852_9879483278432732489_pdf.exe</h4><p>This decoy document is a PDF file. The document itself appears to be scanned from the Council of Ministers of Palestine and relates to an announcement concerning employee regulation.</p><figure class="kg-card kg-image-card"><img src="https://blog.talosintelligence.com/content/images/-RnBkCYh_nBg/WUftot1kwjI/AAAAAAAAAT8/7ZIsg_Qg3wUWp8i_wgQF5MARG3nORhELwCLcBGAs/s640/image4.png" class="kg-image" alt loading="lazy"></figure><h4 id="goal2017-document-goal2017_487886_10152599711675287_250999354_n_354343741352mp4-exe">Goal2017 Document <br>Goal2017_487886_10152599711675287_250999354_n_354343741352mp4.exe</h4><p>Instead of a decoy document, this sample is a decoy video of a music clip "Goal" by the Lebanese singers Myriam Klink and Jad Khalife. This video is particularly controversial as the overt nature of the video led it to be <a href="http://gulfnews.com/news/mena/lebanon/lebanese-pop-singer-s-song-goal-banned-1.1989080">banned</a> by the Lebanese Justice ministry. The sharing or airing of it is subject to a fine of 50 000 000 Lebanese Liras (approximately 33k USD).</p><figure class="kg-card kg-image-card"><img src="https://blog.talosintelligence.com/content/images/-Z1YWVvlDAQY/WUftuybsOuI/AAAAAAAAAUA/MgYlC2x1QUkWguyFZPAdABo9SmNhR_I6wCLcBGAs/s640/image1.png" class="kg-image" alt loading="lazy"></figure><p><br></p><h2 id="micropsia-analysis-for-all-of-these-decoy-documents-the-malware-is-identical-the-only-differences-are-the-sections-containing-the-decoy-documents-themselves-the-malware-is-a-remote-access-trojan-rat-written-in-delphi-named-micropsia-">MICROPSIA Analysis <br> For all of these decoy documents, the malware is identical, the only differences are the sections containing the decoy documents themselves. The malware is a remote access trojan (RAT) written in Delphi named MICROPSIA.</h2><h3 id="features-firstly-the-malware-copies-itself-in-c-programdata-mediaplayer-executelibrary-exe-the-malware-contains-several-resources-one-of-which-is-the-decoy-document-another-is-a-legitimate-binary-developed-by-optimumx-named-shortcut-exe-as-expected-the-purpose-of-this-tool-is-to-create-a-shortcut-it-is-through-creating-a-shortcut-that-the-malware-ensures-its-persistence-">Features <br> Firstly, the malware copies itself in C:\ProgramData\MediaPlayer\ExecuteLibrary.exe. The malware contains several resources, one of which is the decoy document, another is a legitimate binary developed by OptimumX named shortcut.exe. As expected the purpose of this tool is to create a shortcut. It is through creating a shortcut that the malware ensures its persistence:</h3><p>Shortcut.exe /f:"C:\Users\Administrator\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup\D_Windows_v1.lnk" /a:c /t:"C:\ProgramData\MediaPlayer\ExecuteLibrary.exe"</p><p>The malware is a Remote Administration Tool (RAT) which downloads and executes an executable obtained from the Command &amp; Control infrastructure. This executable is downloaded in string format and then modified to become a binary file with the Hex2Bin Delphi API.</p><p>An interesting element is the obfuscation algorithm used to hide the configuration of the RAT. The variables are stored in a custom base64:</p><figure class="kg-card kg-image-card"><img src="https://blog.talosintelligence.com/content/images/-jVsLyDrA1bM/WUftzgWMn8I/AAAAAAAAAUE/0yRKg-hIi_AYtX9CD_JNIzUQ1YDj4NuKwCLcBGAs/s640/image10.png" class="kg-image" alt loading="lazy"></figure><p><br> Once decoded with base64 and with 2 XOR Keys we can obtain the configuration of the malware:</p><p>[{000214A0-0000-0000-C000-000000000046}]<br> Prop3=19,2<br> [InternetShortcut]<br> IDList=<br> URL=file://<br> Mozilla/5.0 (compatible; Googlebot/2.1; +http://www.google.com/bot.html)<br> http://camilleoconnell.website/api/white_walkers/<br> daenerys<br> betriebssystem<br> anwendung<br> mikasa<br> ackerman<br> ginny<br> AV</p><p>We will see later, that this configuration contains the User-Agent, the CC URL and the json keys used for the network communication.</p><p>Additionally the malware is interested by Anti-Virus installed on the system. It uses WMI queries to get this information:</p><ul><li>SELECT * FROM AntiVirusProduct</li><li>SELECT * FROM AntiSpywareProduct</li><li>SELECT * FROM FirewallProduct If an security product is installed this information is sent to the attacker.</li></ul><h3 id="network-communication-all-the-network-parameters-are-stored-in-the-sample-and-can-be-easily-updated-by-the-author-the-cnc-is-a-web-server-http-camilleoconnell-website">Network Communication <br> All the network parameters are stored in the sample and can be easily updated by the author. The CnC is a web server: http://camilleoconnell[.]website</h3><p>The network communication is performed in HTTP. The malware uses an hardcoded User-Agent: Mozilla/5.0 (compatible; Googlebot/2.1; +http://www.google.com/bot.html)</p><p>To register a new infected system the malware perform a POST request to /api/white_walkers/new with data on the compromised system consisting of:</p><ul><li>the filename of the executed malware and the version;</li><li>the version of the infected Operating System;</li><li>the hostname and username encoded in base64. The CC will reply in json format. The json object contains an ID (incremented each time that an infected system is registered) and 3 other boolean values: load_varys, lma and ausfart. Here is an output of a registration:</li></ul><figure class="kg-card kg-image-card"><img src="https://blog.talosintelligence.com/content/images/-c7jdXV_jcVo/WUft5aZFMvI/AAAAAAAAAUI/Ubz3G-hRB6Eblo2bAnEF4Z__as8KmiXgQCLcBGAs/s640/image7.png" class="kg-image" alt loading="lazy"></figure><p>As part of our investigation we believe currently more than 500 systems are already registered on the CC. This number may be a mix of genuinely infected systems and security researcher sandbox systems.</p><p>After a registration, the malware periodically performs HTTP requests to the CC with the following pattern: GET /api/white_walkers/[base64_data_previously_sent]/requests</p><p>The server will reply with a json object. We assume that the server can issue orders to the infected system. Here is an example:</p><figure class="kg-card kg-image-card"><img src="https://blog.talosintelligence.com/content/images/-99p4DZfrd4o/WUft_PKymUI/AAAAAAAAAUM/_9rVBD3KQ1sUsI262ppR-RX5cgY7hI_OQCLcBGAs/s640/image9.png" class="kg-image" alt loading="lazy"></figure><h3 id="reference-to-tv-show-characters-in-the-analysed-variant-we-identify-several-reference-to-tv-show-characters-in-the-network-communication-and-the-urls-used-by-this-actor-">Reference to TV Show Characters <br> In the analysed variant, we identify several reference to TV Show characters in the network communication and the URLs used by this actor:</h3><ul><li>sheldon-cooper[.]info: this URL is a reference to one of the main characters of "The Big Bang Theory" named Sheldon Cooper;</li><li>Camilleoconnell[.]website: this URL is a reference to Camille O'Connell, the main actress of "The Vampire Diaries" and "The Originals";</li><li>Mikasa Ackerman is a json key returned by the CC. And this name is a character in "Attack on Titan";</li><li>/White_Walker/ in the URL is a species in the TV Show "Game of Thrones";</li><li>Deanerys is a variable used during Web request. This is the name of a character in "Game of Thrones";</li><li>Lord_varys is another json key returned by the CC. This is the name of a "Game of Thrones" character.</li></ul><p>The malware author appears to have a real interest for TV shows.</p><h3 id="goethe-s-style-we-identified-the-use-of-german-language-words-in-the-network-communication-with-the-command-and-control-server-">Goethe's Style <br> We identified the use of german language words in the network communication with the Command and Control server.</h3><ul><li>"Betriebssystem" which means Operating System. This variable is used to send the OS version (for example "Windows 7 Service Pack 1 (Version 6.1, Build 7601, 32-bit Edition)")</li><li>"Anwendung" which means Application. This variable is used to send the filename and the version of the malware.</li><li>"Ausfahrt" which means Exit. This is a json key used by the CC during network communication. The key contains a boolean (false/true)</li></ul><p>Obviously, the use of german words does not necessarily means that the author is German. The author could simply be adding german word in order to cover their tracks.</p><h2 id="conclusion-this-spear-phishing-campaign-was-directed-against-palestinian-authorities-and-possibly-against-other-entities-at-least-500-machines-have-been-registered-by-the-cc-infrastructure-which-is-still-operating-indicating-that-this-is-a-successful-campaign-">Conclusion <br> This spear phishing campaign was directed against Palestinian authorities and possibly against other entities. At least 500 machines have been registered by the CC infrastructure, which is still operating, indicating that this is a successful campaign.</h2><p>At Talos, we have in-depth experience of many APT campaigns, in this case one of the most surprising elements is the overt naming convention: the author deliberately uses references to several US TV show and intentionally uses German words for malware communication. We have no indication if these inclusions are to confuse attribution, to mock analysts, or a lapse of trade craft. This is in contrast to the highly convincing decoy documents which appear to be copies of genuine documents relating to the current situation in Palestine which suggests a high degree of professionalism.</p><h2 id="iocs">IOCs <br></h2><h3 id="file-hashes-internetpolicy-r10-9b162f43bcbfaef4e7e7bdffcf82b7512fac0fe81b7f2c172e1972e5fe4c9327">File hashes <br> InternetPolicy.r10:
