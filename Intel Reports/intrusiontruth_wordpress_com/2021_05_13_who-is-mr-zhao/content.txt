<article id="post-797" class="post-797 post type-post status-publish format-standard has-post-thumbnail hentry category-apt-with-no-name tag-apt tag-gssd tag-intrusiontruth-2 tag-mss tag-mssofficer1">
	
	<div class="entry-content">
		
<p>In our last article, we identified a number of front companies used by two Chengdu-based indicted hackers Li Xiaoyu and Dong Jiazhi.&nbsp;</p>



<p>What struck us when reading the US indictment was reference to the Guangdong State Security Department (GSSD). As eager readers of Intrusion Truth will note, we discussed the Guangdong SSD in our very first article series and their use of Boysec as a front company. However we didn’t manage to identify the MSS officers behind APT3. We feel there is unfinished business here and so we set to work to uncover MSS Officer 1. </p>



<p>We started with an address.</p>



<h2 class="wp-block-heading"><strong>GSSD HQ</strong></h2>



<p>Why is the Guangdong Province International Affairs Research Centre (GPIARC) interesting? Well, its claim to fame most recently comes from the 2020 indictment, revealing it as a GSSD cover company. The address: Number 5, 6<sup>th</sup> Crossroad, Upper Nonglin Road, Yuexiu District, Guangzhou, Guangdong Province (越秀区农林上路六横道5号).&nbsp;</p>



<div class="wp-block-image is-style-default"><figure class="aligncenter size-large"><img data-attachment-id="802" data-permalink="https://intrusiontruth.wordpress.com/image-18/" data-orig-file="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-18.png" data-orig-size="566,863" data-comments-opened="0" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;,&quot;orientation&quot;:&quot;0&quot;}" data-image-title="image-18" data-image-description="" data-image-caption="" data-medium-file="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-18.png?w=197" data-large-file="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-18.png?w=566" width="566" height="863" src="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-18.png?w=566" alt="" class="wp-image-802" srcset="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-18.png 566w, https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-18.png?w=98 98w, https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-18.png?w=197 197w" sizes="(max-width: 566px) 100vw, 566px" /></figure></div>



<p>We decided to reach out to our network of contributors, asking about the GPIARC and any previous reference to this company or their known address. We received an interesting response from a trusted source who wishes to remain anonymous. This source, with connections to the Bank of China, was able to provide a number of historic credit card statement sent to the cover address at Upper Nonglin Road. One bank statement in particular stood out.</p>



<h2 class="wp-block-heading"><strong>Zhao Jianfei (赵剑飞)</strong></h2>



<p>On the top left corner on the image below, the corresponding address is Unit 5, 6<sup>th</sup> Crossroad, Upper Nonglin Road, Yuexiu District, Guangzhou. Furthermore, all the transactions appear in Guangzhou, Guangdong.&nbsp;</p>



<p>We know this address is a cover for the GSSD. So, whoever is using this address works directly for the GSSD. So, who is this MSS officer? &nbsp;</p>



<p>Underneath the address is a single name to which the statement is addressed to: Zhao Jianfei (赵剑飞).&nbsp;</p>



<div class="wp-block-image is-style-default"><figure class="aligncenter size-large"><img data-attachment-id="808" data-permalink="https://intrusiontruth.wordpress.com/image-19/" data-orig-file="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-19.png" data-orig-size="807,780" data-comments-opened="0" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;,&quot;orientation&quot;:&quot;0&quot;}" data-image-title="image-19" data-image-description="" data-image-caption="" data-medium-file="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-19.png?w=300" data-large-file="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-19.png?w=700" width="807" height="780" src="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-19.png?w=807" alt="" class="wp-image-808" srcset="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-19.png 807w, https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-19.png?w=150 150w, https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-19.png?w=300 300w, https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-19.png?w=768 768w" sizes="(max-width: 807px) 100vw, 807px" /></figure></div>



<p>Interesting. So, we know Zhao was receiving correspondence about a credit card bill, using the GSSD cover company as the address. It stands to reason that Zhao Jianfei is an MSS officer, working for the Guangdong SSD. Could he be MSS Officer 1?</p>



<h2 class="wp-block-heading"><strong>Asls1027</strong></h2>



<p>An FBI flash memo released on the 21<sup>st</sup> July reveals further information pertaining to the email used by MSS Officer 1 to send Li and Dong zero-day exploits for use in their APT campaign. The memo has redacted the mail provider, but the handle is the bit we need: asls1027.</p>



<div class="wp-block-image is-style-default"><figure class="aligncenter size-large"><img data-attachment-id="812" data-permalink="https://intrusiontruth.wordpress.com/image-20/" data-orig-file="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-20.png" data-orig-size="738,435" data-comments-opened="0" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;,&quot;orientation&quot;:&quot;0&quot;}" data-image-title="image-20" data-image-description="" data-image-caption="" data-medium-file="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-20.png?w=300" data-large-file="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-20.png?w=700" width="738" height="435" src="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-20.png?w=738" alt="" class="wp-image-812" srcset="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-20.png 738w, https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-20.png?w=150 150w, https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-20.png?w=300 300w" sizes="(max-width: 738px) 100vw, 738px" /></figure></div>



<p>Remember when we said one statement in particular from the Bank of China was interesting to us?</p>



<p>Well, turns out that Bank of China sent the credit card statement to the personal email of Zhao Jianfei.&nbsp;</p>



<p>The email address was asls1027@hotmail.com. </p>



<p>Zhao Jianfei is an MSS officer, working for the GSSD and receiving credit card statements to the address of a GSSD cover company. Furthermore, this correspondence was sent to his personal email; the same email account that sent cyber actors a zero-day exploit for use in their illegal activities.</p>



<p>Zhao Jianfei has been directing Li Xiaoyu and Dong Jiazhi by providing them with malware and supporting their APT campaign.</p>



<h2 class="wp-block-heading"><strong>Asls1027’s social media</strong></h2>



<p>As we know, humans are biased and often rely on availability heuristics: we tend to choose the least cognitively demanding option. As such, we tend to reuse email handles, passwords and so on. And it appears our Mr. Zhao falls into this category, reusing his handle across multiple social media sites. </p>



<p>Asls1027 has an interest in cars, posting on the car forum autohome.com.cn. </p>



<p></p>



<p>He also maintains a relatively empty yet bizarre Twitter profile.&nbsp;</p>



<div class="wp-block-image is-style-default"><figure class="aligncenter size-large"><img data-attachment-id="816" data-permalink="https://intrusiontruth.wordpress.com/image-21/" data-orig-file="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-21.png" data-orig-size="689,738" data-comments-opened="0" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;,&quot;orientation&quot;:&quot;0&quot;}" data-image-title="image-21" data-image-description="" data-image-caption="" data-medium-file="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-21.png?w=280" data-large-file="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-21.png?w=689" loading="lazy" width="689" height="738" src="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-21.png?w=689" alt="" class="wp-image-816" srcset="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-21.png 689w, https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-21.png?w=140 140w, https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-21.png?w=280 280w" sizes="(max-width: 689px) 100vw, 689px" /></figure></div>



<p>However none of this provided us with any more information on Zhao Jianfei himself. We know he uses the asls handle and his name is Zhao Jianfei so we decided to get even more creative, and found an interesting profile on Facebook with the stub Asls Zh.</p>



<div class="wp-block-image is-style-default"><figure class="aligncenter size-large"><img data-attachment-id="819" data-permalink="https://intrusiontruth.wordpress.com/image-22/" data-orig-file="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-22.png" data-orig-size="531,943" data-comments-opened="0" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;,&quot;orientation&quot;:&quot;0&quot;}" data-image-title="image-22" data-image-description="" data-image-caption="" data-medium-file="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-22.png?w=169" data-large-file="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-22.png?w=531" loading="lazy" width="531" height="943" src="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-22.png?w=531" alt="" class="wp-image-819" srcset="https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-22.png 531w, https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-22.png?w=84 84w, https://intrusiontruth.wordpress.com/wp-content/uploads/2021/05/image-22.png?w=169 169w" sizes="(max-width: 531px) 100vw, 531px" /></figure></div>



<p>Given the unique of the handle &#8216;asls&#8217;, we strongly believe this profile belongs to our Mr. Zhao. The profile picture was updated in 2014, a similar timeframe to other asls social media posts, as well as Zhao&#8217;s credit card activity in Guangdong. Zh = Zhao. </p>



<p>It seems Zhao was born in Xi’an, Shaanxi Province. Also note Asls Zh’s current residence – Guangzhou, in Guangdong Province. The same location as the Zhao Jianfei’s credit statement.&nbsp;</p>



<p>Asls Zh went to the PLA Information Engineering University to study Computer Science. It fits with what we know about MSS Officer 1, and his ability to deploy zero-day exploits to support criminal hackers.</p>



<h2 class="wp-block-heading"><strong>Conclusion</strong></h2>



<p><strong>Zhao Jianfei is MSS Officer 1.&nbsp;</strong></p>

		document.body.appendChild( script );
		return Promise.resolve();
	}

	function loadExternalScript(handle) {
		if (!urls[handle]) {
			return Promise.resolve();
		}

		return fetches[handle].then(() => {
			return new Promise((resolve, reject) => {
				const script = document.createElement('script');
				script.onload = () => resolve();
				script.onerror = (e) => reject(e);
				script.src = urls[handle];
				document.body.appendChild(script);
			});
		});
	}

	function loadExtra(handle, pos) {
		const count = (scriptExtras[handle] && scriptExtras[handle][pos]) || 0;
		let promise = Promise.resolve();

		for (let i = 0; i < count; i++) {
			promise = promise.then(() => runExtraScript(handle, pos, i));
		}

		return promise;
	}

	function loadWPScript(handle) {
		// Core loads scripts in this order. See: https://github.com/WordPress/WordPress/blob/a59eb9d39c4fcba834b70c9e8dfd64feeec10ba6/wp-includes/class-wp-scripts.php#L428.
		return loadExtra(handle, 'translations')
			.then(() => loadExtra(handle, 'before'))
			.then(() => loadExternalScript(handle))
			.then(() => loadExtra(handle, 'after'));
