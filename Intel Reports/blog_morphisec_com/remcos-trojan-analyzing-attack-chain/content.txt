<h2 style="font-size: 28px;">What is the Remcos Trojan?</h2>
<p>Remcos is a commercial remote access trojan (RAT) developed by <a href="https://breakingsecurity.net/remcos/"><span>BreakingSecurity</span></a>. Remcos has many capabilities, and a free version downloadable directly from BreakingSecurity’s website. Morphisec has previously covered Remcos as the <a href="https://blog.morphisec.com/guloader-the-rat-downloader"><span>payload in Guloader</span></a>, and the payload in the <a href="https://blog.morphisec.com/the-babadeda-crypter-targeting-crypto-nft-defi-communities"><span>Babadeda crypter</span></a>.</p>
<p>The Remcos trojan allows attackers to quickly and easily control an infected computer, steal personal information, and surveil a victim’s activity. All this without investing time in developing a tool with remote administration capabilities.</p>
<h2 style="font-size: 28px;">The Infection Chain</h2>
<p><img src="https://blog.morphisec.com/hs-fs/hubfs/Remcos%20Infection%20chain%20.png?width=587&amp;name=Remcos%20Infection%20chain%20.png" alt="Remcos Infection chain " width="587" loading="lazy" style="width: 587px; margin-left: auto; margin-right: auto; display: block;" srcset="https://blog.morphisec.com/hs-fs/hubfs/Remcos%20Infection%20chain%20.png?width=294&amp;name=Remcos%20Infection%20chain%20.png 294w, https://blog.morphisec.com/hs-fs/hubfs/Remcos%20Infection%20chain%20.png?width=587&amp;name=Remcos%20Infection%20chain%20.png 587w, https://blog.morphisec.com/hs-fs/hubfs/Remcos%20Infection%20chain%20.png?width=881&amp;name=Remcos%20Infection%20chain%20.png 881w, https://blog.morphisec.com/hs-fs/hubfs/Remcos%20Infection%20chain%20.png?width=1174&amp;name=Remcos%20Infection%20chain%20.png 1174w, https://blog.morphisec.com/hs-fs/hubfs/Remcos%20Infection%20chain%20.png?width=1468&amp;name=Remcos%20Infection%20chain%20.png 1468w, https://blog.morphisec.com/hs-fs/hubfs/Remcos%20Infection%20chain%20.png?width=1761&amp;name=Remcos%20Infection%20chain%20.png 1761w" sizes="(max-width: 587px) 100vw, 587px"></p>
<p style="text-align: center;"><span style="font-size: 14px;">Figure 4: Infection chain steps</span></p>
<p>The <span style="font-family: 'Courier New', Courier, monospace;">.xls</span> file writes and executes a new <span style="font-family: 'Courier New', Courier, monospace;">.vbs</span> file. The code inside the file executes a PowerShell command that downloads another <span style="font-family: 'Courier New', Courier, monospace;">.vbs</span> file from a remote server. Once downloaded and saved on the disk, PowerShell executes the file. Next, the newly downloaded <span style="font-family: 'Courier New', Courier, monospace;">.vbs</span> file connects to the C2 server again and fetches an encoded PowerShell command. This PowerShell code is responsible for downloading the next stage and decoding it. This stage contains a .NET injector and the final payload. After decoding, the PowerShell initiates the injector that introduces the final payload, known as Remcos RAT.</p>

<p style="font-size: 18px; text-align: left;">First, the .vbs reads the PowerShell command that is written to F variable, a <span style="font-family: 'Courier New', Courier, monospace;">FileSystemObject</span> CLSID to <span style="font-family: 'Courier New', Courier, monospace;">F2</span>, and a second CLSID (<span style="font-family: 'Courier New', Courier, monospace;">Shell.Application</span>) to <span style="font-family: 'Courier New', Courier, monospace;">F3</span>. Next it creates a new file inside <span style="font-family: 'Courier New', Courier, monospace;">%AppData%</span> using the CLSID object located inside <span style="font-family: 'Courier New', Courier, monospace;">F2</span>. Finally, it writes <span style="font-family: 'Courier New', Courier, monospace;">F</span> to that file and calls the function that executes it. It does so by creating a <span style="font-family: 'Courier New', Courier, monospace;">Shell.Application</span> object using the second CLSID loaded and runs the newly created <span style="font-family: 'Courier New', Courier, monospace;">QAITB.vbs</span> file.</p>
