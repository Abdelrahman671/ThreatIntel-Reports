Click <strong>Start Scan</strong> to find macOS Bundlore or any other malicious software that might be hiding on your machine.</i></span></li></ol></blockquote><h2 id="h2_1">What is OSX.Bundlore</h2><p>macOS Bundlore, <strong>also known as OSX.Bundlore and sometimes Crossrider</strong>, is a form of adware—a type of malware that displays unwanted advertisements and installs software products offered by affiliates. Its primary feature is that <strong>it installs adware applications</strong> in a “bundle,” in other words, together with the applications that the user wants to install themselves.</p><p>&nbsp;</p><p>The authors of macOS Bundlore try to keep up with Apple’s latest security patches. For example, on macOS versions prior to 10.13, macOS Bundlore installed a malicious browser extension that hijacked user search. Now, on macOS versions 10.13 and 10.14, custom user profiles are used to perform the same hack because the previous approach is now blocked by macOS security.</p><h3 id="h3_1">What does Bundlore do</h3><p>macOS Bundlore applications <strong>display intrusive pop-up ads</strong>, which may redirect users to malicious websites or prompt them to submit personal information. Infected software may also collect user-system information, such as IP addresses, queries entered into search engines, URLs visited, pages viewed, passwords, and so on. The adware also reduces browser performance.</p><p>&nbsp;</p><p><strong>The main goal of Bundlore is to earn money for attackers</strong>, who are typically rewarded for all the clicks and impressions that the adware earns. They can also earn affiliate commission for quietly installing certain software on a user’s Mac without their knowledge.</p><h2 id="h2_2">How Bundlore malware is dangerous to your Mac</h2><p>You might assume that Bundlore isn’t as bad as other viruses or <a href="https://mackeeper.com/blog/spyware-on-mac/">spyware on Mac</a>. After all, we’re all used to seeing ads whenever we browse the internet. However, it’s a lot more than just a slight irritation. In addition to slowing down your Mac and overwhelming you with pop-ups and banner ads, Bundlore infections will often lead users to much nastier malware, viruses, and ransomware.</p><p>&nbsp;</p><p>If you accidentally click on an ad displayed by Bundlore, there’s a risk that you’ll be redirected to a site that downloads dangerous software to your computer. This software might hold your data hostage, or steal sensitive information that can be used to blackmail you or steal from you. Whatever the case may be, Bundlore isn’t something you want to have on your Mac.</p><h2 id="h2_3">How does macOS Bundlore get installed on users’ computers</h2><p>macOS Bundlore uses ads of free software or updates to spread. It’ll often tempt unsuspecting users with helpful tools and utilities or big updates for third-party software like Adobe Flash Player. Of course, none of these things are genuine—they’re just designed to fool you into installing the Bundlore virus.</p><p>&nbsp;</p><p>In general, when software is downloaded from unofficial sources (torrents, pop-up ads, unofficial websites), the risk of getting malware like macOS Bundlore is high. Some believe that Macs are immune to threats like this, but that’s simply not true—<a href="https://mackeeper.com/blog/can-macs-get-viruses/">Apple computers get viruses</a> just like Windows PCs.</p><figure class="image"><img class="lazy" loading="lazy"  class="lazy" loading="lazy"  data-src="https://static-cdn.mackeeper.com/mk-blog-upload/images/new_59effaef5d.webp" title="The Bundlore adware on Mac disguised as an Adobe Flash Player update. If you see an alert like this online, close the webpage and move on." alt="The Bundlore adware on Mac disguised as an Adobe Flash Player update. If you see an alert like this online, close the webpage and move on."><figcaption><span class="text-big"><i><strong>Example of a macOS Bundlore dropper download page</strong></i></span></figcaption></figure><h3 id="h3_2">How does macOS Bundlore overcome macOS protection mechanisms</h3><p>Over the years, macOS Bundlore has evolved to overcome the latest security protection mechanisms built into macOS. In earlier versions of the operating system, such as 10.12 and older, Bundlore exploited flaws on macOS by posing as an innocent update to a genuine piece of software.</p><p>&nbsp;</p><p><strong>Apple fixed these flaws in macOS 10.12.2 by moving TCC.db</strong>, an accessibility database that was previously open to exploits, under System Integrity Protection (SIP). This ensures that even with root access to your system, malware cannot change critical system files and settings.</p><p>&nbsp;</p><p>Moreover, <strong>in macOS 10.14</strong>, Apple added Mail, Messages, Safari, Home, iTunes data files, and Time Machine backups to the list of files protected by SIP. Another new security feature is that Apple removed the possibility of downloading third-party extensions that aren't available in the Safari Extensions Gallery.</p><blockquote><p><span class="text-small"><i><strong>Important:&nbsp;</strong></i></span></p><p>&nbsp;</p><p><span class="text-small"><i>Attackers using Bundlore have found <strong>new ways to force their way onto Macs</strong>. In more recent versions, key functionalities are bundled into one package—not split across multiple binaries and bash scripts. Bundlore has also been made compatible with the latest Safari App Extention format (.appex).</i></span></p><p>&nbsp;</p><p><span class="text-small"><i>This makes it easier for Bundlore to pose as an official software update for a popular piece of software and circumvent macOS protections.
