	<div class="container md:my-16 xl:my-24 my-8">
	<div class="entry-content lg:prose-lg mx-auto prose max-w-4xl">
		<p>Newly-discovered variants of an Android spyware that previously was attributed to an advanced persistent threat actor group called C-23 (also known as GnatSpy, FrozenCell, or VAMP) have incorporated new features into their malicious apps that make them more resilient to actions by users, who might try to remove them manually, and to security and web hosting companies that attempt to block access to, or shut down, their command-and-control server domains.</p>
<p>The C-23 threat actor has, in the past, targeted individuals based in the Middle East, particularly in the Palestinian Territories. The group has been active since at least 2017.</p>
<figure id="attachment_79583" aria-describedby="caption-attachment-79583" style="width: 482px" class="wp-caption alignnone"><a href="https://web.archive.org/web/20231208015605/https://news.sophos.com/wp-content/uploads/2021/11/C23_appupdateicon.png"><img decoding="async" class=" wp-image-79583" src="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_appupdateicon.png" alt="" width="482" height="586" srcset="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_appupdateicon.png 600w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_appupdateicon.png?resize=247,300 247w" sizes="(max-width: 482px) 100vw, 482px"/></a><figcaption id="caption-attachment-79583" class="wp-caption-text">The spyware app initially disguises itself as something called &#8220;App Updates&#8221;</figcaption></figure>
<p>The new variants appear in the form of an app that purports to install updates on the target&#8217;s phone, with names that include <strong>App Updates, System Apps Updates, or Android Update Intelligence</strong>. Sophos suspects that the apps are delivered to specific users by means of SMS text messages linking to downloads. To the best of our knowledge, none of the apps have been hosted on Google Play Store, though Sophos did reach out to the Android security team and sent details about the apps to the company.</p>
<p>Once installed, the spyware sends unique, identifiable device parameters to its command-and-control server. One of the newer features of this variant is that it will, initially, use a hardcoded C2 address to communicate, but also contains code that allows the operators of the spyware to push down a new address. This ability can keep the malware functional if one or more of the C2 server domains is taken down. The new variants did not conceal or obfuscate the C2 server address in any way.</p>
<figure id="attachment_79584" aria-describedby="caption-attachment-79584" style="width: 571px" class="wp-caption alignnone"><a href="https://web.archive.org/web/20231208015605/https://news.sophos.com/wp-content/uploads/2021/11/C23_change_c2_code.png"><img decoding="async" class="wp-image-79584 size-full" src="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_change_c2_code.png" alt="" width="571" height="256" srcset="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_change_c2_code.png 571w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_change_c2_code.png?resize=300,135 300w" sizes="(max-width: 571px) 100vw, 571px"/></a><figcaption id="caption-attachment-79584" class="wp-caption-text">Code that can modify the C2 domain on a running installation is a notable new feature</figcaption></figure>
<p>Many of the new variants were found to have been digitally signed by a certificate (with serial number ece521e38c5e9cbea53503eaef1a6ddd204583fa) that Sophos has associated with malware for years.</p>
<h3>Changing disguises after installation</h3>
<p>The first time the user opens the app, it requests that the user grant the app specific permissions to do the kinds of things you&#8217;d expect spyware to do: It requests permissions to record ambient audio, and to access all files stored on the device. <a href="https://web.archive.org/web/20231208015605/https://news.sophos.com/wp-content/uploads/2021/11/C23_record-audio-permissions.png"><img loading="lazy" decoding="async" class="alignnone wp-image-79585 size-medium" src="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_record-audio-permissions.png?w=300" alt="" width="300" https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_record-audio-permissions.png?resize=768,489 768w" sizes="(max-width: 300px) 100vw, 300px"/></a><a href="https://web.archive.org/web/20231208015605/https://news.sophos.com/wp-content/uploads/2021/11/C23_file-permissions.png"><img loading="lazy" decoding="async" class="alignnone size-medium wp-image-79586" src="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_file-permissions.png?w=300" alt="" width="300" height="173" srcset="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_file-permissions.png 918w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_file-permissions.png?resize=300,173 300w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_file-permissions.png?resize=768,442 768w" sizes="(max-width: 300px) 100vw, 300px"/></a>But the apps also use a bit of social engineering to ask the user to grant advanced permissions: notification access, device administrator, and the ability to observe the user&#8217;s actions while interacting with apps.</p>
<p>The app&#8217;s requests appear to justify the need for the additional features, but they&#8217;re lies. For instance, the request to &#8220;Enable Notifications&#8221; claims that the app needs this functionality or else &#8220;you won&#8217;t receive notifications in real time.&#8221;</p>
<p><a href="https://web.archive.org/web/20231208015605/https://news.sophos.com/wp-content/uploads/2021/11/C23_notifications_prompt.png"><img loading="lazy" decoding="async" class="alignnone size-medium wp-image-79588" src="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_notifications_prompt.png?w=300" alt="" width="300" height="173" srcset="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_notifications_prompt.png 1080w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_notifications_prompt.png?resize=300,173 300w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_notifications_prompt.png?resize=768,442 768w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_notifications_prompt.png?resize=1024,589 1024w" sizes="(max-width: 300px) 100vw, 300px"/></a>But that isn&#8217;t what Notification Access permissions do. When prompted to enable this feature, the app pushes the user to a system permissions window that accurately describes what the permission does. The threat actors may assume that the target won&#8217;t carefully read, or understand, the consequences of clicking Allow on this screen.</p>
<p><a href="https://web.archive.org/web/20231208015605/https://news.sophos.com/wp-content/uploads/2021/11/C23_notifications.png"><img loading="lazy" decoding="async" class="alignnone wp-image-79589 " src="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_notifications.png?w=640" alt="" width="550" height="767" srcset="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_notifications.png 1080w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_notifications.png?resize=215,300 215w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_notifications.png?resize=768,1072 768w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_notifications.png?resize=734,1024 734w" sizes="(max-width: 550px) 100vw, 550px"/></a>This permission grants the spyware the ability to read the full-text messages and the names of contacts, from any app, such as Facebook or WhatsApp, as well as dismissing notifications from other apps (such as a mobile anti-malware app&#8217;s warnings) or toggling the Do Not Disturb settings on the phone. The device administrator permission gives the operator of the app the ability to lock the phone, but according to our analysis, the spyware&#8217;s current version has no capability to do this.</p>
<p>The app prompts the user to Enable the device admin permission or &#8220;system won&#8217;t secure your internet connection.&#8221;</p>
<p><a href="https://web.archive.org/web/20231208015605/https://news.sophos.com/wp-content/uploads/2021/11/C23_notification_internet-prompt.png"><img loading="lazy" decoding="async" class="alignnone size-medium wp-image-79592" src="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_notification_internet-prompt.png?w=300" alt="" width="300" height="171" srcset="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_notification_internet-prompt.png 1025w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_notification_internet-prompt.png?resize=300,171 300w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_notification_internet-prompt.png?resize=768,438 768w" sizes="(max-width: 300px) 100vw, 300px"/></a>In reality, the feature the spyware wants the user to enable would let the spyware lock the phone.</p>
<p><a href="https://web.archive.org/web/20231208015605/https://news.sophos.com/wp-content/uploads/2021/11/C23_activate-administrator.png"><img loading="lazy" decoding="async" class="alignnone wp-image-79593" src="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_activate-administrator.png" alt="" width="499" height="526" https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_activate-administrator.png?resize=973,1024 973w" sizes="(max-width: 499px) 100vw, 499px"/></a>The final prompt asks the user to change a setting with a vague warning about something being blocked as a result of battery optimization. <img loading="lazy" decoding="async" class="alignnone wp-image-79594" src="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_avoid-battery-optimization.png" alt="" width="499" height="283" the other prompts, this is also bogus. This prompt redirects the user to enable a feature that permits the spyware to identify what apps you use, when you&#8217;re using them. The spyware sends that information onward to its C2 server.</p>
<p><a href="https://web.archive.org/web/20231208015605/https://news.sophos.com/wp-content/uploads/2021/11/C23_avoid-battery-optimization-observe-actions.png"><img loading="lazy" decoding="async" class="alignnone wp-image-79595" src="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_avoid-battery-optimization-observe-actions.png" alt="" width="499" height="571" srcset="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_avoid-battery-optimization-observe-actions.png 1080w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_avoid-battery-optimization-observe-actions.png?resize=262,300 262w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_avoid-battery-optimization-observe-actions.png?resize=768,879 768w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_avoid-battery-optimization-observe-actions.png?resize=895,1024 895w" sizes="(max-width: 499px) 100vw, 499px"/></a></p>
<p>Once the target has granted all these permissions, the app disguises itself to evade any attempts at manual removal by the user. The method to attain stealth appears to be new to this version: the spyware changes its icon (and name) to disguise itself using an icon of one of the four apps: <strong>Google Play, Youtube, Google, </strong>or <strong>Botim</strong> (a VOIP calling app).</p>
<p><img loading="lazy" decoding="async" class="alignnone wp-image-79178" src="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/10/change_icon.png" alt="" width="767" height="735" srcset="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/10/change_icon.png 767w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/10/change_icon.png?resize=300,287 300w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/10/change_icon.png?resize=32,32 32w" sizes="(max-width: 767px) 100vw, 767px"/>Once this happens, the next time the spyware app is opened, the spyware opens the real app whose disguise it wears, i.e., it opens Chrome if it disguises itself as Chrome, thereby giving an illusion to the user that the app is legit.</p>
<figure id="attachment_79596" aria-describedby="caption-attachment-79596" style="width: 500px" class="wp-caption alignnone"><a href="https://web.archive.org/web/20231208015605/https://news.sophos.com/wp-content/uploads/2021/11/C23_appupdateicon-1.png"><img loading="lazy" decoding="async" class="wp-image-79596" src="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_appupdateicon-1.png?w=600" alt="" width="500" height="608" srcset="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_appupdateicon-1.png 600w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/11/C23_appupdateicon-1.png?resize=247,300 247w" sizes="(max-width: 500px) 100vw, 500px"/></a><figcaption id="caption-attachment-79596" class="wp-caption-text">The spyware app icon appears as &#8220;App Updates&#8221; before the change&#8230;</figcaption></figure>

<p>The apps contain a text string, in the Arabic language, that they send to the command and control server when the icon has been changed. These strings are present in these new versions of the malware.</p>
<blockquote><p>تم تغيير الأيقونة</p></blockquote>
<p>We also found that it tried to install its own version of Botim from the application’s assets – a functionality we believe was meant for future versions, as the samples did not contain (or try to download) any Botim APK file.</p>
<p>Each functionality of the spyware has a command associated with it. The commands are received via Firebase messaging, and the spyware performs the corresponding function as and when instructed.</p>
<p><img loading="lazy" decoding="async" class="alignnone wp-image-79179" src="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/10/commands.png" alt="" width="689" height="659" srcset="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/10/commands.png 689w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/10/commands.png?resize=300,287 300w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/10/commands.png?resize=32,32 32w" sizes="(max-width: 689px) 100vw, 689px"/></p>
<p>The live C2 servers posed as websites for Laravel – a web application framework.</p>
<figure id="attachment_79170" aria-describedby="caption-attachment-79170" style="width: 718px" class="wp-caption alignnone"><img loading="lazy" decoding="async" class="wp-image-79170" src="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/10/donald-grigg.site_.png" alt="" width="718" height="392" srcset="https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/10/donald-grigg.site_.png 898w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/10/donald-grigg.site_.png?resize=300,164 300w, https://web.archive.org/web/20231208015605im_/https://news.sophos.com/wp-content/uploads/2021/10/donald-grigg.site_.png?resize=768,419 768w" sizes="(max-width: 718px) 100vw, 718px"/><figcaption id="caption-attachment-79170" class="wp-caption-text">C2 website posed as website for Laravel (a web application framework)</figcaption></figure>
<p>Yet many of the functionalities of the spyware remain unchanged. The app does the following things:</p>
<ul>
<li>Collects SMS, contacts, call logs</li>
<li>Collects images and documents</li>
<li>Recording audio, incoming and outgoing calls, including WhatsApp calls</li>
<li>Taking screenshots and recording video of the screen</li>
<li>Taking pictures using the camera</li>
<li>Hiding its own icon</li>
<li>Reading notifications from WhatsApp, Facebook, Facebook Messenger, Telegram, Skype, IMO Messenger, or Signal</li>
<li>Canceling notifications from built-in security apps (such as Samsung SecurityLogAgent, Xiaomi MIUI SecurityCenter, Huawei SystemManager), as well as from Android system apps, package Installer, and its own notifications</li>
</ul>

<h3>Don&#8217;t be a spyware victim</h3>
<p>To avoid falling prey to such malicious apps, users should only install apps from trusted sources such as Google Play. Updating Android OS and applications should be done via Android Settings and Google Play respectively, instead of relying on a third-party app.</p>
<p>Users should be particularly wary of apps asking for sensitive permissions such as device admin, notification access, or those requiring superuser/root access. Users can view the apps currently having device admin and notification access permissions by browsing to Settings and searching for &#8220;Device admin apps&#8221; and &#8220;Notification access&#8221; respectively.</p>
<h3>Detections and acknowledgments</h3>
<p>We also advise users to consider installing an antivirus app on their mobile device such as Sophos Intercept X for Mobile that defends their device and data from such threats.  SophosLabs has <a href="https://web.archive.org/web/20231208015605/https://github.com/sophoslabs/IoCs/blob/master/Android_C23-spyware.csv" target="_blank" rel="noopener">published indicators of compromise on its Github page</a>. These samples are detected by Sophos Intercept X for Mobile as <strong>Andr/Spy-BFI</strong>.</p>

					decodeURIComponent( 'You%20have%20been%20subscribed.%20Please%20check%20your%20email%20to%20confirm.' )
				);
				this.subscribed = true;
			} else if (
				data.status === "subscribed"
			) {
				this.setSuccess(
					decodeURIComponent( 'You%20are%20already%20subscribed.%20Thank%20you%21' )
				);
			} else {
				this.setError(
					decodeURIComponent( 'We%20could%20not%20subscribe%20you.%20Please%20try%20again.' )
				);
			}
			})
			.catch((e) => {
			this.setError(
				decodeURIComponent( 'We%20could%20not%20subscribe%20you.%20Please%20try%20again.' )
			);
			})
			.finally(() => {
			this.reset();
			});
		},

		buttonText() {
		return this.loading ? decodeURIComponent( 'Please%20Wait' ) : decodeURIComponent( 'Subscribe' );
		},

		buttonDisabled() {
		return (
			this.subscribeTo.length < 1 ||
			!this.emailIsValid(this.email) ||
			this.loading
		);
		},
	};
	};
</script>
