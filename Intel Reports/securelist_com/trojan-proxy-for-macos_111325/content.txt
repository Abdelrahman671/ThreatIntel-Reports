											<div class="js-reading-content">
												<div class="c-wysiwyg">
													<p>Illegally distributed software historically has served as a way to sneak malware onto victims&#8217; devices. Oftentimes, users are not willing to pay for software tools they need, so they go searching the Web for a &#8220;free lunch&#8221;. They are an excellent target for cybercriminals who realize that an individual looking for a cracked app will be willing to download an installer from a questionable website and disable security on their machine, and so they will be fairly easy to trick into installing malware as well.</p>
<p>We recently discovered several cracked applications distributed by unauthorized websites and loaded with a Trojan-Proxy. Attackers can use this type of malware to gain money by building a proxy server network or to perform criminal acts on behalf of the victim: to launch attacks on websites, companies and individuals, buy guns, drugs, and other illicit goods.</p>
<h2 id="postinstall-script">Postinstall script</h2>
<p>Unlike the original, untampered with, applications typically distributed as a disk image, the infected versions came in the form of .PKG installers. These files are handled by the Installer dedicated utility in macOS, and they can run scripts before and after actual installation. In the examples we gathered, scripts were run only after the application was installed.</p>
<div id="attachment_111328" style="width: 1034px" class="wp-caption aligncenter"><a href="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05164941/Trojan-Proxy-for-macOS-01.png" class="magnificImage"><img loading="lazy" decoding="async" aria-describedby="caption-attachment-111328" class="size-large wp-image-111328" src="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05164941/Trojan-Proxy-for-macOS-01-1024x400.png" alt="Contents of the malware script" width="1024" height="400" srcset="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05164941/Trojan-Proxy-for-macOS-01-1024x400.png 1024w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05164941/Trojan-Proxy-for-macOS-01-300x117.png 300w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05164941/Trojan-Proxy-for-macOS-01-768x300.png 768w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05164941/Trojan-Proxy-for-macOS-01-1536x600.png 1536w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05164941/Trojan-Proxy-for-macOS-01-2048x800.png 2048w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05164941/Trojan-Proxy-for-macOS-01-448x175.png 448w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05164941/Trojan-Proxy-for-macOS-01-370x145.png 370w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05164941/Trojan-Proxy-for-macOS-01-717x280.png 717w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05164941/Trojan-Proxy-for-macOS-01-800x313.png 800w" sizes="(max-width: 1024px) 100vw, 1024px" /></a><p id="caption-attachment-111328" class="wp-caption-text">Contents of the malware script</p></div>
<p>A look at the script code reveals that the /Contents/Resources/ directory contains two suspicious files in addition to the cracked application resources: WindowServer and p.plist. The script replaces the ~/Library/Application Support/WindowServer and ~/Library/LaunchAgents/GoogleHelperUpdater.plist files with the two files from the resources folder, and grants administrator permissions to these. As an installer often requests administrator permissions to function, the script run by the installer process inherits those.</p>
<p>The p.plist (or GoogleHelperUpdater.plist) is a configuration file. Its contents suggest that it imitates a Google configuration file and has only one job: auto-starting the WindowServer file, with a path set to ${VAR}, as a system process after the operating system is loaded.</p>

<h2 id="windowserver">WindowServer</h2>

<p>During our research efforts, we did not receive a server response containing any command but 0x38. An analysis of the program code suggests that the 0x34 command should be accompanied by a message containing the IP address to connect to, the protocol to use and the message to send. The client supports both TCP and UDP connections.</p>

<p>As mentioned earlier, we have discovered several versions of the Trojan, with a number of distinguishing features:</p>
<ul>
<li>Unlike its predecessors, the latest of the versions we know of cannot check its own version or update. The update function is there, but it is not invoked anywhere in the code.
<div id="attachment_111334" style="width: 1030px" class="wp-caption aligncenter"><a href="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05165611/Trojan-Proxy-for-macOS-07.png" class="magnificImage"><img loading="lazy" decoding="async" aria-describedby="caption-attachment-111334" class="size-full wp-image-111334" src="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05165611/Trojan-Proxy-for-macOS-07.png" alt="Snippet of the update function (MD5: 063d956b55da0d18f3f732c2bbd4bc28)" width="1020" height="370" srcset="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05165611/Trojan-Proxy-for-macOS-07.png 1020w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05165611/Trojan-Proxy-for-macOS-07-300x109.png 300w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05165611/Trojan-Proxy-for-macOS-07-768x279.png 768w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05165611/Trojan-Proxy-for-macOS-07-482x175.png 482w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05165611/Trojan-Proxy-for-macOS-07-370x134.png 370w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05165611/Trojan-Proxy-for-macOS-07-772x280.png 772w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/12/05165611/Trojan-Proxy-for-macOS-07-800x290.png 800w" sizes="(max-width: 1020px) 100vw, 1020px" /></a><p id="caption-attachment-111334" class="wp-caption-text">Snippet of the update function (MD5: 063d956b55da0d18f3f732c2bbd4bc28)</p></div></li>
<li>Older versions obtain a C&amp;C IP address by means of a regular DNS request rather than DoH.</li>
<li>All versions of the Trojan write logs to log.txt and dbg.dmp, without cleaning up after shutdown or providing any means for the operators to analyze the logs. Thus, it is possible to ascertain the presence of the malware in the system both by checking the known paths and searching for key tags across all text files.</li>
</ul>
