          <h1>New Arena Crysis Ransomware Variant Released</h1>
          <div class="cz-news-story-title-section">
            <div class="cz-news-title-left-area">
                            By <h6><a rel="author" href="https://www.bleepingcomputer.com/author/lawrence-abrams/" class="author"><span itemprop="author" itemscope itemtype="https://schema.org/Person"><span itemprop="name">Lawrence Abrams</span></span></a></h6>
            </div>
	    <meta itemprop="articleSection" content="Security"><div class="cz-news-title-right-area">
              <ul><li class="cz-news-date">August 25, 2017</li>
                <li class="cz-news-time">03:12 PM</li>
                
                <li class="cz-news-comment"><a href="#comments">78</a></li>
              </ul></div>
          </div>
	  <div class="articleBody">
	   
<p>Yesterday, ID-Ransomware's <a href="https://twitter.com/demonslay335" rel="nofollow">Michael Gillespie</a>&nbsp;discovered a new variant of the Crysis/Dharma ransomware that is appending the .arena extension to encrypted files. It is not known exactly how this variant is being distributed, but in the past Crysis&nbsp;was typically spread by hacking into Remote Desktop Services and manually installing the ransomware.</p>

<p>When this ransomware is installed, it will scan the computer for certain file types and encrypt them. When encrypting a file it will append an extension in the format of .id-[id].[email].arena. For example, a file called <strong><span style="color:#0000ff">test.jpg</span></strong> would be encrypted and renamed to&nbsp;<span style="color:#ff0000"><strong>test.jpg.id-BCBEF350.[chivas@aolonline.top].arena</strong></span>.</p>

<p>It should be noted that this ransomware will encrypt mapped network drives and unmapped network shares. So it is important to make sure your network's shares are locked down so that only those who actually need access have permission.</p>

<p>You can see an example of an encrypted folder below.</p>

<div style="text-align:center">
<figure class="image" data-widget="image" style="display:inline-block"><img alt="Files encrypted with the Crysis Arena Ransomware Variant" height="526" src="https://www.bleepstatic.com/images/news/ransomware/c/crysis/arena/encrypted-files.jpg" width="886"><figcaption><strong>Files encrypted with the Crysis Arena Ransomware Variant</strong></figcaption></figure></div>

<p>While encrypting a computer it will also remove all of the shadow volume copies so that you cannot use them to restore your files. It deletes them by running the <strong>vssadmin &nbsp;delete shadows /all /quiet </strong>command.</p>

<p>The Arena Crysis variant will also create two ransom notes. One is the<strong> info.hta </strong>file, which is launched by an autorun.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Crysis Arena Ransom Note" height="437" src="https://www.bleepstatic.com/images/news/ransomware/c/crysis/arena/ransom-note.jpg" width="800"><figcaption><strong>Crysis Arena Ransom Note</strong></figcaption></figure></div>

<p>The other note is called&nbsp;<strong>FILES ENCRYPTED.txt</strong>.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="FILES Encrypted Ransom Note" height="181" src="https://www.bleepstatic.com/images/news/ransomware/c/crysis/arena/ransom-note-2.jpg" width="681"><figcaption><strong>FILES Encrypted Ransom Note</strong></figcaption></figure></div>

<p>Both of these ransom notes contain instructions to contact <strong>chivas@aolonline.top</strong> in order to get payment instructions.</p>

<p>Finally, the ransomware will configure itself to automatically start when you login to Windows. This allows it to encrypt new files that are created since it was last executed.</p>

<h2>It is not possible to decrypt the Crysis Arena Ransomware&nbsp;Variant</h2>

<p>Unfortunately, at this time it is not possible to decrypt .arena files encrypted by the Crysis Ransomware for free.</p>

<p>The only way to recover encrypted files is via a backup, or if you are incredibly lucky, through Shadow Volume Copies. Though&nbsp;Crysis does attempt to remove Shadow Volume Copies, in rare cases ransomware infections fail to do so for whatever reason. Due to this, if you do not have a viable backup, I always suggest people try as a last resort to&nbsp;<a href="https://www.bleepingcomputer.com/tutorials/how-to-recover-files-and-folders-using-shadow-volume-copies/" target="_blank" rel="nofollow noopener">restore encrypted files from Shadow Volume Copies</a>&nbsp;as well.</p>

<p>For those who wish to discuss the Crysis ransomware or need support, you can use our dedicated <a href="https://www.bleepingcomputer.com/forums/t/607680/crysis-extensionid-numberemailarenacrysis-ransomware-support-topic/" target="_blank" rel="nofollow noopener">Crysis Ransomware&nbsp;Help &amp; Support Topic</a>.</p>

<h2>How to protect yourself from the Crysis Ransomware</h2>

<p>In order to protect yourself from Crysis, or from any ransomware, it is important that you use good computing habits and security software. First and foremost, you should always have a reliable and tested backup of your data that can be restored in the case of an emergency, such as a ransomware attack.</p>

<p>You should also have security software that contains behavioral detections such as&nbsp;<a href="https://www.bleepingcomputer.com/download/emsisoft-anti-malware/" target="_blank" rel="nofollow noopener">Emsisoft Anti-Malware</a>&nbsp;or&nbsp;<a href="https://www.bleepingcomputer.com/download/malwarebytes-anti-malware/" target="_blank" rel="nofollow noopener">Malwarebytes</a>.</p>

<p>Last, but not least, make sure you practice the following good online security habits, which in many cases are the most important steps of all:</p>

<ul><li>Backup, Backup, Backup!</li>
	<li>Do not open attachments if you do not know who sent them.</li>
	<li>Do not open attachments until you confirm that the person actually sent you them,</li>
	<li>Scan attachments with tools like VirusTotal.</li>
	<li>Make sure all Windows updates are installed as soon as they come out! Also make sure you update all programs, especially Java, Flash, and Adobe Reader. Older programs contain security vulnerabilities that are commonly exploited by malware distributors. Therefore it is important to keep them updated.</li>
	<li>Make sure you use have some sort of security software installed.</li>
	<li>Use hard passwords and never reuse the same password at multiple sites.</li>
	<li>If you are using Remote Desktop Services, do not connect it directly to the Internet. Instead make it accessibly only via a VPN.</li>
</ul><p>For a complete guide on ransomware protection, you visit our&nbsp;<a href="https://www.bleepingcomputer.com/news/security/how-to-protect-and-harden-a-computer-against-ransomware/" target="_blank" rel="nofollow noopener">How to Protect and Harden a Computer against Ransomware</a>&nbsp;article.<br>
&nbsp;</p>

  allowed = (((allowed || '') + '')
    .toLowerCase()
    .match(/<[a-z][a-z0-9]*>/g) || [])
    .join(''); // making sure the allowed arg is a string containing only tags in lowercase (<a><b><c>)

    commentsAndPhpTags = /<!--[\s\S]*?-->|<\?(?:php)?[\s\S]*?\?>/gi;

    .replace(tags, function($0, $1) {
      return allowed.indexOf('<' + $1.toLowerCase() + '>') > -1 ? $0 : '';
    });
}
