                                <p>In the first half of 2021, we started to see the REvil ransomware operators pivot to targeting Linux-based systems with a new Linux version of their ransomware, similar to the malware they commonly used on Windows systems. Since then, there have been a few versions of this Linux-based malware.</p><p>In this post, we look at the latest version of their Linux-based ransomware "1.2a".</p><h2>Quick Snapshot:</h2><p>The malicious file is a Linux executable<br>Class: ELF64<br>Type: Dynamically Linked<br>Machine: X86-64<br>Number of section headers: 28<br>Entry Point: 0x401650<br>callq: __libc_start_main@plt<br>MD5: c83df66c46bcbc05cd987661882ff061<br>Yara Rules:&nbsp;</p><p><a href="https://github.com/YaraExchange/yarasigs/blob/master/ransomware/crime_lin_revil.yar">https://github.com/YaraExchange/yarasigs/blob/master/ransomware/crime_lin_revil.yar</a><a href="https://github.com/YaraExchange/yarasigs/blob/master/ransomware/crime_lin_revix.yar"></a><br><a href="https://github.com/YaraExchange/yarasigs/blob/master/ransomware/crime_lin_revix.yar">https://github.com/YaraExchange/yarasigs/blob/master/ransomware/crime_lin_revix.yar</a></p><h2>Introduction</h2><p>The execution of this malware is straightforward. It traverses through the directories specified as targets and encrypts the files present in those directories. Once encryption is complete, it drops a ransom note in the directory with the usual ransom message and instructions on paying the threat actor to get the decryption key.</p><figure class="image ck-widget image-style-block-align-center" contenteditable="false"><img src="https://passle-net.s3.amazonaws.com/Passle/602651b953548812c0fa5fe2/MediaLibrary/Images/60632d6d53548906000a2ec4/2021-11-30-14-09-57-026-61a630b5e5411b0c3c6a3dbc.PNG" style="width: 588px" class="fr-fic fr-dib"></figure><p>This variant of Revix requires a couple of parameters to be passed to execute successfully. It also requires escalated privileges to run and encrypt files on the disk successfully. Additionally, the malware checks the files in the target directories to see if they are already encrypted.</p><p>One of the main targets for this malware is VMware ESX platform's, which we've seen before in a different Linux ransomware from <a href="https://www.trendmicro.com/en_au/research/21/e/darkside-linux-vms-targeted.html">DarkSide</a>.</p><h2>Analysis</h2><p>For this post, we analyzed Revix both statically and dynamically. Both methodologies have been used together throughout the analysis process presented below.</p><p>Let’s take a quick look at a couple of sections of this executable so that we have the offsets to some of the initial calls that can be used for further analysis.</p><figure class="image ck-widget image-style-block-align-center" contenteditable="false"><strong>Header:</strong><strong><img src="https://passle-net.s3.amazonaws.com/Passle/602651b953548812c0fa5fe2/MediaLibrary/Images/60632d6d53548906000a2ec4/2021-11-30-14-10-54-991-61a630ee53548912d026f6f3.png" style="width: 588px" class="fr-fic fr-dib"></strong></figure><p style="margin-left: 80px"><em>Figure 1: Header Information</em></p><p><strong>Section .init:</strong></p><p>This section holds executable instructions that need to be executed before the main program entry point.</p><figure class="image ck-widget image-style-block-align-center" contenteditable="false"><img src="https://passle-net.s3.amazonaws.com/Passle/602651b953548812c0fa5fe2/MediaLibrary/Images/60632d6d53548906000a2ec4/2021-11-30-14-13-17-934-61a6317d49b2040a18559c98.png" style="width: 588px" class="fr-fic fr-dib"></figure><p style="margin-left: 80px"><em>Figure 2: section</em></p><p><strong>Section .text:</strong></p><p>This section contains executable code.</p><figure class="image ck-widget image-style-block-align-center" contenteditable="false"><img src="https://passle-net.s3.amazonaws.com/Passle/602651b953548812c0fa5fe2/MediaLibrary/Images/60632d6d53548906000a2ec4/2021-11-30-14-13-54-027-61a631a253548912d026f8c9.png" style="width: 588px" class="fr-fic fr-dib"></figure><p style="margin-left: 80px"><em>Figure 3: section</em></p><h2><strong>Functions</strong></h2><p>Revix loads several functions upon initialization.
