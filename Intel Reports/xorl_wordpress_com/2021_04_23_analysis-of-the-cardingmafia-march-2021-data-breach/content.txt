	<h2 id="post-4809">Analysis of the CardingMafia March 2021 data&nbsp;breach</h2>
			<p class="comments"><a href="https://xorl.wordpress.com/2021/04/23/analysis-of-the-cardingmafia-march-2021-data-breach/#respond">leave a comment &raquo;</a></p>
	
	<div class="main">
		
<p>One relatively large carding forum, the CardingMafia, was breached and their data were shared in darkweb channels in <a href="https://www.vice.com/en/article/v7m9jx/credit-card-hacking-forum-gets-hacked-exposing-300000-hackers-accounts">late March 2021</a>. I got a hold of this data set and found some interesting information. Of course, I cannot share PII or other sensitive material in a public blog post, but I can share some of the more generic analysis to hopefully provide better insights in this underground carding community, and the overall darkweb carding communities.</p>



<p>First of all, CardingMafia was both a &#8220;shopp&#8221; and an English-speaking forum. This means CardingMafia members were able to have forum discussions on topics of interest but there was also a section of the website where members could buy and/or sell stolen digital goods (compromised accounts, cardholder data, etc.)</p>



<p>The data leak that happened in March 2021 was from the &#8220;user&#8221; table of the SQL server of the forum. In total, there were 297,076 accounts with various details such as username, IP address, creation time, hashed password, email, status, likes, etc. This is the dataset I&#8217;m using for this blog post.</p>



<p></p>



<p>As always, this is a personal research. I&#8217;m neither a lawyer nor I&#8217;m talking on behalf of my employer. This is 100% personal threat research to enrich some of my personal threat intelligence cases and projects.</p>



<div class="wp-block-image"><figure class="aligncenter size-large"><img data-attachment-id="4822" data-permalink="https://xorl.wordpress.com/cardingmafia/" data-orig-file="https://xorl.wordpress.com/wp-content/uploads/2021/04/cardingmafia.jpg" data-orig-size="832,375" data-comments-opened="1" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;,&quot;orientation&quot;:&quot;1&quot;}" data-image-title="cardingmafia" data-image-description="" data-image-caption="" data-medium-file="https://xorl.wordpress.com/wp-content/uploads/2021/04/cardingmafia.jpg?w=300" data-large-file="https://xorl.wordpress.com/wp-content/uploads/2021/04/cardingmafia.jpg?w=700" width="832" height="375" src="https://xorl.wordpress.com/wp-content/uploads/2021/04/cardingmafia.jpg?w=832" alt="" class="wp-image-4822" srcset="https://xorl.wordpress.com/wp-content/uploads/2021/04/cardingmafia.jpg 832w, https://xorl.wordpress.com/wp-content/uploads/2021/04/cardingmafia.jpg?w=150 150w, https://xorl.wordpress.com/wp-content/uploads/2021/04/cardingmafia.jpg?w=300 300w, https://xorl.wordpress.com/wp-content/uploads/2021/04/cardingmafia.jpg?w=768 768w" sizes="(max-width: 832px) 100vw, 832px" /></figure></div>



<p></p>



<p>The top 100 IP addresses with the highest amount of user accounts in CardingMafia are shown below. Each of those IP addresses had anything from 180 to over 13,000 unique forum accounts associated with it.</p>



<p></p>



<p>Although some were automated accounts (probably from darkweb scraping bots), it appears that many legitimate users were also using those services to hide their real IP addresses (proxies, VPNs, hosted jumphosts, etc.)</p>



<div class="wp-block-image"><figure class="aligncenter size-large"><img data-attachment-id="4824" data-permalink="https://xorl.wordpress.com/top10-ips-cardingmafia/" data-orig-file="https://xorl.wordpress.com/wp-content/uploads/2021/04/top10-ips-cardingmafia.jpg" data-orig-size="769,946" data-comments-opened="1" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;,&quot;orientation&quot;:&quot;1&quot;}" data-image-title="top10-ips-cardingmafia" data-image-description="" data-image-caption="" data-medium-file="https://xorl.wordpress.com/wp-content/uploads/2021/04/top10-ips-cardingmafia.jpg?w=244" data-large-file="https://xorl.wordpress.com/wp-content/uploads/2021/04/top10-ips-cardingmafia.jpg?w=700" width="769" height="946" src="https://xorl.wordpress.com/wp-content/uploads/2021/04/top10-ips-cardingmafia.jpg?w=769" alt="" class="wp-image-4824" srcset="https://xorl.wordpress.com/wp-content/uploads/2021/04/top10-ips-cardingmafia.jpg 769w, https://xorl.wordpress.com/wp-content/uploads/2021/04/top10-ips-cardingmafia.jpg?w=122 122w, https://xorl.wordpress.com/wp-content/uploads/2021/04/top10-ips-cardingmafia.jpg?w=244 244w" sizes="(max-width: 769px) 100vw, 769px" /></figure></div>



<p>From those top 100 IP addresses you can see that the vast majority were coming from the US (29%), France (14%) and Ukraine (12%). Again, this doesn&#8217;t necessarily mean that the forum users were actually from these countries, but that they were using services from those countries to access the forum.</p>



<p>To have a more holistic view, the following geo-heatmap is <strong>based on all of the 297,076 IP addresses</strong>. It&#8217;s worth noting here that in reality there were only 56,685 unique IP addresses that were used to open CardingMafia forum accounts.</p>



<p>With this bigger picture view, we get a better understanding of likely the real countries of the forum members since many of them were connecting directly from ISPs. The United States is probably skewed due to the amount of cloud services and proxies, but the rest should be closer to reality.</p>

<ol class="wp-block-list"><li>USA (19393 accounts)</li><li>Germany (2553 accounts)</li><li>United Kingdom (2479 accounts)</li><li>Russia (1841 accounts)</li><li>China (1781 accounts)</li><li>France (1632 accounts)</li><li>Indonesia (1559 accounts)</li><li>Netherlands (1458 accounts)</li><li>Canada (1410 accounts)</li><li>Ghana (1384 accounts)</li><li>Australia (1138 accounts)</li><li>India (1042 accounts)</li><li>Ukraine (1016 accounts)</li><li>Nigeria (992 accounts)</li><li>Brazil (705 accounts)</li><li>Spain (663 accounts)</li><li>Japan (647 accounts)</li><li>Hong Kong (586 accounts)</li><li>Vietnam (572 accounts)</li><li>Singapore (560 accounts)</li></ol>

<p>The email addresses used by the CardingMafia members also provide some interesting insights. For example, there were 294,887 unique email addresses. To avoid clutter, I picked the top 20 email service providers used for those accounts and you can see them here. Unsurprisingly, GMail.com is the top one with 51.7%, followed by Yahoo.com (13.4%) and Hotmail.com (6.3%).</p>

<p>There is something else which is interesting though. There were some email addresses from corporate accounts which again, could be benign but it it could also be an insider threat. I will not list those here but it was everything from small businesses to large multi-national corporations across various industries.</p>



<p>Something else which is notable is that only a tiny percentage of the forum members were using privacy-focused email service providers. For example, using the <a href="https://cybernews.com/secure-email-providers/">top 5 list from CyberNews</a> we have:</p>



<ol class="wp-block-list"><li>ProtonMail: 3155 accounts (1.07% of the total accounts)</li><li>Tutanota: 375 accounts (0.13% of the total accounts)</li><li>Zoho Mail: 60 accounts (0.02% of the total accounts)</li><li>Thexyz: 0 accounts</li><li>Startmail: 4 accounts (0.000014% of the total accounts)</li></ol>



<p>I want to highlight this since we keep on hearing how cyber-criminals are very privacy-/anonymous-aware by many government officials, but the above is a clear indication that the vast majority (over 97%) of the members of a relatively large cyber-criminal forum were using unencrypted email providers.</p>

<p class="has-text-align-center"><strong>Suggestions for threat intelligence production</strong></p>



<p>I didn&#8217;t want this blog post to be just a selection of statistics for a carding forum, so here are some suggestions for turning this data breach into actionable intelligence. There are other use cases too in the more generic threat research space, but I tried to keep it limited to actionable threat intelligence suggestions.</p>



<ul class="wp-block-list"><li>Check if any accounts where created with your domain name(s) as it could be benign (curious employees, threat researchers, etc.) but it could also be an insider threat.</li></ul>



<ul class="wp-block-list"><li>If you are an email service provider, use the dataset to find out which of your users have been using your services to host and/or access cyber-criminal content. Again, it could be benign (threat research, law enforcement, etc.) but it could also lead you to some real cyber-criminals.</li></ul>



<ul class="wp-block-list"><li>If you are a hosting provider, use the IP dataset to find out which of your customers have been using your services to host and/or access cyber-criminal content. Again, it could be benign (threat research, law enforcement, etc.) but it could also lead you to some real cyber-criminals.</li></ul>



<ul class="wp-block-list"><li>If you are a national CERT (or similar entity) use this dataset filtering on your ASNs and TLDs to better understand the threat landscape of your country and potentially even uncover some cyber-criminal activity.</li></ul>



<ul class="wp-block-list"><li>If you are working on de-anonymization or tracking of a cyber-criminal use this dataset for enrichment and pivot searching (e.g. emails, hashed passwords, IP addresses).</li></ul>



<ul class="wp-block-list"><li>Check if any accounts were created from your IP address ranges (there are some originating from corporate networks). That could be anything from a curious individual, to a threat researcher or even an insider threat.</li></ul>



<ul class="wp-block-list"><li>You can use the email addresses of the CardingMafia members to identify and profile high-risk individuals/customers you might have. For example, if you are a payments platform and you have CardingMafia members with accounts on your platform it might worth some proactive fraud investigation.</li></ul>

	function loadExternalScript(handle) {
		if (!urls[handle]) {
			return Promise.resolve();
		}

		return fetches[handle].then(() => {
			return new Promise((resolve, reject) => {
				const script = document.createElement('script');
				script.onload = () => resolve();
				script.onerror = (e) => reject(e);
				script.src = urls[handle];
				document.body.appendChild(script);
			});
		});
	}

	function loadExtra(handle, pos) {
		const count = (scriptExtras[handle] && scriptExtras[handle][pos]) || 0;
		let promise = Promise.resolve();

		for (let i = 0; i < count; i++) {
			promise = promise.then(() => runExtraScript(handle, pos, i));
		}

		return promise;
	}

	function loadWPScript(handle) {
		// Core loads scripts in this order. See: https://github.com/WordPress/WordPress/blob/a59eb9d39c4fcba834b70c9e8dfd64feeec10ba6/wp-includes/class-wp-scripts.php#L428.
		return loadExtra(handle, 'translations')
			.then(() => loadExtra(handle, 'before'))
			.then(() => loadExternalScript(handle))
			.then(() => loadExtra(handle, 'after'));
	}
} )();
