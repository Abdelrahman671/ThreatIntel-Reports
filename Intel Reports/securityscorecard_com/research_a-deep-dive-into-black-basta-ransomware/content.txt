					<h2>Executive Summary of Black Basta Ransomware</h2>
<p>Black Basta ransomware is a recent threat that compiled its first malware samples in February 2022. The ransomware deletes all Volume Shadow Copies, creates a new JPG image set as the Desktop Wallpaper and an ICO file representing the encrypted files. Unlike other ransomware families, the <a href="https://securityscorecard.com/blog/what-is-malware/">malware</a> doesn’t skip files based on their extensions. However, it doesn’t encrypt critical folders that would make the system inoperable.</p>
<p>The files are encrypted using the ChaCha20 algorithm, with the key and nonce being encrypted using the RSA public key that is hard-coded in the sample. The malware can fully or partially encrypt a file depending on its size. The extension of the encrypted files is changed to .basta by the ransomware.</p>

<p>The process constructs the initial state of ChaCha20 using the key, the nonce, and some constant values:</p>
<figure id="attachment_1598" aria-describedby="caption-attachment-1598" style="width: 318px" class="wp-caption aligncenter"><img decoding="async" class="size-full wp-image-1598" src="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-49.png" alt="basta-fig-49" width="318" height="339" srcset="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-49.png 318w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-49-281x300.png 281w" sizes="(max-width: 318px) 100vw, 318px" /><figcaption id="caption-attachment-1598" class="wp-caption-text">Figure 49</figcaption></figure>

<figure id="attachment_1600" aria-describedby="caption-attachment-1600" style="width: 975px" class="wp-caption aligncenter"><img decoding="async" class="size-full wp-image-1600" src="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-51.png" alt="basta-fig-51" width="975" height="107" srcset="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-51.png 975w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-51-300x33.png 300w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-51-768x84.png 768w" sizes="(max-width: 975px) 100vw, 975px" /><figcaption id="caption-attachment-1600" class="wp-caption-text">Figure 51</figcaption></figure>

<figure id="attachment_1601" aria-describedby="caption-attachment-1601" style="width: 975px" class="wp-caption aligncenter"><img decoding="async" class="size-full wp-image-1601" src="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-52.png" alt="basta-fig-52" width="975" height="144" srcset="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-52.png 975w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-52-300x44.png 300w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-52-768x113.png 768w" sizes="(max-width: 975px) 100vw, 975px" /><figcaption id="caption-attachment-1601" class="wp-caption-text">Figure 52</figcaption></figure>

<figure id="attachment_1604" aria-describedby="caption-attachment-1604" style="width: 975px" class="wp-caption aligncenter"><img decoding="async" class="size-full wp-image-1604" src="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-55.png" alt="basta-fig-55" width="975" height="146" srcset="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-55.png 975w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-55-300x45.png 300w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-55-768x115.png 768w" sizes="(max-width: 975px) 100vw, 975px" /><figcaption id="caption-attachment-1604" class="wp-caption-text">Figure 55</figcaption></figure>

<p>In this case, the entire file content is encrypted by the ransomware:</p>
<figure id="attachment_1607" aria-describedby="caption-attachment-1607" style="width: 866px" class="wp-caption aligncenter"><img decoding="async" class="size-full wp-image-1607" src="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-58.png" alt="basta-fig-58" width="866" height="1119" srcset="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-58.png 866w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-58-232x300.png 232w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-58-792x1024.png 792w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-58-768x992.png 768w" sizes="(max-width: 866px) 100vw, 866px" /><figcaption id="caption-attachment-1607" class="wp-caption-text">Figure 58</figcaption></figure>

<p>In this case, the file is partially encrypted. The ransomware encrypts 64 bytes, skips 192 bytes, encrypts 64 bytes again, and so on.</p>
<figure id="attachment_1608" aria-describedby="caption-attachment-1608" style="width: 975px" class="wp-caption aligncenter"><img decoding="async" class="size-full wp-image-1608" src="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-59.png" alt="basta-fig-59" width="975" height="981" srcset="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-59.png 975w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-59-298x300.png 298w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-59-150x150.png 150w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-59-768x773.png 768w" sizes="(max-width: 975px) 100vw, 975px" /><figcaption id="caption-attachment-1608" class="wp-caption-text">Figure 59</figcaption></figure>
<h3>Black Basta Ransomware Case 3 – File size &gt; 4KB</h3>
<p>In this case, the file is partially encrypted. The ransomware encrypts 64 bytes, skips 128 bytes, encrypts 64 bytes again, and so on.</p>
<figure id="attachment_1609" aria-describedby="caption-attachment-1609" style="width: 877px" class="wp-caption aligncenter"><img decoding="async" class="size-full wp-image-1609" src="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-60.png" alt="basta-fig-60" width="877" height="1108" srcset="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-60.png 877w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-60-237x300.png 237w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-60-811x1024.png 811w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-60-768x970.png 768w" sizes="(max-width: 877px) 100vw, 877px" /><figcaption id="caption-attachment-1609" class="wp-caption-text">Figure 60</figcaption></figure>

<figure id="attachment_1610" aria-describedby="caption-attachment-1610" style="width: 975px" class="wp-caption aligncenter"><img decoding="async" class="size-full wp-image-1610" src="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-61.png" alt="basta-fig-61" width="975" height="201" srcset="https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-61.png 975w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-61-300x62.png 300w, https://securityscorecard.com/wp-content/uploads/2024/01/basta-fig-61-768x158.png 768w" sizes="(max-width: 975px) 100vw, 975px" /><figcaption id="caption-attachment-1610" class="wp-caption-text">Figure 61</figcaption></figure>

<p>readme.txt</p>
<p><strong>Files created</strong></p>
<p>%Temp%\fkdjsadasd.ico</p>
<p>%Temp%\dlaksjdoiwq.jpg</p>
<p><strong>Processes spawned</strong></p>
<p>cmd.exe /c “C:\Windows\SysNative\vssadmin.exe delete shadows /all /quiet”</p>
<p>cmd.exe /c “C:\Windows\System32\vssadmin.exe delete shadows /all /quiet”</p>

<p>HKEY_CLASSES_ROOT\.basta</p>
