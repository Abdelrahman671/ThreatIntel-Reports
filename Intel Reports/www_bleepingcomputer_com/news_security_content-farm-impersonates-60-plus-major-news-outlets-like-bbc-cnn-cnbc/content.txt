<article><div class="article_section">
<h1>Content farm impersonates 60+ major news outlets, like BBC, CNN, CNBC</h1>
<div class="cz-news-story-title-section">
<div class="cz-news-title-left-area">
By <h6><a rel="author" href="https://www.bleepingcomputer.com/author/ax-sharma/" class="author"><span itemprop="author" itemscope itemtype="https://schema.org/Person"><span itemprop="name">Ax Sharma</span></span></a></h6>
</div>
<meta itemprop="articleSection" content="Security"><div class="cz-news-title-right-area">
<ul><li class="cz-news-date">March 2, 2024</li>
<li class="cz-news-time">11:31 AM</li>
<li class="cz-news-comment"><a href="#comments">5</a></li>
</ul></div>
</div>
<div class="articleBody">
<p style="text-align:center"><img alt="content farm news scam" height="897" src="https://www.bleepstatic.com/content/hl-images/2024/03/01/image.png" width="1600"></p>
<p>BleepingComputer has discovered a content farm operating some 60+ domains named after popular media outlets, including the BBC, CNBC, CNN, Forbes, Huffington Post, Reuters, The Guardian, and Washington Post, among others.</p>
<p>These "news" websites, which we were able to trace to their proprietor&nbsp;in India, repost articles from credible media and research organizations without attribution.</p>
<p>Beyond that though, their intentions seem multifaceted&mdash;from building SEO for their online gambling&nbsp;ventures to deceptively selling "press release"&nbsp;and "product review" ad slots&nbsp;at hefty prices to unsuspecting users&nbsp;looking to market their products online.</p>
<h2>Content farm operates 60+ 'news' websites</h2>
<p>BleepingComputer has identified a network of more than five dozen "news" websites that impersonate leading media outlets like the BBC, Bloomberg, CNBC, CNN, Crunchbase, Forbes, Huffington Post,&nbsp;The Guardian, The Metro (UK), The Mirror,&nbsp;The Telegraph, Reuters, Washington Times, and Washington Post.</p>
<p>We have released the&nbsp;complete list of these domains in this article below.</p>
<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Websites impersonate popular media outlets like Washington Post, Guardian, CNN, etc." height="384" src="https://www.bleepstatic.com/images/news/u/1164866/2024/Mar/newsfarm/looklike-media-sites2.jpg" width="700"><figcaption><strong>Sites impersonate mainstream media outlets like the Washington Post, Guardian, and CNN</strong><br>
(BleepingComputer)</figcaption></figure></div>
<p>These websites repost existing news articles taken from other sources&nbsp;verbatim under an "admin" author account without proper attribution,&nbsp;effectively plagiarising them from credible media outlets and research organizations.</p>
<p>As an example, notice the following article published on&nbsp;<em>www.guardiannewstoday.com</em>, which is not associated with the widely circulated newspaper,&nbsp;<em>The Guardian.</em></p>
<p>BleepingComputer identified that the article, along with its headline and body, was copied word-to-word from a legitimate source,&nbsp;<a href="https://www.jurist.org/news/2024/02/uk-illegal-migration-act-will-introduce-perma-backlog-of-asylum-seekers-independent-report/" target="_blank" rel="nofollow noopener">jurist.org</a>, which is a legal news and commentary website.</p>
<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="The Guardian lookalike website reposting articles" height="460" src="https://www.bleepstatic.com/images/news/u/1164866/2024/Mar/newsfarm/guardian-fake.jpg" width="600"><figcaption><strong>'Guardian' lookalike website reposting articles from original source, jurist.org</strong><br>
(BleepingComputer)</figcaption></figure></div>
<h2>Spams forums for SEO and to&nbsp;sell press releases for $1000</h2>
<p>The party behind this operation frequently spams&nbsp;forums and comment sections of various kinds of&nbsp;websites with <a href="https://mailchimp.com/resources/what-is-backlinking-and-why-is-it-important-for-seo/#:~:text=If%20you%20know%20only%20little,traffic%20coming%20to%20your%20website." target="_blank" rel="nofollow noopener">backlinks</a>&nbsp;to&nbsp;these&nbsp;domains, in an attempt to boost SEO for these online properties&nbsp;and in turn lend credibility to them.</p>
<p>Of the several examples we found, below are posts made on&nbsp;a gaming forum, and the&nbsp;popular <em>Immigration Boards</em> forum, where&nbsp;community members share experiences about their UK immigration journey.</p>
<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Operator spams forums to boost SEO for these lookalike websites" height="270" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" width="600" data-src="https://www.bleepstatic.com/images/news/u/1164866/2024/Mar/newsfarm/forum-spam.jpg" class="b-lazy"><figcaption><strong>Operator spams forums to boost SEO for these lookalike news sites</strong><br>
(BleepingComputer)</figcaption></figure></div>
<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Multiple forums and news sites spammed" height="629" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" width="550" data-src="https://www.bleepstatic.com/images/news/u/1164866/2024/Mar/newsfarm/forum-spam-google.jpg" class="b-lazy"><figcaption><strong>Several forums and news sites spammed by the 'news' syndicate </strong>(BleepingComputer)</figcaption></figure></div>
<p>BleepingComputer also observed that in some of these comments,&nbsp;the operator of this network appeared&nbsp;to be selling&nbsp;advertorial slots for press releases and product reviews,&nbsp;starting at $50 per&nbsp;post or a "bulk deal" priced at $1000.</p>
<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Advertorial placement spaces for these websites being sold for a thousand dollars" height="464" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" width="400" data-src="https://www.bleepstatic.com/images/news/u/1164866/2024/Mar/newsfarm/selling-press-releases.jpg" class="b-lazy"><figcaption><strong>Advertorial placement spaces for these websites are being sold for a thousand dollars</strong><br>
(BleepingComputer)</figcaption></figure></div>
<p>The lure could be tempting to unsuspecting readers interested in promoting their products, who could&nbsp;be confusing these lookalike websites&nbsp;with legitimate media outlets&nbsp;that they mimic.</p>
<h2>Maintains Google News and&nbsp;social media&nbsp;presence</h2>
<p>We were further surprised to learn that in addition to running over five dozen online properties, the operator maintains Facebook pages for some of these and might&nbsp;have additionally enrolled them as a&nbsp;Google News publisher, much like&nbsp;real media organizations do.</p>
<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Facebook page for a Washington Times News Today site" height="378" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" width="550" data-src="https://www.bleepstatic.com/images/news/u/1164866/2024/Mar/newsfarm/washington-times-lookalike.jpeg" class="b-lazy"><figcaption><strong>Facebook page for the operation's 'Washington Times News Today' website </strong>(BleepingComputer)</figcaption></figure></div>
<p>Notice that the physical address listed on the Facebook page for the so-called 'Washington Times News Today' is <a href="https://www.washingtontimes.com/contact-us/" target="_blank" rel="nofollow noopener">that of an established American daily</a>, <em>Washington Times</em>.</p>
<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Washington Times real office address" height="321" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" width="550" data-src="https://www.bleepstatic.com/images/news/u/1164866/2024/Mar/newsfarm/washington-times-real-address.jpg" class="b-lazy"><figcaption><strong>Washington Times real office address listed by copycat website </strong>(Google Maps)</figcaption></figure></div>
<p>Below is&nbsp;a search&nbsp;result for a Sky Sports (UK) article, a copy of which also appears on&nbsp;<em>CanadianNewsToday.com</em>, a part of this domain syndicate and visible on&nbsp;Google News.</p>

<h2>Operation traced to India</h2>
<p>Many&nbsp;of these domains, according to public WHOIS records, have been operational since at least 2022, with new ones gradually added over time, well into 2024.</p>
<p>BleepingComputer observed that most of&nbsp;these websites use&nbsp;WordPress as their choice of CMS, and&nbsp;share a common registrar and host,&nbsp;<em>One.com</em>.</p>
<p>Based on the contact&nbsp;information listed on some of these&nbsp;websites, we were able to trace them&nbsp;to their operator. Our preliminary analysis suggests that the&nbsp;operation is based in and&nbsp;running&nbsp;out of India.</p>
<p>In addition to abusing well-known media trademarks for SEO marketing, the ultimate goal of these online ventures is apparently&nbsp;to promote online gambling, sports betting, casino games, and&nbsp;crypto enterprises&mdash;which are&nbsp;among the key focus areas&nbsp;of the entity behind this operation.</p>
<p>The operators of this syndicate may be associated with&nbsp;<em>jackpotbetonline.com,&nbsp;</em>a sports betting and gambling company that, according to <a href="https://www.linkedin.com/company/jackpotbetonline/about/" target="_blank" rel="nofollow noopener">its LinkedIn page</a>&nbsp;is&nbsp;based in Gurugram, India, and has been active since at least 2014. The company's social media account on X (formerly Twitter) was <a href="https://twitter.com/jackpotbet" target="_blank" rel="nofollow noopener">suspended</a>.</p>
<p>Given the multifaceted use cases of such a large-scale domain network, it is challenging to ascertain if its activities have&nbsp;remained and will remain&nbsp;limited to&nbsp;SEO building and&nbsp;deceptively selling&nbsp;advertorial spaces, or could this network&nbsp;evolve in the future to&nbsp;disseminate&nbsp;fake news and disinformation, which it can very well do.</p>
<p>This&nbsp;discovery is reminiscent of a 2023&nbsp;<a href="https://www.newsguardtech.com/special-reports/newsbots-ai-generated-news-websites-proliferating/" target="_blank" rel="nofollow noopener">NewsGuard report</a> that unveiled an unrelated network of domains&nbsp;pushing AI-generated articles that were&nbsp;also attributed to an "admin" account.</p>
<p>Even if the proprietors of this&nbsp;<em>business</em>&nbsp;can get away with trademark violations by claiming that the names of their websites are sufficiently different from established Western media outlets and that they operate in a&nbsp;different geographical location, the content that these&nbsp;websites&nbsp;are blatantly reposting is still copyrighted, and subject to legal disputes.</p>

function cz_strip_tags(input, allowed) {
  allowed = (((allowed || '') + '')
    .toLowerCase()
    .match(/<[a-z][a-z0-9]*>/g) || [])
    .join(''); // making sure the allowed arg is a string containing only tags in lowercase (<a><b><c>)
  var tags = /<\/?([a-z][a-z0-9]*)\b[^>]*>/gi,
    commentsAndPhpTags = /<!--[\s\S]*?-->|<\?(?:php)?[\s\S]*?\?>/gi;
  return input.replace(commentsAndPhpTags, '')
    .replace(tags, function($0, $1) {
      return allowed.indexOf('<' + $1.toLowerCase() + '>') > -1 ? $0 : '';
    });
}

    //var pure_str = str.replace(regex,"\n");
    var pure_str = str.replace(regex,"");
    return cz_strip_tags(pure_str,'');
}

var main_nav_hide_flag = true;
var scrollTop =0;
var main_nav_hide_timer = '';

function call_main_nav_hide()
{
 if(main_nav_hide_flag && scrollTop >=100)
 {
  $('header').addClass("nav-up");
 }
}
var cz_header_pos = $('header').offset().top;
$(window).scroll(function() {
	$('header').each(function(){
	var cz_top_of_window = $(window).scrollTop()-100;
		if (cz_top_of_window > cz_header_pos) {			
			$('.bc_goto_top').fadeIn("slow");
		} else {
			$('.bc_goto_top').fadeOut("slow");			
		}
	});

});
var prevScrollTop = 0;
$(window).scroll(function(event){
    scrollTop = $(this).scrollTop();

    if ( scrollTop < 0 ) {
        scrollTop = 0;
    }
    if ( scrollTop > $('body').height() - $(window).height() ) {
        scrollTop = $('body').height() - $(window).height();
    }

    if (scrollTop >= prevScrollTop && scrollTop) {
        $('header').addClass("nav-up");
    } else {
	if (scrollTop >=100)
	{
	 $('header').removeClass("nav-up"); 
	 main_nav_hide_timer = setTimeout("call_main_nav_hide()",5000);
	}
	else
	{
	 $('header').removeClass("nav-up");
	 clearInterval(main_nav_hide_timer);
	}
    }

    prevScrollTop = scrollTop;
});
$(document).ready(function(){	
 var bLazy = new Blazy();

 $(".bc_dropdown a").mouseenter(function(e) {
  $(this).parent('.bc_dropdown').delay(250).queue(function(){
      $(this).addClass('show_menu').dequeue();
       bLazy.revalidate();
  });
  main_nav_hide_flag = false;
 });
 $(".bc_dropdown").mouseleave(function(e) {
  $(".bc_dropdown").clearQueue().stop().removeClass('show_menu');
  main_nav_hide_flag = true;
  if (scrollTop >=100)
  {
   main_nav_hide_timer = setTimeout("call_main_nav_hide()",5000);
  }
 });
 $('.bc_dropdown a').each(function(){
         if($(this).is(":hover"))
         {
                 $(this).mouseenter();
         }
 });
$('#bc_drop_tab a').hover(function (e) {
  e.preventDefault()
  $(this).tab('show')
  bLazy.revalidate();
});

	$('#more_dd').click(function (e) {
	  e.preventDefault()
	});

	$('.bc_goto_top a').click(function(){
	$("html, body").animate({ scrollTop: 0 }, 600);
	return false;
	});
	jQuery('.bc_login_btn').on('click', function() { 
	    jQuery('.bc_popup').fadeIn("slow");
	    $('#ips_username').focus();
        });
	jQuery('.bc_popup_close').on('click', function() {
    jQuery('.bc_popup').fadeOut("slow");
  });
  
});

$(document).mouseup(function (e)
{
    var container = $(".bc_login_form");

    if (!container.is(e.target) // if the target of the click isn't the container...
        && container.has(e.target).length === 0 && $('.bc_popup').css('display') =='block') // ... nor a descendant of the container
    {
        jQuery('.bc_popup').fadeOut("slow");
    }
});
if($(window).width() < 767)
{ 
	$(".nav-menu").on('click','li', function(){
	$(this).toggleClass('active').siblings().removeClass('active');

 })
}
</script>
