    <h1 class="text-center">Threat Roundup for February 4 to February 11</h1>

    <div class="text-center m-1">
        <div class="post-author">
            <span>By </span>
                <a href="https://blog.talosintelligence.com/author/william-largent/">William Largent</a>
        </div>

        <br/>
        <time class="post-datetime" datetime="February 11, 2022 14:28">
            Friday, February 11, 2022 14:28

        <div class="m-3">

    <section class="post-content-wrapper mt-5">
        <div class="post-content">
            <p>Today, Talos is publishing a glimpse into the most prevalent threats we've observed between Feb. 4 and Feb. 11. As with previous roundups, this post isn't meant to be an in-depth analysis. Instead, this post will summarize the threats we've observed by highlighting key behavioral characteristics, indicators of compromise, and discussing how our customers are automatically protected from these threats.</p><p>As a reminder, the information provided for the following threats in this post is non-exhaustive and current as of the date of publication. Additionally, please keep in mind that IOC searching is only one part of threat hunting. Spotting a single IOC does not necessarily indicate maliciousness. Detection and coverage for the following threats is subject to updates, pending additional threat or vulnerability analysis. For the most current information, please refer to your Firepower Management Center, <a href="https://www.snort.org">Snort.org</a>, or <a href="https://www.clamav.net">ClamAV.net</a>.</p><p>For each threat described below, this blog post only lists 25 of the associated file hashes and up to 25 IOCs for each category. An accompanying JSON file can be found <a href="https://talosintelligence.com/resources/371">here</a>that includes the complete list of file hashes, as well as all other IOCs from this post. A visual depiction of the MITRE ATT&amp;CK techniques associated with each threat is also shown. In these images, the brightness of the technique indicates how prevalent it is across all threat files where dynamic analysis was conducted. There are five distinct shades that are used, with the darkest indicating that no files exhibited technique behavior and the brightest indicating that technique behavior was observed from 75 percent or more of the files.</p><p>The most prevalent threats highlighted in this roundup are:</p><!--kg-card-begin: html--><table class="threats-table">

<td class="threat-name-col">  Win.Malware.TinyBanker-9938313-1 </td>
<td>    Malware </td>
<td>    TinyBanker, also known as Zusy or Tinba, is a trojan that uses man-in-the-middle attacks to steal banking information. When executed, it injects itself into legitimate Windows processes such as "explorer.exe" and "winver.exe". When the user accesses a banking website, it displays a form to trick the user into submitting personal information. </td>

<td class="threat-name-col">  Win.Packed.Tofsee-9938395-0 </td>
<td>    Packed </td>
<td>    Tofsee is multi-purpose malware that features a number of modules used to carry out various activities such as sending spam messages, conducting click fraud, mining cryptocurrency, and more. Infected systems become part of the Tofsee spam botnet and are used to send large volumes of spam messages in an effort to infect additional systems and increase the size of the botnet under the operator's control. </td>
</tr>

<td class="threat-name-col">  Win.Dropper.Lokibot-9938416-1 </td>
<td>    Dropper </td>
<td>    Lokibot is an information-stealing malware designed to siphon off sensitive information stored on an infected device. It is modular in nature, supporting the ability to steal sensitive information from a number of popular applications. It is commonly pushed via malicious documents delivered via spam emails. </td>
</tr>
<tr>
<td class="threat-name-col">  Win.Virus.Xpiro-9938457-1 </td>
<td>    Virus </td>

</tr>
<tr>
<td class="threat-name-col">  Win.Dropper.DarkComet-9938488-1 </td>
<td>    Dropper </td>
<td>    DarkComet and related variants are a family of remote access trojans designed to provide an attacker with control over an infected system. Capabilities of this malware include the ability to download files from a user's machine, mechanisms for persistence and hiding, and the ability to send back usernames and passwords from the infected system. </td>
</tr>
<tr>
<td class="threat-name-col">  Win.Worm.Gh0stRAT-9938500-1 </td>
<td>    Worm </td>
<td>    Gh0stRAT is a well-known family of remote access trojans that can provide an attacker with complete control over an infected system. Its capabilities include monitoring keystrokes, collecting video footage from the webcam, and uploading/executing follow-on malware. The source code for Gh0stRAT has been publicly available on the internet for years, significantly lowering the barrier for actors to modify and reuse the code in new attacks. </td>
</tr>
<tr>
<td class="threat-name-col">  Win.Malware.Zbot-9938525-0 </td>
<td>    Malware </td>
<td>    Zbot, also known as Zeus, is a trojan that steals information, such as banking credentials, using methods such as key-logging and form-grabbing. </td>
