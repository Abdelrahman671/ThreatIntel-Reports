			<h1 class="entry-title">Ursnif via LOLbins</h1>
		<div class="entry-meta">
			<span class="posted-on"><a href="https://thedfirreport.com/2020/04/24/ursnif-via-lolbins/" rel="bookmark"><time class="entry-date published" datetime="2020-04-24T12:40:20+00:00">April 24, 2020</time></a></span>		</div><!-- .entry-meta -->

		
	</header><!-- .entry-header -->

	<div class="entry-content">
		<p>Ursnif is a variant of the Gozi malware family has recently been responsible for a growing campaign targeting various entities across North America and Europe. The campaign looks to have started around the 6th of April via a number of domains taking up residence at 8.208.90.28.</p>
<p><img fetchpriority="high" decoding="async" class="alignnone size-full wp-image-214" src="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-002.png" alt="" width="329" height="360" srcset="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-002.png 329w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-002-274x300.png 274w" sizes="(max-width: 329px) 100vw, 329px" /></p>
<p>Overall 16 domains have been pointed to the IP since the start of the campaign.</p>
<p><img decoding="async" class="alignnone size-large wp-image-211" src="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-001-1024x586.png" alt="" width="1024" height="586" srcset="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-001-1024x586.png 1024w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-001-300x172.png 300w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-001-768x439.png 768w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-001-1536x878.png 1536w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-001.png 1558w" sizes="(max-width: 1024px) 100vw, 1024px" /></p>
<p>As of 04/22 these actors have moved their campaign to a new IP: 47.241.106.208</p>
<p><img decoding="async" class="alignnone size-large wp-image-316" src="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-010-1024x537.png" alt="" width="1024" height="537" srcset="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-010-1024x537.png 1024w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-010-300x157.png 300w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-010-768x403.png 768w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-010-1536x806.png 1536w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-010.png 1608w" sizes="(max-width: 1024px) 100vw, 1024px" /></p>
<h3>Initial Access:</h3>
<p>The particular point of interest in this campaign is the effectiveness of the TTP&#8217;s at bypassing many security tools. In the delivery stage the campaign uses compromised email accounts to inject into previous conversations by adding a link and imploring the recipient to check the latest update to the ongoing conversation.</p>
<p><img loading="lazy" decoding="async" class="alignnone wp-image-216 size-full" src="https://thedfirreport.com/wp-content/uploads/2020/04/Ursnif-Campaign1.png" alt="" width="1323" height="602" srcset="https://thedfirreport.com/wp-content/uploads/2020/04/Ursnif-Campaign1.png 1323w, https://thedfirreport.com/wp-content/uploads/2020/04/Ursnif-Campaign1-300x137.png 300w, https://thedfirreport.com/wp-content/uploads/2020/04/Ursnif-Campaign1-1024x466.png 1024w, https://thedfirreport.com/wp-content/uploads/2020/04/Ursnif-Campaign1-768x349.png 768w" sizes="auto, (max-width: 1323px) 100vw, 1323px" /></p>
<p>The link provided is to a Google Drive account, a trusted entity to users, and often not capable of being blocked in many enterprises. The Google Drive link downloads a password protected zip file with a javascript (JS) file inside.</p>
<h3>Execution:</h3>
<p>Upon execution, the JS file will be executed by wscript. Wscript then gives way to Regsrv32 which loads a txt file into memory. The txt file however is actually a DLL file that once loaded into memory runs under the regsrv32 process.</p>
<p><img loading="lazy" decoding="async" class="alignnone size-full wp-image-222" src="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-003.png" alt="" width="974" height="735" srcset="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-003.png 974w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-003-300x226.png 300w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-003-768x580.png 768w" sizes="auto, (max-width: 974px) 100vw, 974px" /></p>
<p>The use of these infection methods were able to bypass several security layers including Windows Defender at the time of run but we witnessed it detect the txt DLL and eat the file on disk while missing the running executable running in memory.</p>
<p><img loading="lazy" decoding="async" class="alignnone size-full wp-image-229" src="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-005.png" alt="" width="941" height="646" srcset="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-005.png 941w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-005-300x206.png 300w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-005-768x527.png 768w" sizes="auto, (max-width: 941px) 100vw, 941px" /></p>
<p>While several infections witnessed during the campaign never moved past beaconing to the Ursnif C2 at 8.208.90.28 with the DLL in memory, some samples proceeded further.</p>
<h3>Persistence:</h3>
<p>For those samples the following behavior occurred.</p>
<p><img loading="lazy" decoding="async" class="alignnone wp-image-226 size-full" src="https://thedfirreport.com/wp-content/uploads/2020/04/Urnsnif-Phase-2.png" alt="" width="1421" height="744" srcset="https://thedfirreport.com/wp-content/uploads/2020/04/Urnsnif-Phase-2.png 1421w, https://thedfirreport.com/wp-content/uploads/2020/04/Urnsnif-Phase-2-300x157.png 300w, https://thedfirreport.com/wp-content/uploads/2020/04/Urnsnif-Phase-2-1024x536.png 1024w, https://thedfirreport.com/wp-content/uploads/2020/04/Urnsnif-Phase-2-768x402.png 768w" sizes="auto, (max-width: 1421px) 100vw, 1421px" /></p>
<p>In the registry location seen referenced below, there could be found more modules for the malware to call upon.</p>
<p><img loading="lazy" decoding="async" class="alignnone size-full wp-image-231" src="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-004.png" alt="" width="630" height="315" srcset="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-004.png 630w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-004-300x150.png 300w" sizes="auto, (max-width: 630px) 100vw, 630px" /></p>
<h3>Command and Control:</h3>
<p><img loading="lazy" decoding="async" class="alignnone size-large wp-image-237" src="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-006-1024x204.png" alt="" width="1024" height="204" srcset="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-006-1024x204.png 1024w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-006-300x60.png 300w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-006-768x153.png 768w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-006-1536x305.png 1536w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-006.png 1796w" sizes="auto, (max-width: 1024px) 100vw, 1024px" /></p>
<p>Initial C2 picked up on the following alerts:</p>
<pre>ETPRO_TROJAN_Ursnif_Variant_CnC_Beacon_12_M2 8.208.90.28
ETPRO_TROJAN_Ursnif_Variant_CnC_Beacon_12_M1 8.208.90.28</pre>
<p>With the TorClient Registry Binary being confirm for its namesake after some time:</p>
<pre class="kbnTableCellFilter__hover">ET_P2P_Tor_Get_Server_Request</pre>
<div data-cell-content="">After around a 24 hour time passage, Ursnif received new activity with alerts triggering for a VNC module and a new C2 IP.</div>
<pre>ETPRO_TROJAN_Possible_Ursnif_VNC_Module_CnC_Beacon 162.244.35.233</pre>
<div data-cell-content="">This then followed with a flurry of new malware dropped to the system. These turned out to include both Cobalt Strike beacons as well as TVRat (Team Viewer RAT).</div>
<div data-cell-content="">Cobalt Strike was delivered in the form of 3 dll&#8217;s loaded into memory again with the help of run32dll.</div>
<div data-cell-content=""><img loading="lazy" decoding="async" class="alignnone wp-image-240 size-full" src="https://thedfirreport.com/wp-content/uploads/2020/04/Cobalt-Stike-23.81.246.22.png" alt="" width="1631" height="606" srcset="https://thedfirreport.com/wp-content/uploads/2020/04/Cobalt-Stike-23.81.246.22.png 1631w, https://thedfirreport.com/wp-content/uploads/2020/04/Cobalt-Stike-23.81.246.22-300x111.png 300w, https://thedfirreport.com/wp-content/uploads/2020/04/Cobalt-Stike-23.81.246.22-1024x380.png 1024w, https://thedfirreport.com/wp-content/uploads/2020/04/Cobalt-Stike-23.81.246.22-768x285.png 768w, https://thedfirreport.com/wp-content/uploads/2020/04/Cobalt-Stike-23.81.246.22-1536x571.png 1536w" sizes="auto, (max-width: 1631px) 100vw, 1631px" /></div>
<div data-cell-content="">Meanwhile TVrat uses the &#8220;legitimate&#8221; access tool Teamviewer to provide remote access to the attacker.</div>
<pre>svcc.exe 99e0fbb8b4d6bbd5fe4eec1530aa51a818d06e245efb2c2fb41199a390a73db8</pre>
<div data-cell-content=""><img loading="lazy" decoding="async" class="alignnone size-full wp-image-243" src="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-007.png" alt="" width="351" height="314" srcset="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-007.png 351w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-007-300x268.png 300w" sizes="auto, (max-width: 351px) 100vw, 351px" /></div>
<pre>1.exe 497129b7b2a940a812b9f3cf3d1a149d903a4179fc75adaf085e4edba533a7c9</pre>
<div data-cell-content="">This exe reaches out to many of the various teamviewer infrastructure:</div>
<div data-cell-content=""><img loading="lazy" decoding="async" class="alignnone size-full wp-image-245" src="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-008.png" alt="" width="1015" height="356" srcset="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-008.png 1015w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-008-300x105.png 300w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-008-768x269.png 768w" sizes="auto, (max-width: 1015px) 100vw, 1015px" /></div>
<div data-cell-content="">At this point the Cobalt Strike and TVrat C2 overtook all previous communications.</div>
<div data-cell-content=""><img loading="lazy" decoding="async" class="alignnone wp-image-247 size-full" src="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-009.png" alt="" width="1810" height="356" srcset="https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-009.png 1810w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-009-300x59.png 300w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-009-1024x201.png 1024w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-009-768x151.png 768w, https://thedfirreport.com/wp-content/uploads/2020/04/ursnif-009-1536x302.png 1536w" sizes="auto, (max-width: 1810px) 100vw, 1810px" /></div>

<div data-cell-content="">
<pre class="kbnTableCellFilter__hover">ETPRO_TROJAN_Cobalt_Strike_Beacon_Observed 23.81.246.22
ETPRO_TROJAN_Cobalt_Strike_Beacon_Observed 93.190.138.35
ET_TROJAN_Win32.Spy/TVRat_Checkin 89.39.107.106</pre>
<div data-cell-content="">
<h3>Action on Objectives:</h3>
<div data-cell-content="">This continued for some time but we did not witness final actions on objectives from the actors.</div>
<div data-cell-content=""></div>
</div>
<h3>Conclusion:</h3>
<div data-cell-content="">Based on the actors capability to bypass security controls and the pivot to new IP infrastructure we expect this campaign to continue for some time. We recommend paying close attention to AV alerts tied to files that you wouldn&#8217;t normally expect AV alerts (like text files). And make sure your network signatures are up to date and monitored as these threats tend to use default or known configurations that are quite noisy if someone is listening.</div>
