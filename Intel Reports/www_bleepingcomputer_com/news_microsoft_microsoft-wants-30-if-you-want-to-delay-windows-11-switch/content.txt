	 <div class="article_section">
          <h1>Microsoft wants $30 if you want to delay Windows 11 switch</h1>
          <div class="cz-news-story-title-section">
            <div class="cz-news-title-left-area">
                            By <h6><a rel='author' href='https://www.bleepingcomputer.com/author/sergiu-gatlan/' class='author'><span itemprop="author" itemscope itemtype="https://schema.org/Person"><span itemprop="name">Sergiu Gatlan</span></span></a></h6>
            </div>
	    <meta itemprop="articleSection" content="Microsoft"><meta itemprop="articleSection" content="Security">
            <div class="cz-news-title-right-area">
              <ul>
                <li class="cz-news-date">October 31, 2024</li>
                <li class="cz-news-time">02:07 PM</li>
                
                <li class="cz-news-comment"><a href="#comments">1</a></li>
              </ul>
            </div>
          </div>
	  <div class="articleBody">
	   
<p style="text-align:center"><img alt="Windows" height="802" src="https://www.bleepstatic.com/content/hl-images/2024/10/31/Windows-10.jpg" width="1600"></p>

<p>&#8203;Microsoft announced today that Windows 10 home users can delay the switch to Windows 11 for one more year if they're willing to pay $30 for Extended Security Updates (ESU).</p>

<p><span style="box-sizing:border-box; margin:0px; padding:0px">The company first announced that Windows 10 home users can buy Extended Security Updates&nbsp;<a href="https://www.bleepingcomputer.com/news/microsoft/microsoft-to-let-windows-10-home-users-buy-extended-security-updates/" target="_blank">in December 2023,</a>&nbsp;but this is the first time that Microsoft put a price on the plan.</span></p>

<p>Windows 10 will reach the end of support on October 14, 2025. After this date, Windows 10 systems will no longer receive bug fixes or security updates patching newly discovered vulnerabilities and defending them from the latest security threats.</p>

<p>However, Long-Term Servicing Branch (LTSB) and Long-Term Servicing Channel (LTSC) releases catering to specialized devices, such as those controlling industrial and medical equipment, will receive updates beyond October 2025.</p>

<p>For instance, Windows 10 IoT Enterprise LTSC 2021 <a href="https://learn.microsoft.com/en-us/lifecycle/products/windows-10-iot-enterprise-ltsc-2021" target="_blank" rel="nofollow noopener">reaches</a> its extended end date on January 13, 2032, while Windows 10 2016 LTSB <a href="https://learn.microsoft.com/en-us/lifecycle/products/windows-10-iot-enterprise-ltsb-2015" target="_blank" rel="nofollow noopener">will reach</a> its extended end date on October 13, 2026.</p>

<p>"The ESU program for consumers will be a one-year option available for $30. Program enrollment will be available closer to the end of support in 2025," said Yusuf Mehdi, Microsoft Executive Vice President and Consumer Chief Marketing Officer.</p>

<p>Statcounter Global Stats data shows over 62% of all Windows systems <a href="https://gs.statcounter.com/os-version-market-share/windows/desktop/worldwide" target="_blank" rel="nofollow noopener">still run Windows 10</a>, while only 33% run Windows 11, three years after its&nbsp;<a href="https://www.bleepingcomputer.com/news/microsoft/windows-11-is-released-what-you-need-to-know-and-new-features/" target="_blank">October 2021 launch</a>.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Windows market share worldwide" height="305" src="https://www.bleepstatic.com/images/news/u/1109292/2024/Desktop%20Windows%20market%20share.png" width="761"><figcaption><em>Windows market share worldwide (Statcounter)</em></figcaption></figure></div>

<p>In an update to a tech community blog post, Microsoft also revealed that it will start offering Extended Security Updates to enterprise customers on November 1, while the first ESU will be delivered in November 2025.</p>

<p>"We'll share more details on availability of the cloud-based Extended Security Update offer on our price list in the future," Redmond <a href="https://techcommunity.microsoft.com/t5/windows-it-pro-blog/when-to-use-windows-10-extended-security-updates/ba-p/4102628" target="_blank" rel="nofollow noopener">said</a>.</p>

<p>Microsoft advises home users who want to switch before Windows 10 reaches the end of support to check if their PC can be upgraded to Windows 11 and buy a new Windows 11 PC if not.</p>

<p>"With the Windows 10 End of Support moment, now is the time to move to Windows 11 with confidence. We understand change is never easy, but we are committed to making this transition as smooth as possible," Mehdi added.</p>

<p>In early June, Microsoft <a href="https://www.bleepingcomputer.com/news/microsoft/microsoft-announces-first-windows-10-beta-build-since-2021/" target="_blank">reopened the Windows 10 beta channel</a>&nbsp;and announced the first Windows 10 Beta build since 2021, almost three years after the last such build was <a href="https://blogs.windows.com/windows-insider/2021/06/17/releasing-windows-10-build-19042-1081-20h2-to-release-preview-channel/" rel="nofollow noopener" target="_blank">pushed to&nbsp;Insiders</a> in the Beta and Release Preview channels.</p>

function cz_strip_tags(input, allowed) {
  allowed = (((allowed || '') + '')
    .toLowerCase()
    .match(/<[a-z][a-z0-9]*>/g) || [])
    .join(''); // making sure the allowed arg is a string containing only tags in lowercase (<a><b><c>)
  var tags = /<\/?([a-z][a-z0-9]*)\b[^>]*>/gi,
    commentsAndPhpTags = /<!--[\s\S]*?-->|<\?(?:php)?[\s\S]*?\?>/gi;
  return input.replace(commentsAndPhpTags, '')
    .replace(tags, function($0, $1) {
      return allowed.indexOf('<' + $1.toLowerCase() + '>') > -1 ? $0 : '';
    });
}
function cz_br2nl(str) {
	var regex = /<br\s*[\/]?>/gi;
    //var pure_str = str.replace(regex,"\n");
    var pure_str = str.replace(regex,"");
    return cz_strip_tags(pure_str,'');
}
