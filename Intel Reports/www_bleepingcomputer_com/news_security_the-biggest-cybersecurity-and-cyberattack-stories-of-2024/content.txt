          <h1>The biggest cybersecurity and cyberattack stories of 2024</h1>
          <div class="cz-news-story-title-section">
            <div class="cz-news-title-left-area">
                            By <h6><a rel="author" href="https://www.bleepingcomputer.com/author/lawrence-abrams/" class="author"><span itemprop="author" itemscope itemtype="https://schema.org/Person"><span itemprop="name">Lawrence Abrams</span></span></a></h6>
            </div>

	  <div class="articleBody">
	   
<p style="text-align:center"><img alt="2024" height="900" src="https://www.bleepstatic.com/content/hl-images/2024/12/29/cyberworld-2024.jpg" width="1600"></p>

<p>2024 was a big year for cybersecurity, with significant cyberattacks, data breaches, new threat groups emerging, and, of course, zero-day vulnerabilities.</p>

<p>Some stories, though, were more impactful or popular with our 31&nbsp;million readers than others.</p>

<p>Below are fourteen of what BleepingComputer believes are the most impactful cybersecurity stories of 2024, with a summary of each. These stories are in no particular order.</p>

<h3 class="top_story" style="color:white!important"><span class="top_num">14. <a href="https://www.bleepingcomputer.com/news/security/internet-archive-hacked-data-breach-impacts-31-million-users/" target="_blank" rel="nofollow noopener">Internet Archive hacked</a></span></h3>

<p>On October 9, the Internet&nbsp;Archive was&nbsp;<a href="https://www.bleepingcomputer.com/news/security/internet-archive-hacked-data-breach-impacts-31-million-users/" target="_blank" rel="nofollow noopener">hit by two different attacks at once</a>&mdash;a data breach where the site's user data for 33 million users was stolen and a DDoS attack by an alleged&nbsp;pro-Palestinian group named SN_BlackMeta.</p>

<p>While both attacks occurred over the same period, they were conducted by different threat actors.&nbsp;</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="JavaScript alert on Internet Archive warning about the breach" height="300" src="https://www.bleepstatic.com/images/news/security/d/data-breaches/w/wayback-machine/js-alert.jpg" width="665"><figcaption><strong>JavaScript alert on Internet Archive warning about the breach</strong><br><em>Source: BleepingComputer</em></figcaption></figure></div>

<p>The threat actors who breached Internet Archive told BleepingComputer that they could do so through an exposed GitLab configuration file containing an authentication token, allowing them to download the Internet Archive source code.</p>

<p>This source code contained additional credentials and authentication tokens, including the credentials to Internet Archive's database management system. This allowed the threat actor to download the organization's user database, further source code, and modify the site.</p>

<h3 class="top_story" style="color:white!important"><span class="top_num">13.</span>&nbsp;<a href="https://www.bleepingcomputer.com/news/security/crowdstrike-update-crashes-windows-systems-causes-outages-worldwide/" target="_blank" rel="nofollow noopener">Bad CrowdStrike updates crashed 8.5 million Wndows devices</a></h3>

<p>On July 19th, 2024, a faulty CrowdStrike Falcon update was pushed out to Windows PCs in the early morning, causing the cybersecurity software's kernel driver to crash the operating system.&nbsp;</p>

<p>This bug <a href="https://www.bleepingcomputer.com/news/security/crowdstrike-update-crashes-windows-systems-causes-outages-worldwide/" target="_blank" rel="nofollow noopener">caused significant global disruptions</a>, impacting approximately 8.5 million Windows systems, who now found that their devices had crashed with no easy way back into the operating system to remove the faulty&nbsp;update other than booting into safe mode.</p>

<p>The bug <a href="https://www.bleepingcomputer.com/news/security/crowdstrike-content-validator-bug-let-faulty-update-pass-checks/" target="_blank" rel="nofollow noopener">stemmed from a flaw in CrowdStrike's content validation process</a>, which failed to detect a defective update. This faulty update triggered a series of system crashes, including endless reboot loops that affected both Windows devices and Windows 365 Cloud PCs.</p>

<p>As CrowdStrike is used by many organizations, it quickly caused widespread disruption, impacting financial firms, airlines, and hospitals worldwide who suddenly found their Windows devices and applications were unavailable.</p>

<p>Microsoft <a href="https://www.bleepingcomputer.com/news/microsoft/microsoft-releases-windows-repair-tool-to-remove-crowdstrike-driver/" target="_blank" rel="nofollow noopener">released a Windows repair tool</a> to help remove the problematic CrowdStrike driver and restore affected systems. Despite this tool, many organizations faced a lengthy recovery process as each device would need to be manually fixed.</p>

<p>Things got worse when the threat actors started getting into the game.</p>

<p>Cybercriminals distributed <a href="https://www.bleepingcomputer.com/news/security/fake-crowdstrike-fixes-target-companies-with-malware-data-wipers/" target="_blank" rel="nofollow noopener">fake CrowdStrike repair tools and manuals that pushed malware</a>, including the <a href="https://www.bleepingcomputer.com/news/security/fake-crowdstrike-repair-manual-pushes-new-daolpu-infostealer-malware/" target="_blank" rel="nofollow noopener">new Daolpu infostealer</a>. These phishing campaigns targeted orgs attempting to recover from the outage, further delaying outages.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Fake CrowdStrike fix pushing info-stealing malware" height="600" src="https://www.bleepstatic.com/images/news/malware/d/Daolpu/fake-crowdstrike-recovery-manual.jpg" width="788"><figcaption><strong>Fake CrowdStrike fix pushing info-stealing malware</strong><br><em>Source: BleepingComputer</em></figcaption></figure></div>

<p>Investors soon <a href="https://www.bleepingcomputer.com/news/legal/crowdstrike-sued-by-investors-over-massive-global-it-outage/" target="_blank" rel="nofollow noopener">filed a lawsuit against CrowdStrike</a>, accusing it&nbsp;of negligence in its quality assurance processes and failing to prevent the release of the defective update.</p>

<p>Microsoft also announced that they would be looking into <a href="https://www.microsoft.com/en-us/security/blog/2024/07/27/windows-security-best-practices-for-integrating-and-managing-security-tools/" target="_blank" rel="nofollow noopener">changing their kernel driver handling policies</a> in response to the incident and encouraged antivirus vendors to <a href="https://www.microsoft.com/en-us/security/blog/2024/07/27/windows-security-best-practices-for-integrating-and-managing-security-tools/" target="_blank" rel="nofollow noopener">limit their use of Kernel drivers</a> to prevent these types of crashes.</p>

<h3 class="top_story" style="color:white!important"><span class="top_num">12.</span>&nbsp;<a href="https://www.bleepingcomputer.com/news/security/biden-bans-kaspersky-antivirus-software-in-us-over-security-concerns/" target="_blank" rel="nofollow noopener">Kaspersky banned in the US&mdash;software automatically replaced with UltraAV</a></h3>

<p>In June, the Biden administration <a href="https://www.bleepingcomputer.com/news/security/biden-bans-kaspersky-antivirus-software-in-us-over-security-concerns/" target="_blank" rel="nofollow noopener">announced an upcoming ban of Kaspersky antivirus software</a>, giving customers until September 29, 2024, to find alternative security software.</p>

<p>The ban not only involved the sale of Kaspersky software in the US, but also prevented the company from delivering antivirus and security updates to customers.</p>

<p>A month later, <a href="https://www.bleepingcomputer.com/news/security/kaspersky-is-shutting-down-its-business-in-the-united-states/" target="_blank" rel="nofollow noopener">Kaspersky began shutting down its operations in the US</a>, telling BleepingComputer that the Biden administration's decisions have made operations "no longer viable."</p>

<p>Kaspersky decided to sell its US customer base to Point Wild (previously Pango)&nbsp;and emailed customers in early September that they would receive a free upgrade to the UltraAV software.</p>

<p>On September 19, Kaspersky users suddenly <a href="https://www.bleepingcomputer.com/news/security/kaspersky-deletes-itself-installs-ultraav-antivirus-without-warning/" target="_blank" rel="nofollow noopener">found their Kaspersky products removed and UltraAV force-installed</a> on their computers whether they wanted it or not.</p>

<p>While in-app notices and emails were sent about the migration, many users either missed them or were unaware, making many customers furious that the software was installed on their devices without permission.</p>

<h3 class="top_story" style="color:white!important"><span class="top_num">11.</span>&nbsp;<a href="https://www.bleepingcomputer.com/news/security/russian-hackers-stole-microsoft-corporate-emails-in-month-long-breach/" target="_blank" rel="nofollow noopener">Russian state-sponsored hackers breached Microsoft's corporate email</a></h3>

<p>In January, <a href="https://www.bleepingcomputer.com/news/security/russian-hackers-stole-microsoft-corporate-emails-in-month-long-breach/" target="_blank" rel="nofollow noopener">Microsoft disclosed</a> that Russian state-sponsored threat actors breached their corporate email servers in November 2023 to steal email from their leadership, cybersecurity, and legal teams.</p>

<p>Some of these emails contained information about the hacking group itself, allowing the threat actors to learn what Microsoft knew about them.</p>

<p>The hacking group, known as&nbsp;Midnight Blizzard (aka Nobelium, or APT29) is believed to be a state-backed cyberespionage group tied to the Russian Foreign Intelligence Service (SVR).</p>

<p>Microsoft <a href="https://www.bleepingcomputer.com/news/microsoft/microsoft-says-russian-hackers-breached-its-systems-accessed-source-code/" target="_blank" rel="nofollow noopener">later disclosed</a> that the threat actors conducted a password-spray attack that allowed access to a legacy non-production test tenant account.</p>

<p>This test tenant account also had access to an OAuth application with elevated privileges in Microsoft's corporate environment, allowing the hackers to steal data from corporate mailboxes.</p>

<p>The hackers breached Microsoft again in March 2024 using information found in the stolen emails, allowing them to steal&nbsp;source code repositories.</p>

<p>It kept getting worse, with <a href="https://www.bleepingcomputer.com/news/security/cisa-orders-agencies-impacted-by-microsoft-hack-to-mitigate-risks/" target="_blank" rel="nofollow noopener">CISA confirming in April</a> that emails between US federal agencies and Microsoft were also stolen in the attack. These emails contained information that let the hackers gain access to some customer's systems.</p>

<h3 class="top_story" style="color:white!important"><span class="top_num">10.</span>&nbsp;&nbsp;<a href="https://www.bleepingcomputer.com/news/security/hackers-leak-27-billion-data-records-with-social-security-numbers/" target="_blank" rel="nofollow noopener">National Public data breach exposed your Social Security Number</a></h3>

<p>In August, almost 2.7&nbsp;billion records of&nbsp;personal information for people in the United States were leaked on a hacking forum, exposing names, social security numbers, all known physical addresses,&nbsp;and possible aliases.</p>

<p>The data was stolen from National Public Data, a company that collects and sells access to personal data for use in background checks, to obtain criminal records, and for private investigators.</p>

<p>Have I Been Pwned's Troy Hunt <a href="https://www.troyhunt.com/inside-the-3-billion-people-national-public-data-breach/" target="_blank" rel="nofollow noopener">analyzed the breach</a> and determined it contained&nbsp;134 million&nbsp;unique email addresses, making this a monstrous data breach.</p>

<p>The threat actors behind the breach attempted to sell it for $3.5 million, but it was eventually <a href="https://www.bleepingcomputer.com/news/security/hackers-leak-27-billion-data-records-with-social-security-numbers/" target="_blank" rel="nofollow noopener">leaked for free</a> on a hacking forum.</p>

<h3 class="top_story" style="color:white!important"><span class="top_num">9.</span>&nbsp;Attacks on edge networking devices run rampant</h3>

<p>This year, we continued to&nbsp;see attacks targeting edge networking devices from various manufacturers, including Fortinet, TP-Link, Ivanti, and Cisco.&nbsp;</p>

<p>These types of devices are valuable targets as they are meant to be exposed to the Internet, and once breached, allow threat actors to pivot into the internal network.</p>

<p>There are too many stories to summarize, so here are a list of the interesting ones:</p>

<ul style="list-style-type:square"><li><a href="https://www.bleepingcomputer.com/news/security/chinese-hackers-breached-20-000-fortigate-systems-worldwide/" target="_blank" rel="nofollow noopener">Chinese hackers breached 20,000 FortiGate systems worldwide</a></li>
	<li><a href="https://www.bleepingcomputer.com/news/security/cisa-cautions-against-using-hacked-ivanti-vpn-gateways-even-after-factory-resets/" target="_blank" rel="nofollow noopener">CISA cautions against using hacked Ivanti VPN gateways even after factory resets</a></li>
	<li><a href="https://www.bleepingcomputer.com/news/security/us-sanctions-chinese-firm-for-hacking-firewalls-in-ragnarok-ransomware-attacks/" target="_blank" rel="nofollow noopener">The Pacific Rim attacks:&nbsp;US sanctions Chinese firm for hacking firewalls in ransomware attacks</a></li>
	<li><a href="https://www.bleepingcomputer.com/news/security/microsoft-chinese-hackers-use-quad7-botnet-to-steal-credentials/" target="_blank" rel="nofollow noopener">Chinese hackers use Quad7 botnet to steal credentials</a></li>
	<li><a href="https://www.bleepingcomputer.com/news/security/cisco-warns-of-nx-os-zero-day-exploited-to-deploy-custom-malware/" target="_blank" rel="nofollow noopener">Cisco warns of NX-OS zero-day exploitation to deploy custom malware</a></li>
</ul><p>It has gotten so bad that the US is considering <a href="https://www.bleepingcomputer.com/news/security/us-considers-banning-tp-link-routers-over-cybersecurity-risks/" target="_blank" rel="nofollow noopener">banning China-made TP-Link routers</a> over cybersecurity concerns.</p>

<h3 class="top_story" style="color:white!important"><span class="top_num">8.</span>&nbsp;<a href="https://www.bleepingcomputer.com/news/security/cdk-global-cyberattack-impacts-thousands-of-us-car-dealerships/" target="_blank" rel="nofollow noopener">CDK Global ransomware attacks takes down the car dealership industry</a></h3>

<p>Car dealership software-as-a-service provider CDK Global <a href="https://www.bleepingcomputer.com/news/security/cdk-global-outage-caused-by-blacksuit-ransomware-attack/" target="_blank" rel="nofollow noopener">suffered a Black Suit ransomware attack</a>, causing the company to shut down its systems and leaving clients unable to operate their business normally.</p>

<p>CDK Global provides clients in the auto industry with a SaaS platform that handles all aspects of a car dealership's operation, including CRM, financing, payroll, support and service, inventory, and back-office operations.</p>

<p>As many of the car dealerships in the US utilize the platform, the outage led to widespread disruption, preventing dealers from tracking and ordering car parts, conducting new sales, and offering financing.</p>

<h3 class="top_story" style="color:white!important"><span class="top_num">7.&nbsp;</span><a href="https://www.bleepingcomputer.com/news/security/mgm-resorts-shuts-down-it-systems-after-cyberattack/" target="_blank" rel="nofollow noopener">The SnowFlake data theft attacks</a></h3>

<p>In May, threat actors began selling data that they claimed was stolen from customers of the Snowflake cloud data platform.</p>

<p>After the attacks were investigated, it was determined that the threat actors didn't breach Snowflake but rather used compromised credentials to log in to customer's SnowFlake accounts.</p>

<p>These credentials are believed to have been stolen through information-stealing malware.</p>

<p>Once they logged into the account, they were able to export the databases and use them to extort companies into paying a ransom for the data not to be publicly released.</p>

<p>AT&amp;T&nbsp;<a href="https://www.bleepingcomputer.com/news/security/massive-atandt-data-breach-exposes-call-logs-of-109-million-customers/" target="_blank" rel="nofollow noopener">disclosed in July</a> that call logs of 109 million customers were exposed during the incident and that the data was accessed from an online database on the company's Snowflake account.</p>

<p>TicketMaster was also impacted, with the threat actors claiming to <a href="https://www.bleepingcomputer.com/news/security/data-of-560-million-ticketmaster-customers-for-sale-after-alleged-breach/" target="_blank" rel="nofollow noopener">steal the data of 560 million customers</a>.</p>

<p style="text-align:center"><img alt="Claiming to sell Ticketmaster data" height="600" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" width="940" data-src="https://www.bleepstatic.com/images/news/u/1109292/2024/Allegedly_stolen_Ticketmaster_data_for_sale.jpg" class="b-lazy"></p>

<p>Data breaches linked to these attacks, which started in April 2024, have affected hundreds of millions of individuals using the services of&nbsp;<a href="https://www.bleepingcomputer.com/news/security/massive-atandt-data-breach-exposes-call-logs-of-109-million-customers/" rel="nofollow noopener" target="_blank">AT&amp;T</a>,&nbsp;<a href="https://www.bleepingcomputer.com/news/security/data-of-560-million-ticketmaster-customers-for-sale-after-alleged-breach/" rel="nofollow noopener" target="_blank">Ticketmaster</a>,&nbsp;<a href="https://www.bleepingcomputer.com/news/security/shinyhunters-claims-santander-breach-selling-data-for-30m-customers/" rel="nofollow noopener" target="_blank">Santander</a>,&nbsp;<a href="https://www.bleepingcomputer.com/news/security/pure-storage-confirms-data-breach-after-snowflake-account-hack/" rel="nofollow noopener" target="_blank">Pure Storage</a>,&nbsp;<a href="https://www.bleepingcomputer.com/news/security/advance-auto-parts-confirms-data-breach-exposed-employee-information/" rel="nofollow noopener" target="_blank">Advance Auto Parts</a>,&nbsp;<a href="https://www.bleepingcomputer.com/news/security/los-angeles-unified-confirms-student-data-stolen-in-snowflake-account-hack/" rel="nofollow noopener" target="_blank">Los Angeles Unified</a>,&nbsp;<a href="https://techcrunch.com/2024/06/07/snowflake-ticketmaster-lendingtree-customer-data-breach/" rel="nofollow noopener" target="_blank">QuoteWizard/LendingTree</a>, and&nbsp;<a href="https://www.bleepingcomputer.com/news/security/neiman-marcus-confirms-data-breach-after-snowflake-account-hack/" rel="nofollow noopener" target="_blank">Neiman Marcus</a>.</p>

<p>In November, the US Department of Justice unsealed an indictment against two people,&nbsp;Connor Riley Moucka and John Erin Binns, who are <a href="https://www.bleepingcomputer.com/news/security/us-indicts-snowflake-hackers-who-extorted-25-million-from-3-victims/" target="_blank" rel="nofollow noopener">accused of being behind the attacks</a>.</p>

<p>The threat actors allegedly extorted $2.5 million as part of these attacks, with <a href="https://www.wired.com/story/atandt-paid-hacker-300000-to-delete-stolen-call-records/" target="_blank" rel="nofollow noopener">Wired reporting</a> that AT&amp;T paid&nbsp;$370,000 for the hackers to delete stolen call records.</p>

<h3 class="top_story" style="color:white!important"><span class="top_num">6.</span>&nbsp;&nbsp;The North Korean IT Worker scheme</h3>

<p>This year, we saw an uptick in North Korean IT workers trying to get jobs in the US and other countries to perform cyberespionage and generate revenue for their country's operations.</p>

<p>In May, the <a href="https://www.bleepingcomputer.com/news/security/five-arizona-ukraine-charged-for-cyber-schemes-infiltrating-over-300-companies-to-benefit-north-koreas-weapons-program/" target="_blank" rel="nofollow noopener">Department of Justice charged five individuals</a>, a US Citizen woman, a Ukrainian man, and three foreign nationals, for their involvement in helping North Korean IT works infiltrate US job markets to generate&nbsp;revenue for North Korea's nuclear weapons program.</p>

<p>In July, email security firm KnowBe4 <a href="https://www.bleepingcomputer.com/news/security/knowbe4-mistakenly-hires-north-korean-hacker-faces-infostealer-attack/" target="_blank" rel="nofollow noopener">mistakenly hired a North Korean hacker</a> as their Principal Software Engineer, who attempted to install information-stealing malware on the network.</p>

<p>In August, the&nbsp;<a href="https://www.bleepingcomputer.com/news/security/us-dismantles-laptop-farm-used-by-undercover-north-korean-it-workers/" target="_blank" rel="nofollow noopener">Justice Department arrested a Nashville man</a> charged with helping North Korean IT workers obtain remote work at companies across the United States and operating a laptop farm they used to pose as U.S.-based individuals.</p>

<p>Both <a href="https://cloud.google.com/blog/topics/threat-intelligence/mitigating-dprk-it-worker-threat" target="_blank" rel="nofollow noopener">Mandiant</a> and <a href="https://www.bleepingcomputer.com/news/security/undercover-north-korean-it-workers-now-steal-data-extort-employers/" target="_blank" rel="nofollow noopener">SecureWorks</a> later released reports on the North Korean IT Worker threat, sharing their tactics and how companies can protect themselves.</p>

<h3 class="top_story" style="color:white!important">5.&nbsp;<a href="https://www.bleepingcomputer.com/news/security/unitedhealth-confirms-optum-hack-behind-us-healthcare-billing-outage/" target="_blank" rel="nofollow noopener">The UnitedHealth Change HealthCare ransomware attack</a></h3>

<p>In February, UnitedHealth subsidiary Change Healthcare <a href="https://www.bleepingcomputer.com/news/security/unitedhealth-confirms-optum-hack-behind-us-healthcare-billing-outage/" target="_blank" rel="nofollow noopener">suffered a massive ransomware attack</a> that caused massive disruption to the US healthcare industry.</p>

<p>The outages prevented doctors and&nbsp;pharmacies from filing claims and&nbsp;prevented pharmacies from accepting discount prescription cards, causing patients to pay full price for medications.</p>

<p>The attack was ultimately <a href="https://www.bleepingcomputer.com/news/security/unitedhealth-subsidiary-optum-hack-linked-to-blackcat-ransomware/" target="_blank" rel="nofollow noopener">linked to the BlackCat ransomware gang</a>, aka ALPHV, who used <a href="https://www.bleepingcomputer.com/news/security/change-healthcare-hacked-using-stolen-citrix-account-with-no-mfa/" target="_blank" rel="nofollow noopener">stolen credentials</a> to breach the company's Citrix remote access service, which did not have multi-factor authentication enabled.</p>

<p>During the attack, the threat actors&nbsp;<a href="https://www.bleepingcomputer.com/news/security/ransomware-gang-claims-they-stole-6tb-of-change-healthcare-data/" target="_blank" rel="nofollow noopener">stole 6 TB of data</a> and ultimately encrypted computers on the network, causing the company to shut down IT systems to prevent the spread of the attack.</p>

<p>The UnitedHealth Group admitted to paying a ransom demand to receive a decryptor and for the threat actors to delete the stolen data. The ransom payment was allegedly $22 million, according to the BlackCat ransomware affiliate who conducted the attack.</p>

<p>The BlackCat ransomware operation was under immense pressure from law enforcement after the Change Healthcare attacks, <a href="https://www.bleepingcomputer.com/news/security/blackcat-ransomware-turns-off-servers-amid-claim-they-stole-22-million-ransom/" target="_blank" rel="nofollow noopener">causing them to shut down</a>.</p>

<p>After UnitedHealth paid an alleged $20 million ransom, the ransomware operation performed an exit scam, stealing all of the money and not sharing any with the affiliate who conducted the attack.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Affiliate stating BlackCat performed an exit scam" height="600" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" width="752" data-src="https://www.bleepstatic.com/images/news/u/1100723/2024/ALPHV_Scam_Affil.png" class="b-lazy"><figcaption>Affiliate stating BlackCat performed an exit scam</figcaption></figure></div>

<p>Unfortunately, the affiliate claimed to still have Change Healthcare's data, which they used to extort the healthcare company again, <a href="https://www.bleepingcomputer.com/news/security/ransomware-gang-starts-leaking-alleged-stolen-change-healthcare-data/" target="_blank" rel="nofollow noopener">this time using RansomHub's extortion site</a>.</p>

<p>Ultimately, the data disappeared from the extortion, likely indicating that another ransom was paid.</p>

<p>In October, UnitedHealth confirmed that&nbsp;<a href="https://www.bleepingcomputer.com/news/security/unitedhealth-says-data-of-100-million-stolen-in-change-healthcare-breach/" target="_blank" rel="nofollow noopener">over 100 million people had their personal and healthcare data stolen</a>, marking this as the largest healthcare data breach in recent years.</p>

<h3 class="top_story" style="color:white!important"><span class="top_num">4.</span>&nbsp;<a href="https://www.bleepingcomputer.com/news/security/lockbit-ransomware-disrupted-by-global-police-operation/" target="_blank" rel="nofollow noopener">LockBit disrupted</a></h3>

<p>On February 19, authorities took down LockBit's infrastructure, which included 34 servers hosting the data leak website and its mirrors, data stolen from the victims, cryptocurrency addresses, decryption keys, and the affiliate panel.</p>

<p>This disruption was part of an international law enforcement operation called <a href="https://www.bleepingcomputer.com/news/security/lockbit-ransomware-disrupted-by-global-police-operation/" target="_blank" rel="nofollow noopener">Operation Cronos</a>.</p>

<div style="text-align:center">
<figure class="image" style="display:inline-block"><img alt="Law enforcement seizure message on LockBit's servers" height="600" src="data:image/gif;base64,R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAICTAEAOw==" width="886" data-src="https://www.bleepstatic.com/images/news/u/1109292/2024/LockBit_seizure_banner_site.jpg" class="b-lazy"><figcaption><strong>Law enforcement seizure message on LockBit's servers</strong><br><em>Source: BleepingComputer</em></figcaption></figure></div>

<p>Five days later, <a href="https://www.bleepingcomputer.com/news/security/lockbit-ransomware-returns-restores-servers-after-police-disruption/" target="_blank" rel="nofollow noopener">LockBit relaunched with new infrastructure</a> and threatened to focus more of its attacks on the government sector.</p>

<p>However, the ransomware gang was never able to return to its previous prominence, with its affiliates moving to other ransomware operations.</p>

<p>Over the past year, law enforcement has continued to target LockBit, identifying and <a href="https://www.bleepingcomputer.com/news/security/us-charges-russian-israeli-as-suspected-lockbit-ransomware-coder/" target="_blank" rel="nofollow noopener">charging seven LockBit ransomware members</a>.</p>

<p>Among those charged, is the primary operator of the ransomware operation, who the <a href="https://www.bleepingcomputer.com/news/security/lockbit-ransomware-admin-identified-sanctioned-in-us-uk-australia/" target="_blank" rel="nofollow noopener">Department of Justice claims</a> is a Russian national named Dmitry Yuryevich Khoroshev,&nbsp;aka 'LockBitSupp' and 'putinkrab'.</p>

<p>LockBit recently began testing a new encryptor called LockBit 4, which does not appear to be much different than its previous version.</p>

<h3 class="top_story" style="color:white!important"><span class="top_num">3. </span><a href="https://www.bleepingcomputer.com/news/microsoft/microsofts-new-windows-11-recall-is-a-privacy-nightmare/" target="_blank" rel="nofollow noopener">Windows 11 Recall: A privacy nightmare?</a></h3>

<p>Microsoft's new AI-powered <a href="https://www.bleepingcomputer.com/news/microsoft/microsofts-new-windows-11-recall-is-a-privacy-nightmare/" target="_blank" rel="nofollow noopener">Windows 11 Recall feature</a> has sparked a lot of concern among the cybersecurity community, with many thinking that it is a massive privacy risk&nbsp;and a new attack vector that threat actors can exploit to steal data.</p>

<p>After receiving tremendous backlash, Microsoft delayed the release of the software to increase its security by requiring users to&nbsp;<a href="https://www.bleepingcomputer.com/news/microsoft/microsoft-makes-windows-recall-opt-in-secures-data-with-windows-hello/" target="_blank" rel="nofollow noopener">opt-in to enable Recall</a> on their computers and that they'll have to confirm they're in front of their PC&nbsp;<a href="https://www.bleepingcomputer.com/news/microsoft/microsoft-makes-windows-recall-opt-in-secures-data-with-windows-hello/" target="_blank" rel="nofollow noopener">via Windows Hello</a> to be able to use it.</p>

<p>Microsoft continued to delay its release while <a href="https://www.bleepingcomputer.com/news/microsoft/microsoft-windows-recall-now-can-be-removed-is-more-secure/" target="_blank" rel="nofollow noopener">adding additional features</a>, such as&nbsp;automatically filtering&nbsp;sensitive content, allowing&nbsp;users to exclude specific apps, websites, or in-private browsing sessions, and it can be removed if needed.</p>

<p>However, after <a href="https://www.bleepingcomputer.com/news/microsoft/microsoft-expands-recall-preview-to-intel-and-amd-copilot-plus-pcs/" target="_blank" rel="nofollow noopener">releasing the software to Windows Insiders</a> for testing, <a href="https://www.tomshardware.com/software/windows/microsoft-recall-screenshots-credit-cards-and-social-security-numbers-even-with-the-sensitive-information-filter-enabled" target="_blank" rel="nofollow noopener">it was discovered</a> that Windows 11 Recall did not properly filter sensitive information, like credit cards.</p>

<p>Microsoft said they continue refining the product as new issues are discovered.</p>

<h3 class="top_story" style="color:white!important"><span class="top_num">2.</span>&nbsp;<a href="https://www.bleepingcomputer.com/news/security/atandt-verizon-reportedly-hacked-to-target-us-govt-wiretapping-platform/" target="_blank" rel="nofollow noopener"><span class="top_num">The 2024 Telecom attacks</span></a></h3>

<p>A Chinese state-sponsored hacking group known as "Salt Typhoon" is <a href="https://www.bleepingcomputer.com/news/security/us-says-chinese-hackers-breached-multiple-telecom-providers/" target="_blank" rel="nofollow noopener">linked to a series of cyberattacks targeting telecommunications firms</a> globally.</p>

<p>These breaches <a href="https://www.bleepingcomputer.com/news/security/white-house-links-ninth-telecom-breach-to-chinese-hackers/" target="_blank" rel="nofollow noopener">compromised at least nine major telecom providers</a>, including AT&amp;T, Verizon, and <a href="https://www.bleepingcomputer.com/news/security/chinese-hackers-breached-t-mobiles-routers-to-scope-out-network/" target="_blank" rel="nofollow noopener">T-Mobile</a>.</p>

<p>The group reportedly focused on infiltrating telecom infrastructure to steal text messages, phone call information, and voicemails from targeted people. The threat actors also targeted the wiretapping platforms used by the US government, raising serious national security concerns.&nbsp;</p>

<p>A White House briefing revealed that Salt Typhoon's operations also impacted telecommunications providers in <a href="https://www.bleepingcomputer.com/news/security/white-house-salt-typhoon-hacked-telcos-in-dozens-of-countries/" target="_blank" rel="nofollow noopener">dozens of countries</a>.</p>

<p>In the US, these attacks prompted concerns about weaknesses in telecom infrastructure and the security of government surveillance platforms.&nbsp;</p>

<p>US lawmakers, including Senator Ron Wyden, have <a href="https://www.bleepingcomputer.com/news/security/wyden-proposes-bill-to-secure-us-telecoms-after-salt-typhoon-hacks/" target="_blank" rel="nofollow noopener">proposed legislation</a> to address vulnerabilities in the nation's telecom infrastructure. The proposed bill aims to establish stricter cybersecurity standards and oversight for telecom providers to prevent similar attacks in the future.</p>

<p>The US government&nbsp;<a href="https://www.bleepingcomputer.com/news/security/atandt-verizon-reportedly-hacked-to-target-us-govt-wiretapping-platform/" target="_blank" rel="nofollow noopener">reportedly plans</a> to ban China Telecom's last active US operations in response to the telecom hacks.</p>

<h3 class="top_story" style="color:white!important"><span class="top_num">1. <a href="https://www.bleepingcomputer.com/news/security/new-moveit-transfer-zero-day-mass-exploited-in-data-theft-attacks/" target="_blank" rel="nofollow noopener">The rise of Infostealers</a></span></h3>

<p>Information-stealing malware campaigns are running rampant this year, used in many different campaigns to steal infected users' browser information, cookies, saved credentials, credit cards, and cryptocurrency wallets.</p>

<p>While infostealers have been around for many years, they have been particularly prominent with threat actors using them in a wide range of campaigns.</p>

<p>These stolen credentials are then used to breach corporate networks, bank accounts, cryptocurrency exchanges, and email accounts.</p>

<p>The number of stories surrounding infostealers is too long to summarize, so instead, here are a few of the ways infostealers were used this year:</p>

<ul style="list-style-type:square"><li><a href="https://www.bleepingcomputer.com/news/security/hacker-hijacks-orange-spain-ripe-account-to-cause-bgp-havoc/" target="_blank" rel="nofollow noopener">Hacker hijacks Orange Spain RIPE account to cause BGP havoc</a></li>
	<li><a href="https://www.bleepingcomputer.com/news/security/global-infostealer-malware-operation-targets-crypto-users-gamers/" target="_blank" rel="nofollow noopener">Global infostealer malware operation targets crypto users, gamers</a></li>
	<li><a href="https://www.bleepingcomputer.com/news/security/windows-vulnerability-abused-braille-spaces-in-zero-day-attacks/" target="_blank" rel="nofollow noopener">Windows vulnerability abused braille "spaces" in zero-day attacks</a></li>
	<li><a href="https://www.bleepingcomputer.com/news/security/malicious-ads-push-lumma-infostealer-via-fake-captcha-pages/" target="_blank" rel="nofollow noopener">Malicious ads push Lumma infostealer via fake CAPTCHA pages</a></li>
	<li><a href="https://www.bleepingcomputer.com/news/security/clever-github-scanner-campaign-abusing-repos-to-push-malware/" target="_blank" rel="nofollow noopener">Clever 'GitHub Scanner' campaign abusing repos to push malware</a></li>
	<li><a href="https://www.bleepingcomputer.com/news/security/cybercriminals-pose-as-helpful-stack-overflow-users-to-push-malware/" target="_blank" rel="nofollow noopener">Cybercriminals pose as "helpful" Stack Overflow users to push malware</a></li>
</ul><p>Unfortunately, for those who become infected with an infostealer, it can lead to devastating financial losses as threat actors steal cryptocurrency and access victims' bank accounts.&nbsp;</p>

<p>The best way to prevent these types of attacks is to enable two-factor authentication with an authenticator app on all accounts that offer the protection. With 2FA enabled, even if a threat actor has your credentials, they won't be able to log in without the code generated by your authenticator.</p>

function cz_strip_tags(input, allowed) {
  allowed = (((allowed || '') + '')
    .toLowerCase()
    .match(/<[a-z][a-z0-9]*>/g) || [])
    .join(''); // making sure the allowed arg is a string containing only tags in lowercase (<a><b><c>)
  var tags = /<\/?([a-z][a-z0-9]*)\b[^>]*>/gi,
    commentsAndPhpTags = /<!--[\s\S]*?-->|<\?(?:php)?[\s\S]*?\?>/gi;
  return input.replace(commentsAndPhpTags, '')
    .replace(tags, function($0, $1) {
      return allowed.indexOf('<' + $1.toLowerCase() + '>') > -1 ? $0 : '';
    });
}
function cz_br2nl(str) {
	var regex = /<br\s*[\/]?>/gi;
    //var pure_str = str.replace(regex,"\n");
    var pure_str = str.replace(regex,"");
    return cz_strip_tags(pure_str,'');
}
