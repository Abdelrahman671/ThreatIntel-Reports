The threat actor used this legitimate application for DLL hijacking purposes, loading a malicious FileAssociation.dll, which is dynamically loaded by jisucommon.exe</p><p><a href="/assets/images/PUBLOAD_Thai/jisucommon.png" class="popup img-link shimmer"><img src="/assets/images/PUBLOAD_Thai/jisucommon.png" alt="sshot" loading="lazy"></a> <a href="/assets/images/PUBLOAD_Thai/fileassocload.png" class="popup img-link shimmer"><img src="/assets/images/PUBLOAD_Thai/fileassocload.png" alt="sshot" loading="lazy"></a></p><p>FileAssociation.dll has a PDB path with a tone of humor, as it has been referenced in previous reports. The username associated with the project is ‘FBI’. <a href="/assets/images/PUBLOAD_Thai/pdb.png" class="popup img-link shimmer"><img src="/assets/images/PUBLOAD_Thai/pdb.png" alt="sshot" loading="lazy"></a></p><p>The legitimate executable and the trojanized DLL are copied to ProgramData for persistence. <a href="/assets/images/PUBLOAD_Thai/copiedfiles.png" class="popup img-link shimmer"><img src="/assets/images/PUBLOAD_Thai/copiedfiles.png" alt="sshot" loading="lazy"></a></p><p>Two types of persistence are implemented: A scheduled task, and via the registry run key.</p><p><u>Registry run key in HKLM and HKCU:</u></p><p><a href="/assets/images/PUBLOAD_Thai/regpersist.png" class="popup img-link shimmer"><img src="/assets/images/PUBLOAD_Thai/regpersist.png" alt="sshot" loading="lazy"></a></p><p><u>Scheduled Task:</u></p><p><a href="/assets/images/PUBLOAD_Thai/schtaskpersist.png" class="popup img-link shimmer"><img src="/assets/images/PUBLOAD_Thai/schtaskpersist.png" alt="sshot" loading="lazy"></a></p><p>The DLL decrypts an embedded shellcode and loads it via a callback function, using EnumPropsExW().</p><p><a href="/assets/images/PUBLOAD_Thai/scdecryptexec.png" class="popup img-link shimmer"><img src="/assets/images/PUBLOAD_Thai/scdecryptexec.png" alt="sshot" loading="lazy"></a></p><p>The decrypted shellcode is a PUBLOAD stager, previously reported by <a href="https://blog.talosintelligence.com/mustang-panda-targets-europe/">Cisco</a> and <a href="https://www.trendmicro.com/en_us/research/22/k/earth-preta-spear-phishing-governments-worldwide.html">TrendMicro</a>. The shellcode implements a common API hashing algorithm using ROR13. The following APIs are resolved via the hashing algorithm:</p><p><a href="/assets/images/PUBLOAD_Thai/loadlibgetproc.png" class="popup img-link shimmer"><img src="/assets/images/PUBLOAD_Thai/loadlibgetproc.png" alt="sshot" loading="lazy"></a> <a href="/assets/images/PUBLOAD_Thai/resapis.png" class="popup img-link shimmer"><img src="/assets/images/PUBLOAD_Thai/resapis.png" alt="sshot" loading="lazy"></a></p><p>The shellcode establishes a socket connection to 45.150.128[.]212:443, and it’s able to receive and execute additional shellcodes.
