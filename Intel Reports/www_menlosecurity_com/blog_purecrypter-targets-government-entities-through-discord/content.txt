class="div-block-284-2"><div class="blog-post-wrapper"><div itemprop="articleBody" class="rich-text-block w-richtext"><h2>Executive Summary</h2><p>Menlo Labs has uncovered an unknown threat actor thatâs leveraging an evasive threat campaign distributed via Discord that features the PureCrypter downloader and targets government entities. The PureCrypter campaign uses the domain of a compromised non-profit organization as a Command and Control (C2) to deliver a secondary payload. The campaign was found to have delivered several types of malware including Redline Stealer, AgentTesla, Eternity, Blackmoon and Philadelphia Ransomware. Our investigation started when Menloâs Cloud Security Platform blocked password-protected archive files across multiple government customers in the Asia-Pacific (APAC) and North America regions.</p><p>Menlo Labs assesses that this threat actor group will continue to use the compromised and taken infrastructure as long as they can, before needing to find a new home. Leaving credentials in malware is an OpSec failure but it leaves a trace for analysts to follow. Fortunately in this case, Menloâs Cloud Security Platform blocked this attack, which allowed Menlo Labs to see it and start to track this actor.</p><h2>Infection Chain</h2><figure class="w-richtext-align-center w-richtext-figure-type-image"><div><img src="https://cdn.prod.website-files.com/65565a6ae1bebedfef4026ca/6571dd5eef38f12cbc5820a1_Menlo_Labs_PureCrypter_discord_blog_Figure_1.jpg" loading="lazy" alt="diagram illustrating download to password protected zip to purecrypter"/></div></figure><h2>Threat Intelligence</h2><p>Threat analysis showed that PureCrypter is downloading a secondary malware, believed to be <a href="https://attack.mitre.org/software/S0331/" target="_blank">AgentTesla</a>.</p><p><a href="https://www.zscaler.com/blogs/security-research/technical-analysis-purecrypter" target="_blank">PureCrypter</a> is an advanced downloader which downloads Remote Access Trojans (RATs) and Infostealers. It has been sold since March 2021 onâhxxps[://]purecoder.sellix.io/.â <a href="https://www.zscaler.com/blogs/security-research/agent-tesla-keylogger-delivered-using-cybersquatting" target="_blank">AgentTesla</a> is an advanced backdoor with capabilities including stealing stored passwords from different browsers, clipboard logging, screen keylogging and screen capturing. It is written in .net and supports all versions of Windows operating system.</p><p>In our investigation, we found that AgentTesla establishes a connection to an FTP server where it stores the stolen victimâs credentials. The FTP server appears to have been taken over and the leaked credentials for the domain were found online, thus suggesting that the threat actors used these credentials to gain access to the server.</p><figure class="w-richtext-align-center w-richtext-figure-type-image"><div><img src="https://cdn.prod.website-files.com/65565a6ae1bebedfef4026ca/6571dd6b38e3c511a1fe285a_Picture1.png" loading="lazy" alt="Screenshot showing collected victim information on FTP server"/></div><figcaption><em>Compromised FTPÂ server showing collected victim information</em></figcaption></figure><figure class="w-richtext-align-center w-richtext-figure-type-image"><div><img src="https://cdn.prod.website-files.com/65565a6ae1bebedfef4026ca/6571dd86f8dfb352873b9e81_Picture2.png" loading="lazy" alt="screenshot showing collected victim information"/></div></figure><p>It&#x27;s also noteworthy that the download link for the secondary malware is from a compromised domain of a non-profit organization whose leaked credentials were also found online. </p><p>A similar sample to theAgentTesla malware we analyzed was discovered in a phishing email with the subject &quot;FW: New Order no. 5959&quot; from Alejandro Gonzalo (e052450f2@891f4e7e1668[.]com). The malicious attachment was named &quot;Nuevo pedido 7887979-800898.gz&quot; and contained FTP server credentials that were the same as those found in the first case.</p><p>Under that same email address, another malicious email titled &quot;New Order&quot; was also uncovered with an attached file called &quot;Ppurchase order6007979-709797790.gz&quot;. Altogether, the Labs team found 106 files using said FTP server.</p><h2>Infection Vector/Technical Details</h2><p>In this campaign, Discord was used to host the payload, and a link to the payload is sent via email. To evade existing defenses, PureCrypter uses password protected ZIP files. Below is a screenshot that shows the poor detection of these password-protected payloads on VT.</p><figure class="w-richtext-align-center w-richtext-figure-type-image"><div><img src="https://cdn.prod.website-files.com/65565a6ae1bebedfef4026ca/6571dd99b075e0899eae1821_purecrypter4.png" loading="lazy" alt="Screenshot showing 1 security vendor and no sandboxes flagged file as malicious"/></div></figure><p>The following steps were taken by the attacker to deliver the payload:</p><ul role="list"><li>An email with Discord app url pointing to malicious password protected zip file is sent to the victim (https://cdn[.]discordapp.com/attachments/1006638283645784218/1048923462128914512/Private_file__dont_share.zip, pwd - 1234, md5- 967f9bc90202925e1f941c8ea1db2c94)</li><li>The ZIP extracts a loader written in .net called PureCrypter (md5 - 5420DCBAE4F1FBA8AFE85CB03DCD9BFC). At the time of investigation the compromised non-profit organizationâs website was down and we didnât get its secondary payload.</li></ul><figure class="w-richtext-align-center w-richtext-figure-type-image"><div><img src="https://cdn.prod.website-files.com/65565a6ae1bebedfef4026ca/6571dda577e4e509515ff3db_purecrypter5.png" loading="lazy" alt="screenshot of code"/></div></figure><p>While we were not able to download the second stage payload from the PureCrypter sample mentioned above (md5 - 5420DCBAE4F1FBA8AFE85CB03DCD9BFC), we were able to identify similar samples which were seen downloading malicious payloads from the compromised non-profit organization.
