items-center mb-8"><img src="/assets/images/sansec-social.svg" alt="Sansec" class="inline-block h-10 mr-3 w-10" width="610" height="458"><div class="leading-relaxed text-gray-600 font-medium text-sm tracking-tight"><p>by Sansec Forensics Team</p><p class="text-xs">Published in <a href="/research" class="underline">Threat Research</a> &minus; September 13, 2022</p></div></div><div class="leading-relaxed text-gray-600 [&_a]:text-ss-teal hover:[&_a]:text-ss-teal-hover text-xl"><p>Fishpig, a vendor of popular Magento-Wordpress integrations, has been hacked. Sansec found that attackers have injected malware in Fishpig software and taken control of Fishpig servers. Online stores running Fishpig software may now have the &quot;Rekoobe&quot; malware installed on their servers, effectively granting store administrator access to attackers.</p></div></div></div></div><section class="bg-white mb-12 my-4"><div class="flex gap-x-6 container mx-auto"><div class="max-w-2xl lg:mx-0 w-full"><div class="-mt-32 -mx-6 italic mb-6 text-center text-gray-400 text-sm"><img class="bg-white border border-gray-300" src="/assets/2023/og/720/og-graphic-10.webp" alt="Magento vendor Fishpig hacked, backdoors added"></div><div class="markdown pt-4"><p><strong>Update 2022-09-13</strong> FishPig has confirmed the incident and <a href="https://fishpig.co.uk/security-announcements/#X20220913">published a status page</a>. It recommends customers to upgrade and/or reinstall all FishPig modules.</p><p></p><div class="markdown-img"><img src="/assets/posts/cache/1bafc143f8c3e6cea87903658a317c9c.png" alt=""></div><p></p><p>Sansec discovered malware in the <a href="https://www.fishpig.co.uk/magento/extensions/block-robots-stop-spam/">Fishpig Magento Security Suite</a> and several other Fishpig extensions for Magento 2. It is likely that all paid Fishpig extensions have been compromised. Free extensions that are hosted on Github seem not to be affected.</p><p>The injected malware will install another piece of malware (&quot;Rekoobe&quot;) which hides as background process on the server.</p><p>The Fishpig distribution server was compromised on or before August 19th. Any Magento store who installed or updated paid Fishpig software since then, is now likely running the Rekoobe malware.</p><p>Fishpig software has over <a href="https://packagist.org/packages/fishpig/">200,000 downloads</a>. It is not known how many stores use the paid extensions.</p><h2 id="next-steps-for-merchants" tabindex="-1">Next steps for merchants</h2><p>FishPig has stated that their repository is fully cleaned. Magento merchants are recommended to:</p><ol><li>Re-install all FishPig extensions per <a href="https://fishpig.co.uk/security-announcements/#X20220913">FishPig instructions</a></li><li>Run a <a href="/pricing">server-side malware scanner</a> to detect installed malware &amp; unauthorized activity. Use coupon <code>FISHPIG</code> to use our scanner one month for free.</li><li>Restart the server to terminate any unauthorized background processes</li></ol><h2 id="attack-details%3A-lic.bin-is-rekoobe" tabindex="-1">Attack details: lic.bin is Rekoobe</h2><p>Attackers have added code to <code>License.php</code>, which is normally used to validate a Fishpig license.
