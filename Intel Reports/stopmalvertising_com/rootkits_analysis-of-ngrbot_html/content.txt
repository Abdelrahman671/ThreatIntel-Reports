<div class="article-intro"><span class="kdropcap">T</span>oday we will have a closer look at <strong>ngrBot</strong>, an IRC bot with rootkit capabilities. The core of ngrBot is an advanced ring3 (usermode) system-wide injection and hooking engine similar to ZeuS and SpyEye.</div>

<p>NgrBot will inject code into almost every running process on the computer and is able to terminate processes. It will install to the user’s Application Data folder under a randomly generated filename using the HDD serial number as the initial key.</p>

<p>The bot is also able to block access to certain domains and redirect domains / IP’s to others.</p>

<p>It’s able to spread via USB devices and Windows Live Messenger. More recently ngrBot has been spotted on Facebook but also on Twitter, using the micro blogging service to spread itself.</p>



<br /><h4>Modules and Features</h4>
<h5>Rootkit</h5>

<p>The rootkit module will attempt to hide the bot’s registry startup key as well as the bot file.</p>

<h5>Ruskill</h5>

<p>The Ruskill module will, if enabled for a download command, monitor the downloaded file as it executes. Ruskill will flag any files that it copies itself to or creates to be deleted at the next system reboot.</p>

<h5>Proactive defense</h5>

<p>The PDef module is an advanced threat detection and removal system. It monitors a range of file and networking API’s to detect and neutralize other threats that are running on the system. Currently this module can detect, block and remove malware that spreads via USB drives, browser exploit packs, and bots that use IRC to communicate.</p>

<h5>DNS Modifier</h5>

<p>This module can block domains from being accessed and redirect domains/IP addresses to others.</p>

<h5>Slowloris</h5>

<p>This module is for web servers running Apache HTTPd. It’s designed to use low bandwidth and to maintain connections as long as possible, thus consuming all available resources.</p>

<h5>Syn Flood</h5>

<p>The syn flood module is good for web servers that Slowloris fails to take down.</p>

<h5>UDP Flood</h5>

<p>This module is ideal for taking home connections offline.</p>

<h5>Internet Explorer Login Grabber</h5>

<p>This module hooks wininet.dll and analyses POST requests made by the IE web browser to capture usernames and passwords on the fly.</p>

<h5>Firefox Login Grabber</h5>

<p>This module hooks nspr4.dll and analyses POST requests made by the Firefox web browser to capture usernames and passwords on the fly.</p>

<h5>FTP Login Grabber</h5>

<p>This module hooks ws2_32!send to grab the FTP logins as they are used.</p>

<h5>USB Spreader</h5>

<p>Waits for USB devices to be inserted and then attemps to infect them using multiple .lnk methods and obfuscated autorun.</p>

<h5>MSN Spreader</h5>

<p>This module hooks ws2_32!send to detect MSN messages being send. It will then monitor outgoing messages and wait for the spoofed number of messages to be sent before replacing one with the set spread message. It has been tested with the msnp10 and msnp21 protocols with msnmsgr.exe, wlcomm.exe, pidgin.exe and msmsgs.exe.</p><br />

<h4>Commands</h4>

<ul><li>Download</li>
<li>Update</li>
<li>Die</li>
<li>Remove</li>
<li>Mute</li>
<li>Version</li>
<li>Visit</li>
<li>Reconnect</li>
<li>Join Channel</li>
<li>Part Channel</li>
<li>Sort Channel by country</li>
<li>Unsort</li>
<li>Module toggle (enable/disable modules)</li>
<li>Statistics (Spreading/login grabbing)</li>
<li>Retrieve all cached logs</li>
<li>Syn</li>
<li>UDP</li>
<li>Slowloris</li>
<li>Stop DDoS</li>
<li>Set MSN Inteerval</li>
<li>Block/Redirect Domain and IP Address</li>
</ul>

<p>The full package containing all the modules is sold for $400. NgrBot can also be obtained a la carte, meaning no modules, pick and choose which modules you want to include.</p><br />

<h4>Analysis of ngrBot</h4>

<p>Upon execution <strong>facebook-pic0008422012.exe</strong> copies itself under a random name based on the HDD serial number (vgbkbf.exe in our analysis), using Kernel32.GetVolumeInformationW, to the C:\Documents and Settings\[username]\Application Data\  folder.</p>

<p><img src="/research/images/ngrbt1.jpg" alt="Analysis of ngrBot" title="Analysis of ngrBot" /></p>
<p><img src="/research/images/ngrbt2.jpg" alt="Analysis of ngrBot" title="Analysis of ngrBot" /></p>

<p>The newly created process vgbkbf.exe will attempt to inject code into the memory space of all running processes.</p>

<p><img src="/research/images/ngrbt3.jpg" alt="Analysis of ngrBot" title="Analysis of ngrBot" /></p>

<p>NgrBot will initiate a communication with its C&C located at <strong>update.jebac.net</strong> through IRC. The domain <strong>api.wipmania.com</strong> is used to retrieve the country code based on the victim’s IP. The HTTP / MSN spread message followed by a link to the binary is transmitted via IRC and Instructions are given to download a list of blocked domains from <strong>data.fuskbugg.se</strong> and a new binary called <strong>milkway.exe</strong> (saved as 1.tmp) from <a href="http://en.wikipedia.org/wiki/RapidShare" target="_blank">RapidShare</a>. The bot will report back to the C&C if the download was succesful or not and if the file was executed.</p>

<p>The file <strong>1.tmp</strong> will request internet access in order to download a binary called <strong>memory.exe</strong> from RapidShare. The file will be saved under a random name in the %temp% folder. </p>

<p><img src="/research/images/ngrbt4.jpg" alt="Analysis of ngrBot" title="Analysis of ngrBot" /></p>

<p>Upon execution the file copies itself as <strong>windows.exe</strong> to the C:\Documents and Settings\[username]\Application Data\  folder.</p>

<p><img src="/research/images/ngrbt5.jpg" alt="Analysis of ngrBot" title="Analysis of ngrBot" /></p>

<p>Once <strong>windows.exe</strong> started, <strong>ngrBot</strong> marks its presence in the infected system with a mutex named <strong>hex-Mutex</strong> and <strong>WeAreWeAre</strong>.</p>

<p><img src="/research/images/ngrbt6.jpg" alt="Analysis of ngrBot" title="Analysis of ngrBot" /></p>

<p>The <strong>Internet Explorer Home page</strong> has been modified to <strong>redirecturls.info</strong> which redirects to another domain.</p>

<p><img src="/research/images/ngrbt8.jpg" alt="Analysis of ngrBot" title="Analysis of ngrBot" /></p>

<p>Examples are:</p>

<ul><li>www.tiptop-article.com</li>
