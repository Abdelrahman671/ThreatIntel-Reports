<p style="font-size:25px">Summary</p>



<p><a href="https://bruteratel.com/">Brute Ratel C4</a> is a Red Team &amp; Adversary Simulation software that can be considered an alternative to Cobalt Strike. In this blog post, we&#8217;re presenting a technical analysis of a Brute Ratel badger/agent that doesn&#8217;t implement all the recent features of the framework. There aren&#8217;t a lot of Brute Ratel samples available in the wild. The malware implements the API hashing technique and comes up with a configuration that contains the C2 server, the user-agent used during the network communications, a password used for authentication with the C2 server, and a key used for encrypting data transmitted to the C2 server. The badger takes control of the infected machine by executing 63 different commands issued by the C2 server. The first 20 commands will be described in this blog post, while the rest of them will be detailed in an upcoming blog post.</p>



<p style="font-size:25px">Technical analysis</p>



<p>SHA256: d71dc7ba8523947e08c6eec43a726fe75aed248dfd3a7c4f6537224e9ed05f6f</p>



<p>This is a 64-bit executable. The malware pushes the code to be executed on the stack in order to evade Antivirus and EDR software:</p>



<figure class="wp-block-image aligncenter size-full"><img fetchpriority="high" decoding="async" width="485" height="313" src="https://cybergeeks.tech/wp-content/uploads/2023/08/1.jpg" alt="" class="wp-image-4667" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/1.jpg 485w, https://cybergeeks.tech/wp-content/uploads/2023/08/1-300x194.jpg 300w" sizes="(max-width: 485px) 100vw, 485px" /><figcaption class="wp-element-caption">Figure 1</figcaption></figure>



<p>It implements the API hashing technique, which uses the &#8220;ROR EDI,0xD&#8221; instruction to compute 4-byte hashes that are compared with pre-computed ones (Figure 2).</p>



<figure class="wp-block-image aligncenter size-full"><img decoding="async" width="710" height="595" src="https://cybergeeks.tech/wp-content/uploads/2023/08/2.jpg" alt="" class="wp-image-4670" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/2.jpg 710w, https://cybergeeks.tech/wp-content/uploads/2023/08/2-300x251.jpg 300w" sizes="(max-width: 710px) 100vw, 710px" /><figcaption class="wp-element-caption">Figure 2</figcaption></figure>



<p>The VirtualAllocEx API is used to allocate a new memory area that will store a DLL file (0x3000 = <strong>MEM_COMMIT</strong> | <strong>MEM_RESERVE</strong>, 0x40 = <strong>PAGE_EXECUTE_READWRITE</strong>):</p>



<figure class="wp-block-image aligncenter size-large"><img decoding="async" width="1024" height="107" src="https://cybergeeks.tech/wp-content/uploads/2023/08/3-1024x107.jpg" alt="" class="wp-image-4671" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/3-1024x107.jpg 1024w, https://cybergeeks.tech/wp-content/uploads/2023/08/3-300x31.jpg 300w, https://cybergeeks.tech/wp-content/uploads/2023/08/3-768x80.jpg 768w, https://cybergeeks.tech/wp-content/uploads/2023/08/3.jpg 1290w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption class="wp-element-caption">Figure 3</figcaption></figure>

<figure class="wp-block-image aligncenter size-large"><img loading="lazy" decoding="async" width="1024" height="45" src="https://cybergeeks.tech/wp-content/uploads/2023/08/57-1024x45.jpg" alt="" class="wp-image-4737" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/57-1024x45.jpg 1024w, https://cybergeeks.tech/wp-content/uploads/2023/08/57-300x13.jpg 300w, https://cybergeeks.tech/wp-content/uploads/2023/08/57-768x34.jpg 768w, https://cybergeeks.tech/wp-content/uploads/2023/08/57.jpg 1292w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption class="wp-element-caption">Figure 57</figcaption></figure>



<p><strong>0x0A32 ID</strong> &#8211; Retrieve the Last-Write time for files/directories</p>



<p>The files are enumerated in the current directory using the FindFirstFileW and FindNextFileW functions:</p>



<figure class="wp-block-image aligncenter size-large"><img loading="lazy" decoding="async" width="1024" height="57" src="https://cybergeeks.tech/wp-content/uploads/2023/08/58-1024x57.jpg" alt="" class="wp-image-4738" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/58-1024x57.jpg 1024w, https://cybergeeks.tech/wp-content/uploads/2023/08/58-300x17.jpg 300w, https://cybergeeks.tech/wp-content/uploads/2023/08/58-768x43.jpg 768w, https://cybergeeks.tech/wp-content/uploads/2023/08/58.jpg 1293w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption class="wp-element-caption">Figure 58</figcaption></figure>



<figure class="wp-block-image aligncenter size-large"><img loading="lazy" decoding="async" width="1024" height="56" src="https://cybergeeks.tech/wp-content/uploads/2023/08/59-1024x56.jpg" alt="" class="wp-image-4739" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/59-1024x56.jpg 1024w, https://cybergeeks.tech/wp-content/uploads/2023/08/59-300x16.jpg 300w, https://cybergeeks.tech/wp-content/uploads/2023/08/59-768x42.jpg 768w, https://cybergeeks.tech/wp-content/uploads/2023/08/59.jpg 1292w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption class="wp-element-caption">Figure 59</figcaption></figure>



<p>For each of the file or directory that matches the pattern, the binary calls the CreateFileW API:</p>



<figure class="wp-block-image aligncenter size-large"><img loading="lazy" decoding="async" width="1024" height="126" src="https://cybergeeks.tech/wp-content/uploads/2023/08/60-1024x126.jpg" alt="" class="wp-image-4740" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/60-1024x126.jpg 1024w, https://cybergeeks.tech/wp-content/uploads/2023/08/60-300x37.jpg 300w, https://cybergeeks.tech/wp-content/uploads/2023/08/60-768x95.jpg 768w, https://cybergeeks.tech/wp-content/uploads/2023/08/60.jpg 1289w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption class="wp-element-caption">Figure 60</figcaption></figure>



<p>The process retrieves the Last-Write time via a function call to GetFileTime:</p>



<figure class="wp-block-image aligncenter size-large"><img loading="lazy" decoding="async" width="1024" height="98" src="https://cybergeeks.tech/wp-content/uploads/2023/08/61-1024x98.jpg" alt="" class="wp-image-4741" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/61-1024x98.jpg 1024w, https://cybergeeks.tech/wp-content/uploads/2023/08/61-300x29.jpg 300w, https://cybergeeks.tech/wp-content/uploads/2023/08/61-768x74.jpg 768w, https://cybergeeks.tech/wp-content/uploads/2023/08/61.jpg 1291w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption class="wp-element-caption">Figure 61</figcaption></figure>



<p>The file time is converted to system time format using FileTimeToSystemTime:</p>



<figure class="wp-block-image aligncenter size-large"><img loading="lazy" decoding="async" width="1024" height="75" src="https://cybergeeks.tech/wp-content/uploads/2023/08/62-1024x75.jpg" alt="" class="wp-image-4742" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/62-1024x75.jpg 1024w, https://cybergeeks.tech/wp-content/uploads/2023/08/62-300x22.jpg 300w, https://cybergeeks.tech/wp-content/uploads/2023/08/62-768x56.jpg 768w, https://cybergeeks.tech/wp-content/uploads/2023/08/62.jpg 1291w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption class="wp-element-caption">Figure 62</figcaption></figure>



<p>Finally, the above time is converted to the currently active time zone:</p>



<figure class="wp-block-image aligncenter size-large"><img loading="lazy" decoding="async" width="1024" height="71" src="https://cybergeeks.tech/wp-content/uploads/2023/08/63-1024x71.jpg" alt="" class="wp-image-4743" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/63-1024x71.jpg 1024w, https://cybergeeks.tech/wp-content/uploads/2023/08/63-300x21.jpg 300w, https://cybergeeks.tech/wp-content/uploads/2023/08/63-768x54.jpg 768w, https://cybergeeks.tech/wp-content/uploads/2023/08/63.jpg 1291w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption class="wp-element-caption">Figure 63</figcaption></figure>



<p><strong>0x3D1D ID</strong> &#8211; Change the Desktop wallpaper</p>



<p>The malicious process opens the &#8220;TranscodedWallpaper&#8221; file that contains the Desktop wallpaper:</p>



<figure class="wp-block-image aligncenter size-large"><img loading="lazy" decoding="async" width="1024" height="203" src="https://cybergeeks.tech/wp-content/uploads/2023/08/64-1024x203.jpg" alt="" class="wp-image-4744" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/64-1024x203.jpg 1024w, https://cybergeeks.tech/wp-content/uploads/2023/08/64-300x60.jpg 300w, https://cybergeeks.tech/wp-content/uploads/2023/08/64-768x152.jpg 768w, https://cybergeeks.tech/wp-content/uploads/2023/08/64.jpg 1285w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption class="wp-element-caption">Figure 64</figcaption></figure>



<p>The above file is filled in with content received from the C2 server (Figure 65).</p>



<figure class="wp-block-image aligncenter size-large"><img loading="lazy" decoding="async" width="1024" height="108" src="https://cybergeeks.tech/wp-content/uploads/2023/08/65-1024x108.jpg" alt="" class="wp-image-4745" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/65-1024x108.jpg 1024w, https://cybergeeks.tech/wp-content/uploads/2023/08/65-300x32.jpg 300w, https://cybergeeks.tech/wp-content/uploads/2023/08/65-768x81.jpg 768w, https://cybergeeks.tech/wp-content/uploads/2023/08/65.jpg 1291w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption class="wp-element-caption">Figure 65</figcaption></figure>

<figure class="wp-block-image aligncenter size-large"><img loading="lazy" decoding="async" width="1024" height="90" src="https://cybergeeks.tech/wp-content/uploads/2023/08/66-1024x90.jpg" alt="" class="wp-image-4746" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/66-1024x90.jpg 1024w, https://cybergeeks.tech/wp-content/uploads/2023/08/66-300x26.jpg 300w, https://cybergeeks.tech/wp-content/uploads/2023/08/66-768x67.jpg 768w, https://cybergeeks.tech/wp-content/uploads/2023/08/66.jpg 1291w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption class="wp-element-caption">Figure 66</figcaption></figure>

<figure class="wp-block-image aligncenter size-large"><img loading="lazy" decoding="async" width="1024" height="64" src="https://cybergeeks.tech/wp-content/uploads/2023/08/67-1024x64.jpg" alt="" class="wp-image-4747" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/67-1024x64.jpg 1024w, https://cybergeeks.tech/wp-content/uploads/2023/08/67-300x19.jpg 300w, https://cybergeeks.tech/wp-content/uploads/2023/08/67-768x48.jpg 768w, https://cybergeeks.tech/wp-content/uploads/2023/08/67.jpg 1293w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption class="wp-element-caption">Figure 67</figcaption></figure>

<figure class="wp-block-image aligncenter size-full"><img loading="lazy" decoding="async" width="850" height="48" src="https://cybergeeks.tech/wp-content/uploads/2023/08/68.jpg" alt="" class="wp-image-4748" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/68.jpg 850w, https://cybergeeks.tech/wp-content/uploads/2023/08/68-300x17.jpg 300w, https://cybergeeks.tech/wp-content/uploads/2023/08/68-768x43.jpg 768w" sizes="(max-width: 850px) 100vw, 850px" /><figcaption class="wp-element-caption">Figure 68</figcaption></figure>

<figure class="wp-block-image aligncenter size-large"><img loading="lazy" decoding="async" width="1024" height="75" src="https://cybergeeks.tech/wp-content/uploads/2023/08/69-1024x75.jpg" alt="" class="wp-image-4749" srcset="https://cybergeeks.tech/wp-content/uploads/2023/08/69-1024x75.jpg 1024w, https://cybergeeks.tech/wp-content/uploads/2023/08/69-300x22.jpg 300w, https://cybergeeks.tech/wp-content/uploads/2023/08/69-768x56.jpg 768w, https://cybergeeks.tech/wp-content/uploads/2023/08/69.jpg 1292w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption class="wp-element-caption">Figure 69</figcaption></figure>
