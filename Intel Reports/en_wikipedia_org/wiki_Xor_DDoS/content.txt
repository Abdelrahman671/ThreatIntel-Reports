					<div id="mw-content-text" class="mw-body-content"><div class="mw-content-ltr mw-parser-output" lang="en" dir="ltr"><div class="shortdescription nomobile noexcerpt noprint searchaux" style="display:none">Linux trojan malware with rootkit capabilities</div>
<p>
<b>XOR DDoS</b> is a Linux Trojan malware with rootkit capabilities that was used to launch large-scale DDoS attacks. Its name stems from the heavy usage of XOR encryption in both malware and network communication to the C&amp;Cs. It is built for multiple Linux architectures like ARM, x86 and x64. Noteworthy about XOR DDoS is the ability to hide itself with an embedded rootkit component which is obtained by multiple installation steps.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1"><span class="cite-bracket">&#91;</span>1<span class="cite-bracket">&#93;</span></a></sup> It was discovered in September 2014 by <a href="/wiki/MalwareMustDie" title="MalwareMustDie">MalwareMustDie</a>, a <a href="/wiki/White_hat_(computer_security)" title="White hat (computer security)">white hat</a> malware research group.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2"><span class="cite-bracket">&#91;</span>2<span class="cite-bracket">&#93;</span></a></sup><sup id="cite_ref-3" class="reference"><a href="#cite_note-3"><span class="cite-bracket">&#91;</span>3<span class="cite-bracket">&#93;</span></a></sup><sup id="cite_ref-4" class="reference"><a href="#cite_note-4"><span class="cite-bracket">&#91;</span>4<span class="cite-bracket">&#93;</span></a></sup> From November 2014 it was involved in massive brute force campaign that lasted at least for three months.<sup id="cite_ref-fireeye_5-0" class="reference"><a href="#cite_note-fireeye-5"><span class="cite-bracket">&#91;</span>5<span class="cite-bracket">&#93;</span></a></sup>
</p><p>In order to gain access it launches a brute force attack in order to discover the password to Secure Shell services on Linux.<sup id="cite_ref-thehackernews_6-0" class="reference"><a href="#cite_note-thehackernews-6"><span class="cite-bracket">&#91;</span>6<span class="cite-bracket">&#93;</span></a></sup> 
Once Secure Shell credentials are acquired and login is successful, it uses root privileges to run a script that downloads and installs XOR DDoS.<sup id="cite_ref-reuters_7-0" class="reference"><a href="#cite_note-reuters-7"><span class="cite-bracket">&#91;</span>7<span class="cite-bracket">&#93;</span></a></sup>
It is believed to be of Asian origin based on its targets, which tend to be located in Asia.<sup id="cite_ref-stateoftheinternet_8-0" class="reference"><a href="#cite_note-stateoftheinternet-8"><span class="cite-bracket">&#91;</span>8<span class="cite-bracket">&#93;</span></a></sup>
</p>
