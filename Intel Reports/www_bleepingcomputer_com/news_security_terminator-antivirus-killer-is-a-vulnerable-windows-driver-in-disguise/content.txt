<div class="article_section">
<h1>Terminator antivirus killer is a vulnerable Windows driver in disguise</h1>
<div class="cz-news-story-title-section">
<div class="cz-news-title-left-area">
By <h6><a rel="author" href="https://www.bleepingcomputer.com/author/sergiu-gatlan/" class="author"><span itemprop="author" itemscope itemtype="https://schema.org/Person"><span itemprop="name">Sergiu Gatlan</span></span></a></h6>
</div>
<meta itemprop="articleSection" content="Security">
<div class="cz-news-title-right-area">
<ul>
<li class="cz-news-date">May 31, 2023</li>
<li class="cz-news-time">03:25 PM</li>
<li class="cz-news-comment"><a href="#comment_form">0</a></li>
</ul>
</div>
</div>
<div class="articleBody">
<p style="text-align:center"><img alt="Cyber smile" height="900" src="https://www.bleepstatic.com/content/hl-images/2022/09/15/cyber-smiley-lower.jpg" width="1600"></p>
<p>A threat actor known as Spyboy is promoting a tool called "Terminator" on a Russian-speaking hacking forum that can allegedly terminate any antivirus, XDR, and EDR platform. However, CrowdStrike says&nbsp;that it's just a fancy Bring Your Own Vulnerable Driver (BYOVD) attack.</p>
<p>Terminator is&nbsp;<a href="https://twitter.com/S0ufi4n3/status/1663059373352943616" target="_blank" rel="nofollow noopener">allegedly capable</a>&nbsp;of bypassing 24 different antivirus (AV), Endpoint Detection and Response (EDR), and Extended Detection and Response (XDR) security solutions, including Windows Defender, on devices running Windows 7 and later,</p>
<p>Spyboy sells the software for prices ranging from $300 for a single bypass to $3,000 for an all-in-one bypass.</p>
<p>"The following EDRs cannot be sold alone: SentinelOne, Sophos, CrowdStrike, Carbon Black, Cortex, Cylance," the threat actor&nbsp;says, with a&nbsp;disclaimer that "Ransomware and lockers are not allowed and I'm not responsible for such actions."</p>
<p>To use Terminator, the "clients" require administrative privileges on the targeted Windows systems and have to trick the user into accepting a User Account Controls (UAC) pop-up that will be displayed when running the tool.</p>
<p>However, as a CrowdStrike engineer <a href="https://www.reddit.com/r/crowdstrike/comments/13wjrgn/20230531_situational_awareness_spyboy_defense/" target="_blank" rel="nofollow noopener">revealed</a> in a Reddit post, Terminator just drops the legitimate, signed Zemana anti-malware kernel driver named <a href="https://www.loldrivers.io/drivers/dfb0270d-4892-4fe5-97ed-0afd2e3fbe52/" target="_blank" rel="nofollow noopener">zamguard64.sys</a> or <a href="https://www.loldrivers.io/drivers/49920621-75d5-40fc-98b0-44f8fa486dcc/" target="_blank" rel="nofollow noopener">zam64.sys</a> into the C:\Windows\System32\ folder with a random name between 4 and 10 characters.</p>
<p>After the malicious driver is written to the disk, Terminator loads it to use its kernel-level privileges to kill off the user-mode processes of AV and EDR software running on the device.</p>
<p>While it is not clear how the Terminator program is interfacing with the driver, a&nbsp;<a data-sk="tooltip_parent" data-stringify-link="https://github.com/irql0/CVE-2021-31728" delay="150" href="https://github.com/irql0/CVE-2021-31728" rel="nofollow noopener noreferrer" target="_blank">PoC exploit was released in 2021</a>&nbsp;that exploits flaws in the driver to execute commands with Windows Kernel privileges, which could be used to terminate normally-protected security software processes.</p>
<p style="text-align:center"><a href="https://twitter.com/S0ufi4n3/status/1663059373352943616" target="_blank" rel="nofollow noopener"><img alt="Soufiane tweet Terminator" height="454" src="https://www.bleepstatic.com/images/news/u/1109292/2023/Soufiane_tweet_Terminator.png" width="557"></a></p>
<p>&#8203;This driver is only being detected by a single anti-malware scanning engine as a vulnerable driver at the moment, according to a&nbsp;<a href="https://www.virustotal.com/gui/file/543991ca8d1c65113dff039b85ae3f9a87f503daec30f46929fd454bc57e5a91" target="_blank" rel="nofollow noopener">VirusTotal scan</a>.</p>
<p>Luckily, Nextron Systems head of research Florian Roth and threat researcher Nasreddine Bencherchali have already shared&nbsp;<a href="https://github.com/magicsword-io/LOLDrivers/blob/main/detections/yara/yara-rules.yar" target="_blank" rel="nofollow noopener">YARA</a>&nbsp;and Sigma (<a href="https://github.com/magicsword-io/LOLDrivers/blob/main/detections/sigma/driver_load_win_vuln_drivers.yml" target="_blank" rel="nofollow noopener">by hash</a>&nbsp;and&nbsp;<a href="https://github.com/magicsword-io/LOLDrivers/blob/main/detections/sigma/driver_load_win_vuln_drivers_names.yml" target="_blank" rel="nofollow noopener">by name</a>) rules that can help defenders detect the vulnerable driver used by the Terminator tool.</p>
<p>This technique is prevalent among threat actors who like to install vulnerable Windows drivers after escalating privileges to bypass security software running on the compromised machines, execute malicious code, and deliver additional malicious payloads.</p>
<p>In Bring Your Own Vulnerable Driver (BYOVD) attacks, as they are known, legitimate drivers signed with valid certificates and capable of running with kernel privileges are dropped on the victims' devices to disable security solutions and take over the system.</p>
<p>A wide assortment of threat groups have used the technique for years, ranging from financially-motivated&nbsp;<a href="https://www.bleepingcomputer.com/news/security/blackbyte-ransomware-abuses-legit-driver-to-disable-security-products/" target="_blank">ransomware gangs</a>&nbsp;to&nbsp;<a href="https://www.bleepingcomputer.com/news/security/lazarus-hackers-abuse-dell-driver-bug-using-new-fudmodule-rootkit/" target="_blank">state-backed hacking outfits</a>.</p>
<p>More recently, Sophos X-Ops security researchers have spotted a&nbsp;<a href="https://www.bleepingcomputer.com/news/security/ransomware-gangs-abuse-process-explorer-driver-to-kill-security-software/" target="_blank">new hacking tool dubbed AuKill</a>&nbsp;used in the wild to disable EDR software with the help of a vulnerable Process Explorer driver before deploying ransomware in BYOVD attacks.</p>

function cz_strip_tags(input, allowed) {
  allowed = (((allowed || '') + '')
    .toLowerCase()
    .match(/<[a-z][a-z0-9]*>/g) || [])
    .join(''); // making sure the allowed arg is a string containing only tags in lowercase (<a><b><c>)
  var tags = /<\/?([a-z][a-z0-9]*)\b[^>]*>/gi,
    commentsAndPhpTags = /<!--[\s\S]*?-->|<\?(?:php)?[\s\S]*?\?>/gi;
  return input.replace(commentsAndPhpTags, '')
    .replace(tags, function($0, $1) {
      return allowed.indexOf('<' + $1.toLowerCase() + '>') > -1 ? $0 : '';
    });
}
function cz_br2nl(str) {
	var regex = /<br\s*[\/]?>/gi;
    //var pure_str = str.replace(regex,"\n");
    var pure_str = str.replace(regex,"");
    return cz_strip_tags(pure_str,'');
}

var prevScrollTop = 0;
$(window).scroll(function(event){
    scrollTop = $(this).scrollTop();

    if ( scrollTop < 0 ) {
        scrollTop = 0;
    }
    if ( scrollTop > $('body').height() - $(window).height() ) {
        scrollTop = $('body').height() - $(window).height();
    }

    if (scrollTop >= prevScrollTop && scrollTop) {
        $('header').addClass("nav-up");

	if (scrollTop >=100)
	{
	 $('header').removeClass("nav-up"); 
	 main_nav_hide_timer = setTimeout("call_main_nav_hide()",5000);
	}
	else
	{
	 $('header').removeClass("nav-up");
	 clearInterval(main_nav_hide_timer);
	}
    }

    prevScrollTop = scrollTop;
});
