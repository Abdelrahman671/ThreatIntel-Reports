<div style="text-align: left;">Posted by <b>Fareed Fauzi</b></div><h2 style="text-align: left;">Introduction</h2><div style="text-align: justify;">In late 2020, a sophisticated SolarWinds attack that hit organizations through the supply chain has recently been disclosed by various sources. This was done via a compromised version of SolarWinds Orion which we called the backdoor with the name &#8220;Sunburst&#8221;.&nbsp;Once the update (include the malicious DLL) is installed, the malicious DLL will be imported and loaded by the legitimate SolarWinds.BusinessLayerHost.exe executable.&nbsp;</div><div style="text-align: justify;"><br /></div><div style="text-align: justify;">Sunburst is a trojan version of a digitally signed SolarWinds Orion plugin named <b><i>SolarWinds.Orion.Core.BusinessLayer.dll</i></b>. The malicious DLL contains a backdoor code used to initiate a function that will do the communication with the victim&#8217;s system via HTTP to the attacker&#8217;s command and control server [1]. The malicious code initiation will give full access to the victim which may retrieve and execute commands that instruct the backdoor to transfer files, remote execution, profile victim&#8217;s system information, and complete control over the affected system.</div>
