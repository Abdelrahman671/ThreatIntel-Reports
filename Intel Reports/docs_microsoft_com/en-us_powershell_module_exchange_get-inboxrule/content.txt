	<div class="margin-block-sm">
		<p>This cmdlet is available in on-premises Exchange and in the cloud-based service. Some parameters and settings may be exclusive to one environment or the other.</p>
<p>Use the Get-InboxRule cmdlet to view Inbox rule properties. Inbox rules are used to process messages in the Inbox based on conditions specified and take actions such as moving a message to a specified folder or deleting a message.</p>
<p>For information about the parameter sets in the Syntax section below, see <a href="/en-us/powershell/exchange/exchange-cmdlet-syntax" data-linktype="absolute-path">Exchange cmdlet syntax</a>.</p>

	</div>

	<h2 id="syntax">Syntax</h2>
	<pre class="wrap"><code class="lang-powershell" dir="ltr">Get-Inbox<wbr>Rule
   [[-Identity] &lt;InboxRuleIdParameter&gt;]
   [-BypassScopeCheck]
   [-DescriptionTimeFormat &lt;String&gt;]
   [-DescriptionTimeZone &lt;ExTimeZoneValue&gt;]
   [-DomainController &lt;Fqdn&gt;]
   [-IncludeHidden]
   [-Mailbox &lt;MailboxIdParameter&gt;]
   [-ResultSize &lt;Unlimited&gt;]
   [-SkipCount &lt;Int32&gt;]
   [-SweepRules]
   [-UseCustomRouting]
   [&lt;CommonParameters&gt;]</code></pre>

	<h2 id="description">Description</h2>
	<p>You need to be assigned permissions before you can run this cmdlet. Although this topic lists all parameters for the cmdlet, you may not have access to some parameters if they're not included in the permissions assigned to you. To find the permissions required to run any cmdlet or parameter in your organization, see <a href="/en-us/powershell/exchange/find-exchange-cmdlet-permissions" data-linktype="absolute-path">Find the permissions required to run any Exchange cmdlet</a>.</p>
<p><strong>Note</strong>: This cmdlet doesn't work for members of View-Only Organization Management role group in Exchange Online or the Global Reader role in Microsoft Entra ID.</p>


	<h2 id="examples">Examples</h2>
	<h3 id="example-1">Example 1</h3>
	
	<pre><code class="lang-powershell">Get-InboxRule -Mailbox Joe@Contoso.com</code></pre>
	<p>This example retrieves all Inbox rules for the mailbox Joe@Contoso.com.</p>

	<h3 id="example-2">Example 2</h3>
	
	<pre><code class="lang-powershell">Get-InboxRule "ReceivedLastYear" -Mailbox joe@contoso.com -DescriptionTimeFormat "MM/dd/yyyy" -DescriptionTimeZone "Pacific Standard Time"</code></pre>
	<p>This example retrieves the Inbox rule ReceivedLastYear from the mailbox joe@contoso.com on which the ReceivedBeforeDate parameter was set when the rule was created. The DescriptionTimeFormat and DescriptionTimeZone parameters are used in this example to specify formatting of the time and the time zone used in the rule's Description property.</p>
