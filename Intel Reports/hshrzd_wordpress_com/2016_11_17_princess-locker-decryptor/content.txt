					<div class="entry-content">
						<p><strong><span style="color:#ff0000">[UPDATE: 19th March 2018] </span><em><span style="color:#ff0000">&#8211; I keep getting e-mails from people asking me why my decryptor doesn&#8217;t work. Please understand, this is an obsolete tool, it was written in 2016 for the FIRST VERSION of&nbsp; Princess Locker. The current version is improved and no longer decryptable.</span><br />
</em></strong></p>
<p><strong>[UPDATE: 28th Nov 2016] <em>&#8211; unfortunately, recently a new variant appeared, that fixed the bug which allowed me crack this ransomware. If generating the key takes more than few minutes,&nbsp; it probably means that you has been infected by the new version of Princess. I am sorry, but I am not capable of helping in such case.<br />
</em></strong></p>
<p><em>If you are a researcher curious how I cracked it, you can see the decryptor&#8217;s source code: <a href="https://github.com/hasherezade/decryptors_archive/tree/master/princesslocker_decrypt" target="_blank" rel="noopener">https://github.com/hasherezade/decryptors_archive/tree/master/princesslocker_decrypt</a></em></p>

<hr>
<h2>HOW TO USE</h2>
<p>In order to use the keygen you must find one file, that you can provide in both forms: unencrypted and encrypted. You also need to supply the added extension. It is beneficial (but not required) to supply the unique ID from your ransom note.<br />
Usage:</p>
<pre>PrincessKeygen.exe [encrypted file] [original file] [added extension] [*unique id]
</pre>
<p>* &#8211; optional parameter</p>
<p><span style="text-decoration:underline"><strong>Example:</strong></span></p>
<p>Read the data from your ransom note:</p>
<p><img data-attachment-id="3063" data-permalink="https://hshrzd.wordpress.com/2016/11/17/princess-locker-decryptor/ransom_id/" data-orig-file="https://hshrzd.wordpress.com/wp-content/uploads/2016/11/ransom_id.png" data-orig-size="468,139" data-comments-opened="1" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;,&quot;orientation&quot;:&quot;0&quot;}" data-image-title="ransom_id" data-image-description="" data-image-caption="" data-medium-file="https://hshrzd.wordpress.com/wp-content/uploads/2016/11/ransom_id.png?w=400" data-large-file="https://hshrzd.wordpress.com/wp-content/uploads/2016/11/ransom_id.png?w=468" loading="lazy" class="alignnone size-full wp-image-3063" src="https://hshrzd.wordpress.com/wp-content/uploads/2016/11/ransom_id.png?w=640" alt="ransom_id"   srcset="https://hshrzd.wordpress.com/wp-content/uploads/2016/11/ransom_id.png 468w, https://hshrzd.wordpress.com/wp-content/uploads/2016/11/ransom_id.png?w=150&amp;h=45 150w, https://hshrzd.wordpress.com/wp-content/uploads/2016/11/ransom_id.png?w=400&amp;h=119 400w" sizes="(max-width: 468px) 100vw, 468px"></p>
<p>And supply them to the keygen:</p>
<pre>PrincessKeygen.exe "square1.bmp.xauwk" "square1.bmp" xauwk ujivtjf25pwt
</pre>
<p><strong>What if you don&#8217;t have any original file?</strong></p>
<p>In case if you don&#8217;t have the original copy of any of your encrypted files, you can use an encrypted file of one of the following formats:</p>
<p>doc, png, gif, pdf, docx, xlsx, ppt, xls</p>
<p>Then, instead of the original file, supply the preprepared header &#8211; you can find the set <a href="https://drive.google.com/open?id=0Bzb5kQFOXkiSR1RoVzBFdUstazg" target="_blank" rel="noopener">here</a>. However, this method may, in some rare cases, produce invalid results &#8211; so, supplying the original file is recommended.</p>
<p>Example:</p>
<p><img data-attachment-id="3078" data-permalink="https://hshrzd.wordpress.com/2016/11/17/princess-locker-decryptor/princess_keygen/" data-orig-file="https://hshrzd.wordpress.com/wp-content/uploads/2016/11/princess_keygen.png" data-orig-size="675,342" data-comments-opened="1" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;,&quot;orientation&quot;:&quot;0&quot;}" data-image-title="princess_keygen" data-image-description="" data-image-caption="" data-medium-file="https://hshrzd.wordpress.com/wp-content/uploads/2016/11/princess_keygen.png?w=400" data-large-file="https://hshrzd.wordpress.com/wp-content/uploads/2016/11/princess_keygen.png?w=640" loading="lazy" class="alignnone size-full wp-image-3078" src="https://hshrzd.wordpress.com/wp-content/uploads/2016/11/princess_keygen.png?w=640" alt="princess_keygen"   srcset="https://hshrzd.wordpress.com/wp-content/uploads/2016/11/princess_keygen.png 675w, https://hshrzd.wordpress.com/wp-content/uploads/2016/11/princess_keygen.png?w=150&amp;h=76 150w, https://hshrzd.wordpress.com/wp-content/uploads/2016/11/princess_keygen.png?w=400&amp;h=203 400w" sizes="(max-width: 675px) 100vw, 675px"></p>
<p><strong>What if you don&#8217;t have the ransom note?</strong></p>
<p>It&#8217;s OK. Just supply the extension &#8211; but be warned that cracking may take a bit longer.</p>
<p><img data-attachment-id="3083" data-permalink="https://hshrzd.wordpress.com/2016/11/17/princess-locker-decryptor/no_note/" data-orig-file="https://hshrzd.wordpress.com/wp-content/uploads/2016/11/no_note.png" data-orig-size="598,385" data-comments-opened="1" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;,&quot;orientation&quot;:&quot;0&quot;}" data-image-title="no_note" data-image-description="" data-image-caption="" data-medium-file="https://hshrzd.wordpress.com/wp-content/uploads/2016/11/no_note.png?w=400" data-large-file="https://hshrzd.wordpress.com/wp-content/uploads/2016/11/no_note.png?w=598" loading="lazy" class="alignnone size-full wp-image-3083" src="https://hshrzd.wordpress.com/wp-content/uploads/2016/11/no_note.png?w=640" alt="no_note"   srcset="https://hshrzd.wordpress.com/wp-content/uploads/2016/11/no_note.png 598w, https://hshrzd.wordpress.com/wp-content/uploads/2016/11/no_note.png?w=150&amp;h=97 150w, https://hshrzd.wordpress.com/wp-content/uploads/2016/11/no_note.png?w=400&amp;h=258 400w" sizes="(max-width: 598px) 100vw, 598px"></p>
<hr>
<p>Check if your output file is valid. If so, save the key and use it to decrypt rest of your files, with the help of PrincessDecryptor.</p>
<p>Usage:</p>
<pre>PrincessDecryptor.exe [key] [ransom extension] [*file/directory] 
</pre>
<p>* &#8211; optional parameter &#8211; default is current directory</p>
