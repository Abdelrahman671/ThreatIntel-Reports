    <div class="node-full__body blog-content__body">  <h3 aria-level="3" paraeid="{5e42ce3a-aa2f-4079-8bb4-91f2cca9c4a6}{138}" paraid="353122204" role="heading">Key findings&nbsp;</h3>

<ul>
	<li paraeid="{5e42ce3a-aa2f-4079-8bb4-91f2cca9c4a6}{144}" paraid="588113332">Proofpoint has observed an increase in malware delivery via TryCloudflare Tunnel abuse.&nbsp;</li>
	<li paraeid="{5e42ce3a-aa2f-4079-8bb4-91f2cca9c4a6}{154}" paraid="870065390">The activity is financially motivated and delivers exclusively remote access trojans (RATs).&nbsp;&nbsp;</li>
	<li paraeid="{1e2799d2-7b02-4d39-8feb-88fa0b2fb13c}{185}" paraid="230331499">Since initial observation, the threat activity set behind the campaigns has modified tactics, techniques, and procedures in attempts to bypass detection and improve efficacy.&nbsp;</li>
	<li paraeid="{5e42ce3a-aa2f-4079-8bb4-91f2cca9c4a6}{170}" paraid="6549620">Proofpoint does not attribute this activity to a tracked TA, but research is ongoing.&nbsp;</li>
</ul>

<h3 aria-level="3" paraeid="{5e42ce3a-aa2f-4079-8bb4-91f2cca9c4a6}{176}" paraid="849584846" role="heading">Overview&nbsp;</h3>

<p paraeid="{5e42ce3a-aa2f-4079-8bb4-91f2cca9c4a6}{182}" paraid="878739094">Proofpoint is tracking a cluster of cybercriminal threat activity leveraging Cloudflare Tunnels to deliver malware. Specifically, the activity abuses the TryCloudflare feature that allows an attacker to create a one-time tunnel without creating an account. Tunnels are a way to remotely access data and resources that are not on the local network, like using a virtual private network (VPN) or secure shell (SSH) protocol.&nbsp;&nbsp;</p>

<p paraeid="{5e42ce3a-aa2f-4079-8bb4-91f2cca9c4a6}{204}" paraid="1961165670">First observed in February 2024, the cluster increased activity in May through July, with most campaigns leading to Xworm, a remote access trojan (RAT), in recent months.&nbsp;&nbsp;</p>

<p paraeid="{5e42ce3a-aa2f-4079-8bb4-91f2cca9c4a6}{216}" paraid="281045025">In most campaigns, messages contain a URL or attachment leading to an internet shortcut (.URL) file. When executed, it establishes a connection to an external file share, typically via WebDAV, to download an LNK or VBS file. When executed, the LNK/VBS executes a BAT or CMD file that downloads a Python installer package and a series of Python scripts leading to malware installation. In some cases, file staging leverages the <a href="https://attack.mitre.org/techniques/T1105/" rel="noreferrer noopener" target="_blank">search-ms</a> protocol handler to retrieve the LNK from a WebDAV share. Typically in campaigns, a benign PDF is displayed to the user to appear legitimate.&nbsp;&nbsp;&nbsp;</p>

<p paraeid="{fb2b37cb-4ab7-4cdc-8589-2dbf0fc2438a}{2}" paraid="1037406766">In June and July, nearly all observed campaigns delivered Xworm, but previous campaigns also delivered AsyncRAT, VenomRAT, GuLoader, and Remcos. Some campaigns will lead to multiple different malware payloads, with each unique Python script leading to the installation of a different malware.&nbsp;</p>

<p paraeid="{fb2b37cb-4ab7-4cdc-8589-2dbf0fc2438a}{2}" paraid="1037406766"><img alt="Malware observed in related campaigns leveraging “trycloudflare” tunnels.&nbsp;&nbsp;" data-entity-type="file" data-entity-uuid="f3a3c810-f10a-4baf-903e-25d54e40ad68" height="687" src="/sites/default/files/inline-images/Screenshot%202024-07-25%20at%2011.56.08%E2%80%AFAM_0.png" width="929" loading="lazy"></p>

<p paraeid="{fb2b37cb-4ab7-4cdc-8589-2dbf0fc2438a}{2}" paraid="1037406766"><em>Malware observed in related campaigns leveraging “trycloudflare” tunnels.&nbsp;&nbsp;</em></p>

<p paraeid="{fb2b37cb-4ab7-4cdc-8589-2dbf0fc2438a}{65}" paraid="1402929524">Campaign message volumes range from hundreds to tens of thousands of messages impacting dozens to thousands of organizations globally. In addition to English, researchers observed French, Spanish, and German language lures. Xworm, AsyncRAT, and VenomRAT campaigns are often higher volume than campaigns delivering Remcos or GuLoader. Lure themes vary, but typically include business-relevant topics like invoices, document requests, package deliveries, and taxes.&nbsp;&nbsp;</p>

<p paraeid="{fb2b37cb-4ab7-4cdc-8589-2dbf0fc2438a}{99}" paraid="568078709">While the tactics, techniques and procedures (TTPs) of the campaigns remain consistent, the threat actor does appear to modify different parts of the attack chain to increase sophistication and defense evasion. For example, initial campaigns used little to no obfuscation in their helper scripts. The scripts often included detailed comments about the functionality of the code. However, this changed in June when the threat actors began to incorporate obfuscation in their code.&nbsp;</p>

<p paraeid="{fb2b37cb-4ab7-4cdc-8589-2dbf0fc2438a}{139}" paraid="1037717467"><img alt="Helper script without obfuscation (May 2024 campaign example)." data-entity-type="file" data-entity-uuid="8d25664b-9e6b-4748-9a16-c0192d92caa6" height="834" src="/sites/default/files/inline-images/Screenshot%202024-07-25%20at%2012.05.42%E2%80%AFPM.png" width="1012" loading="lazy"></p>

<p paraeid="{fb2b37cb-4ab7-4cdc-8589-2dbf0fc2438a}{160}" paraid="1919149148"><em>Helper script without obfuscation (May 2024 campaign example).&nbsp;</em></p>

<p paraeid="{fb2b37cb-4ab7-4cdc-8589-2dbf0fc2438a}{160}" paraid="1919149148"><em><img alt="Helper script with obfuscation (June 2024 campaign example).&nbsp;&nbsp;" data-entity-type="file" data-entity-uuid="4dbe5c8f-2c7a-4eee-a98c-2ba26eea5096" height="621" src="/sites/default/files/inline-images/Screenshot%202024-07-25%20at%2012.06.50%E2%80%AFPM_0.png" width="1026" loading="lazy"></em></p>

<p aria-level="3" paraeid="{fb2b37cb-4ab7-4cdc-8589-2dbf0fc2438a}{245}" paraid="105555160" role="heading"><span style="font-size:12pt"><span style="line-height:116%"><span style="font-family:Aptos,sans-serif"><em>Helper script with obfuscation (June 2024 campaign example).</em> </span></span></span></p>

<p aria-level="3" paraeid="{fb2b37cb-4ab7-4cdc-8589-2dbf0fc2438a}{245}" paraid="105555160" role="heading">Threat actor abuse of <a href="https://developers.cloudflare.com/cloudflare-one/connections/connect-networks/do-more-with-tunnels/trycloudflare/" rel="noreferrer noopener" target="_blank">TryCloudflare</a> tunnels <a href="https://www.bleepingcomputer.com/news/security/malicious-pypi-packages-create-cloudflare-tunnels-to-bypass-firewalls/" rel="noreferrer noopener" target="_blank">became popular in 2023</a> and appears to be increasing among cybercriminal threat actors. Each use of TryCloudflare Tunnels will generate a random subdomain on trycloudflare[.]com, for example ride-fatal-italic-information[.]trycloudflare[.]com. Traffic to the subdomains is proxied through Cloudflare to the operators’ local server.&nbsp;</p>

<h3 aria-level="3" paraeid="{fb2b37cb-4ab7-4cdc-8589-2dbf0fc2438a}{245}" paraid="105555160" role="heading">Campaign examples&nbsp;</h3>

In this campaign, tax-themed messages contained URLs leading to a zipped .URL file. The campaign targeted organizations in law and finance and included less than 50 total messages.&nbsp;</p>

<h3 aria-level="2" paraeid="{a70f6812-0e32-4519-95a8-29922581265b}{23}" paraid="1478802782" role="heading">Attribution&nbsp;</h3>

<p paraeid="{a70f6812-0e32-4519-95a8-29922581265b}{29}" paraid="344042252">Based on the tactics, techniques and procedures (TTPs) observed in campaigns, Proofpoint assesses they can be attributed to one cluster of related activity. Researchers have not attributed a specific threat actor to this activity, but research is ongoing. &nbsp;</p>

<h3 paraeid="{a70f6812-0e32-4519-95a8-29922581265b}{29}" paraid="344042252">Why it matters&nbsp;&nbsp;</h3>

<p paraeid="{a70f6812-0e32-4519-95a8-29922581265b}{43}" paraid="1838028986">The use of Cloudflare tunnels provide the threat actors a way to use temporary infrastructure to scale their operations providing flexibility to build and take down instances in a timely manner. This makes it harder for defenders and traditional security measures such as relying on static blocklists. Temporary Cloudflare instances allow attackers a low-cost method to stage attacks with helper scripts, with limited exposure for detection and takedown efforts.&nbsp;</p>

<p paraeid="{a70f6812-0e32-4519-95a8-29922581265b}{79}" paraid="1849682704">Attackers’ use of Python scripts for malware delivery is notable. Packaging Python libraries and an executable installer alongside the Python scripts ensures the malware can be downloaded and run on hosts that did not previously have Python installed. Organizations should restrict the use of Python if it is not required for individuals’ job functions. This is not the first-time researchers have observed software packages delivered alongside malware files. In recent months Proofpoint has observed campaigns delivering Java-based malware that bundle a JAR and the Java Runtime Environment (JRE) inside a ZIP to ensure the correct software is installed before executing the downloader or dropper.&nbsp;</p>

<p paraeid="{a70f6812-0e32-4519-95a8-29922581265b}{151}" paraid="579202293">The attack chain requires significant victim interaction in order to detonate the final payload, including clicking on the malicious link, double clicking on multiple files such as the LNK or VBS files, and unzipping compressed scripts. This gives the recipient multiple opportunities to identify suspicious activity and disrupt the attack chain before successful execution.&nbsp;&nbsp;</p>

<p paraeid="{a70f6812-0e32-4519-95a8-29922581265b}{171}" paraid="331722508">Threat actors are increasingly using WebDAV and Server Message Block (SMB) for payload staging and delivery as the cybercriminal ecosystem continues to experiment with different TTPs. Organizations should restrict access to external file sharing services to only known, safelisted servers.&nbsp;&nbsp;</p>
