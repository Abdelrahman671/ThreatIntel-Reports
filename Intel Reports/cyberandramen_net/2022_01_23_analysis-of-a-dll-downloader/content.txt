			data = JSON.parse( e.data );
		} catch( e ) {
			wpcom_remote_login_final_cleanup();
			return;

			return;

		if ( data.msg === 'REQUEST_ACCESS' ) {
			console.log( 'request access: safari' );

			iframe.setAttribute( 'scrolling', 'no' );

<h2 class="wp-block-heading" id="summary">Summary</h2>



<ul class="wp-block-list"><li>SHA256: dedb8516befa4a5088000b8c7f699dae7f33761403dd355a14684ac89ff56a9a</li></ul>



<ul class="wp-block-list"><li>Filename: Unknown</li><li>Filetype: DLL</li><li>File size: 39KB</li></ul>



<p>From here on, the above DLL will be referred to as &#8220;downloader.dll&#8221;. The file is capable of:</p>



<ul class="wp-block-list"><li>Downloading files</li><li>Interacting with a C2 server</li></ul>



<h2 class="wp-block-heading" id="malware-overview">Malware Overview</h2>



<p>This is an older file that was first identified around October 2021. </p>



<p>Downloader.dll is a downloader capable of downloading a file from a hardcoded command and control (C2) server. A follow-on stage was not identified.</p>



<p>The file disguises itself as an extension for Foxit Reader software. Foxit is a software that develops document formatting tools and is based in the U.S. and China.</p>



<figure class="wp-block-image size-large"><img data-attachment-id="277" data-permalink="https://cyberandramen.net/2022-01-23_16-39/" data-orig-file="https://cyberandramen.net/wp-content/uploads/2022/01/2022-01-23_16-39.png" data-orig-size="713,971" data-comments-opened="0" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;,&quot;orientation&quot;:&quot;0&quot;}" data-image-title="2022-01-23_16-39" data-image-description="" data-image-caption="" data-medium-file="https://cyberandramen.net/wp-content/uploads/2022/01/2022-01-23_16-39.png?w=220" data-large-file="https://cyberandramen.net/wp-content/uploads/2022/01/2022-01-23_16-39.png?w=713" width="713" height="971" src="https://cyberandramen.net/wp-content/uploads/2022/01/2022-01-23_16-39.png?w=713" alt="" class="wp-image-277" srcset="https://cyberandramen.net/wp-content/uploads/2022/01/2022-01-23_16-39.png 713w, https://cyberandramen.net/wp-content/uploads/2022/01/2022-01-23_16-39.png?w=110 110w, https://cyberandramen.net/wp-content/uploads/2022/01/2022-01-23_16-39.png?w=220 220w" sizes="(max-width: 713px) 100vw, 713px" /><figcaption>Figure 1</figcaption></figure>



<p>Upon running the DLL, a request is made to online-manual.c1[.]biz, and a file is downloaded to %TEMP%. Looking at the strings output of the file, there is a large base64 encoded string that when decoded, appears to be a batch file as seen below.</p>



<pre class="wp-block-code"><code>@echo off

cd /d %TEMP%
:WAITING
timeout /t 1
if not exist "a.log" (goto WAITING)
del /f /q "a.log"
install.bat
del /f /q "%~dpnx0"</code></pre>



<p>Again analyzing just the strings, we see a call to run the batch file via the following command:</p>



<pre class="wp-block-code"><code>cmd /c cd /d "%TEMP%" &amp;&amp; temp.bat</code></pre>



<p>One can only assume that the downloaded file is run with another command run via the DLL file:</p>



<pre class="wp-block-code"><code>cmd /c expand "%s" -F:* "%s" &amp;&amp; del /f /q "%s" &amp;&amp; echo OK &gt; a.log</code></pre>



<p>The encoded strings represented by &#8220;%s&#8221; are likely the downloaded file. We can see from the above output that the file is deleted upon the above command completing.</p>



<h2 class="wp-block-heading" id="network-indicators">Network Indicators</h2>

<p>The above infrastructure was tied to a possible Konni campaign by Black Lotus Labs in late November 2021.</p>

<h2 class="wp-block-heading" id="basic-snort-suricata-rule">Basic Snort/Suricata Rule</h2>



<p>alert tcp $HOME_NET any -&gt;  any any (msg: &#8220;Probaly shouldnt run in production. Possible Konni DLL download URL,&#8221;; content: &#8220;index.php/user_id=&#8221;; content:&#8221;&amp;type=&#8221;; threshold:type limit, track by_src, count 1, seconds m; sid:999999</p>
