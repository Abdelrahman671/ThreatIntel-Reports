						<h1 class="post-title single_blog_inner__Title">
							Kimsuky Group&#8217;s Malware  Disguised as Lecture Request Form (MSC, HWP)						</h1>

					<!-- Article content -->
					<div class="entry-content clearfix">
						
						<p><span style="font-size:10.0pt;">Recently, malware disguised as a lecture request form targeting specific users was identified.&nbsp;</span></p>
<p>The distributed files include Hangul Word Processor (HWP) documents and files in MSC format, which download additional malicious files. Decoy document files used to disguise as legitimate documents have been found to sometimes contain personal information, suggesting that the malware targets specific users. Although the final malicious behaviors have not been confirmed, a script containing the threat actor&#8217;s commands is stored and continuously executed on the user&#8217;s PC, which could lead to various malicious behaviors such as information leakage and downloading additional malicious code.</p>
<p>The format of the malicious URLs used by this malware is similar to those analyzed in AhnLab&#8217;s ASEC blog post in 2023, &#8220;Malicious Batch File (*bat) Disguised as a Document Viewer Being Distributed (Kimsuky) <a href="https://asec.ahnlab.com/en/55219/">[1]</a>&#8220;, suggesting that it is suspected to be the same attack group.</p>
<p>&nbsp;This document explains the behavior of the distributed files and their subsequent actions.</p>
<figure class="image">
 <img decoding="async" src="https://image.ahnlab.com/atip/content/image/20241101/UVVv211zJC9cSabrzdTQsjrSM77DiRZMvBpnTTqV.png"><br />
</figure>
<p style="text-align:center;">Figure 1. Malware operation process</p>
<p>&nbsp;</p>
<p>A characteristic of the operation process is the use of a legitimate executable file and a malicious script file disguised as the executable&#8217;s manifest file. Additionally, the malware uses Google Drive to receive additional malicious commands by inserting encoded malicious commands into the title of uploaded files to carry out malicious behaviors.</p>
<p>&nbsp;</p>
<p>The malware is suspected to be primarily distributed through spear phishing, with Hangul Word Processor (HWP) documents and MSC files being identified. The confirmed distributed file names are as follows.</p>
