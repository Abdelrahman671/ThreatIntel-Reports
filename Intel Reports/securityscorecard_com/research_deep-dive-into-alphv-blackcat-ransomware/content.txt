					<h2>Executive summary</h2>
<p>ALPHV/BlackCat is the first widely known ransomware written in Rust. The <a href="https://securityscorecard.com/research/apt28s-stealer-called-credomap/">malware</a> must run with an access token consisting of a 32-byte value (–access-token parameter), and other parameters can be specified. The ransomware comes with an encrypted configuration that contains a list of services/processes to be stopped, a list of whitelisted directories/files/file extensions, and a list of stolen credentials from the victim environment. It deletes all Volume Shadow Copies, performs privilege escalation using the CMSTPLUA COM interface, and enables “remote to local” and “remote to remote” symbolic links on the victim’s<br />
machine.</p>
<p>The files are encrypted using the AES algorithm, with the AES key being encrypted using the RSA public key contained in the configuration. The extension of the encrypted files is changed to uhwuvzu by the malware.</p>
<h2>Analysis and findings</h2>
<p>SHA256: 847fb7609f53ed334d5affbb07256c21cb5e6f68b1cc14004f5502d714d2a456</p>
<p>The malware can run with one of the following parameters:</p>
