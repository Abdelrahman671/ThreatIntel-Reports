<h1 class="blog-banner__heading">
      <span>Ugg Boots 4 Sale: A Tale of Palestinian-Aligned Espionage </span>

  </h1>

    </div>

    <div class="node-full__body blog-content__body">  <p><meta charset="utf-8"></p>

<h3>Key Takeaways&nbsp;&nbsp;</h3>

<ul>
	<li>TA402, a likely Palestinian-aligned <a data-entity-substitution="canonical" data-entity-type="node" data-entity-uuid="3c00835f-2d9f-46b7-9029-de06c9ce450b" href="/us/threat-reference/advanced-persistent-threat" title="Advanced Persistent Threat">advance persistent threat</a> actor, has recently engaged in campaigns leveraging a new implant, dubbed by Proofpoint analysts as NimbleMamba.&nbsp;</li>
	<li>NimbleMamba is likely a replacement for the groups previously used LastConn implant.&nbsp;&nbsp;</li>
	<li>These campaigns have a complex attack chain that leverages geofencing and URL redirects to legitimate sites in order to bypass detection efforts.&nbsp;</li>
</ul>

<h3>Overview&nbsp;</h3>

<p>In late 2021, Proofpoint analysts identified a complex attack chain targeting Middle Eastern governments, foreign policy think tanks, and a state-affiliated airline. Over three months, Proofpoint observed three subtle variations of this attack chain. Proofpoint attributes these campaigns to TA402, an actor commonly tracked as&nbsp;<a href="https://www.proofpoint.com/us/blog/threat-insight/new-ta402-molerats-malware-targets-governments-middle-east" target="_blank">Molerats</a>&nbsp;and believed to be operating in the interest of the Palestinian Territories. Based on Proofpoints research, TA402 is a persistent threat to organizations and governments in the Middle East, routinely updating not only their <a data-entity-substitution="canonical" data-entity-type="node" data-entity-uuid="5312929a-c85d-4291-a361-e34378e7fdff" href="/us/threat-reference/malware" title="Malware">malware</a> implants, but also their delivery methods. After publication of Proofpoints TA402 research in June 2021, TA402 appeared to halt its activities for a short period of time, almost certainly to retool. Proofpoint researchers believe they used that time to update their implants and delivery mechanisms, using malware dubbed NimbleMamba and BrittleBush. TA402 also regularly uses geofencing techniques and varied attack chains which complicate detection efforts for defenders.&nbsp;&nbsp;</p>

<h3>Campaign Details&nbsp;</h3>

<p><img alt="TA402" data-entity-type="file" data-entity-uuid="1339ca66-d61d-4083-be29-1e0cdab43a94" src="/sites/default/files/inline-images/Screen%20Shot%202022-02-07%20at%202.46.54%20PM.png" width="1158" height="501" loading="lazy"></p>

<p><em>Figure 1. TA402 attack chain November 2021 to January 2022.&nbsp;</em></p>

<p>In the recently observed campaigns, TA402 used <a data-entity-substitution="canonical" data-entity-type="node" data-entity-uuid="cb24684f-298a-4be0-a321-0cf284caa322" href="/us/threat-reference/spear-phishing" title="Spear Phishing">spear phishing emails</a> containing links that often lead to malicious files. Proofpoint observed three different URL types in those campaigns.&nbsp;</p>

<p><strong>Variation 1: Actor-Controlled Domain (November 2021)&nbsp;</strong></p>

<p>In a November 2021 campaign, TA402 masqueraded as the Quora website while using an actor-controlled Gmail account with an actor-controlled domain. The malicious URL, such as https[:]//www[.]uggboots4sale[.]com/news15112021.php, in the <a data-entity-substitution="canonical" data-entity-type="node" data-entity-uuid="3d41a44d-bc21-401b-8912-4f84e4e683ce" href="/us/threat-reference/phishing" title="Phishing">phishing email</a> was geofenced to the targeted countries. If the target's IP address fits into the targeted region, the user would be redirected to the RAR file download containing the latest TA402 implant, NimbleMamba. If outside the target area, the user would be redirected to a legitimate news site, Figure 2.&nbsp;</p>

<p><img alt="Redirect" data-entity-type="file" data-entity-uuid="3df1a0ec-a943-4613-98ff-0ca4b9b28014" src="/sites/default/files/inline-images/Screen%20Shot%202022-02-07%20at%202.38.50%20PM.png" width="703" height="434" loading="lazy"></p>

<p><em>Figure 2. Benign redirect to legitimate news site https[:]www[.]emaratalyoum[.]com.&nbsp;</em></p>

<p><strong>Variation 2: Dropbox URL (December 2021)&nbsp;</strong></p>

<p>In December 2021, TA402 used multiple phishing pretenses, including clickbait medical lures and ones allegedly sharing confidential geopolitical information. TA402 continued to use an actor-controlled Gmail account but shifted to Dropbox URLs to deliver the malicious RAR files containing NimbleMamba. This shift away from actor-controlled domains meant that TA402 could no longer geofence their payloads. Proofpoint discovered that TA402 is not only abusing Dropbox services for delivery of NimbleMamba, but also for malware command and control (C2). Proofpoint has shared our investigation and analysis with Dropbox prior to publication, and they took the needed actions for neutralizing the activity within their organization.&nbsp;</p>

<p><strong>Variation 3: WordPress Redirect Actor-Controlled Domain (December 2021/January 2022)&nbsp;</strong></p>

<p>In their latest campaigns, TA402 continued to use lure content customized for each of their targets but slightly adjusted their attack chain by inserting an additional actor-controlled WordPress URL. That WordPress site (Figure 3), which impersonates a news aggregator of the legitimate news site from Variation 1, likely redirects to the download site of the malicious RAR files containing NimbleMamba if the visitor is coming from an IP within the targeted region. If the source IP address does not align with the target region, the URL will redirect the recipient to a benign website, typically an Arabic language news website (Figure 2).&nbsp;</p>

<p><img alt="WordPress site " data-entity-type="file" data-entity-uuid="39c12a5d-fd50-43aa-bbcb-89884e451d10" src="/sites/default/files/inline-images/Screen%20Shot%202022-02-07%20at%202.39.17%20PM.png" width="699" height="521" loading="lazy"></p>

<p><em>Figure 3. Example WordPress site (https[:]//emaratalyoumcom[.]wordpress[.]com/) impersonating an Arabic language news aggregator.&nbsp;</em></p>

<p>The use of geofenced URLs, Dropbox URLs and then redirect URLs demonstrate TA402s determination to blend in with legitimate email traffic and infect targets with NimbleMamba.&nbsp;</p>

<h3>Malware Analysis: NimbleMamba&nbsp;&nbsp;</h3>

<p>Each variant of TA402s attack chain led to a RAR file containing one or multiple malicious compressed executables. These executables include a TA402 implant Proofpoint dubbed NimbleMamba and oftentimes an additional trojan Proofpoint named BrittleBush. NimbleMamba is almost certainly meant to replace LastConn, which Proofpoint&nbsp;<a href="https://www.proofpoint.com/us/blog/threat-insight/new-ta402-molerats-malware-targets-governments-middle-east" target="_blank">reported</a>&nbsp;about in June 2021. LastConn was likely an updated version of the SharpStage malware,&nbsp;<a href="https://www.cybereason.com/hubfs/dam/collateral/reports/Molerats-in-the-Cloud-New-Malware-Arsenal-Abuses-Cloud-Platforms-in-Middle-East-Espionage-Campaign.pdf" target="_blank">reported</a>&nbsp;by Cybereason in December 2020. While NimbleMamba and LastConn have some similarities, such as being written in C#, base64 encoding within the C2 framework, and use of the Dropbox API for C2 communication, there appears to be little code overlap between the two.&nbsp;&nbsp;</p>

<p>NimbleMamba uses guardrails to ensure that all infected victims are within TA402s target region. NimbleMamba uses the Dropbox API for both command and control as well as exfiltration. The malware also contains multiple capabilities designed to complicate both automated and manual analysis. Based on this, Proofpoint assesses NimbleMamba is actively being developed, is well-maintained, and designed for use in highly targeted intelligence collection campaigns.&nbsp;&nbsp;</p>

<p>NimbleMamba is written in C# and delivered as an obfuscated .NET executable using third-party obfuscators. Both samples analyzed used the&nbsp;<a href="https://www.red-gate.com/products/dotnet-development/smartassembly/" target="_blank">SmartAssembly</a>&nbsp;obfuscator. Additionally, the malware does basic virtual machine checks to avoid detection by looking for common strings that indicate a sample is running in a virtual environment.&nbsp;</p>

<h3>Guardrails&nbsp;</h3>

<p>NimbleMamba contains multiple guardrails to ensure that the malware only executes on targeted machines. It uses the following IP resolving web services to check the users IP address and determine if it fits into the target region. This is done to avoid detection and analysis.&nbsp;</p>

<ul>
	<li>api[.]ipify[.]com (<a href="https://www.ipify.org/" target="_blank">https://www.ipify.org</a>)&nbsp;</li>
	<li>myexternalip[.]com (<a href="https://myexternalip.com/" target="_blank">https://myexternalip.com</a>)&nbsp;</li>
	<li>ip-api[.]com (<a href="https://ip-api.com/" target="_blank">https://ip-api.com</a>)&nbsp;</li>
	<li>api[.]ipstack[.]com (<a href="https://ipstack.com/" target="_blank">https://ipstack.com</a>)&nbsp;</li>
</ul>

<p>If the machine is unable to connect to those services, the malware will keep calling the addresses in random order, thus putting the execution in an endless loop in closed network environments.&nbsp;</p>

<p>ACSS contains the obfuscated Dropbox account API auth key that is used for C2 communication. The malware then takes the external IP address, username and computer name retrieved earlier, writes them as comma-separated strings, base64 encodes them with stripped padding bytes and then reverses the string. The resulting string is used as a folder name that is created on the Dropbox account using their API with the API key deobfuscated (Figure 6) from the JustPasteIt post.&nbsp;&nbsp;</p>

<p>From there, the malware starts communicating with Dropbox to obtain a RAR file and a decoy file that is immediately displayed to the user if present. The decoy file is often an office document or PDF. The RAR file is password-protected with a password stored as the second comma-separated value in the OOOO argument from the JustPasteIt paste and dropped to the folder pointed by the first parameter in OOOO. The downloaded RAR file contains two additional executables, an updated sample of NimbleMamba along with an executable that contains a screenshot of the functionality. This technique allows for TA402 to serve additional payloads to targeted NimbleMamba victims.&nbsp;</p>

<p>NimbleMamba has the traditional capabilities of an intelligence-gathering trojan and is likely designed to be the initial access. <h3>Conclusion&nbsp;</h3>

<p>TA402 continues to be an effective threat actor that demonstrates its persistence with its highly targeted campaigns focused on the Middle East. Based on the variations between campaigns delivering NimbleMamba, along with the historical pattern of developing new malware post disclosure, Proofpoint judges with moderate confidence that TA402 will continue to update both their implants and infection chains to complicate defensive efforts.&nbsp;&nbsp;</p>
