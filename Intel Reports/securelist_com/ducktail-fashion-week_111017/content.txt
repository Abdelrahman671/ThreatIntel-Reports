											<div class="js-reading-content">
												<div class="c-wysiwyg">
													<p>Ducktail is a malware family that has been active since the second half of 2021 and aims to steal Facebook business accounts. <a href="https://labs.withsecure.com/publications/ducktail" target="_blank" rel="noopener">WithSecure</a> and <a href="https://gridinsoft.com/blogs/ducktail-malware-analysis/" target="_blank" rel="noopener">GridinSoft</a> have covered Ducktail attacks: the infostealer spread under the guise of documents relating to well-known companies&#8217; and brands&#8217; projects and products. Both public reports attribute the Ducktail attacks to a group that presumably hails from Vietnam. We have analyzed a recent campaign that ran between March and early October 2023 and targeted marketing professionals. An important feature that sets it apart is that, unlike previous campaigns, which relied on .NET applications, this one used Delphi as the programming language.</p>
<h2 id="infection">Infection</h2>
<p>The campaign saw the bad actor send out an archive containing images of new products by bona fide companies along with a malicious executable disguised with a PDF icon. When started, the malware would open a real, embedded PDF file that contained the job details. The attack was tailored to target marketing professionals looking for a career change. The choice of victims and the distinctive means used by the threat actor led us to assume early on that the campaign was about spreading a new version of Ducktail.</p>
<p>The malware would install a browser extension capable of stealing Facebook business and ads accounts, likely for subsequent sale.</p>
<h2 id="ducktail-and-the-malicious-extension">Ducktail and the malicious extension</h2>
<p>We examined a large number of archives from the latest campaign: in each case, a copy of Ducktail was emailed in the name of a major clothing company.</p>
<div id="attachment_111022" style="width: 677px" class="wp-caption aligncenter"><a href="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165544/Ducktail_01.png" class="magnificImage"><img loading="lazy" decoding="async" aria-describedby="caption-attachment-111022" class="size-full wp-image-111022" src="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165544/Ducktail_01.png" alt="The contents of the malicious archive" width="667" height="443" srcset="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165544/Ducktail_01.png 667w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165544/Ducktail_01-300x199.png 300w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165544/Ducktail_01-263x175.png 263w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165544/Ducktail_01-370x246.png 370w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165544/Ducktail_01-422x280.png 422w" sizes="(max-width: 667px) 100vw, 667px" /></a><p id="caption-attachment-111022" class="wp-caption-text">The contents of the malicious archive</p></div>
<p>If opened by an interested victim, the malicious file saves a PowerShell script named param.ps1 and a PDF decoy locally to C:\Users\Public. The script uses the default PDF viewer on the device to open the decoy, pauses for five minutes, and then terminates the Chrome browser process.</p>
<p>While the script stands by, the parent executable saves a malicious library named libEGL.dll to C:\Users\Public\Libraries\ and then loads it. When launched, the library goes over every LNK file that it finds in:</p>
<ul>
<li>C:\ProgramData\Microsoft\Windows\Start Menu\Programs\,</li>
<li>C:\ProgramData\Microsoft\Internet Explorer\Quick Launch\User Pinned\TaskBar\,</li>
</ul>
<p>and on the desktop, altering the launch string for all Chromium-based browsers (Google Chrome, Edge, Vivaldi, Brave) by adding the following code: <code>--load-extension="C:\Users\%USERNAME%\AppData\Local\Google\Chrome\User Data\fjoaledfpmneenckfbpdfhkmimnjocfa"</code></p>
<p>Some of the library strings required for the malicious code to run are encrypted with the AES-CBC key &#8220;gnghfn47n467n43b&#8221; and the initialization vector &#8220;dakfhskljh92384h&#8221;.</p>
<div id="attachment_111023" style="width: 417px" class="wp-caption aligncenter"><a href="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165619/Ducktail_02.png" class="magnificImage"><img loading="lazy" decoding="async" aria-describedby="caption-attachment-111023" class="size-full wp-image-111023" src="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165619/Ducktail_02.png" alt="The use of the strings containing the AES key and initialization vector as featured in the code" width="407" height="147" srcset="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165619/Ducktail_02.png 407w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165619/Ducktail_02-300x108.png 300w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165619/Ducktail_02-370x134.png 370w" sizes="(max-width: 407px) 100vw, 407px" /></a><p id="caption-attachment-111023" class="wp-caption-text">The use of the strings containing the AES key and initialization vector as featured in the code</p></div>
<p>In addition to launching the library, the parent file saves malicious browser extension files to C:\Users\%USERNAME%\AppData\Local\Google\Chrome\User Data\fjoaledfpmneenckfbpdfhkmimnjocfa. The extension disguises itself with the Google Docs Offline icon and description text, while the directory that features in the path (fjoaledfpmneenckfbpdfhkmimnjocfa) is used by the bona fide extension NordVPN. It is worth noting that other variants of the malware may use different paths to host the extension.</p>
<div id="attachment_111026" style="width: 838px" class="wp-caption aligncenter"><a href="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165940/Ducktail_03_eng.png" class="magnificImage"><img loading="lazy" decoding="async" aria-describedby="caption-attachment-111026" class="size-full wp-image-111026" src="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165940/Ducktail_03_eng.png" alt="The malicious extension as seen in Google Chrome (left) and the authentic Google Docs Offline extension (right)" width="828" height="226" srcset="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165940/Ducktail_03_eng.png 828w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165940/Ducktail_03_eng-300x82.png 300w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165940/Ducktail_03_eng-768x210.png 768w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165940/Ducktail_03_eng-641x175.png 641w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165940/Ducktail_03_eng-370x101.png 370w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08165940/Ducktail_03_eng-800x218.png 800w" sizes="(max-width: 828px) 100vw, 828px" /></a><p id="caption-attachment-111026" class="wp-caption-text">The malicious extension as seen in Google Chrome (left) and the authentic Google Docs Offline extension (right)</p></div>
<p>The core exception script is obfuscated. It constantly sends the details of all open browser tabs to the command-and-control (C&amp;C) server, and if detecting Facebook-related URLs, checks for ads and business accounts to try and steal them. In particular, the extension snatches cookies and details of accounts that the victim is signed in to on the device. To bypass two-factor authentication, the extension uses Facebook API requests and Vietnam&#8217;s 2fa[.]live service, which offers various auxiliaries for generating one-time access codes, among other things. This is probably how the hackers log in after the user&#8217;s authentication session has expired. Stolen credentials and cookies are forwarded to a C&amp;C server registered in Vietnam.</p>
<div id="attachment_111027" style="width: 1034px" class="wp-caption aligncenter"><a href="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08170011/Ducktail_04_eng.png" class="magnificImage"><img loading="lazy" decoding="async" aria-describedby="caption-attachment-111027" class="size-large wp-image-111027" src="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08170011/Ducktail_04_eng-1024x282.png" alt="Malicious file usage flowchart" width="1024" height="282" srcset="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08170011/Ducktail_04_eng-1024x282.png 1024w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08170011/Ducktail_04_eng-300x83.png 300w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08170011/Ducktail_04_eng-768x211.png 768w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08170011/Ducktail_04_eng-636x175.png 636w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08170011/Ducktail_04_eng-370x102.png 370w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08170011/Ducktail_04_eng-1017x280.png 1017w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08170011/Ducktail_04_eng-800x220.png 800w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2023/11/08170011/Ducktail_04_eng.png 1439w" sizes="(max-width: 1024px) 100vw, 1024px" /></a><p id="caption-attachment-111027" class="wp-caption-text">Malicious file usage flowchart</p></div>
<p>In this campaign, in addition to the main script, the malware would save to the extension folder a script named jquery-3.3.1.min.js, a corrupted version of the core script from prior attacks.</p>
<h2 id="ducktail-attack-geography">DuckTail attack geography</h2>
<p>According to our telemetry, cybercriminals most often attacked users in India. Our solutions also stopped infection attempts on devices of users in Kazakhstan, Ukraine, Germany, Portugal, Ireland, Greece, Jordan, Pakistan, Vietnam, UAE, USA, Peru and Chile.</p>
