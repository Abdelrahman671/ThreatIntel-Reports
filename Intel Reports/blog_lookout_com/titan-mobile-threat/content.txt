class="text-rich-text break w-richtext"><p>Lookout researchers have recently observed cybercriminals evolving the way they operate to reflect the multitude of platforms people now use to access information: you may read and triage email on a smartphone, switch to a laptop to crank out some work, then flip over to a tablet to catch up on social media or watch a video. To the attackers, it&#x27;s a numbers game. The more devices they can reach, the more likely they are to compromise a target. Given the rise of mobile productivity, this means that we are seeing a growing number of attackers add mobile capabilities to their toolkits.</p><p>The latest threat to follow this trend is Titan, a family of sophisticated Android surveillanceware apps surfaced by Lookout&#x27;s automated analysis that, based on command and control infrastructure, is linked to the same actors behind Operation Tropic Trooper. Tropic Trooper is a long running campaign, first <a href="https://www.trendmicro.de/cloud-content/us/pdfs/security-intelligence/white-papers/wp-operation-tropic-trooper.pdf">reported</a> in 2016, that executed targeted desktop attacks against enterprises and military units in Taiwan and the Philippines. All Lookout customers are protected from Titan.</p><h2>Capabilities</h2><p>Titan usually comes with busybox and various native libraries that provide a range of functionality, from automated gathering of a user&#x27;s data to being able to execute attacker specified instructions as superuser. Over time, Titan has evolved considerably with a distinct trend of malicious code shifting first from Java to native libraries, then moving into second stage components. Analysis of Titan variants found that they contained the following capabilities:</p><ul role="list"><li>Retrieve call history</li><li>Retrieve text messages</li><li>Retrieve contact information</li><li>Retrieve a list of Installed packages</li><li>Track device location</li><li>Take a photo with the device camera when a user is first present or when instructed by an attacker</li><li>Record all calls or only calls to attacker specified numbers</li></ul><ul role="list"><li>Block text messages to attacker specified numbers</li><li>Take a screenshot</li><li>Send a text message</li><li>Execute attacker specified commands as root</li><li>Upload specific files</li><li>Download attacker specified files</li></ul><p><br/>Titan hasn&#x27;t been seen to trojanize legitimate applications and doesn&#x27;t contain any legitimate functionality.
