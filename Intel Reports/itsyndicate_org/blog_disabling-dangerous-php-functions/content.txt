data-block-key="nqd8"><b>escapeshellarg:</b> Escape a string to be used as a shell argument</p><p data-block-key="b7ba5"><b>escapeshellcmd:</b> Escape shell metacharacters</p><p data-block-key="cc0qh"><b>proc_open:</b> Execute a command and open file pointers for input/output</p><p data-block-key="eep4h"><b>proc_close:</b> Close a process opened by proc_open() and return the exit code of that process</p><p data-block-key="56q34"><b>ini_alter:</b> Alias of ini_set()</p><p data-block-key="3p0sh"><b>dl:</b> Loads a PHP extension at runtime</p><p data-block-key="a9gpp"><b>popen:</b> Opens process file pointer</p><p data-block-key="qkn5"><b>parse_ini_file:</b> Parse a configuration file</p><p data-block-key="25ctj"><b>show_source:</b> Alias of
