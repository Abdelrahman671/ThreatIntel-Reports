							<div class="content "><h1 id="del">Del</h1><div class="display-flex justify-content-space-between align-items-center flex-wrap-wrap page-metadata-container">
										<div class="margin-right-xxs">
											<ul class="metadata page-metadata" data-bi-name="page info" lang="en-us" dir="ltr"><li>Article</li><li>
														<time class="is-invisible" data-article-date aria-label="Article review date" datetime="2016-08-31T00:00:00.000Z" data-article-date-source="ms.date">08/31/2016</time>

											<h2 id="ms--in-this-article" class="title is-6 margin-block-xs">In this article</h2>
										</nav><!-- <content> --><p>&nbsp;</p>
<p>Applies To: Windows Vista, Windows Server 2008, Windows Server 2008 R2, Windows Server 2012, Windows 8</p>
<p>Deletes one or more files. This command is the same as the <strong>erase</strong> command.</p>
<p>For examples of how to use this command, see <a href="">Examples</a>.</p>
<h2 id="syntax">Syntax</h2>
<pre><code>del [/p] [/f] [/s] [/q] [/a[:]&lt;Attributes&gt;] &lt;Names&gt;
erase [/p] [/f] [/s] [/q] [/a[:]&lt;Attributes&gt;] &lt;Names&gt;
</code></pre>
<h2 id="parameters">Parameters</h2>
<table>
<colgroup>
<col>
<col>
</colgroup>

<th><p>Parameter</p></th>
<th><p>Description</p></th>

<tr class="odd">
<td><p>&lt;Names&gt;</p></td>
<td><p>Specifies a list of one or more files or directories. Wildcards may be used to delete multiple files. If a directory is specified, all files within the directory will be deleted.</p></td>

<td><p>Prompts for confirmation before deleting the specified file.</p></td>

</tr>

</tr>

<h2 id="remarks">Remarks</h2>
<div class="WARNING">
<p>Warning</p>
<p>If you use <strong>del</strong> to delete a file from your disk, you cannot retrieve it.</p>
</div>
<ul>
<li><p>If you use <strong>/p</strong>, <strong>del</strong> displays the name of a file and sends the following message:</p>
<p>FileName, Delete (Y/N)?</p>
<p>To confirm the deletion, press Y. To cancel the deletion and display the next file name (that is, if you specified a group of files), press N. To stop the <strong>del</strong> command, press CTRL+C.</p>
</li>
<li><p>If you disable command extensions, <strong>/s</strong> displays the names of any files that were not found instead of displaying the names of files that are being deleted (that is, the behavior is reversed).</p>
</li>
<li><p>If you specify a folder in <em>Names</em>, all of the files in the folder are deleted. For example, the following command deletes all of the files in the \Work folder:</p>
<pre><code>del \work
</code></pre>
</li>
<li><p>You can use wildcards (<strong>*</strong> and <strong>?</strong>) to delete more than one file at a time. However, to avoid deleting files unintentionally, you should use wildcards cautiously with the <strong>del</strong> command. For example, if you type the following command:</p>
<pre><code>del *.*
</code></pre>
<p>The <strong>del</strong> command displays the following prompt:</p>
<p>Are you sure (Y/N)?</p>
<p>To delete all of the files in the current directory, press Y and then press ENTER. To cancel the deletion, press N and then press ENTER.</p>
</li>
</ul>
<div class="NOTE">
<p>Note</p>
<p>Before you use wildcard characters with the <strong>del</strong> command, use the same wildcard characters with the <strong>dir</strong> command to list all the files that will be deleted.</p>
</div>
<ul>
<li>The <strong>del</strong> command, with different parameters, is available from the Recovery Console.</li>
</ul>
<h2 id="examples">Examples</h2>
<p>To delete all the files in a folder named Test on drive C, type either of the following:</p>
<pre><code>del c:\test
del c:\test\*.*
</code></pre>
<p>To delete all files with the .bat file name extension from the current directory, type:</p>
<pre><code>del *.bak
</code></pre>
<p>To delete all read-only files in the current directory, type:</p>
<pre><code>del /a:r *.*
</code></pre>
