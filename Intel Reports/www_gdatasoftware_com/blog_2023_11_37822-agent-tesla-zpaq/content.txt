id="blogpage-content-container"> <div class="row"> <div class="column fce small-12"> <p class="intro-text">A new variant of Agent Tesla uses the uncommon compression format ZPAQ to steal information from approximately 40 web browsers and various email clients. But what exactly is this file compression format? What advantage does it provide to threat actors? And why it is assumed that the version of Agent Tesla is “new”?</p> </div> </div> <div class="row hide-for-large"> <div class="column small-12"> <div class="pagenews-progress"> <div class="reading-time"> <span class="eta"></span> <span class="word-count"></span> </div> <div data-sticky-container id="mobile-progress-container" class="hide-on-large"> <progress value="0" class="sticky" data-sticky data-margin-top="0" data-sticky-on="small" data-margin-bottom="0" data-top-anchor="mobile-progress-container" data-btm-anchor="site-footer:top"></progress> </div> </div> <nav class="mobile-menu"> <ul class="pagenews-menu"> <li> <a href="#c231333" data-scroll-link> <span>ZPAQ compression format and what it hides</span> </a> </li> <li> <a href="#c231336" data-scroll-link> <span>Sounds like stolen data</span> </a> </li> <li> <a href="#c231337" data-scroll-link> <span>Telegram as C&amp;C</span> </a> </li> <li> <a href="#c231341" data-scroll-link> <span>Takeaways</span> </a> </li> <li> <a href="#c231342" data-scroll-link> <span>Information for fellow researchers</span> </a> </li> </ul> </nav> </div> </div> <div class="row"> <div data-magellan-target="c231333" class="small-12 column"><div class="row"> <div id="c231333" class="fce column large-12 small-12 frame-layout-0 " ><header class=" "><h2 class=" "> ZPAQ compression format and what it hides </h2></header><p>On November 1, 2023, researcher Xavier Mertens <a href="https://isc.sans.edu/diary/rss/30366" class="notion-link-token notion-focusable-token notion-enable-hover" data-token-index="1" rel="noopener noreferrer">reported</a> a phishing attempt on one of his honeypots. What's noteworthy is that a threat actor used the ZPAQ archive and .wav file extension to infect the system with Agent Tesla. </p><p><a href="https://mattmahoney.net/dc/zpaq.html" target="_blank" rel="noreferrer">ZPAQ is a file compression format </a>that offers a better compression ratio and journaling function compared to widely used formats like ZIP and RAR. That means that ZPAQ archives can be smaller, saving storage space and bandwidth when transferring files. However, ZPAQ has the biggest disadvantage: limited software support. There are GUI unpackers that support this format, for example, <a href="https://peazip.github.io/paq-files-compression-utility.html" class="notion-link-token notion-focusable-token notion-enable-hover" data-token-index="1" rel="noopener noreferrer">Peazip</a>, but ZPAQ can be extracted primarily with a command-line tool that does not make it easy to work with, especially for users without technical expertise.</p></div> <div id="c231334" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-intext-right"><div class="image-gallery-wrap"><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><a href="/fileadmin/_processed_/7/2/ATesla_Illu1_ede467b8a1.png" class="fancybox"><img class="image-embed-item " src="/fileadmin/_processed_/7/2/ATesla_Illu1_278514beeb.png" width="300" height="162" loading="lazy" alt="" /></a><figcaption class="image-caption"> Extraction of the .NET executable with ZPAQ cmd tool and comparison of size. </figcaption></figure></div></div></div><p>The initial file was found in an email called "Purchase Order pdf.zpaq”. As you can see from the file name, the threat actor is attempting to deceive us into believing that the archive contains a PDF file with important information. After using the command-line extraction tool ZPAQ it turns out that the 6KB archive suddenly "weighed" 1GB after extraction. After a deep look into the executable, it turned out that the file is a .NET executable with async methods and that is bloated with zero bytes. One of the indicators is 0 entropy in the overlay section. The analysis of the executable in a hex editor proved that 90% of the sample is filled with zero bytes.</p></div></div> <div id="c231335" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-center ce-below"><div class="ce-bodytext"><p>Threat actors may prefer to use bloated executable files due to their significant advantage: the inability to upload such files to automatic scanning systems, Virus Total, sandboxes, etc. This technique allows them to bypass traditional security measures and increase the effectiveness of their attack. </p></div><div class="ce-gallery row small-up-1 medium-up-2 large-up-2"><div class="column column-block"><figure class="image"><a href="/fileadmin/_processed_/0/d/ATesla_Illu2_dca23ed819.png" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/_processed_/0/d/ATesla_Illu2_084447a2ed.png" width="346" height="300" loading="lazy" alt="" /></a><figcaption class="image-caption"> DetectItEasy. Overlay size and entropy level. (Click to enlarge) </figcaption></figure></div><div class="column column-block"><figure class="image"><a href="/fileadmin/_processed_/8/0/ATesla_Illu3_f75bcab43f.png" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/_processed_/8/0/ATesla_Illu3_5ad0754c9a.png" width="453" height="300" loading="lazy" alt="" /></a><figcaption class="image-caption"> Malcat. Hex editor view. Overlay has 0 bytes. </figcaption></figure></div></div></div></div> </div></div> <div data-magellan-target="c231336" class="small-12 column"><div class="row"> <div id="c231336" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-intext-left"><header class=" "><h2 class=" "> Sounds like stolen data </h2></header><div class="image-gallery-wrap"><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><a href="/fileadmin/_processed_/9/2/ATesla_Illu4_3438c754f7.png" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/_processed_/9/2/ATesla_Illu4_d1efcc5e52.png" width="300" height="97" loading="lazy" alt="" /></a><figcaption class="image-caption"> ILSpy. Link to download malicious component and the string that is responsible for decryption (Click to enlarge) </figcaption></figure></div></div></div><p>The main function of the unarchived .NET executable is to download a file with .wav extension and decrypt it (3DES algorithm).<br> Waveform Audio File Format (shortened as .wav) is a popular audio file format standard. However, in that case, it is unrelated to audio, and the threat actor simply used this file extension to hide the presence of malicious content. One possible reason is covert communication: using commonly used file extensions disguises the traffic as normal, making it more difficult for network security solutions to detect and prevent malicious activity.</p></div></div> </div></div> <div data-magellan-target="c231337" class="small-12 column"><div class="row"> <div id="c231337" class="fce column large-12 small-12 frame-layout-0 " ><div class="media-object stack-for-small textmedia--beside-right" data-columns="1"><div class="media-object-section"><div class="inner"><header class=" "><h2 class=" "> Telegram as C&C </h2></header><p>Due to the widespread legal usage of Telegram, its traffic is often allowed through firewalls, making it a useful medium for covert communication. Unlike other messaging platforms, Telegram's API does not require the installation of Telegram software on target devices. Its chat interface simplifies conversation.&nbsp;<br><br> The Telegram API of the threat actor was found among the configuration data of the payload. Unfortunately, it was not possible to retrieve information about the Telegram bot malware because of authorization problems. It can indicate that the token has been already changed, removed, or expired at the moment of the analysis.</p></div></div><div class="media-object-section "><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><a href="/fileadmin/_processed_/1/a/ATesla_Illu5_65f4c72853.png" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/_processed_/1/a/ATesla_Illu5_756318f173.png" width="300" height="49" loading="lazy" alt="" /></a><figcaption class="image-caption"> Example of the simplest communication with a Telegram Bot via Windows command line. </figcaption></figure></div><div class="column column-block"><figure class="image"><a href="/fileadmin/_processed_/e/d/ATesla_Illu6_d51b071a7f.png" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/_processed_/e/d/ATesla_Illu6_35ba0e268b.png" width="300" height="195" loading="lazy" alt="" /></a></figure></div></div></div></div></div> <div id="c231338" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-intext-right"><header class=" "><h2 class=" "> Another Agent Tesla </h2></header><div class="image-gallery-wrap"><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><a href="/fileadmin/_processed_/3/d/ATesla_AccessData_c1ec49ae42.png" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/_processed_/3/d/ATesla_AccessData_43fff7dcab.png" width="400" height="412" loading="lazy" alt="" /></a><figcaption class="image-caption"> Configuration data (click to enlarge) </figcaption></figure></div></div></div><p>Agent Tesla is a .NET-based information stealer that emerged around 2014. Over time, it has undergone multiple updates, evolving in terms of both capabilities and evasion techniques. In this specific case, Agent Tesla was obfuscated with the .NET Reactor (<a href="https://www.youtube.com/watch?v=3t9BEdA4ch8" target="_blank" rel="noreferrer">my colleague Karsten has done an in depth analysis of this in a video</a>&nbsp;- the link will open in a new window), and several rounds of de-obfuscation were necessary to make the code clearer. The analysis revealed that it possesses the following functions:</p><ul class="list-black-square"><li>targetting sensitive data of around 40 different web browsers</li><li>stealing credentials from popular email clients</li><li>screen logging</li><li>keylogging</li><li>gathering system information</li><li>capturing sensitive data of VPN tools</li></ul><p>From a capabilities standpoint, it doesn't offer anything significantly new. However, after analysis of similar samples, all of them have a similar .NET class with configuration data. The way to submit the stolen data, persistence variables, keylogger variable and etc. are kept in this class. Other samples had the same structure, but just different methods to deliver information to the threat actor</p></div></div> <div id="c231339" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-intext-left"><div class="image-gallery-wrap"><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><a href="/fileadmin/web/general/images/blog/2023/11/ATesla_Illu7.png" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/_processed_/b/b/ATesla_Illu7_f688881750.png" width="300" height="162" loading="lazy" alt="" /></a><figcaption class="image-caption"> Communication ways in similar samples. (Click to enlarge) </figcaption></figure></div></div></div><p>The way to submit the stolen data, persistence variables, keylogger variable and etc. are kept in this class. Other samples had the same structure, but just different methods to deliver information to the threat actor. It was noticed that besides Telegram, the threat actor uses FTP and SMTP. And the list can be much bigger, because since 30.09.2023 more than 700 versions of this variant were observed on VirusTotal. As is customary in casses like this, the data being used is associated with compromised websites, the accerss credentials to which were likely acquired through an access broker who specializes in selling those types of accounts..</p></div></div> <div id="c231340" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-center ce-below"><div class="ce-bodytext"><p>One of the TelegramAPIs from the oldest observed sample is still active. Here the information that was found about this communication channel:</p></div><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><a href="/fileadmin/_processed_/d/8/ATesla_Illu9_320e4964fc.png" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/_processed_/d/8/ATesla_Illu9_761ce10395.png" width="1436" height="145" loading="lazy" alt="" /></a><figcaption class="image-caption"> Active Telegram API from the oldest observed file (as of September 30, 2023) </figcaption></figure></div></div></div></div> </div></div> <div data-magellan-target="c231341" class="small-12 column"><div class="row"> <div id="c231341" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-center ce-above"><div class="ce-bodytext"><header class=" "><h2 class=" "> Takeaways </h2></header><p>The usage of the ZPAQ compression format raises more questions than answers. The assumptions here are that either threat actors target a specific group of people who have technical knowledge or use less widely known archive tools, or they are testing other techniques to spread malware faster and bypass security software. However, it is definitely a good example that even very specific archive formats or widely spread file extensions like .wav can be used for malicious purposes. </p><p>Like any other stealer, Agent Tesla can harm not only private individuals but also organizations. It has gained popularity among cybercriminals for many reasons including ease of use, versatility, affordability on the Dark Web, and so on. It is worth mentioning that cybersecurity professionals and organizations are constantly working on developing countermeasures and detection techniques to minimize its effects. To protect your devices, it is essential to have malware protection, maintain a high level of security awareness, and regularly update software. </p></div></div></div> </div></div> <div
