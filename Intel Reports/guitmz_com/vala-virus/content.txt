    <div itemprop="articleBody">
      
      <p>Vala is an object-oriented programming language with a self-hosting compiler that generates C code and uses the GObject system. Vala is syntactically similar to C# and and rather than being compiled directly to assembly or to another intermediate language, Vala is source-to-source compiled to C, which is then compiled with a platform&rsquo;s standard C compiler, such as GCC.</p>

<p>You can also create VAPI files which are basically native C (not C++) functions you can import to Vala code (I will show an example later). Being a language that is converted into plain and pure C, Vala code can also run on Windows (with the necessary code optimizations, of course).</p>

<p>Anyway I was decided to write a prepender in this language, the first (binary) virus ever written so far in Vala. It&rsquo;s named <a href="http://www.virusradar.com/en/Linux_Zariche/detail">Linux.Zariche</a> and there are two variants available so far.</p>

<ul>
<li><a href="http://www.virusradar.com/en/Linux_Zariche.A/description">Linux.Zariche.A</a> original release, simple ELF infector (prepender).</li>
<li><a href="http://www.virusradar.com/en/Linux_Zariche.B/description">Linux.Zariche.B</a> uses AES encryptation via external library (vapi).</li>
</ul>

<p><p>I will explain parts of the code and then add a download link for the full file below (you can check the GitHub repository too at <a href="https://github.com/guitmz/vala-zariche">https://github.com/guitmz/vala-zariche</a>). Shall we start?</p>

<p>The very first thing I&rsquo;m going to do is to declare my global variables and specify the library I&rsquo;m using for most of the functions, which is <b>libgee</b>. I&rsquo;m also importing the native C <b>exit()</b> function as you can see.</p>

<p>Compilation instructions are the following (tested on a x86_64 system, but should work on x86):</p>

<blockquote>
<p>valac filename.vala &ndash;pkg=gee-1.0 &ndash;pkg=gio-2.0 &ndash;pkg=nettle &ndash;pkg=posix</p>
</blockquote>

<p>Where Vala &gt;= 0.20. In order to use the &ndash;pkg=nettle flag, place your nettle.vapi file inside Vala&rsquo;s library folder, usually something like <b>/usr/share/vala-0.20/vapi</b> for example.</p>
