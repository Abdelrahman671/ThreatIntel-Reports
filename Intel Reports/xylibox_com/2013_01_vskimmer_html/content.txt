<div class="text" style="background-color: #f0f0f0; border: 1px solid #d0d0d0; color: #000066; font-family: monospace;">
vSkimmer - Virtual Skimmer<br />
<br />
Functions:<br />
- Track 2 grabber<br />
- HTTP Loader (Download &amp; Execute)<br />
- Update bot itself<br />
<br />
Working Modes:<br />
- Online: If internet is reachable it will try to bypass firewalls and communicate to a the control panel.<br />
- Offline: If internet is not reachable it wait for a specific pendrive/flashdrive plugged in and copy logs to it.<br />
<br />
Server coded in PHP (can be modified on request to send logs to remote server, via smtp, etc.. )<br />
Client coded in C++ no dependencies, 66kb, cryptable. (can be customized) </div>
