<h4 class="wp-block-heading" id="h-aurora-stealer-background"><strong>Aurora Stealer Background</strong></h4>



<p class="has-text-align-right wp-author has-black-color has-text-color has-small-font-size">March 24, 2023<br> <span style="color:#FFF">By</span> <strong>Saharsh Agrawal</strong></p>



<p>Aurora Stealer a non-friendly program in your neighborhood is developed in <a href="https://go.dev/" target="_blank" rel="noreferrer noopener">Go-lang</a>. The malware was developed for threat actors to steal victims&#8217; sensitive data. Since its initial release, there have been many adaptations to the malware. It started as a botnet having info-stealer capabilities but currently it is a full-time info-stealer which is being sold by the threat actor labelled Cheshire on Russian speaking dark-net forums mentioned by cybersecurity firm <a href="https://blog.sekoia.io/aurora-a-rising-stealer-flying-under-the-radar/" target="_blank" rel="noreferrer noopener">Sekoia</a>.</p>



<p>The malware has a knack for stealing the victim&#8217;s browser data including cookies, passwords, login data and many more. It has the horizon to steal data from most browsers. The stealer also targets crypto wallets to pilfer victim&#8217;s data. The malware does not stop here, it can exfiltrate files and data stored in different software from the victim&#8217;s machine. Once all the super-sensitive data is possessed by the stealer it delivers the data to the attacker.</p>



<h4 class="wp-block-heading" id="h-operating-model"><strong>Operating Model</strong></h4>



<div class="wp-block-image is-style-default"><figure class="aligncenter size-large is-resized"><img fetchpriority="high" decoding="async" src="https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/Isoflow-_-Network-diagrams-1024x541.png" alt="Aurora Stealer Attack Chain" class="wp-image-3449" width="768" height="406" srcset="https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/Isoflow-_-Network-diagrams-1024x541.png 1024w, https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/Isoflow-_-Network-diagrams-300x158.png 300w, https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/Isoflow-_-Network-diagrams-768x406.png 768w, https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/Isoflow-_-Network-diagrams-1536x811.png 1536w, https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/Isoflow-_-Network-diagrams.png 1647w" sizes="(max-width: 768px) 100vw, 768px" /><figcaption><em>Figure1: Attack</em> <em>Chain of Aurora Stealer</em></figcaption></figure></div>



<p>The malware is being spread among victims in the hospitality industry through phishing emails as seen by <a href="https://www.trendmicro.com/vinfo/us/threat-encyclopedia/spam/3721/fake-roadway-map-leads-to-aurora-stealer" target="_blank" rel="noreferrer noopener">Trendmirco</a>. Besides phishing campaigns, the threat actors also have utilized SEO poisoning technique to deliver the malware via malicious ads of notepad++ according to <a href="https://isc.sans.edu/diary/Malicious+Google+Ad+Fake+Notepad+Page+Aurora+Stealer+malware/29448" target="_blank" rel="noreferrer noopener">SANS</a> finding.</p>



<p>Upon the detonation of the malware, our observations revealed the implementation of multiple sophisticated techniques by the malware to evade detection by antivirus software. One such technique involved the incorporation of <a href="https://attack.mitre.org/techniques/T1027/001/" target="_blank" rel="noreferrer noopener">binary padding</a>, resulting in an increase in the file size allowing the malware to circumvent the antivirus software and evade detection.</p>

<p class="has-text-align-center .wp-block-image figcaption has-text-color has-small-font-size" style="color:#5f5f5f"><em>Figure2: &nbsp;Fake Adobe Acrobat Installer Downloaded from SEO Poisoning Campaign</em></p>



<p>After successful infiltration, the malware utilizes Windows Management Instrumentation (WMI) commands to gather pertinent system information. This behavior was consistently observed in all samples of Aurora stealer that we  analyzed. Additionally, independent research conducted by <a href="https://blog.cyble.com/2023/01/18/aurora-a-stealer-using-shapeshifting-tactics/" target="_blank" rel="noreferrer noopener">Cyble</a> and other industry experts has reported this observation.</p>



<div class="wp-block-image"><figure class="aligncenter size-large"><img loading="lazy" decoding="async" width="933" height="298" src="https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/wmic.jpg" alt="WMI Commands" class="wp-image-3589" srcset="https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/wmic.jpg 933w, https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/wmic-300x96.jpg 300w, https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/wmic-768x245.jpg 768w" sizes="(max-width: 933px) 100vw, 933px" /><figcaption><em>Figure2.1 : WMI Commands getting Executed</em></figcaption></figure></div>



<p>We also captured Aurora stealer&#8217;s actions that were aimed at extracting sensitive information from files and folders located in various browsers, cryptocurrency wallets, and drives, as illustrated below.</p>



<div class="wp-block-image"><figure class="aligncenter size-large is-resized"><img loading="lazy" decoding="async" src="https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/New-Project-1024x576.jpg" alt="Aurora Stealer - Stealing Browser and Crypto Data" class="wp-image-3587" width="768" height="432" srcset="https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/New-Project-1024x576.jpg 1024w, https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/New-Project-300x169.jpg 300w, https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/New-Project-768x432.jpg 768w, https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/New-Project-1536x864.jpg 1536w, https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/New-Project.jpg 1920w" sizes="(max-width: 768px) 100vw, 768px" /><figcaption><em>Figure3: Stealing Browser and Crypto Data</em></figcaption></figure></div>



<p>Aurora leverages PowerShell to facilitate the execution of its second stage, employing the following command:</p>



<pre class="wp-block-code"><code> powershell "" "start-process C:\Users\Admin\AppData\Local\Temp\BK2c7L9Si8.exe" </code></pre>



<p>During the second stage of Aurora, an entry to the &#8220;run keys&#8221; in the registry was observed, enabling the malware to automatically execute its third stage DLL via RUNDLL32.exe at each user login.</p>



<div class="wp-block-image wp-image"><figure class="aligncenter size-large"><img loading="lazy" decoding="async" width="1024" height="152" src="https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/reg-1024x152.jpg" alt="Aurora Stealer - Run Key Added for DLL Execution" class="wp-image-3598" srcset="https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/reg-1024x152.jpg 1024w, https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/reg-300x45.jpg 300w, https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/reg-768x114.jpg 768w, https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/reg-1536x228.jpg 1536w, https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/reg.jpg 1554w" sizes="(max-width: 1024px) 100vw, 1024px" /><figcaption><em>Figure4: Run Key Added for DLL Execution</em></figcaption></figure></div>



<p>The third stage of the malware executes an encoded PowerShell command allowing it to delay execution for certain duration using the following command before it exfiltrates the data to the attacker.</p>



<pre class="wp-block-code"><code>C:\Windows\System32\WindowsPowerShell\v1.0\powershell.exe" -enc UwB0AGEAcgB0AC0AUwBsAGUAZQBwACAALQBTAGUAYwBvAG4AZABzACAAMwA0AA==</code></pre>



<p>Once all the data and information are collected, the malware employs advanced encoding techniques to conceal the information and prepare it for shipment to the attacker over the C2 channel. The attacker can retrieve the encoded data and use it for nefarious purposes, such as identity theft or financial fraud.</p>



<div class="wp-block-image"><figure class="aligncenter size-large is-resized"><img loading="lazy" decoding="async" src="https://research.loginsoft.com/wp-content/uploads/sites/2/2023/03/FmfrZFjWYAQy5vo-1024x578.jpg" alt="Aurora Stealer - C2 Panel" class="wp-image-3484" width="900" height="504" /><figcaption><em>Figure5: C2 Panel; source: <a href="https://twitter.com/silascutler/status/1614518491557642243" target="_blank" rel="noreferrer noopener">twitter</a></em></figcaption></figure></div>



<h4 class="wp-block-heading" id="h-detection"><strong>Detection</strong></h4>



<p id="h-the-infernal-behavior-of-this-malware-can-be-detected-using-sigma-rules-to-begin-with-we-attribute-different-wmi-commands-getting-executed-in-sequence-to-detect-aurora-stealer-malware">The infernal behavior of this malware can be detected using SIGMA rules.&nbsp; <br>To begin with, we attribute different WMI commands getting executed in sequence to detect Aurora Stealer malware.</p>



<pre class="wp-block-code"><code>logsource:
 category: process_creation
 product: windows
detection:
 selection:
 Image|endswith: '\wmic.exe'
 CommandLine|contains: 
 - 'wmic os get Caption'
 - 'wmic path win32_VideoController get name'
 - 'wmic cpu get name'
 condition: selection
</code></pre>



<p>One detection would be to capture changes in ‘run key’ registry which execute DLL using RUNDLL32.exe</p>



<pre class="wp-block-code"><code>logsource:
 product: windows
 category: registry_set
detection:
 selection:
 EventType: SetValue
 TargetObject|contains: '\REGISTRY\MACHINE\Software\Microsoft\Windows\CurrentVersion\RunOnce'
 Details|re: .*rundll32.exe.C:\\\\Windows\\\\system32\\\\advpack.dll,DelNodeRunDLL32 \\"C:\\\\Users\\\\.*\\AppData\\\\Local\\\\Temp\\\\IXP&#091;0-9]{1,3}.TMP.*
 condition: selection
</code></pre>



<p>Based on the general behavior followed by Aurora and different malwares, we have developed several detection methods. We also observed changes in the configuration of Windows Defender, where the path &#8216;C:\Program Data&#8217; is excluded.</p>



<pre class="wp-block-code"><code>logsource:
 product: windows
 category: registry_set
detection:
 selection:
 EventType: SetValue
 TargetObject|contains: '\Microsoft\Windows Defender\Exclusions'
 Details: 'C:\Program Data\'
 condition: selection
</code></pre>



<p>Another detection would be by tracing event id `5007`, which can be activated by enabling Windows Defender operation logs. This particular event is triggered whenever there is a modification made to the configuration of Microsoft Defender Antivirus.</p>



<pre class="wp-block-code"><code>logsource:
 product: windows
 service: windefend
detection:
 selection:
 EventID: 5007
 NewValue|contains: 'C:\Program Data\'
 condition: selection
</code></pre>



<p>Additionally, one clever way to detect this malware is to monitor the execution of obfuscated PowerShell commands.</p>



<p>The same campaign can be detected using <a href="https://www.loginsoft.com/blog/2020/06/25/explicating-the-concepts-of-osquery/" target="_blank" rel="noreferrer noopener">OSQUERY</a> rules. Below are some rules that will help you keep your organization safe.</p>



<pre class="wp-block-code"><code>query_wmic:
    SELECT name,
