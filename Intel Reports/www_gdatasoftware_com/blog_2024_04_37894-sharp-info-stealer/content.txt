Infostealers are a favorite of the cybercrime scene. And even though outsiders might think that everything has been said and done as far as new malware goes, there is still the occasional new player who enters the game.&nbsp;Threat researcher <a href="https://twitter.com/hashtag/SHARPILRAT" target="_blank" rel="noreferrer">Yogesh Londhe reported on X (formerly Twitter)</a>&nbsp;a new sample that is called âSharpil RAT.exeâ. Additionally, it collects geolocation of the victim and user information from the Minecraft gaming server âVime Worldâ.&nbsp;&nbsp;</p><p>To connect to the Telegram bot it uses the method âParseLastMessageâ that parses a JSON string and extracts the value associated with the last occurrence of the âtextâ in the JSON object received from <a href="https://core.telegram.org/#bot-api" target="_blank" rel="noreferrer">Telegram bot API</a>.&nbsp;&nbsp;</p><p>After that, it validates if the received argument is equal to the values in the application and based on the argument starts information-gathering actions.&nbsp;&nbsp;</p><table border="1" data-tablelook="1184" data-tablestyle="MsoTableGrid" class="contenttable"><tbody><tr><td data-celllook="0"><p>Sharpil RAT commands&nbsp;</p></td><td data-celllook="0"><p>Actions&nbsp;</p></td></tr><tr><td data-celllook="0"><p>/browsers&nbsp;</p></td><td data-celllook="0"><p>Search installed browsers&nbsp;&nbsp;</p></td></tr><tr><td data-celllook="0"><p>/system&nbsp;</p></td><td data-celllook="0"><p>System information: username, clipboard, CPU, RAM, GPU, IP Geolocation, BSSID, HDD, MAC address, BIOS caption. Considering that, it can be classified as a remote-controlled stealer. The main difference is that a remote access trojan can be used to not only steal but also gain full control of a remote system in order to run any code the attacker wants. It is how ransomware infections often start. Once a system has been compromised, the access to said system is sold by specialized brokers either to the highest bidder or as part of a bulk package.</p></div></div> <div id="c232597" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-center ce-above"><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><a href="/fileadmin/_processed_/5/3/G_DATA_Blog_SharpInfostealer_Image2_77b3249448.png" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/_processed_/5/3/G_DATA_Blog_SharpInfostealer_Image2_968ce9b5fb.png" width="800" height="116" loading="lazy" alt="" /></a><figcaption class="image-caption"> ParseLast Message method </figcaption></figure></div></div><div class="ce-bodytext"></div></div></div> </div></div> <div data-magellan-target="c232591" class="small-12 column"><div class="row"> <div id="c232591" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-intext-right"><header class=" "><h2 class=" "> Sharp Project and Sharp Stealer </h2></header><div class="image-gallery-wrap"><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><a href="/fileadmin/_processed_/c/8/G_DATA_Blog_SharpInfostealer_Image3_13eb854af5.png" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/_processed_/c/8/G_DATA_Blog_SharpInfostealer_Image3_6ea07cc855.png" width="300" height="154" loading="lazy" alt="" /></a><figcaption class="image-caption"> The Telegram channel of Sharp Stealer </figcaption></figure></div></div></div><p>As the file was neither encrypted nor obfuscated, it was possible to obtain information about the threat actor based on the configuration data and Telegram API token. The Telegram channel was created on the 3rd of April, 2024, and since that two messages with small updates have been posted.&nbsp;&nbsp;</p></div></div> <div id="c232592" class="fce column large-12 small-12 frame-layout-0 " ><div class="ce-textpic ce-intext-left"><div class="image-gallery-wrap"><div class="ce-gallery row small-up-1 medium-up-1 large-up-1"><div class="column column-block"><figure class="image"><a href="/fileadmin/web/general/images/blog/2024/04/G_DATA_Blog_SharpInfostealer_Image4.png" class="fancybox"><img class="image-embed-item thumbnail" src="/fileadmin/_processed_/9/8/G_DATA_Blog_SharpInfostealer_Image4_b79df85b94.png" width="250" height="140" loading="lazy" alt="" /></a><figcaption class="image-caption"> VirusTotal rating of unencrypted sample (left) and encrypted sample (right).
