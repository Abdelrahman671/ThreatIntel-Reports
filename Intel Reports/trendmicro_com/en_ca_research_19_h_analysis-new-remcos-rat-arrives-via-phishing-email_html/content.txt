		<div>
			<p>In July, we came across a phishing email purporting to be a new order notification, which contains a malicious attachment that leads to the remote access tool Remcos RAT (detected by Trend Micro as BKDR_SOCMER.SM). This attack delivers Remcos using an AutoIt wrapper that incorporates various obfuscation and anti-debugging techniques to evade detection, which is a common method for distributing known malware.</p>
<p>Remcos RAT emerged in 2016 being peddled as a service in hacking forums — advertised, sold, and offered cracked on various sites and forums. The RAT appears to still be actively pushed by cybercriminals. In 2017, we reported spotting Remcos being <a href="https://blog.trendmicro.com/trendlabs-security-intelligence/cve-2017-0199-new-malware-abuses-powerpoint-slide-show/" target="_blank">delivered</a> via a malicious PowerPoint slideshow, embedded with an exploit for CVE-2017-0199. Recently, the RAT has made its way to phishing emails.</p>
<p>The malicious actor behind the phishing email appears to use the email address rud-division@alkuhaimi[.]com (with a legitimate domain) and the subject &quot;RE: NEW ORDER 573923&quot;. The email includes the malicious attachment using the ACE compressed file format, <i>Purchase order201900512.ace</i>, which has the loader/wrapper <i>Boom.exe</i>.</p>
<h3><b><span class="body-subhead-title">Analyzing the wrapper/loader</span></b></h3>

<h3 style="text-align: left;"><b><span class="body-subhead-title">Examining the main payload, Remcos RAT</span></b></h3>
<p>Originally marketed as a remote access tool that legitimately lets a user control a system remotely, Remcos RAT has since been used by cybercriminals. Once the RAT is executed, a perpetrator gains the ability to run remote commands on the user’s system. In a past campaign, for instance, the tool was seen with a <a href="https://blog.trendmicro.com/trendlabs-security-intelligence/cve-2017-0199-new-malware-abuses-powerpoint-slide-show/" target="_blank">variety of capabilities</a>, which includes downloading and executing commands, logging keys, logging screens, and capturing audio and video using the microphone and webcam.</p>

<h3 style="text-align: left;"><b><span class="body-subhead-title">Prevention and Trend Micro Solutions</span></b></h3>
<p>To defend against threats like Remcos RAT that use email-based attacks, we advise users to refrain from opening unsolicited emails — especially those with attachments — from unknown sources. Users should also exercise caution before clicking on URLs to avoid being infected with malware. For enterprises, if an anomaly is suspected in the system, report the activity to the network administrator immediately. We also recommend these best practices for added protection:</p>
<ul>
<li><span class="rte-red-bullet">Learn how to <a href="https://www.trendmicro.com/vinfo/us/security/news/cybercrime-and-digital-threats/best-practices-identifying-and-mitigating-phishing-attacks" target="_blank">identify phishing emails</a> and spot indicators of unwanted emails (i.e., misspellings, odd vocabulary)</span></li>
<li><span class="rte-red-bullet">Update applications and systems regularly</span></li>
