													<p>In 2018, researchers at Cisco Talos published a <a href="https://blog.talosintelligence.com/2018/04/gravityrat-two-year-evolution-of-apt.html">post</a> on the spyware GravityRAT, used to target the Indian armed forces. The Indian Computer Emergency Response Team (CERT-IN) first <a href="https://nic-cert.nic.in/NIC_CERT/pdf/13-Advisory%20for%20Malicious%20Targeted%20Attack%20Campaign.pdf">discovered</a> the Trojan in 2017. Its creators are believed to be Pakistani hacker groups. According to our information, the campaign has been active since at least 2015, and previously targeted Windows machines. However, it underwent changes in 2018, with Android devices being added to the list of targets.</p>
<h2 id="malicious-guide">Malicious guide</h2>
<p>In 2019, on VirusTotal, we encountered a curious piece of Android spyware which, when analyzed, seemed connected to GravityRAT. The cybercriminals had added a spy module to Travel Mate, an Android app for travelers to India, the source code of which is available on <a href="https://project-travel-mate.github.io/">Github</a>.</p>
<p style="text-align: center"><a href="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16083352/image1-1.png" class="magnificImage"><img loading="lazy" decoding="async" class="aligncenter size-full wp-image-99099" src="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16083352/image1-1.png" alt="" width="718" height="901" srcset="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16083352/image1-1.png 718w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16083352/image1-1-239x300.png 239w" sizes="(max-width: 718px) 100vw, 718px" /></a><strong><em>Clean Travel Mate app on Google Play</em></strong></p>
<p>The attackers used a version of the app published on Github in October 2018, adding malicious code and changing the name to Travel Mate Pro.</p>
<p><a href="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16083629/image2-1.png" class="magnificImage"><img loading="lazy" decoding="async" class="aligncenter size-full wp-image-99100" src="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16083629/image2-1.png" alt="" width="363" height="647" srcset="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16083629/image2-1.png 363w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16083629/image2-1-168x300.png 168w" sizes="(max-width: 363px) 100vw, 363px" /></a></p>

<p style="text-align: center"><strong><em>List of Trojan classes</em></strong></p>
<p>The spyware&#8217;s functions are fairly standard: it sends device data, contact lists, e-mail addresses, and call and text logs to the C&amp;C server. In addition, the Trojan searches for files in the device memory and on connected media with the extensions .jpg, .jpeg, .log, .png, .txt, .pdf, .xml, .doc, .xls, .xlsx, .ppt, .pptx, .docx, and .opus, and sends these to C&amp;C as well.</p>
<p>The malware does not resemble a &#8220;typical&#8221; Android spy in that the choice of app is rather specific and the malicious code is not based on that of any known spyware app, as is often the case. As such, we decided to look for connections with known APT families.</p>
<p><a href="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16083748/image5-1.png" class="magnificImage"><img loading="lazy" decoding="async" class="aligncenter size-full wp-image-99103" src="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16083748/image5-1.png" alt="" width="644" height="56" srcset="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16083748/image5-1.png 644w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16083748/image5-1-300x26.png 300w" sizes="(max-width: 644px) 100vw, 644px" /></a></p>

<h3 id="android-versions">Android versions</h3>
<p><strong>SavitaBhabi</strong> exists for Windows and Android.</p>
<p><a href="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16092901/image21.png" class="magnificImage"><img loading="lazy" decoding="async" class="aligncenter size-full wp-image-99119" src="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16092901/image21.png" alt="" width="966" height="613" srcset="https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16092901/image21.png 966w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16092901/image21-300x190.png 300w, https://media.kasperskycontenthub.com/wp-content/uploads/sites/43/2020/10/16092901/image21-768x487.png 768w" sizes="(max-width: 966px) 100vw, 966px" /></a></p>
<p>The Windows version is based on .NET. The functionality is standard: the Trojan checks if it is running on a virtual machine and if security software is installed on the computer, transmits information about the computer to the server, and receives commands in response. It uses Windows Task Scheduler to launch the payload. Communication with the server is through POST requests to download.savitabhabi.co[.]in/A5739ED5.php.</p>
<p>The second file, downloaded from the same site, is the Android app Savitabhabi.apk, which is an adult comic strip with an embedded spyware module. Unlike the Travel Mate Pro version, this time it seems that the cybercriminals took a bottom-up approach and wrote the app themselves.</p>

<h3 id="conclusion">Conclusion</h3>
<p>In 2019, <em>The Times of India</em> published an <a href="https://timesofindia.indiatimes.com/city/lucknow/pakistan-spy-lured-98-targets-with-bots/articleshow/69867201.cms">article</a> about the cybercriminal methods used to distribute GravityRAT during the period 2015-2018. Victims were contacted through a fake Facebook account, and asked to install a malicious app disguised as a secure messenger in order to continue the conversation. Around 100 cases of infection of employees at defense, police, and other departments and organizations were identified.</p>
<p>It is safe to assume that the current GravityRAT campaign uses similar infection methods — targeted individuals are sent links pointing to malicious apps.</p>
<p>The main modification seen in the new GravityRAT campaign is multiplatformity: besides Windows, there are now versions for Android and macOS. The cybercriminals also started using digital signatures to make the apps look more legitimate.</p>
