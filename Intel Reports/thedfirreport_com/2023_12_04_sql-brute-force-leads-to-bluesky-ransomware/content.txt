<h1 class="entry-title">SQL Brute Force Leads to BlueSky Ransomware</h1>
<div class="entry-meta">
<span class="posted-on"><a href="https://thedfirreport.com/2023/12/04/sql-brute-force-leads-to-bluesky-ransomware/" rel="bookmark"><time class="entry-date published" datetime="2023-12-04T01:55:56+00:00">December 4, 2023</time></a></span> </div>
</header>
<div class="entry-content">
<p>In December 2022, we observed an intrusion on a public-facing MSSQL Server, which resulted in BlueSky ransomware. First discovered in <a href="https://twitter.com/Kangxiaopao/status/1541627210611978240" target="_blank" rel="noreferrer noopener">June 2022</a>, BlueSky ransomware has code links to <a href="https://unit42.paloaltonetworks.com/bluesky-ransomware/" target="_blank" rel="noreferrer noopener">Conti and Babuk</a> ransomware.</p>
<p>While <a href="https://x.com/1ZRR4H/status/1543339315756994563?s=20" target="_blank" rel="noreferrer noopener">other reports</a> point to malware downloads as initial access, in this report the threat actors gained access via a MSSQL brute force attack. They then leveraged Cobalt Strike and Tor2Mine to perform post-exploitation activities. Within one hour of the threat actors accessing the network, they deployed BlueSky ransomware network wide.</p>
<h2 id="case-summary" class="wp-block-heading"><a href="#case-summary">Case Summary</a></h2>
<p>In the month of December 2022, we observed a cluster of activity targeting MSSQL servers. The activity started with brute force password attempts for the MS SQL &#8220;sa&#8221; (System Administrator) account on an internet facing server. Upon successfully discovering the password, the threat actors enabled &#8220;xp_cmdshell&#8221; on the SQL server. The &#8220;xp_cmdshell&#8221; allows users with sysadmin privilege to execute shell commands on the host.</p>
<p>Using &#8220;xp_cmdshell&#8221; the threat actors first executed a PowerShell command on the SQL server. The command contained base64 encoded content, which, upon execution, established a connection to a Cobalt Strike command and control server. This activity was immediately followed by injection into the legitimate process winlogon. The injected process then spawned PowerShell and cmd to perform SMB scans and discovery using <a href="https://github.com/Kevin-Robertson/Invoke-TheHash/blob/master/Invoke-SMBExec.ps1#L1" target="_blank" rel="noreferrer noopener">SMBexec</a>.</p>
<p>The PowerShell session was then seen making a connection to a Tor2Mine stager server. This was followed by execution of a PowerShell script which performed a variety of operations, such as checking privileges of the active user, disabling of AV solutions, and dropping of a miner payload named java.exe. <a href="https://news.sophos.com/en-us/2021/12/02/two-flavors-of-tor2mine-miner-dig-deep-into-networks-with-powershell-vbscript/" target="_blank" rel="noreferrer noopener">Tor2Mine</a> is a Monero-mining campaign that is based on XMRigCC. Depending upon the privileges of the user, the script also performs creation of scheduled tasks and Windows services to maintain persistence on the host.</p>
<p>Around 15 minutes after initial access, the threat actors then moved laterally toward domain controllers and file shares using remote service creation. These services were used to execute the same PowerShell commands, download and execute the Tor2Mine malware. Upon establishing access to one of the domain controllers the threat actors performed similar activity as observed on the beachhead.</p>
<p>After roughly 30 minutes after initial access, the BlueSky ransomware binary was dropped and executed on the beachhead. The execution worked as intended which resulted in the ransomware spreading to all devices in the network over SMB. The time to ransomware in this case was 32 minutes.</p>
<h3 class="wp-block-heading">Threat Actor Profile:</h3>
<p><strong>Cobalt Strike</strong></p>
<p>The Cobalt Strike server observed in this intrusion was first observed on December 16th 2022 and remained active through January 17th 2023. We saw the server then return for a second time frame from April 6th 2023 though April 15th 2023. This data was provided via the <a href="https://thedfirreport.com/services/threat-intelligence/">Threat Intel tracking services</a> of The DFIR Report.</p>
<p><a href="https://thedfirreport.com/wp-content/uploads/2023/12/19208-001.png"><img fetchpriority="high" decoding="async" class="aligncenter size-full wp-image-23975" src="https://thedfirreport.com/wp-content/uploads/2023/12/19208-001.png" alt width="2190" height="307" srcset="https://thedfirreport.com/wp-content/uploads/2023/12/19208-001.png 2190w, https://thedfirreport.com/wp-content/uploads/2023/12/19208-001-300x42.png 300w, https://thedfirreport.com/wp-content/uploads/2023/12/19208-001-1024x144.png 1024w, https://thedfirreport.com/wp-content/uploads/2023/12/19208-001-768x108.png 768w, https://thedfirreport.com/wp-content/uploads/2023/12/19208-001-1536x215.png 1536w, https://thedfirreport.com/wp-content/uploads/2023/12/19208-001-2048x287.png 2048w" sizes="(max-width: 2190px) 100vw, 2190px" /></a></p>
<p><strong>Tor2Mine</strong></p>
<p>The PowerShell scripts involved in this case as well as infrastructure for the Tor2Mine server were observed being reused in May 2023 with the PaperCut NG <a href="https://cve.circl.lu/cve/CVE-2023-27350" target="_blank" rel="noreferrer noopener">CVE-2023-27350</a> exploit as the initial access source. In that intrusion no ransomware was observed. The linked case data is available for <a href="https://thedfirreport.com/services/threat-intelligence/">All Intel</a> subscribers in event 21132 (<span class="quickSelect">c39d59d8-8bae-49f5-8b29-de5c13b61899</span>).</p>

<h2>Analysts</h2>
<p>Analysis and reporting completed by <a href="https://twitter.com/yatinwad" target="_blank" rel="noreferrer noopener">@yatinwad</a></p>
<h2 id="initial-access" class="wp-block-heading"><a href="#initial-access">Initial Access</a></h2>
<p>The initial access occurred via a brute-force attack, where the threat actors mainly targeted the System Admin (&#8220;sa&#8221;) account.</p>
<p>During the intrusion, we observed over 10,000 failed attempts before successful login.</p>
<p>SQL Server event ID 18456 Failure Audit Events in the Windows application logs:</p>

<h2 id="impact" class="wp-block-heading"><a href="#impact">Impact</a></h2>
<p>The BlueSky ransomware binary named <code>vmware.exe</code> was dropped on the beachhead, which upon execution, resulted in network wide ransomware. This was accomplished using SMB with the ransomware connecting to host over port 445 to encrypt files.</p>
<p><a href="https://thedfirreport.com/wp-content/uploads/2023/12/19208-033.png"><img loading="lazy" decoding="async" class="aligncenter size-full wp-image-24011" src="https://thedfirreport.com/wp-content/uploads/2023/12/19208-033.png" alt width="1425" height="814" srcset="https://thedfirreport.com/wp-content/uploads/2023/12/19208-033.png 1425w, https://thedfirreport.com/wp-content/uploads/2023/12/19208-033-300x171.png 300w, https://thedfirreport.com/wp-content/uploads/2023/12/19208-033-1024x585.png 1024w, https://thedfirreport.com/wp-content/uploads/2023/12/19208-033-768x439.png 768w" sizes="(max-width: 1425px) 100vw, 1425px" /></a></p>
