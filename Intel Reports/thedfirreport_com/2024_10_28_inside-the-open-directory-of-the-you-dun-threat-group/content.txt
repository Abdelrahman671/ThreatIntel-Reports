<h2 id="key" class="wp-block-heading"><a href="#key">Key Takeaways</a></h2>



<ul class="wp-block-list">
<li>Analysis of an open directory found a Chinese speaking threat actor&#8217;s toolkit and history of activity.</li>



<li>The threat actor displayed extensive scanning and exploitation using WebLogicScan, Vulmap, and Xray, targeting organizations in South Korea, China, Thailand, Taiwan, and Iran.</li>



<li>The Viper C2 framework was present as well as a Cobalt Strike kit which included TaoWu and Ladon extensions.</li>



<li>The Leaked LockBit 3 builder was used to create a LockBit payload with a custom ransom note that included reference to a Telegram group which we investigated further in the report.</li>
</ul>

<h2 id="case-summary" class="wp-block-heading"><a href="#case-summary">Case Summary</a></h2>
<p><!-- /wp:heading -->

<!-- wp:paragraph --></p>
<p>The DFIR Report&#8217;s Threat Intel Team detected an open directory in January 2024 and analyzed it for trade craft and threat actor activity. Once reviewed, we identified it was related to the Chinese speaking hacking group that call themselves &#8220;You Dun&#8221;.</p>
<p><!-- /wp:paragraph -->

<!-- wp:paragraph --></p>
<p>The threat actor conducted various activities using the host we investigated which included reconnaissance and web exploitation activities. Using tools such as WebLogicScan, Vulmap, and Xray, they were able to identify numerous vulnerable servers. They exploited several websites running Zhiyuan OA software and used SQLmap to conduct SQL injection.</p>
<p><!-- /wp:paragraph -->

<!-- wp:paragraph --></p>
<p>We found evidence of several successful exploitation attempts. After gaining access we found use of further tools to try and use various exploits to elevate privileges on the compromised hosts, including the use of <a href="https://github.com/liamg/traitor" target="_blank" rel="noreferrer noopener">traitor</a> for Linux privilege escalation exploits and <a href="https://github.com/cdk-team/CDK" target="_blank" rel="noreferrer noopener">CDK</a> for docker and kubernetes privilege escalation.</p>
<p><!-- /wp:paragraph -->

<!-- wp:paragraph --></p>
<p>Both Cobalt Strike and Viper framework files were visible in the open directory. A zip archive that contained the files for a Cobalt Strike team server, included the plugins TaoWu and Ladon which extend the capabilities of the framework greatly. The DFIR Report <a href="https://thedfirreport.com/services/threat-intelligence/" target="_blank" rel="noreferrer noopener">Threat Intel Team</a> tracked the server as hosting active command and control from January 18th through Feburary 10th of 2024. Using data from the leaked server we identified a cluster of eight IP address&#8217;s all being used to proxy the command and control for the same threat actor and active for the same time frame.</p>
<p><!-- /wp:paragraph -->

<!-- wp:paragraph --></p>
<p>The threat actor also utilized the leaked LockBit 3 ransomware builder to create a custom binary LB3.exe. The ransom note the LockBit binary produced provided the contact details as a Telegram group &#8220;You_Dun&#8221; administrated by &#8220;EVA&#8221;. The group responsible also use the name &#8220;Dark Cloud Shield Technical Team&#8221;. This group appears to be involved in selling &#8220;penetration testing&#8221; according to their channels, but also engaged in illicit data sales, DDOS, and based on the LockBit binary, also use ransomware to earn a payday.</p>
